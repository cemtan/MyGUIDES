#### DIRECTORIES
---
---



- **The installation directory**: /opt/hitachi.
- **The installation destination for Common Services**: /opt/hitachi/CommonService.
- **User data for Common Services is stored in the following user data directory**: /var/installation-directory-of-Common-Services
- **Command Control Interface (required by Ops Center API Configuration Manager)**: /opt/hitachi/ConfManager/HORCM
