cluster-extend-status

Report whether the cluster is extended

Syntax

cluster-extend-status

Description

Reports whether a unified NAS module cluster has been extended for disaster recovery.

This command prints "enabled" if the cluster has been extended by 'cluster-extend-create', or joined to an extended cluster by 'cluster-extend-join'.

Otherwise it prints "disabled".

Applies To

Cluster wide.

See Also

cluster-extend-create cluster-extend-delete cluster-extend-join cluster-extend-unconfigure

Privilege Level

User
