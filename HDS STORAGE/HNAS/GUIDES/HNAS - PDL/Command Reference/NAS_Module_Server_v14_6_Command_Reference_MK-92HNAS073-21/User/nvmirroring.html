nvmirroring

Queries the state of NVRAM mirroring

Syntax

nvmirroring

Description

Displays the current state of NVRAM mirroring. File system modifying operations are stored in NVRAM before being committed to storage so that no data is lost in the event of server or storage failure. In a high-availability cluster, the contents of the NVRAM is also mirrored to another server in the cluster enabling it to seamlessly takeover in the event of a server failure. Under normal circumstances, the NVRAM will always be mirrored when in a high-availability cluster.

Note this command may return inconsistent information across the nodes in the cluster if NVRAM mirroring is changing state at the time the command is issued.

Applies To

Cluster node
