dcerpc-default-state-show

Show DCE/RPC default state.

Syntax

dcerpc-default-state-show

Description

Display DCE/RPC default state. When DCE/RPC is enabled by default, registered DCE/RPC Services are started automatically at boot.

Examples

dcerpc-default-state-show
DCE/RPC default state : enabled.

Applies To

Cluster node

See Also

dcerpc-service-show

User
