rest-server-show

Show the RestAPI server configuration.

Syntax

rest-server-show

Description

Shows the RestAPI server configuration.

For further information about the RestAPI server, use 'man rest-server'.

Examples

$ rest-server-show
RestAPI server is disabled.

Applies To

Cluster wide

See Also

rest-server rest-server-mode rest-server-start rest-server-status rest-server-stop

Privilege Level

User
