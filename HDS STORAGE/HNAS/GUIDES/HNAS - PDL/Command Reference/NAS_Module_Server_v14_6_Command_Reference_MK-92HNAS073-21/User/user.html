user

Manage management users

Syntax

user add [<username> <password> <level>]|del <username>|delall|list

Description

This command is only for the creation and deletion of the management user named 'supervisor'. The maintenance utility must be used for all other modifications.

If "user add" is used without options, then the command prompts for a username, level and password, in that order. A final interactive password confirmation is required for all "user add" invocations, even if the username, password and level are specified, in that order, on the command line.

"user list" displays all users of equal or lower level.

The only available option for both level and username is 'supervisor'. "user del" requires a username to be supplied.

Examples

To add a user called 'supervisor' as a 'supervisor' level user:

user add supervisor password supervisor

To show currently configured users:

user list

To show currently logged-in users:

who

To show the user logged in on the current console:

whoami

Applies To

Cluster wide

See Also
