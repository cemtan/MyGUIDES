ftp-stats

Report/reset FTP statistics

Syntax

ftp-stats [-p] [-r]

Description

The ftp-stats command reports statistics about FTP server activity.

Options:

-p, --persistent

Shows the statistics since boot. Resetting the statistics does not affect the values returned with the -p option.

-r, --reset

Reset statistics.

Examples

Display the FTP statistics since last reset:

ftp-stats

Display statistics since boot:

ftp-stats -p

Reset the FTP statistics:

ftp-stats -r

Privilege Level

User
