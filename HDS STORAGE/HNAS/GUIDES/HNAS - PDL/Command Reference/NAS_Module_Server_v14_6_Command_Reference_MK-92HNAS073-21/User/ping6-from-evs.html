ping6-from-evs

Ping remote system

Syntax

ping6-from-evs [-c <count>][-I <source-IP-address>[%interface]][-s <packetsize>] <IP-address or hostname>[%interface]

Description

ping6-from-evs works in a similar way to ping6 except that it runs on the cluster node that the EVS currently resides on. If routing-by-evs is enabled, then the choice of source address is limited to the source addresses configured on the EVS. If routing-by-evs is disabled, then the choice of source address can be any of the source addresses currently hosted on the cluster node hosting the EVS.

ping6-from-evs determines whether a given host is alive by sending a number of ICMPv6 protocol ECHO_REQUEST datagrams using the IPv6 protocol.

Options:

-c <count>

The number of ECHO_REQUESTS to send. The default is 4; the minimum is 1.

-I <source-ip-address>[%interface]

Set the source IP address of the request to the supplied IP address. An optional source interface name may be supplied; for a link local source address an interface name must be supplied either here or on the destination. If an interface is specified both here and as part of the destination, then the two interface names must be the same.

-s <packetsize>

Set the number of data bytes to be sent. The default is 32.

Examples

To ping an IPv6 address once:

vn 2 ping6-from-evs -c 1 2001:db8:220a:480:1::
2001:db8:220a:480:1:: (2001:db8:220a:480:1::) ...
32 bytes from 2001:db8:220a:480:1:: icmp_seq=0 time <1 ms
--
1 packets transmitted, 1 packets received, 0% packet loss
round-trip min/avg/max = 0/0/0 ms

To ping an IPv6 link local address:

vn 1 ping6-from-evs fe80::baac:6fff:fe80:5962%ag1
fe80::baac:6fff:fe80:5962%ag1 (fe80::baac:6fff:fe80:5962) ...
32 bytes from fe80::baac:6fff:fe80:5962 icmp_seq=0 time=4 ms
32 bytes from fe80::baac:6fff:fe80:5962 icmp_seq=1 time=1 ms
32 bytes from fe80::baac:6fff:fe80:5962 icmp_seq=2 time=1 ms
32 bytes from fe80::baac:6fff:fe80:5962 icmp_seq=3 time=1 ms
--
4 packets transmitted, 4 packets received, 0% packet loss
round-trip min/avg/max = 1/1/4 ms

Applies To

EVS
