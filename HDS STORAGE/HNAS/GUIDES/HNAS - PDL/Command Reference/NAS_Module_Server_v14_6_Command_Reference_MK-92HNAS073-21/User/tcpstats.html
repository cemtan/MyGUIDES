tcpstats

Report/reset TCP statistics

Syntax

tcpstats [-r | -n] [-p <name> | -t]

Description

There are a number of IP related stats commands that display information about different protocols:

  • tcpstats displays statistics for TCP protocol packets;
  • udpstats displays statistics for UDP protocol packets;
  • icmpstats displays statistics for ICMP protocol packets;
  • ip-stats-other displays statistics for all other IP protocol packets.

The tcpstats command takes the following options:

-r, --reset

Reset statistics

-n, --nr

Non-resettable statistics

-p, --port

Port name eg. ag1

-t, --total

Total for mgmnt1 and all aggregations

Applies To

Cluster node

Privilege Level

User
