query

Queries the specified file system

Syntax

query [-r | --reset] <file-system>

Description

Displays the current state of the specified file system.

Usage:

query [-r | --reset] <file-system>

<file-system>

The file system to query.

Options:

[-r | --reset]

Reset the data read/written counters.

Examples

An example of the output for an unmounted file system:

server:$ query fs1
Device ID                       : 1024
Current state                   : Not mounted
Available for use               : Yes
Formatted                       : Yes
Mounted                         : No
Thin provisioned                : No
File system version             : 2
Read only                       : No
WORM                            : No
Dedupe                          : Supported and enabled
Will mount automatically        : No
Status of last mount attempt    : Success
Size                            : 5.68 GB (6098518016 bytes)
Block size                      : 4 KB (4096 bytes)
Free space                      : 3.518 GB (3777138688 bytes)

An example of the output for a mounted file system:

server:$ query fs1
Device ID                       : 1024
Current state                   : FS mounted
Available for use               : Yes
Formatted                       : Yes
Mounted                         : Yes
Thin provisioned                : No
File system version             : 2
Read only                       : No
WORM                            : No
Dedupe                          : Supported and enabled
Will mount automatically        : Yes
Status of last mount attempt    : Success
Size                            : 5.68 GB (6098518016 bytes)
Block size                      : 4 KB (4096 bytes)
Free space                      : 3.518 GB (3777208320 bytes)
Data read                       : 2.047 MB (0 B/second)
Data written                    : 2.039 MB (0 B/second)
Ops/second                      : 0

An example of the output for a mounted file system for which thin provisioning is enabled:

server:$ query fs0
Device ID                       : 1025
Current state                   : FS mounted
Available for use               : Yes
Formatted                       : Yes
Mounted                         : Yes
Thin provisioned                : Yes

WORM                            : No
Dedupe                          : Not supported
Will mount automatically        : Yes
Status of last mount attempt    : Success
Size                            : 203 GB (218238025728 bytes)
Block size                      : 32 KB (32768 bytes)
Free space                      : 201 GB (215867326464 bytes)
Thin Size                       : 5.00 TB (5496752832512 bytes)

Data written                    : 0 B (0 B/second)

The thin provisioning (TP) discussed here is the file system-level TP discussed in the 'filesystem-thin' man page, rather than the storage-level TP discussed in 'hdp'.
