sscat

Cat file

Syntax

sscat [--binary] file

Description

Print file.

The command takes one parameter - the source filenames, which may include the flash name.

If the file is binary, specify --binary option before the filename to help the filter flush the standard output frequently.

Examples

sscat --binary zipped.txt.gz | unzip
