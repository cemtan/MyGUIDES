replication-csv-active-fs

Show file systems with active object replication sessions

Syntax

replication-csv-active-fs [--source|--target]

Description

Lists file systems with active object replication sessions.

Usage:

-s, --source

Show only active object replication source sessions.

-t, --target

Show only active object replication target sessions.

Examples

server:$ replication-csv-active-fs
span0fs

Applies To

EVS

See Also

evs replication-show-active

Privilege Level

Supervisor
