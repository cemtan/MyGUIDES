lacp-set-timeout-short

Select short LACP timeout

Syntax

lacp-set-timeout-short <interface>

Description

The 'lacp-set-timeout-short', 'lacp-set-timeout-long' and 'lacp-set-timeout-default' commands configure the LACP Timeout parameter for an aggregation. The default LACP Timeout configuration uses a short timeout.

Changing the LACP Timeout value for an active aggregation may cause a temporary change to the link state resulting in a short interruption of service to any EVS using the aggregation.

To display the configuration use the 'lacp-show' command.

Examples

To configure use of a long LACP Timeout for an aggregation:

    lacp-set-timeout-long ag1

Applies To

Cluster wide
