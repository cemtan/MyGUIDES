cluster-ping

Ping cluster nodes and quorum devices

Syntax

cluster-ping [-p|-s] [-i <ipaddr>] [-c|-q]

Description

The cluster-ping command can be used to confirm network connectivity with other cluster nodes or quorum devices. This command sends network "ping" messages to locate such systems on the network. The cluster nodes use either primary or secondary interconnect link to respond to the "ping" messages.

Options:

-p, --primary

Pings through the primary intercluster link.

-s, --secondary

Pings through the secondary intercluster link.

-i --ipaddr

Specifies the IP address of a specific cluster node.

-c, --cluster node

Displays only Cluster Nodes.

-q, --quorum

Displays only Quorum Devices.
