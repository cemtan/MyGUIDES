utility-progress

Display the progress of one or more utilities running on the either the supplied file system, the currently selected file system, or all file systems

Syntax

utility-progress [-f <file-system>]

Description

utility-progress gives an indication of the progress of the one or more utilities. Each generated report is appropriate to the utility it pertains to. The report is sent to console it was invoked on.

The progress as a percentage may give erratic results if a file system only contains a few very large directories, or if the Free Root Onodes list is unusually large in the case of checkfs.

Note: utility-progress may be run with the file system mounted or unmounted.

NOTE: This command supersedes the following obsolete commands: checkfs-progress check-directory-progress fixfs-progress

Examples

To check the progress of the utility running on file system myFS:

utility-progress -f myFS

To check the progress of all running utilities:

utility-progress

NOTE: When invoked without arguments, this command will run on the cluster node hosting the currently selected EVS. The progress for all the utilities running on all the EVSs hosted on that cluster node will be reported.

Applies To

EVS

Privilege Level

Supervisor
