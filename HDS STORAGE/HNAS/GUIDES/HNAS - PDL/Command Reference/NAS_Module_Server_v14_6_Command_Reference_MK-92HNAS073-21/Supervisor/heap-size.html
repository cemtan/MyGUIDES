heap-size

Display heap size

Syntax

heap-size [-r|--reset]

Description

The heap-size command tells you the amount of memory allocated to the heap by the kernel (which the command calls "Heap capacity"), the total size of all the blocks of heap currently allocated (which the command calls "Momentary heap size") and the maximum value of this "Momentary heap size" since heap-size was last run with the -r argument (which the command calls "Maximum heap size"). The capacity minus the size is the free space. Momentary and minimum values for this are also displayed, corresponding to the momentary and maximum size values, to save manual subtraction where the free space is the real interest.

Examples

server:$ fsm heap-size
Heap capacity 2.57 GB (2757787648 B).
Momentary heap size 2.15 GB (2306617344 B).
Momentary heap free 430 MB (451170304 B).
Maximum heap size 2.26 GB (2422423264 B).
Minimum heap free 320 MB (335364384 B).
server:$

Applies To

Cluster node
