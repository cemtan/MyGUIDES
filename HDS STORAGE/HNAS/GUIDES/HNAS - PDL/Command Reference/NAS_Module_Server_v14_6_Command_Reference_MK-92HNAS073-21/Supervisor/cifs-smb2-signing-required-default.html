cifs-smb2-signing-required-default

Sets SMB2 signing required to the default value

Syntax

cifs-smb2-signing-required-default

Description

Restores the default value for SMB2 signing required. The default setting is signing not required.

Examples

$ cifs-smb2-signing-required-default


SMB2 signing is NOT required on EVS 1

Applies To

EVS

See Also

cifs-smb2-signing-required-disable cifs-smb2-signing-required-enable cifs-smb2-signing-status

Privilege Level

Supervisor
