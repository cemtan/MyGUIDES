nfsv3-wsize-get

Display the maximum number of bytes per NFSv3 WRITE request the client can send.

Syntax

nfsv3-wsize-get --tcp | --udp | --all

Description

Displays the maximum number of bytes per NFSv3 WRITE request that a client can send whilst writing data to a file. These values may be set using nfsv3-wsize-set.

Usage:

nfsv3-wsize-get

nfsv3-wsize-get --all

Displays all the currently configured values.

nfsv3-wsize-get --tcp

Display the currently configured value for TCP transports.

nfsv3-wsize-get --udp

Display the currently configured value for UDP transports.

Applies To

Cluster wide

See Also

nfsv3-rsize-get nfsv3-rsize-set nfsv3-wsize-set

Privilege Level

Supervisor
