replication-policy-list

List the object replication policies

Syntax

replication-policy-list [-n <num-entries>] [<policy id|source fs label>]|[<policy name> <source fs label>]

Description

Lists policies for object replication.

<policy id>

ID of the policy.

<policy name>

Name of the policy, which must be unique.

<source fs label>

Source file system label.

-n, --max <num-entries>

Lists at most num-entries policies.

Examples

replication-policy-list

Lists all object replication policies on this EVS.

replication-policy-list fs0

Lists all object replication policies for source file system fs0.

replication-policy-list DefaultPolicy fs0

List the object replication policy named 'DefaultPolicy' for source file system fs0.

Applies To

EVS

See Also
