smu-service-enable

Enable the SMU service

Syntax

smu-service-enable

Description

Enable the SMU service.

Examples

smu-service-enable


     cluster node 1:
     Attempting to apply modifications...
     Embedded SMU enabled successfully.
     Successfully applied configuration modifications.
     The SMU has been enabled.
     cluster node 2:
     The SMU has been enabled.

Applies To

Cluster wide
