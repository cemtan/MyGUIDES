cluster-annotation-set

Sets comments for cluster annotations

Syntax

cluster-annotation-set <name> <comment>

Description

The cluster-annotation-set command assigns a 'comment' to a 'name' that will be returned by the Cluster Annotations feature.

Cluster annotations are persistent cluster-wide attributes whose values (called 'comments') can be set, updated and queried, for example for use by an external application.

cluster-annotation-set <name> <comment>

Sets the cluster annotation <name> to be <comment>. If the annotation specified doesn't already exist, it will be created.

Examples

cluster-annotation-set evs/01/service/smb True

Applies To

Cluster wide

See Also
