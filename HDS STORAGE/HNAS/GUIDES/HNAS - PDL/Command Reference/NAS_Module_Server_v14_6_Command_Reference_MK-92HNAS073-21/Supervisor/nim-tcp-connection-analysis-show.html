nim-tcp-connection-analysis-show

Display TCPAN per-connection data gathering status

Syntax

nim-tcp-connection-analysis-show

Description

The nim-tcp-connection-analysis-show command is used to display the current status of per-connection TCP traffic data gathering.

Applies To

Cluster node

See Also

nim-tcp-connection-analysis-disable nim-tcp-connection-analysis-enable nim-tcp-connection-dump

Privilege Level

Supervisor
