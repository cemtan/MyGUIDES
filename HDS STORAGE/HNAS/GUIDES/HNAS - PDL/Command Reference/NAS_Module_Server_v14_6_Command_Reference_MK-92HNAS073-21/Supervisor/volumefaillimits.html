volumefaillimits

Sets the volume failure limit.

Syntax

volumefaillimits [-c <max failures> -t <time period in secs>]

Description

Shows or sets the limit to the rate of failures allowed before a file system will no longer be automatically mounted.

Options:

-c <max failures>

number of failures allowed in time window

-t <time period in secs>

the time window

Examples

To see the volume failure limits configuration: volumefaillimits

Privilege Level

Supervisor
