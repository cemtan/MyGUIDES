pluggable-transceiver-dump

Dump data from pluggable transceivers

Syntax

pluggable-transceiver-dump [--diags | --data] [[--port-family (net|cluster|fc) [--port-number <port-number>]] | [--port-name <port-name>] | [--type (xfp|sfp)]] [--hex]

Description

Displays information about pluggable transceivers (e.g. XFPs and SFPs).

There are four mutually exclusive ways of specifying which pluggable transceivers to dump information for:

  1. Specify a port name with the --port-name option.
  2. Specify a port family ('net', 'cluster' or 'fc') with the --port-family option, and optionally specify a (zero based) port number with the --port-number option.
  3. Specify the type of pluggable transceivers to dump information for with the --type option.
  4. Specify none of the above to dump information for all pluggable transceivers.

Options:

--data

Show only information from the non-volatile data EEPROMs of pluggable transceivers.

--diags

Show only information from the volatile diagnostic EEPROMs of pluggable transceivers.

--hex

Dump transceiver EEPROM data as raw hexadecimal output.

--port-family <net|cluster|fc>

Show information only for pluggable transceivers attached to ports belonging to the given family.

--port-name port

Show information only for a pluggable transceiver attached to the specified port.

--port-number n

Show information only for a pluggable transceiver attached to port n (zero based) of the family specified with the --port-family option.

--type <sfp|xfp>

Show information only for pluggable transceivers of the specified type. Note that 'sfp' specifies both SFP and SFP+.

Examples

pluggable-transceiver-dump
Shows information for all pluggable transceivers.


pluggable-transceiver-dump --port-family net
Shows information for all pluggable transceivers attached to the network ports.


pluggable-transceiver-dump --port-name fc2
Shows information for any pluggable transceiver attached to the second fibre
channel port.


pluggable-transceiver-dump --port-family cluster --port-number 0 --diags
Shows information from the diagnostics EEPROM of any pluggable transceiver
attached to the first cluster port.


pluggable-transceiver-dump --type sfp
Shows information for any SFPs or SFP+s attached to the server. This is
equivalent to the (deprecated) sfps command.

Applies To

Cluster node
