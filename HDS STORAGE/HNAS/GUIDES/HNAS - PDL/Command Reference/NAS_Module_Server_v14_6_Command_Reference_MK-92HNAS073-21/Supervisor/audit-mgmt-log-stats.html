audit-mgmt-log-stats

Show management audit log statistics

Syntax

audit-mgmt-log-stats [-r]

Description

Displays or resets statistics about entries written to the management audit log on a single cluster node.

Options

-r

Reset all management audit log statistics. This clears only the statistics counters, and has no effect on audit log content.

If no option is given, this command displays management audit log statistics accumulated on the cluster node since the most recent statistics reset, or since the cluster node was last booted.

Examples

To display currently collected management audit log statistics for the presently selected cluster node:

audit-mgmt-log-stats

To display current statistics for cluster node 2:

cn 2 audit-mgmt-log-stats

To reset management audit log statistics counters on the present cluster node:

audit-mgmt-log-stats -r

Applies To

Cluster node

Privilege Level

Supervisor
