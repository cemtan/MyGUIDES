route-host-add

Add a host route

Syntax

route-host-add --gateway <gateway> [--mtu <mtu>] <host>

Description

The route-host-add command adds a host route to the routing table.

Options

The following options are recognised:

-g, --gateway <gateway-address>

This is the gateway for the route. It must be supplied.

-m, --mtu <mtu-size>

The MTU to be used for this route. This value must lie between 68 and 9600 inclusive for IPv4 routes and between 1280 and 9600 inclusive for IPv6 routes. If the MTU is not specified a default of 1500 will be applied.

<host>

This is the remote host to which this route applies. It must be supplied.

Examples

$ route-host-add 10.1.2.3 -g 10.1.3.4
Route cache flushed.
$

Creates a host route to get to 10.1.2.3 via 10.1.3.4.

Applies To

Cluster wide
