nat-mapping-clear

Delete all the NAT mappings

Syntax

nat-mapping-clear

Description

Clear the list of NAT mappings.

NAT mappings allow UIs for devices on a private network to be launched from the embedded SMU's System Monitor page. This command provides a way of deleting all NAT mappings from the registry, and hence from the server's firewall.

WARNING: After using this command, the embedded SMU's System Monitor may still list devices from the private management network, but will not be able to launch their UIs. These device will need to be removed manually from the embedded SMU's System Monitor. If it is required to synchronize the NAT mappings in the registry with the private network devices in the embedded SMU's System monitor, an alternative approach would be to remove a device from the System Monitor, and re-add it.

Examples

nat-mapping-clear

Clears the entire list of NAT mappings.

Applies To

Cluster wide
