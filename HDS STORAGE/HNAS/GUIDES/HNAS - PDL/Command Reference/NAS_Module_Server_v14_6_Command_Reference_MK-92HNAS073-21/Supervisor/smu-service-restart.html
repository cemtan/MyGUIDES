smu-service-restart

Restart the SMU service

Syntax

smu-service-restart

Description

Restart the SMU service.

The entire SMU service (webserver and database) will be restarted.

The restart will be carried out on each node at a time. If the restart is not possible on a node, it will still be carried out on the remaining nodes.

Examples

server:$ smu-service-restart
cluster node 1:
Restarting smu
smu restarted successfully.
cluster node 2:
Restarting smu
smu restarted successfully.
server:$

Applies To

Cluster wide

See Also
