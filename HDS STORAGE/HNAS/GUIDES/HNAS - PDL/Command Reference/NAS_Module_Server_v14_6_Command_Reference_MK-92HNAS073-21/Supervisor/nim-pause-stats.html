nim-pause-stats

Display NIM MAC PAUSE stats

Syntax

nim-pause-stats [-i <name>] [-w] [-e] [-c] [-h] [-m N]

Description

The nim-pause-stats command displays data about MAC PAUSE frame activity. Each line of the displayed data contains the time at which the data was gathered, the number of VLSI clock cycles for which a PAUSE was in effect and the percentage of the sample for which a PAUSE was in effect. Note that the VLSI clock cycle time varies according to the hardware in use.

-i <name>, --interface <name>

Specifies the interface for which MAC PAUSE frame information should be displayed. If not specified, all available information is displayed.

-w, --switch

Displays information about MAC PAUSE frames sent by the server to a connected switch.

-e, --server

Displays information about MAC PAUSE frames sent to the server by a connected switch.

If neither -w nor -e are specified, both will be assumed.

-c, --current

Displays current data: less than 15 minutes old and accumulated each second.

-h, --historical

Displays historical data: between 15 minutes and 48 hours old, accumulated in blocks of 15 minutes.

If neither -h nor -c are specified, both will be assumed.

-m N, --max N

Display at most N entries for each selected port

Examples

To display MAC PAUSE frame data for ag1:

    nim-pause-stats -i ag1

Applies To

Cluster node

Privilege Level

Supervisor
