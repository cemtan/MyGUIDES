explain-wfs-status

Explain a given WfsStatus

Syntax

explain-wfs-status <status>

Description

This command provides more detail about the specified WFS status.

<status>

Explain the specified status. The status may be specified as a numerical value or an error code.

Examples

explain-wfs-status ERR_WFILE_SET_LENGTH_ON_NON_FILE


explain-wfs-status 0x314

Privilege Level

Supervisor
