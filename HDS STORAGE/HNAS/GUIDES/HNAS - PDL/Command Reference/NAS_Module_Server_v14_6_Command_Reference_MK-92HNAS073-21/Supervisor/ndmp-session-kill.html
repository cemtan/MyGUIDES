ndmp-session-kill

Kill the NDMP session - any current operation will fail

Syntax

ndmp-session-kill <session_number>

Description

The command immediately terminates an NDMP session identified by the session id parameter aborting any current operation. The ndmp-session-list command should be used to check that the correct session identifier is used. If there is an outstanding SCSI request to a tape drive or media changer then the device will not be freed until that SCSI request has completed.

Examples

Kill session 21:

ndmp-session-kill 21

See Also

ndmp-session-list

Privilege Level
