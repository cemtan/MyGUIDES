cluster-annotation-unset-all

Unsets all cluster annotations

Syntax

cluster-annotation-unset-all

Description

The cluster-annotation-unset-all command removes all items that would be returned by the Cluster Annotations feature.

Cluster annotations are persistent cluster-wide attributes whose values (called 'comments') can be set, updated and queried, for example for use by an external application.

cluster-annotation-unset-all

Removes all the annotations.

Examples

cluster-annotation-unset-all

Applies To

Cluster wide

See Also
