ndmp-session-list

List active NDMP sessions

Syntax

ndmp-session-list [-t|--stack-trace] [-c|--children] [<session_number>]

Description

Options:

--stack-trace

Displays the current stack trace for each session.

--children

Displays information for all child processes of the NDMP session.

The command displays information about all NDMP sessions on this node or a single NDMP session specified by the optional session number parameter. This includes details of any current operations and any backup devices that are owned by the session.

Examples

List all NDMP sessions on this node. This example shows a single recovery from tape.

server-a:$ ndmp-session-list
NDMP session 2: EVS=1, pid=623, tape drive=/dev/mt_d0l1
- Tape:Active -> Data:Active (/vol1/test_fs/hl_dar_rec/z_link3)
server-a:$
