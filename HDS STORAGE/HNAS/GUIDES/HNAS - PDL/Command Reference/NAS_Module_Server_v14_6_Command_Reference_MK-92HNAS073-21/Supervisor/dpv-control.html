dpv-control

Display DPV control

Syntax

dpv-control [show|on|off]

Description

The dpv-control command controls the sending of all classes of DPV packets.

The command takes effect immediately but it is not persistent across a reboot. The DPV tests are intended to be operational except when explicitly disabled, so, on boot, all pingers are enabled. Note that the DPV-CNS-PINGER will be enabled but will only send packets in a cluster with more than one head.

Examples

To display DPV summary information:

    dpv-control

To display all DPV information:

    dpv-control show

To enable all DPV pingers:

    dpv-control on

To disable all DPV pingers:

    dpv-control off

Applies To

Cluster node

Privilege Level

Supervisor
