smb2-session-connection-list

Display the connections used by SMB2+ sessions

Syntax

smb2-session-connection-list (--ipaddr <ipaddr> | --session <session id> | --all) [--verbose]

Description

Lists the connections that are bound to an SMB2, or higher, dialect session.

Usage:

smb2-session-connection-list (--ipaddr <ipaddr> | --session <session id> | --all) [--verbose]

Options are:

-v, --verbose

In addition to summarising the connections, displays the credentials information for each session.

-s, --session <session id>

A specific session ID to list the connections of.

-i, --ipaddr <ip address>

Restricts output to only display sessions from the specified peer IP address.

IPv4 or IPv6 addresses can be used.

--all

Lists the connections of all sessions.

Using this option may take a considerable time to complete, depending on the number of active sessions.

Examples
