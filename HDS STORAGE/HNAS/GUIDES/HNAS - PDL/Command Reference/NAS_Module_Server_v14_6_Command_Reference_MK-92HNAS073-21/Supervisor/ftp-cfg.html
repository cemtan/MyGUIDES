ftp-cfg

Show/modify FTP general configuration

Syntax

ftp-cfg [-t <time-out>] [-n <on|off>] [-s <on|off>] [-a <ro|rw>]

Description

The ftp-cfg command is used to configure the FTP server.

Options:

-t, --timeout

Sets the FTP session inactivity time-out value in minutes. The valid range is 15 to 14400 minutes (10 days), inclusive.

-n, --ntsecurity

Controls whether FTP logins are authenticated using a domain controller in the configured CIFS domain.

-s, --nissecurity

Controls whether FTP logins are authenticated using a NIS server in the configured NIS domain.

If both --ntsecurity and --nissecurity are enabled then a login will be authenticated against a domain controller first, and only if this fails will an attempt be made to authenticate against a NIS server.

-a, --anonuser

Controls whether anonymous FTP sessions are read-only or read-write.

Examples

Display FTP configuration:

ftp-cfg

Change FTP session timeout to 15 minutes:

ftp-cfg -t 15

Enable authentication against a CIFS domain controller:

ftp-cfg -n on

Disable authentication against a NIS server:

ftp-cfg -s off

Change FTP anonymous logins to read-only:

ftp-cfg -a ro

Applies To

EVS security context

ftp-user
