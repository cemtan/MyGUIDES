file-clone-stat

Display information about a clone

Syntax

file-clone-stat [-f <file-system>] <clone> [--verbose]

Description

Displays information about a clone object, specifically, the handles of the Snapshot File objects it depends on.

-f, --file-system

The file system to work on. The currently selected file system is used if this option is not specified.

<clone>

The path of the clone.

-v, --verbose

Display human-readable info on the file handle for each reported object.

Examples

   $ file-clone-stat /clone

     SnapshotFile: FileHandle[0000000000006b19000626d26826ffffffffffffffffffff]
     SnapshotFile: FileHandle[0000000000006ed1000626d16826ffffffffffffffffffff]
