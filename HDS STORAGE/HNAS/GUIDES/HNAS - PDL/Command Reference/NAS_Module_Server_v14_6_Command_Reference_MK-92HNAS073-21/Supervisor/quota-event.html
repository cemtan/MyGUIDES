quota-event

Enable/disable adding quota events to the event log

Syntax

quota-event [--on | --off]

Description

Enable/disable adding quota events to the event log

Usage:

quota-event [--on | --off]

--on

Add quota events to the event log

--off

Don't add quota events to the event log (unless no email list is configured for the file system or virtual volume)

Examples

quota-event --on

Applies To

Cluster wide

See Also

default-quota fs-email quota virtual-volume

Privilege Level

Supervisor
