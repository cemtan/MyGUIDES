namespace-capacity

Displays the capacity of a cluster or EVS namespace

Syntax

namespace-capacity <cluster or EVS namespace>
Alias:       cns-capacity

Description

Displays the capacity of the cluster or EVS namespace.

Examples

Display the capacity of the cluster namespace 'cns0':

namespace-capacity cns0
Cluster namespace
Name   : cns0
Objects: 1 link, 2 directories
Used   : 3/520 (0.58%)

Applies To

Cluster wide

See Also

namespace-list

Privilege Level

Supervisor
