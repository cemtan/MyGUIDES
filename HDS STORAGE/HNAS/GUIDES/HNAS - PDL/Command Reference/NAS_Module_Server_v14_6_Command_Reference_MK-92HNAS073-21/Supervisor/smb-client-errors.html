smb-client-errors

Report SMB client protocol errors

Syntax

smb-client-errors [-r|--reset]

Description

The smb-client-errors command is used to display errors seen on all SMB client connections. This information can be useful when debugging SMB client connectivity problems. SMB client connections are used for connections to domain controllers, virus scanners and file filtering agents.

Usage:

--reset, -r

Resets the error log.

Applies To

Cluster node

See Also

protocol-error-logging smb-client-stats

Privilege Level

Supervisor
