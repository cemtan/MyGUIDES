router-dump-by-evs

dumps the routes by EVS

Syntax

router-dump-by-evs 
Alias:       dump-interface-routes-by-evs

Description

The router-dump-by-evs command displays the routes that the routing subsystem maintains for each EVS. Along with the test-route-by-evs command, it can be used to help diagnose networking problems.

The command displays information for each of the EVSs currently hosted on the cluster node on which it is run. The command provides the interface routes, which define subnets that are directly connected via one of the interfaces configured on an EVS, and the configured routes that apply to the EVS.

Examples

To display the EVS routes:

server:$ router-dump-by-evs
