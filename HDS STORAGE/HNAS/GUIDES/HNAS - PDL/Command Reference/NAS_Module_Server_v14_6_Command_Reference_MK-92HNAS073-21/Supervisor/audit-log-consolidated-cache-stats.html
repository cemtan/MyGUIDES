audit-log-consolidated-cache-stats

Display basic file system audit log consolidated cache statistics

Syntax

audit-log-consolidated-cache-stats [--reset]

Description

The audit-log-consolidated-cache-stats command displays basic statistics about file system audit log consolidated cache activity.

Options:

-r, --reset

Reset statistics.

Examples

Reset the file system audit log consolidated cache statistics for the current EVS:

audit-log-stats -r

Display the file system audit log consolidated cache statistics for the current EVS:

audit-log-stats

Applies To

EVS
