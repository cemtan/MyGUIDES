cifs-smb2-orphaned-durable-open-files

Display the orphaned durable open files

Syntax

cifs-smb2-orphaned-durable-open-files

Description

The cifs-smb2-orphaned-durable-open-files command is used to display the list of orphaned durable open files on the cluster node.

Examples

cifs-smb2-orphaned-durable-open-files

Applies To

Cluster node
