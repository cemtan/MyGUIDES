sd-restore-cod

Write saved span config back to a single system drive

Syntax

sd-restore-cod [--ssfs] [--denied] [--fscod] [--spancod] <filename> <SD-device-ID>

Description

Restores a single-SD Cod backup (an SDC) made by the 'sd-back-up-cod' command. Cod backups are described in the 'cod' man page.

Specify --spancod or -s if you wish to write the span Cod to the SD; specify --fscod or -f if you wish to write the filesystem Cod (the filesystem catalogue and the chunk table). You must specify at least one of these switches; you can, if you wish, specify both.

Before restoring filesystem Cod, see the warnings in the 'cod' man page about filesystem expansion. If a filesystem has been expanded since the Cod-backup was made, restoring filesystem Cod will corrupt that filesystem; you may be able to recover using 'filesystem-replay-expansions'.

'sd-restore-cod' will prevent you from writing Cod back to the wrong SD. If you are logged on as Dev, you can override this constraint by specifying --othersd. This may possibly be useful if SD Luids have changed.

If you're using ssc, the file is read from the ssc filesystem (i.e. on the computer running the ssc client); you can choose to read it from the server's SSFS file system instead by providing the --ssfs or -F switch. In all other environments the file is always read from the SSFS file system.

If the file is read from SSFS, it is, by default, read from the cluster node where the command is run. To read it from another server's SSFS file system, use the syntax detailed in the 'ssfs' man page.

Specifying the --denied switch lets you restore Cod on to an SD to which access has been denied. In combination with 'sd-back-up-cod' and the Cod converter, this lets you modify the Cod on an SD before allowing access to the SD and loading the Cod into memory.

This command does not restore backups created by the 'span-back-up-cod' command. Use 'span-restore-cod' for that.

Examples

sd-restore-cod --spancod zero.sdc 0
sd-restore-cod --fscod --ssfs zero.sdc 0

Applies To

Cluster node
