run-level

Show system run level

Syntax

run-level 
Alias:       rl

Description

Reports the "run level" of the server, and the status of various services. Only really useful when diagnosing booting problems.

A service is reported as started when its listeners have been set up. This is in contrast to service-running which reports a service as started when it is able to accept incoming connections.

Examples

server$: run-level
Runlevel completed:
  MMB: Running
Service status:
    CNS: Disabled
  ReadCache: Unlicensed
  iSCSI: Unlicensed
   CIFS: Started
    NFS: Started
    FTP: Disabled
