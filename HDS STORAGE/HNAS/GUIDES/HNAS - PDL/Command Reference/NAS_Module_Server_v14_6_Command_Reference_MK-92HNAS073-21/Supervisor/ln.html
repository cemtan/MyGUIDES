ln

Create a link to file system object

Syntax

ln [-s] <target> <link>

Description

WARNING: This command bypasses all security checks and doesn't ask for confirmation, so make sure you know what you're doing.

The ln command is used to create a link to a file system object. The default action is to create a hard link.

Usage:

-s, --symbolic

Create a symbolic link instead of a hard link.

Examples

To create a symbolic link '/foo/link' to the file '/bar/file':

ln -s /bar/file /foo/link
