migration-cloud-rule-delete

Delete an external migration rule

Syntax

migration-cloud-rule-delete <rule name>|<rule id>

Description

Deletes cloud external migration rule.

<rule id>

ID of the rule.

<rule name>

Name of the rule.

Examples

migration-cloud-rule-delete JpgFilesRule

Deletes the cloud external migration rule named 'JpgFilesRule' for this cluster. Will fail in case rule is in use by some policy.

Applies To

Global security context

See Also

migration-cloud-rule-create migration-cloud-rule-delete-all migration-cloud-rule-list migration-cloud-rule-modify
