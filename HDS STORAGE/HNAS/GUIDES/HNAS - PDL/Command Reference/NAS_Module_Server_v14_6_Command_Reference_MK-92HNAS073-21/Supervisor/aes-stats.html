aes-stats

show or reset the AES stats

Syntax

aes-stats [-r] [-v]

Description

The aes-stats command resets or displays AES encryption performance statistics.

Usage:

  aes-stats [-r] [-v]

Options:

-r, --reset

Reset the statistics.

-v, --verbose

Show detailed statistics.

Examples

  aes-stats

Shows the simple statistics.

  aes-stats -r

Resets the statistics.

  aes-stats -v

Shows the detailed statistics

Applies To

Cluster node.

Privilege Level

Supervisor
