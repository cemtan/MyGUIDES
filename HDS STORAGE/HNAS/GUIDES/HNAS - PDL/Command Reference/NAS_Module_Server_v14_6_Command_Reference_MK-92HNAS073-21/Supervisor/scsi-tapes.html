scsi-tapes

Display information about SCSI tape devices

Syntax

scsi-tapes [[device id][-c, --clear]] |
[<-v, --verb>[-c, --clear][-i, --ignore][-p, --port id][-o, --order]] | 
[<-e, --ext> [-c, --clear][-i, --ignore][-p, --port id][-o, --order]] | 
[[-a, --all] [-c, --clear][-l, --luns][-m, --mirror][-o, --order][-r, --paddr]]

Description

Displays information about all tape devices the server has discovered.

Options:

-a, --all

Shows all LUNs/tape devices, including those that are disconnected/removed.

-c, --clear

Clear device error information before displaying.

-e, --ext

Show extended information (verbose format without error information).

l, --luns

For summary information, show all connected LUNs if device has multiple LUNs.

-o, --order

List devices in ascending Device ID order (default is vendor/(system id)/revision/device id order).

-p, --port

Show verbose information about all devices that have a nexus with the given SCSI port id.

-r, --paddr

For summary information, show Host port number, port id and online status of the devices' ports, as Hport:Port:Addrs.

-s, --pstat

For summary information, show port id and online status for each of the devices' ports.

-v, --verb

Show verbose information about all devices.

Examples

Show extended tape device information:

scsi-tapes -e

Show verbose information about all tape devices:

scsi-tapes -v

Applies To

Cluster node

See Also

fc-ports scsi-devices scsi-disks

Privilege Level

Supervisor
