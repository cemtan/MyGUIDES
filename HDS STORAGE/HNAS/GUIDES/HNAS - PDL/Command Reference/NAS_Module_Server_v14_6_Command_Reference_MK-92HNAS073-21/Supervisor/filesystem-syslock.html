filesystem-syslock

Set a filesystem into syslocked mode: read-only for protocols, read/write for NDMP

Syntax

filesystem-syslock <filesystems-see-'man fs-spec'>

Description

Sets file systems into syslocked mode, where they are read-only to protocols (such as NFS and CIFS) but read-write to NDMP.

You can specify one or more file systems using the flexible syntax described in the 'fs-spec' man page.

The status of a syslocked, mounted file system is shown in 'filesystem-list' and 'span-list -v' as 'SysLk'.

Examples

filesystem-syslock Accounts

Applies To

Cluster wide

See Also
