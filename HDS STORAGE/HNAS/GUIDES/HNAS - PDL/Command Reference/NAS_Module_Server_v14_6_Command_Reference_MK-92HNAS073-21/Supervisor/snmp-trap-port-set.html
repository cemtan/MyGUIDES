snmp-trap-port-set

Change SNMP trap sender port

Syntax

snmp-trap-port-set <port>

Description

The snmp-trap-port-set command allows the SNMP trap port (to which the SNMP agent sends traps) to be changed. The default SNMP trap port is 162.

Examples

Set the SNMP trap sender port to 163:

snmp-trap-port-set 163

Applies To

Cluster wide
