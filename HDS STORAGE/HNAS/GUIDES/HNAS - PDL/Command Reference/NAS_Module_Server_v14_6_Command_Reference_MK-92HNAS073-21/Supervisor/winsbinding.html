winsbinding

Configure WINS registration for a specific network interface

Syntax

winsbinding [--enable | --disable] <network interface>
	<interface>       network interface name
	-e, --enable      enable WINS registration
	-d, --disable     disable WINS registration

Description

Used to configure WINS registration for a specific network interface.

Examples

To enable WINS registration on network interface ag1:

winsbinding -e ag1

To disable WINS registration on network interface ag2:

winsbinding -d ag2

To display the existing WINS registration setting on network interface ag1:

winsbinding ag1
