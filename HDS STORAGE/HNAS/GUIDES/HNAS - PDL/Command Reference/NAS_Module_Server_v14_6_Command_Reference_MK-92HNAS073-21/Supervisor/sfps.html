sfps

Dump data from SFPs (deprecated)

Syntax

sfps [--diags | --data] [--port-family (net|cluster|fc)] [--port-number <port-number>] [--port-name <port-name>] [--type (xfp|sfp)] [--hex]

Description

Note - this command is deprecated in favour of pluggable-transceiver-dump.

Dump information on all SFPs on the current board, from the devices' EEPROM, including diagnostics where applicable.

This command is only valid on boards that have SFPs.

Examples

sfps

See Also

pluggable-transceiver-dump

Privilege Level

Supervisor
