cluster-node-route-net-delete

Delete a cluster network route

Syntax

cluster-node-route-net-delete --pnode <pnode> --gateway <gateway> <network>

Description

The cluster-node-route-net-delete command deletes a network route from the routing table but only for the specified pnode.

Options

The following options are recognised:

-g, --gateway <gateway-address>

This is the gateway for the route. It must be supplied.

<network>

This is the peer network to which this route applies. It must be supplied.

-p, --pnode <pnode>

The pnode from which this cluster-node-route will be removed. It must be supplied.

Examples

$ cluster-node-route-net-delete 192.168.0.0/16 --gateway 10.1.1.1 --pnode 2
Route cache flushed.
$

Deletes a route for the network 192.168.0.0/16 that applies only on pnode 2.

Applies To

Cluster wide
