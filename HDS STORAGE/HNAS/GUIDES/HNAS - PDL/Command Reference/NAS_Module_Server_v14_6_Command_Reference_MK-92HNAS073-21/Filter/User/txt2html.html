txt2html

Minimal text-to-HTML filter

Syntax

... | txt2html

Description

Piping a command's output through 'txt2html' will make it safe to include in an HTML document. It will

  • Escape the following characters: < & >
  • Escape bytes 0x7F and above
  • Preserve Newline
  • Strip out Carriage Return and all other Ascii control characters

'txt2html' doesn't produce a self-contained HTML document.

'txt2html' can't be used as a command.

Examples

server:$ echo 'a>b' | txt2html
a&gt;b
server:$ echo a $'\xc3'$'\xa9' b | txt2html
a &#xC3;&#xA9; b
server:$
