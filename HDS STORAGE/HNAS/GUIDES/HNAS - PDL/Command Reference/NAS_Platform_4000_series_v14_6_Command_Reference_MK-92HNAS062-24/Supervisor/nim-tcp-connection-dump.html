nim-tcp-connection-dump

Display the TCPAN per-connection data

Syntax

nim-tcp-connection-dump [-i <conn-id>] | [-s <local-addr>] [-p <local-port>] [-d <remote-addr>] [-g <remote-port>]
Alias:       nim-tcp-connection-tcpan

Description

The nim-tcp-connection-dump command is used to display the most recently gathered traffic data for the specified TCP connection.

The following options are available:

-i

Specifies the connection ID to select

-s

Specifies the source (local) address to select

-d

Specifies the destination (remote) address to select

-p

Specifies the source (local) port to select

-g

Specifies the destination (remote) port to select

Applies To

Cluster node
