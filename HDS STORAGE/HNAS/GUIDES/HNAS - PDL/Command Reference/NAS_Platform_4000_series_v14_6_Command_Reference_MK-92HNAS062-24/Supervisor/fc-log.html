fc-log

dump/clear FC logs

Syntax

fc-log [-d, --dump][-c, --clear][-s, --size][-h, --help]

Description

Dump or clear the contents of the fibre channel (FC) log or logs.

There is a separate log for each host port; it logs FC commands transmitted and responses received from the given host port.

If no host port number is specified, host port 1 FC log is assumed.

Console output may be stopped or aborted with a key press or by CTRL-C.

The syntax of a log entry indicates the FC frame that was transmitted or received as follows:

Sent/Rxed

FC frame was sent or received by the server host port.

Frame Type

The name of the command frame sent to the FC target port, or the name of the response frame received by the FC target port.

xxxxxx

Address of the target port that the FC command was sent to or a response was received from.

<OXID>

The OXID of the exchange.

[Sn]

n = the server internal session number of the exchange.

STATUS

The status of the session.

OFFLINE

The remote port is offline or is not responding and has been logged out.

POLLING

The remote port is not responding so the host port is periodically polling it.

INACTIVE

The remote port is an initiator port.

DISCOVERY

The session is attempting to establish a login session with the remote port.

PENDING

Attempting to re-establish an active login session with the remote target port.

ACTIVE

The session has successively established a login session with the target port.

Options:

-d, --dump

Dump the contents of the FC log for the given host port for all host ports. The last 20 entries are output if no range is specified.

-h, --hport <1..n>

Specify host port log to dump.

-H, --Hports

Dump logs for all host ports.

-p, --port <addrs>

Filter by the target port address.

-r, --rev

Output entries in reverse order, oldest last.

-a, --all | -n <[-]no of entries to dump>

Output all entries, last n entries, or first n entries if n is negative. If n = 0, dump entire log; by default, dump last 20 entries.

-c, --clear

Clear the log, default is host port 1 log.

-h, --hport <1..n>

Specify host port log to be cleared.

-C, --Conf

Do not request confirmation before clearing the log.

-s, --size

Show the current and maximum log size.

-h, --help

Show help.

Examples

Dump the entire FC log for host port 1:

fc-log -d -a

Dump last 100 entries from host port 2 log for target port on FC address 10200:

fc-log -d -h 2 -p 0x10200 -n 100

Dump first 20 entries from host port 2 log for target port on FC address 610c00:

fc-log -d -h 2 -p 610c00 -n -10

Dump all entries from host port 1 log:

fc-log -d -h 1 -a

Dump all entries from all host port logs:

fc-log -d -H -a

Clear the host port 1 log, no confirmation:

fc-log -c -h 1 -C

Clear all host port logs, no confirmation:

fc-log -c -H -C

Applies To

Cluster node
