checkpoint-stats

Dumps stats on checkpoints for specified file system

Syntax

checkpoint-stats [-r|--reset] [-s|--summary] [--logged [--raw]] [-f <file-system>|--file-system <file-system>]
Alias:       cpstats

Description

Dumps checkpoint-related statistics, either for all file systems or a given file system.

Options:

--logged

Dumps information gathered about the previous sixteen checkpoints.

--raw

When used in conjunction with the --logged option, outputs raw VLSI and software timings for the last sixteen checkpoints.

-s, --summary

Outputs tables summarizing the timings taken during checkpoints.

-r, --reset

Resets the statistics.

-f, --file-system

Only outputs or resets statistics for a single file system. If this option isn't used, the command will output statistics for all file systems in the current EVS.

With no options specified, the command will output detailed statistics for all the file systems in the current EVS.

Examples

To see the checkpoint statistics for file system "fs1":

checkpoint-stats --file-system fs1

Applies To

EVS

Privilege Level

Supervisor
