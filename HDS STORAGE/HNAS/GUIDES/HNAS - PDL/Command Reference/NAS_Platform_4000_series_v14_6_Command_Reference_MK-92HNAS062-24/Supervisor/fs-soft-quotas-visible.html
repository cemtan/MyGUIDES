fs-soft-quotas-visible

Get/Set whether soft quotas affect freespace/capacity reporting

Syntax

fs-soft-quotas-visible [on|off]

Description

Configures whether quotas with soft limits affect file system freespace/capacity reporting.

When this option is disabled only quotas with hard limits affect the freespace/capacity counts returned to a client.

Usage:

fs-soft-quotas-visible [on|off]

Examples

fs-soft-quotas-visible on

Applies To

Cluster wide

See Also

fs-user-quotas-visible

Privilege Level

Supervisor
