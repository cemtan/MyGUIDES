cifs-close

Closes files opened over CIFS

Syntax

cifs-close <file-system> <path>

Description

The cifs-close command can be used to forcibly close files that were opened over CIFS. This may be useful if, for example, a client is holding a file open, preventing other clients from being able to access it.

Usage:

cifs-close <file-system> <path>

Examples

cifs-close fs0 \company\myfiles\mysharedfile.dat

Privilege Level

Supervisor
