event-log-definitions

Display event definitions

Syntax

event-log-definitions operation
where 'operation' is one of:
    display [-c] event_id
    list [-g group_name] [-c]
    groups

Description

The event-log-definitions command displays event definitions.

Options

display [-c] event_id

display definition for specific event id

-c, --csv

list in comma-separated format

list [-g group_name] [-c]

list event definitions

-g, --group

show definitions for specific group

-c, --csv

list in comma-separated format

groups

list the names of event groups

Examples

To display definition of event 9995:

event-log-definitions display 9995

To list definitions of events in system group:

event-log-definitions list -g System

To list the event groups:

event-log-definitions groups

Applies To

Cluster node
