rpc-errors

Report RPC protocol errors

Syntax

rpc-errors [-r|--reset]

Description

The rpc-errors command is used to report the errors that have been returned at the ONC/RPC layer. ONC/RPC is used to transport application layer protocols such as NFS, MOUNT, NLM and NSM.

Usage:

--reset, -r

Resets the error log.

Applies To

Cluster node

See Also

protocol-error-logging

Privilege Level

Supervisor
