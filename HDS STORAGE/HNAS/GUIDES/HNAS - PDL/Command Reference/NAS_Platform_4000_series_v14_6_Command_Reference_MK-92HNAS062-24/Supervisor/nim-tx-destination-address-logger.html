nim-tx-destination-address-logger

Display Discovery log

Syntax

nim-tx-destination-address-logger

Description

The nim-tx-destination-address-logger command displays a record of recent Discovery operations.

The columns record the following information:

Operation

Operation being traced. An operation may be logged at it's start and end, in which case the suffix "-D" meaning "Done" is added to the Operation name of the end log.

Local IP

The server IP address to which this entry relates.

Peer IP

The IP address of a remote machine to which this entry relates.

Next Hop MAC address

The MAC address to which all traffic for this entry will be sent. If the remote machine is within the same subnet, this will be a MAC address of that machine; otherwise it will be a MAC address of a local gateway. If the address is not yet resolved it will be 00-00-00-00-00-00.
