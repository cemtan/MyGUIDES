home-directory-perf

Display Home Directory performance statistics

Syntax

home-directory-perf [[--reset] | [--summary] [--verbose] [--file-system <label> | --all-file-systems]] [<request-pattern>]

Description

The home-directory-perf command displays performance statistics about file system home directory activity.

Usage:

home-directory-perf [OPTION...] [<request-pattern>]

--all-file-systems, -a

Show per file system stats for all file systems.

--file-system, -f

Show per file system stats for the specified file system.

--reset, -r

Reset the statistics.

--summary, -s

Only display a summary.

--verbose, -v

Display verbose information.

<request-pattern>

Only show statistics for requests with names that contain the given string.

Examples

Display the performance statistics for the file system "Span0FS":

home-directory-perf -f Span0FS

Reset the performance statistics:

home-directory-perf -r

Applies To

Cluster node

See Also
