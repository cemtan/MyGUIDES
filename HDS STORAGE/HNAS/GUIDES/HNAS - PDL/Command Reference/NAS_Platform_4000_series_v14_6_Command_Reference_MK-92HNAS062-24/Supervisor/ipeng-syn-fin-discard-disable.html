ipeng-syn-fin-discard-disable

Allow packets with syn and fin flags both set

Syntax

ipeng-syn-fin-discard-disable

Description

The ipeng-syn-fin-discard-* commands control how packets are handled if the SYN and FIN flags are both set. It is a global setting, acting on all ports.

This feature is not usually necessary, and hence is turned off by default. Only use under instruction from your support provider.

The commands available are as follows.

ipeng-syn-fin-discard-enable

Packets which have the SYN and FIN flags both set should be discarded.

ipeng-syn-fin-discard-disable

Allow packets which have both the SYN and FIN flags set.

ipeng-syn-fin-discard-show

Display whether the feature is currently enabled or not.

Examples

To start discarding packets with SYN and FIN flags set: $ ipeng-syn-fin-discard-enable $

To no longer discard packets with SYN and FIN flags set: $ ipeng-syn-fin-discard-disable $

Displaying current feature setting: $ ipeng-syn-fin-discard-show Parameter Value Default ----------------------------------------- ---------- ---------- Syn Fin Discard Enabled: No No $

Applies To

Cluster wide
