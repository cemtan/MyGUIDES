virusscan-max-file-size-enable

Enable the virus scanning maximum file size feature

Syntax

virusscan-max-file-size-enable

Description

Turn on the maximum file size for virus scanning. Use "virusscan-max-file-size-show" to check the current setting.

Scanning large files for viruses can take a noticeable time and/or consume limited bandwidth. Use this command in conjunction with virusscan-max-file-size-set to limit virus scanning only to files smaller than some maximum you specify.

Examples

To enable a maximum file size limit:

virusscan-max-file-size-enable

Applies To

EVS

See Also

virusscan-max-file-size-disable virusscan-max-file-size-set virusscan-max-file-size-show

Privilege Level

Supervisor
