ndmp-device-stats

Dump the stats for the specified ndmp device

Syntax

ndmp-device-stats -i <device number> | -n <device name>

Description

Displays the NDMP device performance statistics for the specified device. The stats are reset when a device is allocated to an NDMP session, so using this command after the end of an NDMP session will show you the stats for the last session that used the device. The stats for the device associated with a session will also be displayed by "ndmp-session-stats -p", and reset by "ndmp-session-stats -r" and therefore also reset when a performance-info-report is taken. Whilst the command will attempt to output statistics for a changer (as opposed to a tape) device, it is extremely unlikely that any statistics have been collected for a changer device as the locations in the code that log statistics are nearly all tape device specific.

Options:

--device-name, -n <device name>

Display stats for the device specified by its name.

--device-number, -i <device number>

Display stats for the device specified by its number. The device number for a specific device is displayed in the output of the "ndmp-devices-list" command.

Examples

Display the stats for device number 1:

ndmp-device-stats -i 1

Display the stats for device "Tape Drive 2":

ndmp-device-trace  --device-name "Tape Drive 2"

Applies To

Cluster node

See Also

ndmp-devices-list ndmp-session-stats performance-info-report

Privilege Level

Supervisor
