virusscan-icap-engine-delete

Delete an ICAP virus scanning engine

Syntax

virusscan-icap-engine-delete [--confirm] <host>

Description

The virusscan-icap-engine-delete command is used to remove an ICAP virus scanning engine from the current ICAP engine list. If the current virus scanning mode is RPC then no action will be taken. Note that if virus scanning is on and all virus scan engines are deleted, CIFS access to files might be denied.

Options:

--confirm

If the --confirm option is not supplied, the user will be prompted to confirm the virusscan-icap-engine-delete request if this is the last ICAP scan engine registered and in use.

<host>

A hostname, or an IP address, that identifies the host running the ICAP virus scanning engine.

Examples

To remove the ICAP engine icap.example.com from the engine list:

Using a hostname:

virusscan-icap-engine-delete icapengine

Alternatively:

virusscan-icap-engine-delete icapengine.example.com

Alternatively, using an IP address:

virusscan-icap-engine-delete 192.168.43.100

Applies To

EVS
