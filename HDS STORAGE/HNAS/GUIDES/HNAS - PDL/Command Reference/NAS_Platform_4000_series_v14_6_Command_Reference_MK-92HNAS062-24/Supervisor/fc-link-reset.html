fc-link-reset

Reset one or all of the FC interfaces

Syntax

fc-link-reset [-i <1|...|N>] [-t <time in ms>]
Alias:       fclinkreset

Description

The fc-link-reset command allows you to reset one or all of the Fibre Channel interfaces. Command options are:

-i <interface_number>

Specify the FC interface to reset. If this option isn't set then the command applies to all available interfaces. Interfaces are numbered from 1.

-t <time_in_ms>

If this option is passed then the specified interface (see -i option) or all interfaces will be reset for the specified period. This option is not normally required and the period defaults to one second.

Examples

Reset all interfaces:

fc-link-reset

Reset interface 1:

fc-link-reset -i 1

Applies To

Cluster node

Privilege Level

Supervisor
