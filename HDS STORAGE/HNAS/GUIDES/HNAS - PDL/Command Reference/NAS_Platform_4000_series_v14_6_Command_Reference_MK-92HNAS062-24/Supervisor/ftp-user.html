ftp-user

Report, create, delete & modify FTP user

Syntax

ftp-user operation
where 'operation' is one of:
list [<username>]
add [-e] <username> <file-system> <initdir>
del <username>
delall [--confirm]
mod [-e] [-f <file-system>] [-i <dir>] <username>

Description

The ftp-user command is used to configure a user's FTP access.

Options:

list [<username>]

List all or single user.

add [-e] <username> <file-system> <initdir>

Add a new user, specifying file system and initial directory.

<username> can be an NT/UNIX name, or a special name that designates anonymous access (either "anonymous" or "ftp").

-e, --ensure

Ensure 'initdir' exists.

del <username>

Delete a user.

delall [--confirm]

Delete all users.

mod [-e] [-f <fs>] [-i <dir>] <username>

Modify a user's configuration.

-f, --file-system

Change file system to 'fs'.

-i, --initdir

Change initial directory to 'dir'.

-e, --ensure

Ensure 'dir' exists.

Examples

Display all FTP users:

ftp-user list

Add user 'john' on file system 'fs1' with initial directory '/johndir':

ftp-user add -e john fs1 /johndir

Delete user 'rob':

ftp-user del rob

Delete all users:

ftp-user delall --confirm

Change file system to 'fs2' for user 'john':

ftp-user mod -f fs2 john
