ndmp-fc-device-number-set

Assign a number to Fibre Channel Device used by NDMP

Syntax

ndmp-fc-device-number-set [--override] <fc_node_wwn> <assigned_number>

Description

The Device Number assigned by NDMP to a Fibre Channel Device is used in the NDMP device names generated by NDMP. See ndmp-devices for details.

This command changes the Device Number used to refer to a specific Fibre Channel Device. The Fibre Channel Device is identified by its Fibre Channel node name. (Use the 0x prefix to indicate this is in hexadecimal.)

The --override option must be used if the Device Number is currently mapped to a different Fibre Channel Device. If this is used the other Fibre Channel Device will be renumbered automatically.

Examples

Override an existing device numbering to replace an old tape library:

server(NIM):$ ndmp-fc-device-number-set --override 0x5005084000278001 4
The number assignment was registered
The following reassignment resulted from override action:
FC Node WWN (hexadecimal): 0x5005084000278000   Assigned Number: 5

Applies To

Cluster wide
