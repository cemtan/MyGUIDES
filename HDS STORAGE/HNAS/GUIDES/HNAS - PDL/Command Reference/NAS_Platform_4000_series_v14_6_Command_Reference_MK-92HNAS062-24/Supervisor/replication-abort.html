replication-abort

Abort an object replication

Syntax

replication-abort { <replication guid> | <policy name> <source fs label> }

Description

Aborts an object replication.

<replication guid>

GUID of the object replication session to abort.

<policy name>

Policy name of the object replication to abort. Note that using a policy name will only perform a source abort. Provide a guid for a target abort.

<source fs label>

Name of the source file system of the object replication to abort. Required with a policy name.

Examples

replication-abort fd8d1540-71c8-11ca-9001-c5d1939ee7a7

Aborts the object replication session with GUID fd8d1540-71c8-11ca-9001-c5d1939ee7a7.

replication-abort DefaultPolicy fs0

Aborts the object replication with policy named 'DefaultPolicy' and source file system named 'fs0'.

Applies To

EVS
