backup-device-deny-access

Deny access to one or more tape/media changer devices

Syntax

backup-device-deny-access <backup-devices-see-'man sd-spec'>

Description

Prevents the server or cluster from using the specified backup device.

Examples

backup-device-deny-access 800

Applies To

Cluster wide

See Also

backup-device-allow-access backup-devices-list sd-spec

Privilege Level

Supervisor
