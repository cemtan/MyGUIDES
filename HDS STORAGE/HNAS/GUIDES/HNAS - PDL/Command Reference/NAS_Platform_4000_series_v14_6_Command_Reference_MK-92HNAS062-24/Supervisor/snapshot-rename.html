snapshot-rename

Rename a manually created snapshot.

Syntax

snapshot-rename --file-system <file-system> <current-snapshot-name> <new-snapshot-name>

Description

Renames a manually created snapshot of a specified file system.

--file-system, -f <file-system>

Select file system by label.

Snapshots created by invoking a snapshot rule, or internally generated by the system, cannot be renamed.

Examples

server:$ snapshot-rename --file-system fs1 testB monday
Information: Snapshot renamed - ("monday" on fs1) was previously called "testB"
