ca-certificate-show

Show the installed custom certificates.

Syntax

ca-certificate-show [--verbose | --pem]

Description

The ca-certificate-show command displays the custom certificates contained in the system trusted certificate store.

By default the command will output a summary list of the installed certificates.

Options:

-p, --pem

Output the certificates in base64 PEM format.

-v, --verbose

Output the certificates in human readable format.

Examples

To list the installed certificates:

$ ca-certificate-show


Status                 Valid from                Valid unti                      Name
------   ------------------------   ------------------------   ----------------------
 Valid   Oct 20 11:29:06 2017 GMT   Oct 20 11:29:06 2022 GMT   hcp-01.example.com.crt
 Valid   Nov 22 11:23:07 2017 GMT   Nov 22 11:23:07 2022 GMT   hcp-02.example.com.crt

Applies To

Cluster wide
