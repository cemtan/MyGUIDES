nfs-hostname

Get, Set or Clear NFS server's hostname

Syntax

nfs-hostname [[--clear] | [-c] | [<hostname>]]

Description

This command is used to display, set or clear the DNS hostname used by NFS to create its GSS-API credential.

Clearing the name results in the empty string being saved as the name. Setting and clearing are mutually exclusive.

Examples

nfs-hostname server1.example.com


nfs-hostname --clear


nfs-hostname -c

Applies To

EVS
