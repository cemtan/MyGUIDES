nis-broadcast-mode

Configure NIS to use broadcast mode or not

Syntax

nis-broadcast-mode on | off

Description

Displays and manipulates whether the NIS client locates potential servers using broadcasts.

Examples

To display the current setting of NIS broadcast mode:

nis-broadcast-mode

To permit the NIS client to find servers via broadcast:

nis-broadcast-mode on

To prevent the NIS client from finding servers via broadcast:

nis-broadcast-mode off

Applies To

EVS
