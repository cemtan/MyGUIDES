routing-by-evs-show

Show state of routing by EVS

Syntax

routing-by-evs-show

Description

The routing-by-evs-show command shows whether the routing engine is using routing-by-evs or not.

Examples

To show whether routing by EVS is in effect:

server:$ routing-by-evs-show
routing-by-EVS is disabled

Applies To

Cluster wide

See Also

routing-by-evs routing-by-evs-disable routing-by-evs-enable

Privilege Level

Supervisor
