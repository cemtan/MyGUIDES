network-protocol-performance-monitor-set-defaults

Set network protocol performance monitor statistics back to factory default settings.

Syntax

network-protocol-performance-monitor-set-defaults [--confirm]

Description

Wipes any network protocol performance monitor configuration settings back to their defaults. Any manually configured log sizes will be cleared and the auto logging feature will be enabled.

With the configuration reset, some stored statistics may be lost depending on how the auto logging level compares to the previous manual configuration.

Options:

-c, --confirm

An option to override the question on whether you really wish to wipe the configuration.

Examples

Reset network protocol performance monitor back to defaults, without asking for confirmation.

    network-protocol-performance-monitor-set-defaults -c

Applies To

Cluster wide
