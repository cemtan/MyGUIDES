cluster-create

Create an Active-Active cluster

Syntax

cluster-create [-c | --confirm] <cluster name> <QD name/ipaddr>

Description

The cluster-create command is used to create a new cluster.

See the 'cluster-interfaces' man page for guidelines on configuring the cluster network.

Usage:

-c, --confirm

Confirms cluster creation.

<cluster name>

Specifies the cluster name.

<QD name/ipaddr>

Specifies the quorum device's name or IP address.

Examples

server:$ cluster-create MY-CLUSTER MY-QD
Cluster name: MY-CLUSTER
Quorum name: MY-QD
This command will update the configuration and reboot the system.
Do you want to proceed?(Y/N)[N]:
y

Applies To

Cluster node

See Also

cluster-interfaces cluster-join cluster-node-delete cluster-show cluster-unconfigure

Privilege Level

Supervisor
