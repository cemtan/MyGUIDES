smtp-retry-connection

Retry the connection to the SMTP server now

Syntax

smtp-retry-connection

Description

The smtp-retry-connection command tells the SMTP client to attempt to connect to the SMTP server and attempt to deliver queued messages immediately, disregarding any wait period it may currently be undergoing due to a previous failure to communicate with an SMTP server.

Examples

smtp-retry-connection

Applies To

Cluster node

See Also

smtp-clear-queue smtp-statistics

Privilege Level

Supervisor
