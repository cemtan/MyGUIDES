vlan-interface-delete-all

Delete all VLAN interfaces

Syntax

vlan-interface-delete-all --confirm [--verbose]

Description

The vlan-interface-delete-all command tries to delete all existing VLAN interfaces.

The following options are recognised:

-c, --confirm

The --confirm option is required to verify that the command should remove all VLAN interfaces.

-v, --verbose

The --verbose option causes successful interface deletions to be listed. Deletion failures are listed even if the --verbose option is not specified.

Examples

$ vlan-interface-delete-all --confirm
$

Deletes all vlan interfaces.

Applies To

Cluster wide
