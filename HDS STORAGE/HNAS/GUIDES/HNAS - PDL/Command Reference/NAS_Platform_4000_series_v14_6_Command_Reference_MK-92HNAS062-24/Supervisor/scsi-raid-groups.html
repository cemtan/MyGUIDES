scsi-raid-groups

Display the Raid Groups for each storage rack or the Raid Group information for a SD.

Syntax

scsi-raid-groups [device id] [--unknowns]

Description

Display Raid group information for all storage racks or for the specified SD. Each Raid group relates to one or more of the discovered storage devices.

The information displayed includes the following: the Raid group type; the group number; a sub group number (for Raid group type only); the Raid level; the group status; and the members of the group, showing the device ID, the external (host) LUN and the internal LUN.

The group type may be one of the following: Raid group; DP pool; external group; snapshot volume; or unknown.

Note, Raid group information will only be retrieved from HDS storage arrays.

Options:

device id

Display the Raid group information for the specified device ID. This option is used primarily to check the cached Raid group information for the SD.

-u, --unknowns

Include the Raid Groups with unknown type as well.
