vlan-interface-show

Display all VLAN interfaces

Syntax

vlan-interface-show [--interface <interface>]

Description

The vlan-interface-show command lists all existing VLAN interfaces.

The following options are recognised:

-i, --interface <interface>

Only display VLAN interfaces associated with the specified aggregation interface.

Examples

$ vlan-interface-show
ag1-vlan0100
ag1-vlan0300
ag2-vlan0039
ag2-vlan0100
$

Lists the existing VLAN interfaces.

Applies To

Cluster wide
