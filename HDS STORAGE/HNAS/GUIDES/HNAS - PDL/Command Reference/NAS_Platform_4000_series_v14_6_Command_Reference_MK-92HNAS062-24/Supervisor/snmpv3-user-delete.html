snmpv3-user-delete

Delete an SNMPv3 user

Syntax

snmpv3-user-delete <username>

Description

The snmpv3-user-delete command can be used to remove an SNMPv3 user from the system so they can no longer be used to authenticate SNMPv3 requests. Once a user has been deleted they cannot be recovered though a user of the same name can be re-added (man snmpv3-user-add).

Examples

Delete an SNMPv3 user named "existingUser":

snmpv3-user-delete existingUser

Applies To

Cluster wide
