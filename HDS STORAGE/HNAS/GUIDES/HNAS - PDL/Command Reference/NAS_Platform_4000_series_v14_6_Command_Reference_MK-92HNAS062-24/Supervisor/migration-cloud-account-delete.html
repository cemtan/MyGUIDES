migration-cloud-account-delete

Delete an external cloud migration account

Syntax

migration-cloud-account-delete <account name>|<account id>

Description

Deletes a cloud migration account representation on this cluster.

<account name>

Name of the cloud account created with migration-cloud-account-create

<account id>

ID of the account created with migration-cloud-account-create. This value can be obtained by specifying the account name with the migration-cloud-account-list command.

Examples

migration-cloud-account-delete HNAS-S3target

Deletes cloud account called HNAS-S3target.

Applies To

Global security context

See Also

migration-cloud-account-create migration-cloud-account-list migration-cloud-account-modify migration-cloud-destination-delete

Privilege Level
