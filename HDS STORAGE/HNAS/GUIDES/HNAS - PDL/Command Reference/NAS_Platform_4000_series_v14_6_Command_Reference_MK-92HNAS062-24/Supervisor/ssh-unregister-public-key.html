ssh-unregister-public-key

Delete a stored SSH public key

Syntax

ssh-unregister-public-key <user-name> <ip-address> [<dsa|rsa>]

Description

Deletes an SSH public key from the server's authorized list, given the user-name, ip-address and type of key.

Options:

user-name

Name of user to delete

ip-address

IP address to delete

dsa|rsa

Type of key to delete

Examples

ssh-unregister-public-key johnsmith 10.1.1.99 dsa

Deletes dsa public key for johnsmith login from ip-address 10.1.1.99.

ssh-unregister-public-key johnsmith 10.1.1.99 rsa

Deletes rsa public key for johnsmith login from ip-address 10.1.1.99.

ssh-unregister-public-key johnsmith 10.1.1.66

Deletes both dsa and rsa public keys for johnsmith login from ip-address 10.1.1.66.

Privilege Level

Supervisor
