replication-delete-record

Deletes the replication record (Use only on instruction from your support provider)

Syntax

replication-delete-record -f <fs_label> -n <replic_name> [-d]

Description

This command is only intended for use by SMU scripts and support. Do NOT use this command except under guidance from your support provider. We recommend that the SMU be used for all replication configuration.

Options:

-f, --file-system <fs_label>

Label of the file system addressed.

-n, --name <replic_name>

Name of the replication.

-d, --destination

If set, deletes destination record.

Applies To

EVS

Privilege Level

Supervisor
