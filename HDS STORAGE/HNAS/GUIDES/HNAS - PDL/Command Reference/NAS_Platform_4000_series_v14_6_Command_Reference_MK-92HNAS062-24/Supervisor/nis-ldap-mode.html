nis-ldap-mode

Display / update common NIS/LDAP mode info

Syntax

nis-ldap-mode [ -m  nis | ldap]

Description

The nis-ldap-mode command permits you to select whether LDAP or traditional NIS servers are used to satisfy NIS queries.

With no arguments, the nis-ldap-mode command displays current NIS configuration info.

Examples

Display the current NIS mode settings:

nis-ldap-mode

Set NIS mode to NIS (non-LDAP servers):

nis-ldap-mode -m nis

Set NIS mode to use LDAP servers. Server addresses must be set via the ldap-server command:

nis-ldap-mode -m ldap
ldap-server add 10.2.20.152

Applies To
