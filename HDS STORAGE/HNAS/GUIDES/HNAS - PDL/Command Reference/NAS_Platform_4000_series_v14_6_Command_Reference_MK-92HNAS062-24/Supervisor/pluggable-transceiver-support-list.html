pluggable-transceiver-support-list

Show to the supported parts list(s)

Syntax

pluggable-transceiver-support-list [--port-family <port-family>]

Description

Lists the pluggable transceiver (XFP, SFP or SFP+) models officially supported by this firmware. Additionally, if any transceiver models have been accepted by the administrator (non-default support) then these are also listed.

Transceiver ports are grouped into families ('net', 'cluster' and 'fc'). The list of supported transceiver models differs between families; the 'net' and 'cluster' families use the same list while the 'fc' family uses a different list.

Options:

--port-family <net|cluster|fc>

Only display model lists applicable to transceivers attached to ports belonging to the given family. The 'net' and 'cluster' options are aliases of each other.

Examples

pluggable-transceiver-support-list
Lists all supported pluggable transceiver models.


pluggable-transceiver-support-list --port-family fc
Lists supported pluggable transceiver models for the 'fc' port family.

Applies To

Cluster wide
