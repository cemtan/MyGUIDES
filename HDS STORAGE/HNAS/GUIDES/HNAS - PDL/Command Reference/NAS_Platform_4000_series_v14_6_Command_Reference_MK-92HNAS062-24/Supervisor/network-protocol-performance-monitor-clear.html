network-protocol-performance-monitor-clear

Clear all statistics stored by the network protocol performance monitor.

Syntax

network-protocol-performance-monitor-clear [--confirm]

Description

Discards all current network protocol performance monitor statistics from the node. Configuration is preserved, and logging continues as before so the statistics should start re-filling with the next sample.

Options:

-c, --confirm

An option to override the question on whether you really wish to clear all currently stored stats.

Examples

Discard all current network protocol performance monitor statistics.

    network-protocol-performance-monitor-clear -c

Applies To

Cluster node.
