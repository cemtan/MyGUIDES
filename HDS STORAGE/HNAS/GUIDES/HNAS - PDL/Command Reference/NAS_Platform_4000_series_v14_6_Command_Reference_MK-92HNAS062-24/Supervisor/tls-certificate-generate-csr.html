tls-certificate-generate-csr

Generate and display a Certificate Signing Request (CSR)

Syntax

tls-certificate-generate-csr [--human-readable]
Alias:       ssl-get-csr

Description

The tls-certificate-generate-csr command generates and displays the SSL certificate signing request (CSR) for the current SSL certificate used for management https, SMU and remote PAPI access.

Options:

--human-readable

Print the CSR in a human readable form. This command is unlikely to need be specified and most likely you will need to supply the CSR in the default form to your CA.

Examples

tls-certificate-generate-csr

To generate and display the current certificate signing request

tls-certificate-generate-csr --human-readable

To generate and display the current certificate signing request in human readable form

Applies To

Cluster wide
