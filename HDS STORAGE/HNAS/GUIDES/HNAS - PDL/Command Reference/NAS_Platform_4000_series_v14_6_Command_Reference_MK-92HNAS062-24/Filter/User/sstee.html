sstee

Write to console and file

Syntax

... | sstee <file>

Description

Command output may be piped through sstee to write it simultaneously to the console and the specified file. It may not be used as a command.

There is just one parameter to sstee, the ssfs filename.

Examples

Dump the management log to the "tmp" file:

mgmnt-log-show | sstee tmp

Dump the management log to the "tmp" file, not outputting any text to the console:

mgmnt-log-show | sstee tmp | devnull

Privilege Level

User
