fc-stats

Show FC statistics

Syntax

fc-stats [-a] [-r] [-R]
Alias:       fcstats

Description

This is a legacy command retained for backwards compatibility. It is superseded by:

  • storage-request-stats
  • storage-throughput-stats
  • fc-link-congestion
  • fc-link-errors

See those commands' man pages for detailed descriptions of the output from fc-stats.

Options

-a, --all

display statistics for all interfaces, not just enabled ones

-r, --reset

reset statistics - equivalent to:

storage-request-stats -r; storage-throughput-stats -r; fc-link-errors -r

-R, --reset-throughput

reset throughput statistics - equivalent to:

storage-throughput-stats -r

Applies To

Cluster node

See Also

fc-link-congestion fc-link-errors storage-request-stats storage-throughput-stats

Privilege Level

User
