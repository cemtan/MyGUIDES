ifconfig

displays network configuration

Syntax

ifconfig [-v] [interface]

Description

ifconfig displays the status of the currently active interfaces.

If a single interface is provided, it displays the status of that interface only.

-v

Shows all network interfaces. These include interfaces which are used internally within the server.

The various fields displayed are:

Link encap

A value that indicates link encapsulation: 1 means Ethernet.

HWAddr

MAC Address of this interface.

inet addr

IP addresses for this interface.

Bcast

Broadcast address corresponding to the specified IP address.

mask

Network mask corresponding to the specified IP address.

inet6 addr

IPv6 addresses for this interface.

Scope

Whether the corresponding address is of Link-local or Global scope.

Link

Current physical link status; can be UP or DOWN.

Admin

UP if an IP address has been assigned to this interface, otherwise DOWN.

MTU

Maximum size IP packet that can be transmitted over the interface without requiring fragmentation. This value is not negotiated but will change in response to MTU settings set by the ipadv command. The fileserving interfaces support differing MTUs for TCP and non-TCP traffic. The MTU specified for cluster interfaces does not apply to traffic generated by the VLSI (for example, NVRAM traffic, which requires a jumbo-frame capable network that supports at least 9216 byte frames).

Metric

A value used to determine which interface to route over when multiple interfaces provide a suitable path. Lower values have higher priority.

txqueuelen

Maximum number of simultaneous messages that may be queued on the interface awaiting transmission.
