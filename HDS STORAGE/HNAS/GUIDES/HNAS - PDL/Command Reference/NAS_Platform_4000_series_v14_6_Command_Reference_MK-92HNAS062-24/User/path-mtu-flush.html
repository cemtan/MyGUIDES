path-mtu-flush

Flush Path MTU Table

Syntax

path-mtu-flush [-i <interface>]

Description

The 'path-mtu-flush' command flushes the contents of the Path MTU table. Path MTU Discovery only applies to IPv6 traffic and operates in accordance with RFC-1981.

Options:

-i, --interface <name>

Limits the flush operation to table entries associated with the named interface.

Examples

To clear the Path MTU table of all entries for interface 'ag1':

    path-mtu-flush -i ag1

Applies To

Cluster node
