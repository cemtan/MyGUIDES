host

Host lookup

Syntax

host hostname

Description

The host command performs a host name lookup using the nameservices configured by the nsorder command. It can translate names into IP addresses and IP addresses into names.

Examples

To resolve www.example.com into an IP address:

host www.example.com

To resolve an IP address into a name:

host 198.175.96.33

Applies To

EVS security context
