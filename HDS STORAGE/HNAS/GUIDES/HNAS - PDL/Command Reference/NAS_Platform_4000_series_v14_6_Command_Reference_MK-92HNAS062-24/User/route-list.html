route-list

Display standard route registry configuration

Syntax

route-list

Description

The route-list command displays the configured standard route entries as stored in the registry. The format matches that used by the route command.

If this command is issued when the cluster node is in cluster-route mode the following warning will be issued as a reminder that the routes displayed are not currently active:

  NOTE: None of these standard routes is currently in effect.

Examples

$ route-list
Using standard routing
Destination                    Gateway                    MTU     Flags
0.0.0.0/0                      172.18.254.254             Default G
1.2.3.4/32                     172.0.0.0                  Default H
::/0                           fdca:f995:220a:480:1:1:2e:1 Default G
fe80::1/128                    fdca:f995:220a:480:1:1:2e:1 Default H
fe80::/120                     fdca:f995:220a:480:1:1:2e:1 Default
fdca:f995:220a:480:1:1:2e:1/128 fdca:f995:220a:480:1:1:2e:1 Default H

Lists the current routes.

Applies To

Cluster wide
