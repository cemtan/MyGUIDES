dnsdomainname

Define DNS domain name

Syntax

dnsdomainname operation
where 'operation' is one of
	show
	set domainname
	clear

Description

The dnsdomainname command allows the DNS domain name to be manipulated.

Note:

Changing the domain name will trigger creation of a new self-signed SSL/TLS certificate for the SOAP HTTPS management server and (if present) for the embedded SMU. However, if you had installed a custom certificate it will not be affected (but you may want to consider updating it to match the new domain name).

Examples

To show the current DNS domain name:

dnsdomainname show

To set the DNS domain name:

dnsdomainname set example.com

To clear the DNS domain name:

dnsdomainname clear

Applies To

EVS security context
