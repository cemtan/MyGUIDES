cifs-dc-stats

Report Domain Controller statistics

Syntax

cifs-dc-stats [--reset | -r] [--boot | -b]

Description

The cifs-dc-stats command reports domain controller usage statistics.

Usage:

[--reset | -r]

Resets the statistics.

[--boot | -b]

Without this option, the command displays the statistics since they were last reset. With this option, the statistics since boot are displayed.

Applies To

Cluster node

See Also
