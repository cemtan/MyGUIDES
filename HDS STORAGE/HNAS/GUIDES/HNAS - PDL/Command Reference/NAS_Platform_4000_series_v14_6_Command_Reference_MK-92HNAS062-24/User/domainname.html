domainname

Get or set the current NIS domain.

Syntax

domainname [domainname]

Description

Displays or sets the domain served by the current NIS master (either classic or LDAP).

If NIS is configured to use LDAP, then any settings made with the ldap-search-config command will modify or override the use of the domainname command. In general, ldap-search-config offers a more flexible means of configuring the server's LDAP search behavior.

Examples

To display the current domain name:

domainname

To set the current domain name to "uk":

domainname uk

When using LDAP, to set the domain for LDAP RDN DC=some,DC=example,DC=com:

domainname some.example.com

Applies To

EVS security context
