iscsi-lu

Manages iSCSI Logical Units

Syntax

iscsi-lu operation
Where operation is:
list [<name>]
add [-c <comment>] [-e] <name> <file-system> <path> [<size n[K|M|G|T|P]>]
clone [-c <comment>] [-e] <source> <name> <path> 
del [-d] [-f] <name>
delall [-c]
expand <name> <new size n[K|M|G|T|P]>
mod [-n <name>] [-c <comment>] <name>
mount [-e] <name>
unmount [-f] <name>

Description

This command manages iSCSI logical units. A logical unit is an area of storage that holds data for an iSCSI target.

Usage:

iscsi-lu list [<name>]

Lists one/all logical units.

name

The name of the logical unit to list.

iscsi-lu add [-c <comment>] [-e] <name> <file-system> <path> [<size n[K|M|G|T|P]>]

Creates a new logical unit.

--comment, -c

Set Comment.

--ensure, -e

Create path to file if it doesn't exist.

name

The name of the logical unit to create.

file-system

The name of the file system on which to create the logical unit.

path

The path to the logical unit's underlying file. Logical units' files have '.iscsi' extensions, so if the path doesn't already end with '.iscsi', the extension will be added automatically.

size

The size of the logical unit.

iscsi-lu clone [-c <comment>] [-e] <src> <name> <path>

Clones an existing logical unit.

--comment, -c

Set Comment.

--ensure, -e

Create path to clone if it doesn't exist.

src

Alias of logical unit to clone

name

The name of the logical unit to create.

path

The path to the logical unit's underlying file (a .iscsi extension will be added automatically if necessary). This file will be a clone of the existing logical unit's underlying file.

iscsi-lu expand <name> <new size n[K|M|G|T|P]>

Expands a logical unit.

name

The name of the logical unit to expand.

size

The new size of the logical unit.

iscsi-lu del [-d] [-f] <name>

Deletes a logical unit.

--delete-underlying-file, -d

Delete the logical unit's underlying file. If this option is chosen, the logical unit's data will be permanently deleted. The default is to delete the logical unit, but to preserve its underlying file.

--force, -f

Force delete even if underlying file system is not available.

name

The name of the logical unit to delete.

iscsi-lu delall [-c]

Deletes all logical units. The command effectively performs a series of 'iscsi-lu del -df' commands, meaning that each logical unit's underlying file will be deleted. So, this command will permanently delete every logical unit and all underlying data files.

--confirm, -c

Don't ask for confirmation.

iscsi-lu mod [-n <name>] [-c <comment>] <name>

Modifies a logical unit.

--new-alias, -n

Update the alias.

--comment, -c

Update the comment.

name

The name of the logical unit to modify.

iscsi-lu unmount [-f] <name>

Unmounts a logical unit.

--force, -f

Force unmount even if in use.

iscsi-lu mount [-e] <name>

Mounts a logical unit.

--ensure, -e

Ensure the underlying file exists.

Examples

iscsi-lu list
iscsi-lu add -e lu1 fs1 /marketing/mylu.iscsi 512M
iscsi-lu clone -c "Clone of lu1" lu1 lu3 /marketing/mylu_clone.iscsi
iscsi-lu expand lu1 1G
iscsi-lu del -f lu1
iscsi-lu delall
    Confirm (y/n)?
iscsi-lu delall -c
iscsi-lu mod -n lu2 lu3
iscsi-lu unmount -f lu2
iscsi-lu mount -e lu2

Applies To

EVS
