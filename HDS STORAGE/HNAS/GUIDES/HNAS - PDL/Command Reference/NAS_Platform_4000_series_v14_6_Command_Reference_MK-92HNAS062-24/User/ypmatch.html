ypmatch

Prints the value of a key from an NIS map

Syntax

ypmatch [-k] key mapname

Description

Displays the value of a key in the specified yp database.

ypmatch [-k] key map

Use the -k option to display the relevant key as well as the value (useful if the key isn't part of the value).

Examples

Display the value of the key "test" in the passwd.byname map:

ypmatch test passwd.byname

Applies To

EVS
