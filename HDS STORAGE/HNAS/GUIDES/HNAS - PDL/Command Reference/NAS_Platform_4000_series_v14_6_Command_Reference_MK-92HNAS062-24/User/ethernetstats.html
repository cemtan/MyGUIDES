ethernetstats

Report/reset ethernet statistics

Syntax

ethernetstats [-r|--reset] [-n] [-p <name> | -t ]

Description

Displays the statistics for each ethernet port.

Note the statistics described below are not necessarily available on all interfaces. In particular, the various "bytes last second" statistics are only available on file-serving interfaces.

No statistics are provided for the management interfaces.

The 'ethernetstats' command takes the following options:

-r, --reset

Reset statistics

-n, --nr

Display non-resettable statistics

-p, --port <port>

Port name eg. ag1

-t, --total

Total ethernet statistics across file-serving and management ports.

The following information is displayed:

General

Mac Address

This is the interface MAC address. It will not be displayed if it is invalid.

RX Statistics

Last Reset

The time (if any) at which the interface statistics were last reset.

Bytes All

A count of all bytes received at the MAC interface.

Bytes OK

A count of all bytes received at the MAC interface that are accepted without error.

Packets OK

A count of all packets received at the MAC interface that are accepted without error.

Instantaneous bytes/sec

A count of bytes received in the last second. This figure is updated once per second.

Peak bytes/sec

The highest value of "Instantaneous bytes/sec" since the last reset.

Packets Unicast

A count of all unicast packets received at the MAC interface that are accepted without error.

Packets Broadcast

A count of all broadcast packets received at the MAC interface that are accepted without error.

Packets Multicast

A count of all multicast packets received at the MAC interface that are accepted without error.

Packets Oversized

A count of packets received that are greater than the maximum frame size permitted for the interface.

Packets Undersized

A count of packets received that are smaller than the minimum frame size permitted for the interface.

Packets Fragmented

A count of packets received that are smaller than the minimum frame size permitted for the interface and also have either a CRC error or an alignment error.

Jabbers

A count of frames received with alignment errors.

Collisions

A count of corrupted frames received caused by collisions. Should be zero on 1GbE and 10GbE interfaces.

CRC Errors

A count of frames received with an invalid FCS.

All Errors

A count of all frames received with errors.

Packets Dropped

A count of received packets discarded because of problems discovered in the MAC chip or CRC errors.

Packets No Proto

A count of packets received that are smaller than the minimum frame size permitted for the interface and are subsequently rejected by software rather than the MAC.

TX Statistics

Last Reset

The time (if any) at which the interface statistics were last reset.

Bytes All

A count of all bytes transmitted by the MAC interface.

Bytes OK

A count of bytes successfully transmitted by the interface.

Packets OK

A count of packets successfully transmitted by the interface.

Instantaneous bytes/sec

A count of bytes transmitted in the last second. This figure is updated once per second.

Peak bytes/sec

The highest value of "Instantaneous bytes/sec" since the last reset.

Packets Unicast

A count of all unicast packets transmitted by the MAC interface.

Packets Broadcast

A count of all broadcast packets transmitted by the MAC interface.

Packets Multicast

A count of all multicast packets transmitted by the MAC interface.

Packets Dropped

A count of packets that were not transmitted by the MAC interface.

Collisions Single

A count of packets successfully sent after a single collision. Should be zero on 1GbE and 10GbE interfaces.

Collisions Multiple

A count of packets successfully sent after multiple collisions. Should be zero on 1GbE and 10GbE interfaces.

Collisions Excessive

A count of packets not sent after more than 15 collisions. Should be zero on 1GbE and 10GbE interfaces.

Collisions Late

A count of packets successfully sent after a collision more than 512 bit times into the transmission of the frame. Should be zero on 1GbE and 10GbE interfaces.

Totals

Bytes Total All

Total of all received and transmitted bytes, including errors.

Bytes Total OK

Total of all bytes received and transmitted without error.

Packets Total OK

Total of all packets received and transmitted without error.

Examples

ethernetstats

Display ethernet statistics for all ports

ethernetstats -r

Reset all ethernet statistics

ethernetstats -n

Display ethernet non-resettable statistics for all ports

ethernetstats -p ge1

Display ethernet statistics for gige-1 port

ethernetstats -t

Display total ethernet statistics for all ports

Applies To

Cluster node

Privilege Level

User
