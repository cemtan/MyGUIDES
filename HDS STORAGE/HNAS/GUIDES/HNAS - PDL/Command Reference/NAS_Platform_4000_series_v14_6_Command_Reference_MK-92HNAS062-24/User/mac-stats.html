mac-stats

Display MAC chip statistics

Syntax

mac-stats 	[-p] [interface-name]

Description

The mac-stats command provides detailed statistics (where available) from the various onboard high-speed MAC interfaces.

interface-name

Specifies the interface for which statistics or MAC PAUSE frame information should be displayed. If not specified, information is displayed for all relevant interfaces.

-p

Specifies that only PAUSE frame information should be displayed.

Examples

To display MAC statistics for c1:

    mac-stats c1

To display MAC PAUSE statistics for tg1:

    mac-stats -p tg1

To display all MAC statistics for all relevant interfaces:

    mac-stats

Applies To

Cluster node

Privilege Level

User
