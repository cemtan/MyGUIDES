getmacid

Report the MAC ID

Syntax

getmacid

Description

The getmacid command prints the MAC ID of the server. This is a unique ID for the server that is used internally for various checks. Your support provider may ask for this ID.

Note that this MAC ID does not correspond to any of the MAC addresses the server uses on the network.

Examples

server:$ getmacid
MAC ID is D0-7D-AA-E4-2B-50

Applies To

Cluster node

Privilege Level

Any
