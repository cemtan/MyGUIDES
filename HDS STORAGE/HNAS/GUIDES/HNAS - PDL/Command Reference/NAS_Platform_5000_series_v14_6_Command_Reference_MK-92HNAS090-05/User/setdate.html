setdate

Display/set system date

Syntax

setdate [dd mm yyyy]

Description

Displays or sets the system date.

If the embedded SMU is enabled, the command will restart the webserver after a new date is set.

Options:

dd

Day to set

mm

Month to set

yyyy

Year to set

Examples

setdate

Displays the current system date.

setdate 1 4 2000

Sets the system date to 1st April 2000.

See Also

settime tz-set

User
