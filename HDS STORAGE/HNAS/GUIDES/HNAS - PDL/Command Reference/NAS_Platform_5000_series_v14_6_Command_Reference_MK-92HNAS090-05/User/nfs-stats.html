nfs-stats

Report or reset NFS statistics

Syntax

nfs-stats [-r|--reset] [-b|--boot] [-v|--version <2|3|4>]

Description

Displays counts of the NFS requests that have been received by a server since it was rebooted. The counts are broken down by NFS version and procedure.

Usage:

nfs-stats [-r] [-b] [-v <2|3|4>]

-r, --reset

Reset NFS statistics.

-b, --boot

Show NFS statistics since boot. These values are maintained over a reset of the statistics.

-v, --version <version>

Shows the statistics for the particular NFS version. The supported version options are 2, 3 and 4.

If this option is not supplied, statistics for all NFS versions are displayed.

Examples

To display the NFS request count since boot:

nfs-stats -b

To display the NFS version 3 request count only:

nfs-stats -v 3

Applies To

Cluster node
