smb-client-stats

Report SMB client statistics

Syntax

smb-client-stats [--reset | -r] [--boot | -b]
Alias:       cifs-client-stats

Description

The smb-client-stats command reports SMB client connection usage statistics. SMB client connections are used for connections to domain controllers, virus scanners and file filtering agents.

Usage:

[--reset | -r]

Resets the statistics.

[--boot | -b]

Without this option, the command displays the statistics since they were last reset. With this option, the statistics since boot are displayed.

Applies To

Cluster node

Privilege Level

User
