ndmp-status

Display or change NDMP status

Syntax

ndmp-status [--start | --abort] [--boot-enable | --boot-disable]

Description

Display or change NDMP status.

If issued with no parameters it displays the current NDMP status.

The following options can be used to control the current running state:

--start

If NDMP is not running, request that it starts.

--abort

Request that NDMP processing is terminated immediately. This will disrupt any ongoing replication, backup, recovery, or adc copy so should only be used when the system is idle.

The following options can be used to control the NDMP action at boot time:

--boot-enable

If set, NDMP will start as usual at boot time.

--boot-disable

NDMP will not be started at boot time.

Examples

ndmp-status

Displays the current status of NDMP.
