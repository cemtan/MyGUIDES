fc-link-errors

Show FC link error stats

Syntax

fc-link-errors [-a] [-r]

Description

This command is used to display or reset statistics on low-level Fibre Channel link errors.

Options

-a, --all

display statistics for all interfaces, not just enabled ones

-r, --reset

reset statistics

Error details

Loss of Signal

As reported by the internal transceiver hardware or a pluggable optics module.

Bad Received Character

FC frame transmission code errors. May indicate a poor link.

Loss of Synchronisation

The FC controller detects an incoming signal but is unable to synchronise with it.

Link Failure

Miscellaneous link failures not covered by other counts.

Bad CRC

FC frames received with an invalid CRC.

Protocol Error

FC Primitive Sequence protocol errors.

Examples

server:$ fc-link-errors
