nameservice-host

Formerly used to provide board specific host name lookup

Syntax

nameservice-host host

Description

The nameservice-host command is an alternative name for the host command and is provided for backwards compatibility with existing scripts.

Examples

To resolve www.example.com into an IP address:

nameservice-host www.example.com

To resolve an IP address into a name:

nameservice-host 198.175.96.33

Applies To

EVS security context

See Also
