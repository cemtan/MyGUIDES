ntp

NTP configuration

Syntax

ntp operation
where 'operation' is one of:
	list
	add server1 [server2] ... [server5]
	del server1 [server2] ... [server5]

Description

The ntp command displays and manipulates configuration information for NTP, which keeps the system clock synchronized to Internet standard time servers.

Operations:

list

Lists the currently configured NTP servers.

add

Adds one or more NTP servers for synchronization. A maximum of five servers can be configured.

del

Stops using one or more NTP servers for time synchronization.

Examples

To display the currently configured NTP servers:

ntp list

To add an NTP server:

ntp add 10.1.1.14

To delete an NTP server:

ntp del 10.1.1.14

Applies To

Cluster wide

Privilege Level

User
