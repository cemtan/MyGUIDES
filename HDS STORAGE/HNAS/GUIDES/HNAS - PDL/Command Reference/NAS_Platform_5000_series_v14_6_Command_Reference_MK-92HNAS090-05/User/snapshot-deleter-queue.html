snapshot-deleter-queue

Show work on the deleter queue

Syntax

snapshot-deleter-queue [<file system>] [--verbose]

Description

This command displays the snapshot deleter's current work queue, including snapshots undergoing or queued for deletion.

<file system>

Show snapshots on the specified file system only.

--verbose

Shows details of each snapshot including deletion progress, if available. Snapshot deletion progress is only available for block-based snapshots.

Examples

server:$ snapshot-deleter-queue fs1
 1 registered volume(s), namely: 11
 1 modifiable volume(s), namely: 11


Number of views on modifiable volumes that are awaiting or undergoing deletion is 1


"snap1" on fs1


Dumped 1 of 1


No bitmap-scrubbable volumes
 No root-onode-scrubbable volumes
 No restorable volumes

Applies To

EVS
