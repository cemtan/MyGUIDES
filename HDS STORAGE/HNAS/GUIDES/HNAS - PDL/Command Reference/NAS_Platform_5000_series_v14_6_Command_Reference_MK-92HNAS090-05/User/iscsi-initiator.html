iscsi-initiator

Manage iSCSI initiators

Syntax

iscsi-initiator operation
Where operation is:
list
add <name> <secret>
del <name>
delall [-c]
set <name> <new-secret>

Description

If configured for mutual authentication, an iSCSI target must authenticate with each initiator that connects to it. This command provides the iSCSI service with authentication information for each iSCSI initiator.

iscsi-initiator list
    Displays configured initiators.


iscsi-initiator add <name> <secret>
    Adds a new initiator.
    name
        The name of the initiator to add.
    secret
        The initiator's secret.


iscsi-initiator del <name>
    Deletes an initiator.
    name
        The name of the initiator to delete.


iscsi-initiator delall [OPTIONS]
    Deletes all configured initiators.


    OPTIONS
    --confirm, -c
        Don't ask for confirmation.


iscsi-initiator set <name> <new-secret>
    Update an initiator's secret.
    name
        The name of the initiator to update.
    new-secret
        The new secret.

Examples

iscsi-initiator list
iscsi-initiator add iqn.2004-03.com.example:initiator1 secretsecret1
iscsi-initiator del iqn.2004-03.com.example:initiator1
iscsi-initiator delall -c
iscsi-initiator set iqn.2004-03.com.example:initiator1 secretsecret2
