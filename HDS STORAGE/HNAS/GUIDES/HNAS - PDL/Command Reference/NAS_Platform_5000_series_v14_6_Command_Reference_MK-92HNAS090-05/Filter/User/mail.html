mail

Mail command output

Syntax

... | mail [-s subject] recipient ...

Description

Command output can be piped through mail to send it to the specified email address. mail may not be used as a command.

Options:

-s subject

The subject of the email.

recipient

One or more recipients email addresses.

Examples

Send the current event log:

event-log-show | mail -s "Latest event log" support@example.com

It is possible to send the output of multiple commands by combining mail with ssrun:

ssrun -c "ver ; ssls" | mail -s "Version and ssfs listing" support@example.com

Alternatively commands can be combined in a script and then run with ssrun:

server:$ ssmk test.scr
Enter data, ending with "." on a line by itself
echo first command
echo second command
.
server:$ ssrun test.scr | mail -s "script output" support@example.com
server:$ ssrm test.scr
Deleted test.scr
server:$
