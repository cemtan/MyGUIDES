hex2bin

Undoes hexdump

Syntax

... | hex2bin

Description

The hex2bin filter converts hexdumps, like those from the eponymous command, back to the original data.

Examples

The output can be piped into the hexdump filter with switches to simplify the hexdump for scripted transformation. Contrast:

server:$ hexdump .bss 512 | grep vtable"|"same
Using ANY symbol '.bss', size 0 B at address 0x9c00000
                     vtable for FeatureSwitch+0x10
                                             vtable for FeatureTelcSynchronizer+0x10
                                             vtable for SimplerPersistentConfig<unsigned int>+0x10
                     vtable for SimplerPersistentConfig<bool>+0x10
                     vtable for SimplerPersistentConfig<std::string>+0x10
..... same (for 2 more lines, 32 more bytes) .....
                     vtable for LogTrace+0x18
                                             vtable for LogTrace+0x68
..... same (for 2 more lines, 32 more bytes) .....
                     vtable for LogTrace+0xd8
server:$

... with:

server:$ hexdump .bss 512 | hex2bin | hexdump -v --no-symbols | grep vtable"|"same
Using ANY symbol '.bss', size 0 B at address 0x9c00000
server:$
