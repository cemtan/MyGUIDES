chassis-drive-status

Display the chassis drive status

Syntax

chassis-drive-status

Description

The system hardware includes hard disk drives on which the server software and firmware images are loaded. These drives are configured into RAID 1 volumes for resiliency.

This command displays a summary of the current status and percentage usage of each volume. Where the volume does not have an associated filesystem mounted such as a swap volume, the usage figures are donated by - entries.

Note: This command does not display information about storage connected to the Fiber channel ports of the system.

Status values:

Unknown

No status information is available

Good

Volume is fault tolerant and all members are synchronized.

Degraded

One member of the volume has failed.

Invalid

The volume has only a single member.

Rebuilding

The volume is being rebuilt. The volume is not fault tolerant until the rebuild completes.

Examples

$ chassis-drive-status
chassis-drive-status
Device  Status        % Used  Size (4k blks) Used (4k blks) % Rebuild
------  ------------  ------  -------------- -------------- ------------
0       Good          29      3846436        1124784        Synchronized
1       Good          62      12302144       7681668        Synchronized
2       Good          -       -              -              Synchronized

Applies To

Cluster node
