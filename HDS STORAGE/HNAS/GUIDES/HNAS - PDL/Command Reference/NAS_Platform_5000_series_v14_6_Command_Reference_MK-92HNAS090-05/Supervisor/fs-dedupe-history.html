fs-dedupe-history

Show the history of dedupe runs for a file system

Syntax

fs-dedupe-history [-a|--all] <file-system>

Description

This command shows the history of dedupe runs for a mounted file system that supports dedupe. Note that the file system keeps records of only the five most recent runs in the registry.

If the option -a or --all is specified, the command shows the current run and all recent runs in the registry, starting with the most recent run first. Otherwise, it shows only the current run and the most recent run.

Examples

1. Show current run and last run of dedupe for file system 'fs1':

    server:$ fs-dedupe-history fs1
    Current run:
      Dedupe job type         : incremental
      Start time              : 2012-06-01 16:07:56-07:00
      Current time            : 2012-06-01 16:07:56-07:00
      Current job state       : Running
      Data processed          : 2.42 MB
      Data deduped            : 1.17 MB
      Throughput              : 220 MB/s


    Last run:
      Dedupe job type         : incremental
      Start time              : 2012-05-23 18:02:20-07:00
      End time                : 2012-05-23 18:02:20-07:00
      End job state           : Finished
      Data processed          : 50.7 MB
      Data deduped            : 34.8 MB
      Throughput              : 186 MB/s
