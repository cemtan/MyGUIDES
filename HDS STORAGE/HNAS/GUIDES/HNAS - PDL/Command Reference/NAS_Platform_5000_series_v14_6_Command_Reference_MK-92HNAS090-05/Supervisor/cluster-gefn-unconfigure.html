cluster-gefn-unconfigure

Unconfigure GEfN cluster

Syntax

cluster-gefn-unconfigure

Description

Converts a 4 node HNAS 5000 series or above cluster that has previously been converted to a GAD Enhanced for NAS (GEfN) cluster back to a standard 4 node cluster.

If this is done in a geographically separated (stretched) cluster, the cluster will no longer be site resilient in the case of a site-wide outage.

Applies To

Cluster wide.

Examples

1. Convert a GEfN cluster back to a standard 4 node cluster:

server-1:$ cluster-gefn-unconfigure

See Also
