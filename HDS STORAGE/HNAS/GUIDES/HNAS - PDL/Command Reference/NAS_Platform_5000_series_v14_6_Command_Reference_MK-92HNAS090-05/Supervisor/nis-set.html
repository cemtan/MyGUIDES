nis-set

Configure the NIS client settings

Syntax

nis-set  [-t timeout] [-r wait]

Description

Displays and manipulates some NIS client parameter settings.

-t sets the bind timeout. This is the length of time, in milliseconds, for which the NIS client will wait for a response from a server after sending a bind request. If no response is received within this time, the target is assumed not to have been an NIS server. The acceptable range is 100 to 10000 milliseconds.

-r sets the bind retry period. This is the time, in minutes, between attempts to rebind to the current NIS server. The acceptable range is 1 to 15 minutes.

Examples

To display the current NIS client settings:

nis-set

To change the bind timeout to 200ms:

nis-set -t 200

To change the rebind period to 2 minutes and the bind timeout to 199ms:

nis-set -r 2 -t 199

EVS
