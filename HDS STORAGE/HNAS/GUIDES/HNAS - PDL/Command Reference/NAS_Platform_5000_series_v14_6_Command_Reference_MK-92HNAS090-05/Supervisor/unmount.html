unmount

Unmounts the specified file system

Syntax

unmount [[--deworm] <file-system> | --all]
Dev-level options: [--ok-if-unmounted]

Description

The unmount command will make the specified file system unavailable to the configured file serving protocols.

An unmounted file system will not be mounted automatically the next time the EVS it is assigned to comes online.

Options:

--deworm

Turn a WORM file system into a non-WORM file system, if it's empty.

--all, -a

Unmount all the mounted file systems assigned to the EVS in context. Cannot be used with the --deworm option.

Privilege Level

Supervisor
