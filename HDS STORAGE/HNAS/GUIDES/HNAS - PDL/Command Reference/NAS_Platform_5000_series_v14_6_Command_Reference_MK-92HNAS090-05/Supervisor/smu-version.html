smu-version

Report the embedded SMU's version

Syntax

smu-version

Description

Display the version of the embedded SMU.

If the SMU is not installed the output will state that fact.

Examples

smu-version

Applies To

Cluster wide

See Also

smu-get-diagnostic smu-service-disable smu-service-enable smu-service-restart smu-service-status smu-uninstall
