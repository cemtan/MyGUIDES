sstar

Create archive from list of files

Syntax

sstar <tarfile> <file> [<file> ...]

Description

Creates an archive file from the specified list of files.

The command takes two or more parameters - the name of the archive file followed by the list of files to be archived.

Examples

sstar test.tar test1

Creates an archive file, test.tar, which includes only the test1 file.

sstar test.tar test* extra1 extra2

Creates an archive file, test.tar, which includes all files beginning with 'test', plus file extra1 and extra2.
