irdp

Configure/Display ICMP Router Discovery

Syntax

irdp [show] | discover | update | flush | stop

Description

The 'irdp' command controls the operation of the server's implementation of the ICMP Router Discovery Protocol (IRDP). Without any options, the 'irdp' command displays the current status of IRDP.

In discovery mode routers are noted but not passed on to the routing module. In update mode routers are made known to the routing module. When IRDP is stopped, all routers that have been made known to the routing module will be removed.

Operations

show

Display the IRDP information.

Note: Dynamic routers are only displayed for the cluster node on which the command is executed.

discover

Configure IRDP to operate in the discovery mode.

update

Configure IRDP to operate in the update mode.

stop

Stop IRDP operation and remove IRDP-learned routers.

flush

Remove any IRDP-learned routers but leave IRDP state otherwise unchanged.

Note: This operation only applies to the cluster node on which the command is executed.

Examples

To enable updates for IRDP:

$ irdp update
$

Applies To

Cluster wide
