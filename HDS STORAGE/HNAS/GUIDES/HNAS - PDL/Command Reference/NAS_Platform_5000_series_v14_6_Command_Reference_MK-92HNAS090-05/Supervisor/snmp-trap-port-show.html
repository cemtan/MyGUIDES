snmp-trap-port-show

Report SNMP trap sender port

Syntax

snmp-trap-port-show

Description

The snmp-trap-port-show command reports the port to which the SNMP agent send traps.

Examples

Show the SNMP trap sender port:

snmp-trap-port-show

Applies To

Cluster wide
