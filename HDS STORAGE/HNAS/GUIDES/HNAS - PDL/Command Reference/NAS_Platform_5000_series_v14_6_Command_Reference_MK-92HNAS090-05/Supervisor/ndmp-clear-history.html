ndmp-clear-history

Clear NDMP history information

Syntax

ndmp-clear-history

Description

The command does not take any parameters.

The command clears the history of NDMP backups and adc copies for the current EVS. The first backup or copy of a directory following this command will be a full backup/copy rather than an incremental copy.

This may be useful in cases where a previous NDMP backup or adc copy has been lost. For instance if the tape containing a previous backup has been accidentally overwritten. (The command does affect all backups though.)

Note: This command does not affect the history of incremental replications created through the SMU interface.

Examples

ndmp-clear-history

Applies To

EVS

Privilege Level

Supervisor
