sd-back-up-cod

Read a single system drive's span and filesystem config into a file

Syntax

sd-back-up-cod [-F | --ssfs] [-d | --denied] <system-drives-see-'man sd-spec'> [<filename>]
Alias:       sd-backup-cod

Description

Takes single-SD Cod backups (SDCs) from one or more system drives (SDs). Cod and Cod backups are described in the 'cod' man page.

If you don't specify a filename and you're using the ssc filesystem, the server will construct a default name based on the SD device ID, a timestamp, and the filename extension '.sdc'. If you're using the SSFS filesystem (even if you're running the ssc client), specifying a filename is mandatory.

If you're using ssc, the new file is created on the ssc file system (i.e. on the computer running the ssc client); you can choose to create the file in the server's SSFS file system instead by providing the --ssfs or -F switch. In all other environments the file is always created in the SSFS file system.

If the file is created on SSFS, it is, by default, created on the cluster node where the command is run. To create it on another server's SSFS file system, use the syntax detailed in the 'ssfs' man page.

If you're using ssc, you can specify multiple SD device IDs. See the 'sd-spec' man page for more details. You must rely on the default filename if you do this. If you need to specify your own filenames, back up one SD's Cod at a time by running multiple 'sd-back-up-cod' commands.

Specifying the --denied switch lets you back up the Cod from an SD to which access has been denied. In combination with 'sd-restore-cod' and the Cod converter, this lets you modify the Cod on an SD before allowing access to the SD and loading the Cod into memory.

Examples

server:$ sd-back-up-cod 0
Creating sd0000--2007-08-07--17-45-55.sdc
Wrote 58.0 KB (59435 B) in 124 ms at 468 KB/s (479314 B/s)
server:$ sd-back-up-cod 0 Zero.sdc
Wrote 58.0 KB (59435 B) in 145 ms at 400 KB/s (409896 B/s)
server:$ sd-back-up-cod --ssfs 0 Zero.sdc
server:$ sd-back-up-cod 0-3
Creating sd0000--2007-08-07--17-46-17.sdc
Wrote 58.0 KB (59435 B) in 142 ms at 409 KB/s (418556 B/s)
Creating sd0001--2007-08-07--17-46-17.sdc
Wrote 58.0 KB (59435 B) in 110 ms at 528 KB/s (540318 B/s)
Creating sd0002--2007-08-07--17-46-17.sdc
Wrote 58.0 KB (59435 B) in 105 ms at 553 KB/s (566047 B/s)
Creating sd0003--2007-08-07--17-46-17.sdc
Wrote 58.0 KB (59435 B) in 140 ms at 415 KB/s (424535 B/s)
server:$
