fc-link

Enable/disable FC link

Syntax

fc-link [<interface> [enable|disable]]
Alias:       fclink

Description

The fc-link command is used to enable or disable a Fibre Channel interface on the server. The settings apply to all nodes in a cluster. With no arguments it will display the current state of all interfaces.

Examples

Disable Fibre Channel interface 1:

fc-link 1 disable

Applies To

Cluster wide

See Also

fc-link-reset fc-link-speed fc-link-status fc-link-type

Privilege Level

Supervisor
