banner-set-enabled

Enable or disable the login banner

Syntax

banner-set-enabled [<on-off>]

Description

Enable or disable the login banner.

Options:

on

Enables the login banner.

off

Disables the login banner.

Examples

banner-set-enabled on


banner-set-enabled off

Applies To

Cluster wide

See Also

banner-get banner-get-enabled banner-set

Privilege Level

Supervisor
