file-streams

Displays summary info about a file's data streams

Syntax

file-streams <file name or handle>

Description

The file-streams command is used to display information about a file's data streams.

When information is written to a file, it is stored in the file's unnamed data stream. Normally, this will be the only data stream that the file has.

However, CIFS provides the ability to create and access named data streams, which are additional data streams belonging to the file. Named data streams are usually used to store information related to the file, such as version or summary information.

Applies To

EVS

Privilege Level

Supervisor
