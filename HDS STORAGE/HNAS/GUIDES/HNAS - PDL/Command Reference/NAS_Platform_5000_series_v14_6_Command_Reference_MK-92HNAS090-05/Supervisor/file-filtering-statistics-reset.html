file-filtering-statistics-reset

Reset the file filtering statistics that are displayed by the file-filtering-statistics-show command

Syntax

file-filtering-statistics-reset

Description

This command resets the statistics shown by the file-filtering-statistics-show command. Use "file-filtering-statistics-show" to show the current statistics.

Examples

To reset statistics:

file-filtering-statistics-reset

Applies To

EVS

See Also

file-filtering-statistics-show

Privilege Level

Supervisor
