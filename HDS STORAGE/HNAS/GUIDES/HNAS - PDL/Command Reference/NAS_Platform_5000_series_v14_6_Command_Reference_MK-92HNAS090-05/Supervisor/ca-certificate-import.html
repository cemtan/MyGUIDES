ca-certificate-import

Add a custom certificate to the trusted certificate store for use in authenticating other servers to HNAS, when HNAS is acting as a client, for example, in Data Migrator to Cloud feature.

Syntax

ca-certificate-import --path <path> [--prompt]

Description

The ca-certificate-import command adds a custom X509 certificate, contained in a PEM formatted file, into the system trusted certificate store.

The command requires the user to be connected to the system using ssc to access a host-based file containing the certificate.

Options:

--path

The path to the file containing the PEM formatted certificate to import.

--prompt

Output information about the certificate including the MD5 and SHA1 fingerprints to verify the certificate before confirming the import operation.

Examples

To import a certificate:

$ ca-certificate-import --path /certificates/hcp8-uk-01.example.com.crt

Applies To

Cluster wide
