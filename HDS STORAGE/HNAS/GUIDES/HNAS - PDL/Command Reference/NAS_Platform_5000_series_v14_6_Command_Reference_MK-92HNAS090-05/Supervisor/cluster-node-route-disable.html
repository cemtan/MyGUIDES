cluster-node-route-disable

Disable cluster-node-route configuration

Syntax

cluster-node-route-disable

Description

The cluster-node-route-disable command is used to terminate cluster-node-route functionality and return to standard route mode.

Examples

To revert to standard route functionality:

$ cluster-node-route-disable
$

Applies To

Cluster node

See Also

cluster-node-route-enable cluster-node-route-list

Privilege Level

Supervisor
