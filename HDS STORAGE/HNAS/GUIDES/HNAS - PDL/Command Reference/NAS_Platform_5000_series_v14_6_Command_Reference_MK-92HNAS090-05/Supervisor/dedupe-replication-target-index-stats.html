dedupe-replication-target-index-stats

Show or reset stats for the access of the dedupe index on replication target file systems.

Syntax

dedupe-replication-target-index-stats [--reset | --verbose]

Description

The dedupe-replication-target-index-stats command shows or resets stats for the access of the dedupe index on replication target file systems.

This is a diagnostic command, intended for use by your service organization.

Usage:

  dedupe-replication-target-index-stats [-r] [-v]

Options:

-r, --reset

Reset the statistics.

-v, --verbose

Show detailed statistics.

Examples

  dedupe-replication-target-index-stats

Shows the simple statistics.

  dedupe-replication-target-index-stats -r

Resets the statistics.

  dedupe-replication-target-index-stats -v

Shows the detailed statistics

Applies To

Cluster node.

Privilege Level

Supervisor
