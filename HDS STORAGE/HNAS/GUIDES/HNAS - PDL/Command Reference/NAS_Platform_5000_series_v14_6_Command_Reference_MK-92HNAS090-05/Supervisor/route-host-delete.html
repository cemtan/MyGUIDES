route-host-delete

Delete a host route

Syntax

route-host-delete --gateway <gateway> <host>

Description

The route-host-delete command deletes a host route from the routing table.

Options

The following option is recognised:

-g, --gateway <gateway-address>

This is the gateway for the route. It must be supplied.

<host>

This is the remote host to which this route applies. It must be supplied.

Examples

$ route-host-delete 10.1.2.3 -g 10.1.3.4
Route cache flushed.
$

Deletes a host route to 10.1.2.3 via 10.1.3.4.

Applies To

Cluster wide
