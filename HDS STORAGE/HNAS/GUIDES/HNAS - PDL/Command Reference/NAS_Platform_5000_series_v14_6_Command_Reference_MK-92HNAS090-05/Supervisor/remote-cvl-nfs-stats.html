remote-cvl-nfs-stats

Display statistics for remote cross volume links over NFS

Syntax

remote-cvl-nfs-stats [--reset]

Description

The remote-cvl-nfs-stats command displays statistics about remote cross volume link activity using the NFS protocol.

Options:

-r, --reset

Reset statistics.

Examples

Reset all remote cross volume link over NFS statistics:

remote-cvl-nfs-stats -r

Display all remote cross volume link over NFS statistics:

remote-cvl-nfs-stats

Applies To

Cluster node
