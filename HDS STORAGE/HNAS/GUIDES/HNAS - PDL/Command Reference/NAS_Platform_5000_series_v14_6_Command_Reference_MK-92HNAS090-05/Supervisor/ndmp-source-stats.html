ndmp-source-stats

Show NDMP source stats for this node

Syntax

ndmp-source-stats [-r|--reset]

Description

This command displays the accumulated statistics of file system accesses made by NDMP operations at the source end of transfers. This includes NDMP backups, adc copies, replications and data migrations. It shows only the data related to reading from file systems.

By monitoring the increases in these figures the load on the file system from NDMP processes can be assessed.

These statistics can be reset using the -r parameter.

The statistics for writing to file systems can be displayed using the ndmp-target-stats command. Session level stats can be displayed using ndmp-session-stats, but that will only display stats for sessions that are currently active.

Examples

Applies To

Cluster node
