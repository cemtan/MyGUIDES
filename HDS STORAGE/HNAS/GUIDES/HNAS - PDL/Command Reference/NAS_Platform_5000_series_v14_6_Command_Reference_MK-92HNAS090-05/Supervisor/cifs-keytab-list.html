cifs-keytab-list

Lists keytabs for ADS CIFS names

Syntax

cifs-keytab-list [--all | -a] [ <name1> [ <name2> [ <name3> ... ] ] ]

Description

The cifs-keytab-list command lists the keytab used by ADS CIFS names for Kerberos authentication.

Up to eight ADS CIFS names on the current evs can be specified

cifs-keytab-list <name1> <name2> <name3> ... <name8>

Otherwise, --all lists the keytabs for all ADS CIFS names on the current evs

cifs-keytab-list --all

Examples

To list the keytabs for all ADS CIFS names configured for the current evs

cifs-keytab-list --all

To list the keytab for a particular ADS CIFS name configured for the current evs

cifs-keytab-list <name>

Applies To

EVS
