ssput

Put file

Syntax

ssput <local file> <server file>

Description

The ssput command transfers data from a client running ssc to the server flash. This command is only available on ssc connections.

The command takes two parameters - the client filename and the server flash filename.

Examples

$ ssc 192.168.0.1 ssput 4.0.824q.pkg //simm/4.0.824q.pkg
Wrote 181.4 MB (190172872 B) in 54.13 s at 3.351 MB/s (3513326 B/s)
51.58 s spent in ssfs-write-throttle.
Transferred local file 4.0.824q.pkg to server file //simm/4.0.824q.pkg
[ssput took 54 s.]
$

Applies To

Cluster node
