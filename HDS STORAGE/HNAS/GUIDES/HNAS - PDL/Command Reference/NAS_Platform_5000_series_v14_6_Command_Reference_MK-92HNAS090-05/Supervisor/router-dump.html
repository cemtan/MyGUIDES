router-dump

dumps the whole of the router

Syntax

router-dump

Description

The router-dump command displays internal routing tables including both the global and pnode routing information. Along with the test-route command, it can be used to help diagnose networking problems. As well as providing similar output to the route command, it also prints interface routes. Interface routes define subnets that are directly connected via one of the interfaces.

This is a per-board command and displays different output on each of the boards.
