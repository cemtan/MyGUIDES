krb5-kdc-ipaddress

Get/Set gss/krb5 KDC IP Address

Syntax

krb5-kdc-ipaddress [ip-address] [-r]

Description

The krb5-kdc-ipaddress command configures which Key Distribution Center (KDC) the Kerberos V5 GSS-API security mechanism uses.

Usage of this command should not generally be required in a properly configured environment because KDCs are identified dynamically using DNS and the knowledge HNAS has about the Kerberos realm that is in scope. This command can be used to override such dynamic selection and manually specify a KDC instead. The reset option of this command can be used to clear the manually configured KDC address and revert to the dynamic selection mechanism.

Usage:

krb5-kdc-ipaddress [ip-address] [-r]

ip-address

The IP Address of the Key Distribution Center.

-r, --reset

Clears the address of the Key Distribution Center

Examples

To set the address of the Key Distribution Center:

  krb5-kdc-ipaddress 172.18.1.1
  krb5-kdc-ipaddress fdca:f995:220a:480:2::1

To clear the address of the Key Distribution Center:

  krb5-kdc-ipaddress -r

To display the address of the Key Distribution Center:

  krb5-kdc-ipaddress

Applies To

EVS security context
