cluster-node-route-enable

Enable cluster-node-route configuration

Syntax

cluster-node-route-enable

Description

The cluster-node-route-enable command is used to activate cluster-node-route functionality.

Note: It is advised that RIP and IRDP be disabled when cluster-node-route functionality is in effect.

Examples

To switch to using cluster-node-route functionality:

$ cluster-node-route-enable
$

Applies To

Cluster wide

See Also

cluster-node-route-disable cluster-node-route-list cluster-node-route-status irdp rip

Privilege Level

Supervisor
