vlan-interface-delete

Delete a VLAN interface

Syntax

vlan-interface-delete [--interface <interface>] <if>

Description

The vlan-interface-delete command deletes a previously created VLAN interface.

Options

The following options are recognised:

if

If --interface is not specified, this must be the full vlan interface name. If --interface specifies an aggregation interface, then this must specify just the vlan number.

-i, --interface <interface>

This switch specifies the physical aggregation interface on which traffic will pass.

Examples

$vlan-interface-delete ag2-vlan0100
$

Deletes the vlan interface for VLAN 100 on ag2.

$ vlan-interface-delete --interface ag2 100
$

Also deletes the vlan interface for VLAN 100 on ag2.

Applies To

Cluster wide
