vlan-interface-create

Create a VLAN interface

Syntax

vlan-interface-create --interface <interface> <vlan>

Description

The vlan-interface-create command creates an interface that will send and receive traffic marked for the specified VLAN.

Options

The following options are recognised:

-i, --interface <interface>

This switch specifies the physical aggregation interface on which traffic will pass.

vlan

This specifies the VLAN tag that will be apply to the vlan interface.

Examples

$ vlan-interface-create --interface ag2 100
Created ag2-vlan0100
$

Creates a vlan interface for VLAN 100 on ag2.

Applies To

Cluster wide
