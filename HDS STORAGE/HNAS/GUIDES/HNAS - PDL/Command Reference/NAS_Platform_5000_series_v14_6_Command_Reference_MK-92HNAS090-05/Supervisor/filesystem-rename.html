filesystem-rename

Change a filesystem's label

Syntax

filesystem-rename <old-instance-name> <new-base-name>
Alias:       mvfs

Description

Changes the base name of a filesystem. If the filesystem resides on a storage-based snapshot, the system automatically produces a new instance name.

The filesystem need not be unmounted.

You can specify the filesystem by its old instance name, its device ID, or its zero-based volume number.

Examples

filesystem-rename Wages Salaries

Applies To

Cluster wide

See Also

Privilege Level
