network-protocol-performance-monitor-dump

Dump raw network protocol stats

Syntax

network-protocol-performance-monitor-dump [<filesystem name>]...

Description

This command exposes some performance statistics which are gathered on a per-filesystem basis.

The values are totalled up over a ten second period. This command outputs the values for the previous ten second period, not the set of values currently being accumulated.

Note that the output is intended for gathering data via a script, and hence is not all that friendly for humans to read.

Parameters are a list of filesystem names to dump statistics for, or a numeric value giving a VLSI statistics buffer number from 0 to 127.

If no parameter is given, all mounted filesystems are shown.

Examples

Show statistics for filesystem ExampleFS1:

    network-protocol-performance-monitor-dump ExampleFS1

Show statistics for all mounted filesystems:

    network-protocol-performance-monitor-dump

Applies To

Cluster node

Privilege Level

Supervisor
