i2c-dump

Dump I2C devices

Syntax

i2c-dump [--verbose] [--dimms] [--known] [<min> [<max>]]

Description

Dumps information, onto the console, on devices which are accessible via the I2C. These include DIMMs (CPU and VLSI), serial EEPROMs, temperature sensors, PSUs, batteries, SFPs, etc.

Options:

--verbose, v

verbose dump

--dimms, d

dump dimms only

--known, k

dump all known devices

Optional positional parameters:

min

device number to start at, in hexadecimal

max

device number to end at, in hexadecimal
