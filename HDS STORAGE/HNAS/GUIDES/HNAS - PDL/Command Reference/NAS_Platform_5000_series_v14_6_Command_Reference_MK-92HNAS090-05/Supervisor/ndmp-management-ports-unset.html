ndmp-management-ports-unset

Unset NDMP management relay ports

Syntax

ndmp-management-ports-unset

Description

Remove the NDMP management port settings so that the SMU resumes use of standard connections to the Service EVSs when controlling Replication and Data Migration.

See ndmp-management-ports-set for a full description of the use of these settings.

Examples

Applies To

Cluster wide

See Also

ndmp-management-ports-display ndmp-management-ports-set

Privilege Level

Supervisor
