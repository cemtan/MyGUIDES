last-few-packets

Display details on the last few packets received

Syntax

last-few-packets [-r]

Description

The last-few-packets command displays information about the last few network protocol requests that have been received by the server.

Information includes the time at which each request was received, the IP address of the client that issued each request, the evs the request was associated with, the network protocol, command type and (optionally) the packet data. Network protocols that are currently reported include CIFS, NFS, iSCSI and object replication.

Usage:

last-few-packets [OPTION]

-r, --reset

Reset/clear the command cache.

-a, --all

Displays a maximum of 1024 requests per fiber, rather than a maximum of 1024 requests in total.

Examples

m1-merc9:$ last-few-packets -a
LastFewPackets: (Oldest first)
No.  Time        Source                                 EVS  Protocol Operation
0    T-16.61 h   192.168.10.230:1025                    2    SMB      115 (SMBsesssetupX)
1    T-16.61 h   192.168.10.230:1027                    1    SMB      114 (SMBnegprot)
2    T-16.61 h   192.168.10.230:1027                    1    SMB      115 (SMBsesssetupX)
3    T-16.61 h   192.168.10.230:1028                    3    SMB      114 (SMBnegprot)
4    T-16.61 h   192.168.10.230:1028                    3    SMB      115 (SMBsesssetupX)
