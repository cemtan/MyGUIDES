lockd-state

Displays information on NLM locks for the specified file

Syntax

lockd-state [-u] <path> [<client IP address>]

Description

The lockd-state command is used to view or remove network lock manager (NLM) locks and other state.

Usage:

lockd-state [OPTION] <path> [<client IP address>]

<path>

The file whose locks and other state are to be displayed (or removed if the -u option is present).

<client IP address>

Only display (or remove if the -u option is present) locks and other state belonging to given client. This must be supplied if the -u option is present.

--unlock, -u

Remove all NLM locks and other state held by the client specified by <client IP address> for the file identified by <path>.

Applies To

EVS

Privilege Level

Supervisor
