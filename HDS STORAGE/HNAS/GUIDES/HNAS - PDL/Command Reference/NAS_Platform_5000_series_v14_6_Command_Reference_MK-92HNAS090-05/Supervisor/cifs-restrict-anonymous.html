cifs-restrict-anonymous

Enables or disables support for restricting anonymous CIFS users.

Syntax

cifs-restrict-anonymous [--enable|--disable]

Description

The cifs-restrict-anonymous command allows you to configure whether to restrict anonymous user access over CIFS.

Changes made by this command take effect immediately but do not affect existing CIFS sessions.

Usage:

cifs-restrict-anonymous enable

Anonymous users are restricted and only authenticated users are allowed access over CIFS.

cifs-restrict-anonymous disable

Anonymous users are not restricted and are allowed access over CIFS.

cifs-restrict-anonymous

Shows the setting for the current EVS.

The default setting is disabled.

Applies To

EVS security context

See Also
