give-domain-administrators-local-administrator-rights

Add all domain administrators to the local administrators group

Syntax

give-domain-administrators-local-administrator-rights [yes|no]

Description

The give-domain-administrators-local-administrator-rights command controls automatically granted rights for CIFS domain administrators.

With no arguments, the command displays the current setting. The default is "yes". The setting is persistent and takes immediate effect across the current EVS security context (cluster-wide or per-EVS).

Every time a CIFS user is authenticated, the domain administrator for the user's domain is added to the local administrators group if this setting is enabled.

Examples

$ give-domain-administrators-local-administrator-rights
the default, shared security context (via EVS 2): Giving domain administrators local administrator rights.
$


$ give-domain-administrators-local-administrator-rights no
the default, shared security context (via EVS 2): Not giving domain administrators local administrator rights.
$


$ give-domain-administrators-local-administrator-rights yes
the default, shared security context (via EVS 2): Giving domain administrators local administrator rights.
$


$ give-domain-administrators-local-administrator-rights
the default, shared security context (via EVS 2): Giving domain administrators local administrator rights.
$

Applies To

EVS security context
