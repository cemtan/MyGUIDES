global-symlink-list

List the configured global symlinks

Syntax

global-symlink-list [-a]

Description

See the global-symlink-add man page for a general description of global symlinks.

The global-symlink-list command is used to display the global symlink translation table.

Usage:

global-symlink-list [OPTION]...

-a, --all-evss

Display the global symlink translation table for all configured EVSs. The current EVS's table will be displayed first, with all other EVSs appearing in order afterwards. If the -a option is omitted, the command will target the current EVS only. This option cannot be used if multi-tenancy is enabled.

Applies To

EVS
