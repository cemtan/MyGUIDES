file-clone-create

Clone a file's unnamed stream.

Syntax

file-clone-create [-f <file-system>] <source> <dest>

Description

Creates a new file and makes its data stream a clone of the source file's data stream.

-f, --file-system

The file system to work on. The currently selected file system is used if this option is not specified.

<source>

The path of the source file.

<dest>

The path to the new destination file.

Examples

To clone "./production-database.dat":

$ file-clone-create ./production-database.dat /tmp/test-database.dat

Applies To

File system
