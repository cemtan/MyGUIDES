nis-cache-list

Lists the NIS cache contents

Syntax

nis-cache-list [--verbose] [--key <name>] [--map <name>]
Alias:       ldap-cache-list

Description

Displays the NIS cache contents and its current size.

For NIS/YP cache the current size represents the total number of cache keys, whereas for NIS/LDAP it also includes the total number of cache records.

Options are:

--verbose

By default, the output is limited to 1000 entries. Use --verbose option to display the entire contents.

--key <name>

Display entries matching the key.

--map <name>

Display entries matching the map. This option is only applicable for NIS/YP cache. It can be combined with --key option.
