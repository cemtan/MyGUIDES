dedupe-replication-target-support

Enable or disable support for deduplication on replication target filesystems

Syntax

dedupe-replication-target-support [--enable | --disable]

Description

Enable or disable support for deduplication on replication target filesystems.

If enabled (the default) then deduplication can be performed on replication target filesystems that have been configured with dedupe support.

If disabled, then deduplication will not be performed on any replication target filesystem regardless of their individual configurations.

Deduplication of replication targets is not supported on HNAS-3080, HNAS-3090, HNAS-4020, or HNAS-4040 systems.

Options:

--enable

Enable support for deduplication on replication target filesystems.

--disable

Disable support for deduplication on all replication target filesystems.

If neither the enable or disable option is specified, the current setting is displayed.

Examples

$: dedupe-replication-target-support --enable
Support for deduplication on replication target filesystems is enabled

Applies To

Cluster wide
