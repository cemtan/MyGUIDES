nis-cache-ttl

Displays or sets the NIS cache TTL

Syntax

nis-cache-ttl [<duration>(ms|s|m|h|d)]
Alias:       ldap-cache-ttl

Description

Displays or sets the NIS cache TTL.

The NIS cache TTL is the maximum duration that NIS query results will be cached. If no TTL value is set, the default is 15 minutes. Permissible values range between 300 milliseconds and 45 days. The default units for the set action are seconds. Updating the NIS cache TTL will result in NIS cache flush.

Examples

Display the NIS cache TTL:

        $ nis-cache-ttl
        NIS cache TTL: 15 min
        $

Set a new NIS cache TTL:

        $ nis-cache-ttl 1d
        NIS cache TTL: 24 h
        $


        $ nis-cache-ttl 600
        NIS cache TTL: 10 min
        $

Applies To

EVS security context
