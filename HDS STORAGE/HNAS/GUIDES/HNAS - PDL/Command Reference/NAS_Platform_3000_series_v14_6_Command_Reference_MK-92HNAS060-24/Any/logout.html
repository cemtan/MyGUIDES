logout

Logout from the console

Syntax

logout

Description

The logout command does not disconnect from the current console but does remove its authenticated status. The login command can then be used to authenticate as a different user. Use the exit command to disconnect too.

Examples

server:$ whoami
User supervisor currently logged in.
server:$ devuser on; logout; login dev ed6nO9rTw0CJOu5o
server:$ whoami
User dev currently logged in.
server:$ exit
Connection closed by foreign host.
$

Applies To

Console

See Also
