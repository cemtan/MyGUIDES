package-get-version

Get version of a package

Syntax

package-get-version <package-file>

Description

The package-get-version command displays the version number of the specified package.

Examples

package-get-version mercury-package_6.5.1816.00.tar.gz

Applies To

Cluster node

See Also

package-list package-remove package-set-default package-upgrade

Privilege Level

Supervisor
