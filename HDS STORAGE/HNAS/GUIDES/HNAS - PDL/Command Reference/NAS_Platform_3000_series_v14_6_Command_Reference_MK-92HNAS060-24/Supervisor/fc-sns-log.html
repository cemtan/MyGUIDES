fc-sns-log

dump/clear FC (SNS) logs

Syntax

fc-sns-log <[-d, --dump] | [-c, --clear] | [-s, --size] | [-h, --help]>

Description

Dump or clear the contents of the fibre channel (FC) SNS log.

The fc-sns-log logs FC commands transmitted and responses received from the fabric name server port on address FFFFFC. There is a separate log for each host port. If no host port number is specified, host port 1 FC log is assumed.

Console output may be stopped or aborted with a key press or by CTRL-C.

The syntax of a log entry

Sent/Rxed

FC frame was sent or received by the server host port.

Frame Type

Name of the command frame sent to the FC target port or the name of the response frame received by the FC target port.

xxxxxx

Address of the target port that the FC command was sent to or a response was received from, which is always FFFFFC.

<OXID>

The OXID of the exchange.

[Sn]

n = the server internal session number of the exchange.

STATUS

The status of the session:

OFFLINE

The remote port is offline or is not responding and has been logged out.

POLLING

The remote port is not responding so the host port is periodically polling it.

INACTIVE

The remote port is an initiator port.

DISCOVERY

The session is attempting to establish a login session with the remote port.

PENDING

Attempting to re-establish an active login session with the remote target port.

ACTIVE

The session has successively established a login session with the target port.

Options:

-d, --dump

Dump the contents of the FC log for the given host port. The last 20 entries are output if no range is specified.

-h, --hport <1..n>

Specify host port log to dump.

-r, --rev

Output entries in reverse order, oldest last.

-a, --all | [-n <[-]no of entries to dump>

Output all entries, last n entries, or first n entries if n is negative. If n = 0, dump entire log; by default, dump last 20 entries.

-c, --clear

Clear the log, default is host port 1 log.

-h, --hport <1..n>

Specify host port log to be cleared.

-C, --Conf

Do not request confirmation.

-s, --size

Show the current and maximum log size.

-h, --help

Show help.

Examples

Dump the entire FC SNS log for host port 1:

fc-sns-log -d -a

Dump last 100 entries from host port 2 SNS log:

fc-sns-log -d -h 2 -n 100

Dump first 10 entries from host port 2 SNS log:

fc-sns-log -d -h 2 -n -10

Clear the host port 1 SNS log, no confirmation:

fc-sns-log -c -h 1 -C

Applies To

Cluster node
