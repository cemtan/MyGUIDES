msconnections

List management server connections

Syntax

msconnections

Description

The msconnections command dumps information about current connections to the following management servers.

HTTP

Remote management (API access)

HTTPS

Remote management (secure API access)

LoopbackHTTP

Local management via the loopback interface (API access)

ss

Statistics Server

ssc

Command line (CLI)

vss

VSS hardware provider DLL connection

To see statistics about non-current management server connections, use the msstats command.

Examples

mercurymg110d(HNAS-3090):$ msconnections
Active:
  SSC(0x00007F88A61DA680) clo: 0 rc: 0 sock: 0x00007F88342A8FB0 peer: 203.0.113.0 age: 36457ms
  SSC(0x00007F88A5EC24B0) clo: 0 rc: 0 sock: 0x00007F88342A9280 peer: 203.0.113.0 age: 2152ms
  HTTPS(0x00007F88A60FF120) clo: 0 rc: 0 sock: 0x00007F88342A8CE0 peer: 127.0.0.1 age: 255723ms
Closing:
  (none)
mercurymg110d(HNAS-3090):$

Applies To

Cluster node
