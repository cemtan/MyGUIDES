chgrp

Change the group associated with a file

Syntax

chgrp group <file1> [<file2>...]

Description

The chgrp command is used to change the group ownership of a file.

Usage:

group

Specify the unix id or unix name of the new group owner of the file(s).

Examples

Change the group ownership of file 'bar' to group 'foo':

chgrp foo bar

Applies To

EVS

See Also

chown

Privilege Level

Supervisor
