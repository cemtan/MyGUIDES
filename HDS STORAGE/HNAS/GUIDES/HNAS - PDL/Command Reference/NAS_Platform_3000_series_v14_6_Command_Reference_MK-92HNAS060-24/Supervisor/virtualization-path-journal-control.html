virtualization-path-journal-control

Controls the virtualization path journal.

Syntax

virtualization-path-journal-control -t <fs label> -n <association name> [ --journal-enable | --journal-disable | --email-logs --recipient <email-address> [--start-date <YYYY-MM-DD>] [--end-date <YYYY-MM-DD>] | --delete-logs [--start-date <YYYY-MM-DD>] [--end-date <YYYY-MM-DD>] [--confirm] ]

Description

Manages journal of files deleted from remote server for a virtualization path association.

Options:

-t, --target <fs label>

Target file system.

-n, --name <association name>

Logical name of the association.

-e, --journal-enable

Enables a journal for a virtualization path association.

-d, --journal-disable

Disables a journal for a virtualization path association.

-m, --email-logs

Sends logs by email.

-u, --delete-logs

Deletes logs.

-r, --recipient <email-address>

Specifies an email address to send logs to.

-s, --start-date <YYYY-MM-DD>

First log date to send by email or to delete (optional). If not provided, defaults to first log file existing.

-f, --end-date <YYYY-MM-DD>

Last log date to send by email or to delete (optional). If not provided, defaults to current date.

-c, --confirm

Skips interactive confirmation for --delete-logs

Examples

virtualization-path-journal-control -t fs1 -n assoc1

Provides information about journal state.

virtualization-path-journal-control -t fs1 -n assoc1 --journal-enable

Enables journal for the virtualization path for file system fs1.

virtualization-path-journal-control -t fs1 -n assoc1 --email-logs --recipient user@example.com --start-date 2012-09-01 --end-date 2012-09-30

Sends log of all deleted files in September for the virtualization path for file system fs1.
