apppro-stats

Display AppPro statistics

Syntax

apppro-stats [-r]

Description

The apppro-stats command displays basic statistics about AppPro management call activity. The AppPro SOAP service is used by external management applications to control certain features of the server.

Options:

-r, --reset

Reset statistics.

Examples

Reset AppPro statistics:

apppro-stats -r

Applies To

Cluster node

Privilege Level

Supervisor
