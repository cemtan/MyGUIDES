migration-cloud-destination-delete

Delete an external cloud migration destination

Syntax

migration-cloud-destination-delete <destination name>|<destination id>

Description

Deletes cloud migration destination representation on this cluster system.

<destination name>

Name of destination to delete.

<destination id>

ID of destination to delete.

Examples

migration-cloud-destination-delete CloudDest

Deletes destination CloudDest

Applies To

Global security context

See Also

migration-cloud-destination-create migration-cloud-destination-list migration-cloud-destination-modify migration-cloud-policy-delete migration-delete-external-path

Supervisor
