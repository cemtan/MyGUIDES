namespace-ls

Displays the contents of a cluster or EVS namespace

Syntax

namespace-ls [--links] [--verbose] <cluster or EVS namespace> [<path>]
Alias:       cns-ls

Description

Displays the contents of the cluster or EVS namespace. If no path is specified then the contents of the root of the cluster or EVS namespace are displayed.

Usage:

namespace-ls [--links] [--verbose] <cluster or EVS namespace> [<path>]

-l, --links

Lists the links in any directory below the given path. Don't include directories in the listing.

-v, --verbose

Reports the EVS hosting a link's target volume. Applicable for cluster namespaces only.

Examples

Display the contents at the root of the cluster namespace 'cns0':

namespace-ls cns0
.
..
home

Display the contents at the root of the EVS namespace 'evs1':

namespace-ls evs1
.
..
home

Display the links to file systems beneath the root of the cluster namespace 'cns0':

namespace-ls -l cns0
/home/fs1 -> fs1:/
/home/fs0 -> fs0:/

Display the links to file systems beneath the root of the EVS namespace 'evs1':

namespace-ls -l evs1
/home/fs1 -> fs1:/
/home/fs0 -> fs0:/

Applies To

Cluster wide
