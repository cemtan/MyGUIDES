virusscan-exclusion-list-disable

Disable the virus scanning exclusion list

Syntax

virusscan-exclusion-list-disable

Description

The virusscan-exclusion-list-disable command disables the use of the virus scan exclusion list in determining whether or not to scan a file.

In this state, whether a file is scanned or not is determined by the state of the inclusion list.

Applies To

EVS

See Also
