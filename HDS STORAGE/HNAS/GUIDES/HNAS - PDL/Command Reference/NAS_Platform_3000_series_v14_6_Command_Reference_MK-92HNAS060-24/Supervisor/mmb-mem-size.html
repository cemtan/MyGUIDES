mmb-mem-size

Show the amount of memory on the MMB

Syntax

mmb-mem-size [-c|--current]

Description

Shows the current MMB memory size.

All memory sizes are given in megabytes, and have been rounded down to the nearest megabyte if necessary.

Options:

-c, --current

Show the current MMB memory size.

Examples

mg34-1(MMB):$ mmb-mem-size
Current MMB memory size     8192 MB

Displays the current amount of memory installed on the MMB.

Applies To

Cluster node
