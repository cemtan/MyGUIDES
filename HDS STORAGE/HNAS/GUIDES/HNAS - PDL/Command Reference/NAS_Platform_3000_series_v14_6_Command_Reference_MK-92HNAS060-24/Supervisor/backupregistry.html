backupregistry

Backup the registry file to the current directory

Syntax

backupregistry [-v] [-f <filename>]

Description

Uses SSC to make a backup copy of the server's registry configuration files on a client system.

This command is provided along with the "restoreregistry" command for use in a disaster recovery scenario; contact your support provider before attempting a registry restore operation.

Options:

-v, --verbose

Displays extra information

-f, --file <filename>

Stores file as 'filename' instead of default (registry.tgz)

Examples

mylinuxsystem:~/tmp$ssc 192.168.43.70


Console
Server name : MYSERVER
MAC ID : 10-A7-2D-49-D2-F5
server:$ backupregistry
Wrote 9.25 KB (9475 B) in 94 ms at 98.4 KB/s (100797 B/s).
Transferred registry to local file registry.tgz


server:$ backupregistry -f mybackup.tgz
Wrote 9.21 KB (9437 B) in 45 ms at 204 KB/s (209711 B/s).
Transferred registry to local file mybackup.tgz

Applies To

Cluster wide
