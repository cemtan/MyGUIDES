service-running

Display service status

Syntax

service-running

Description

Reports the state of the file system services. A service is reported as started if it can accept incoming connections.

The run-level command also reports file system service states. However run-level reports whether the service listeners have been set up, not whether the service is ready to accept incoming connections.

Examples

server:$ service-running
NFS is currently started on evs 1
NFS is currently started on evs 2
CIFS is currently started on evs 1
CIFS is currently started on evs 2
iSCSI is currently started on evs 1
iSCSI is currently started on evs 2
