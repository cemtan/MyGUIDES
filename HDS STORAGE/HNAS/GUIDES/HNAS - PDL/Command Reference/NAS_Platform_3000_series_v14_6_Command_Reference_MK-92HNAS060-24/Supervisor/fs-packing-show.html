fs-packing-show

Displays the raw and effective packing flags in the latest on-disk DSB of a mounted or unmounted file system

Syntax

fs-packing-show <file-system>

Description

Display the raw and effective packing flags in the latest on-disk dynamic superblock (DSB) of a file system. The file system can be mounted or unmounted. Note that the packing flags in the current checkpoint are always in sync with the packing flags in the latest on-disk DSB while the file system is mounted.

There are two packing flags: onode packing and data packing. Both the flags are normally in the same state, either enabled or disabled. File system packing can be supported only on file systems with dedupe support. Thus, the effective packing is disabled on file systems without dedupe support regardless of the raw values of the packing flags.

Options:

<file-system>

File system label.

Examples

Show file system packing flags for fs, which supports dedupe:

   $ fs-packing-show fs
   Information from the latest DSB on disk for file system 'fs':
   	Bitmap type             : reference counted
   	Raw onode packing       : enabled
   	Raw data packing        : enabled
   	Effective onode packing : enabled
   	Effective data packing  : enabled

The raw packing flags can also be seen, if they are enabled, in the output of the dsb command:

   $ dsb -f fs
   File system is WFS-2 and has 128 DSBs.
   WFS-2 DSB @ index 124, offset 0x3470000 / 0x3878000 (2021-09-20 16:51:48-07:00)
