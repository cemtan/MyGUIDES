mkdir

Creates directories

Syntax

mkdir [-p|--parents] [-v|--verbose] [--] <dir1> [<dir2>...]

Description

WARNING: This command bypasses all security checks and doesn't ask for confirmation, so make sure you know what you're doing.

The mkdir command is used to create one or more directories. If a directory name begins with a dash, it cannot be specified as the first argument to mkdir. It should be specified after a stand-alone option "--" or after another directory name that does not begin with a dash.

Usage:

--parents, -p

Do not fail if the directory exists, make parent directories as needed.

--verbose, -v

Display a status message for each directory.

--

Optional to signify the end of options and hence all subsequent arguments are directory names to create. Useful if the first directory name begins with a dash.

Examples

1. To create the directories 'dir1', 'dir2' and 'dir3' in the current working directory:

   mkdir dir1 dir2 dir3

2. Create parent directories if needed:

   mkdir -p xdir/ydir/zdir

3. Two ways to create a directory whose name begins with a dash:

   mkdir -- -testdir  # after stand-alone option --
   mkdir adir -bdir   # after another directory

EVS
