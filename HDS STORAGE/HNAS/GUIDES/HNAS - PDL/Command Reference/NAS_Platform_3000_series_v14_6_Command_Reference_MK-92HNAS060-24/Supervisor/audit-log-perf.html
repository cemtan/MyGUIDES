audit-log-perf

Display file system audit log performance statistics

Syntax

audit-log-perf [--reset] [--graph] [<file-system>]

Description

The audit-log-perf command displays performance statistics about file system audit log activity.

Options:

-r, --reset

Reset performance statistics.

-g, --graph

Display statistics graphically.

file-system

File system label.

Examples

Display the remote audit log performance statistics graphically for file system fs0:

audit-log-perf --graph fs0

Reset the remote audit log performance statistics for file system fs1:

audit-log-perf -r fs1
