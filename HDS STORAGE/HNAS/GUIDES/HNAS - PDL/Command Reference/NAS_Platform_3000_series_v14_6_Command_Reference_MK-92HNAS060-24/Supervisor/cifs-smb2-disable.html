cifs-smb2-disable

Disables the SMB2 service.

Syntax

cifs-smb2-disable

Description

The cifs-smb2-disable command is used to disable the SMB2 service for the current evs. It has been superseded by the smb-min-supported-version and smb-max-supported-version commands.

Examples

cifs-smb2-disable

Applies To

EVS

See Also

cifs-smb2-enable cifs-smb2-status evs-select smb-max-supported-version smb-min-supported-version

Privilege Level

Supervisor
