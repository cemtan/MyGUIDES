rpc-config

Configure RPC

Syntax

rpc-config [-c on|off]

Description

The rpc-config command is used to configure ONC/RPC behavior. When invoked without any arguments the current configuration is displayed.

Usage:

rpc-config [OPTION...]

-c, --can-timeout-conns (on|off)

This option configures whether or not TCP connections transporting ONC/RPC traffic are allowed to time out.

Applies To

Cluster wide

Privilege Level

Supervisor
