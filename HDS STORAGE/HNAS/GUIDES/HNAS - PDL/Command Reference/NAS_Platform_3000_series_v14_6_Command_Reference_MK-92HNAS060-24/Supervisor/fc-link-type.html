fc-link-type

Set or display FC link port type

Syntax

fc-link-type [-i <1|...|N>] [-t <N|NL>]
Alias:       fclinktype

Description

The fc-link-type command allows you to query and set the link type of the Fibre Channel interfaces. The link types supported are "N" for fabric switch attachment and "NL" for direct connect arbitrated loop operation. When setting the link type the setting should always be consistent with the physical connectivity. Failure to observe this could result in errors being reported in the event log and/or loss of access to storage devices. The settings apply to all nodes in a cluster. Command options are:

-i <interface_number>

Specify the FC interface to query or set. If this option isn't set then the command applies to all available interfaces. Interfaces are numbered from 1.

-t N|NL

If this option is passed then the specified interface (see -i option) or all interfaces will be set to the specified type.

Examples

Query the link type on all interfaces:

fc-link-type

Set the link type on interface 1 to NL:

fc-link-type -i 1 -t NL

Set the link type on all interfaces to N:

fc-link-type -t N

Applies To

Cluster wide

See Also

fc-link fc-link-speed fc-link-status

Privilege Level

Supervisor
