hard-links

Displays info about a file's hard links

Syntax

hard-links [-f <fs>] [-v] [-l] <file name or handle>

Description

The hard-links command displays information about a file's hard links.

Usage:

hard-links [OPTION...] <file name or handle>

-f, --file-system <file system>

Look for the file on the given file system. The default is to use the console's current file system.

-l, --live

If the file is in a snapshot, show the links for the equivalent file in the live file system.

-v, --verbose

Display extended information about each hard link.

Examples

hard-links -v /home/myfiles/file0

EVS
