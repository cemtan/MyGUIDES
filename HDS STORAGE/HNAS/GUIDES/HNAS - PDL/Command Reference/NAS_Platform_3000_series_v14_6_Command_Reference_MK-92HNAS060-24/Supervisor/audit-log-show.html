audit-log-show

Display the file system audit log for a file system

Syntax

audit-log-show [-o] [-f <backup-file>] <file-system> [count]

Description

Options:

-f, --file <backup-file>

Read records from the named backup file system audit log file instead of the current audit log. The full path name from the root of the file system must be provided.

-o, --oldfirst

Display audit records oldest to newest (default: newest to oldest).

file-system

File system label.

count

Maximum number of audit records to display (optional).

The audit-log-show command displays file system audit log records from the current audit log for a file system, or from a backup audit log file that has been saved to the same file system.

If count is not specified and there are many records available to be displayed, the command will seek confirmation before continuing.

Examples

To show 5 records from file system fs0:

audit-log-show fs0 5

To show all the records from the backup audit log file "backup.evt" in the directory "logs" on file system fs0:

audit-log-show --file logs/backup.evt fs0

Applies To

EVS
