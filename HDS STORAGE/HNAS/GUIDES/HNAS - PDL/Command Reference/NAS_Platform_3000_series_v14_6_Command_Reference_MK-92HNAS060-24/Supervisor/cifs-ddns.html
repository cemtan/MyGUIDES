cifs-ddns

Enable/disable ADS CIFS name registration with DDNS

Syntax

cifs-ddns [on|off]

Description

The cifs-ddns command allows you to configure whether the server attempts to register its ADS-mode CIFS serving names with DDNS.

By default, DDNS registration of ADS-mode CIFS names is enabled. Note that DDNS registration of NT4-mode CIFS names is never attempted.

Separate cifs-ddns configurations are kept for individual EVSs according to the current per-EVS security settings.

Changes made by this command take effect immediately.

Examples

To display the present CIFS serving name DDNS registration setting:

cifs-ddns

To disable CIFS name DDNS registration:

cifs-ddns off

To enable CIFS name DDNS registration:

cifs-ddns on

Applies To

EVS security context

See Also

cifs-name console-context dnsserver evs-security evs-select

Privilege Level

Supervisor
