dump-logged-stats

Dump logged statistics to the console

Syntax

dump-logged-stats <-header>

Description

This command is used to display the internally logged statistics as comma-separated rows. Logged statistics are gathered on a per board basis.

Options:

-header

Display the column headers

Examples

To dump the statistics with a header:

dump-logged-stats -header

To dump the statistics from all boards

for-each-board dump-logged-stats

Applies To

Cluster node

See Also

for-each-board start-logging-stats stats-logger stop-logging-stats

Privilege Level

Supervisor
