cluster-node-route-gateway-delete

Delete a gateway route

Syntax

cluster-node-route-gateway-delete --pnode <pnode> <gateway>

Description

The cluster-node-route-gwateway-delete command deletes a gateway from the routing table but only for the specified pnode.

Options

The following options are recognised:

-p, --pnode <pnode>

The pnode from which this cluster-node-route will be removed. It must be supplied.

<gateway>

This is the gateway that identifies the route. It must be supplied.

Examples

$ cluster-node-route-gateway-delete 10.1.2.3 --pnode 2
Route cache flushed.
$

Deletes a gateway 10.1.2.3 that applied only on pnode 2.

Applies To

Cluster wide
