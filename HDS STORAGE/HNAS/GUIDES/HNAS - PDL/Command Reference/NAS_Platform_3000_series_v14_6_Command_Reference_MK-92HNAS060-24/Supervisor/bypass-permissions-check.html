bypass-permissions-check

Disable fs access control

Syntax

bypass-permissions-check [on|off]

Description

Used to display/modify whether file permissions are enforced.

To display the current setting, run the command with no arguments. To change the current setting, specify the on or off parameter.

The setting is persistent and takes immediate effect across the current EVS security context (cluster-wide or per-EVS).

Examples

server$ bypass-permissions-check
default, shared security context: bypass-permissions-check: on
server$ bypass-permissions-check off
default, shared security context: bypass-permissions-check: off
server$ bypass-permissions-check on
default, shared security context: bypass-permissions-check: on
server$

Applies To

EVS security context
