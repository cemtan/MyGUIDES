migration-cloud-rule-list

List an external migration rules

Syntax

migration-cloud-rule-list [<rule name>]|[rule id>]

Description

Lists cloud external migration rules.

<rule id>

ID of the rule.

<rule name>

Name of the rule, which must be unique.

Examples

migration-cloud-rule-list


       Label                                                     Rule     Case sensitive     In use by Policies
------------     ----------------------------------------------------     --------------     ------------------
    rule_dir                                 INCLUDE (<PATH /dir2/*>)                 No                    Yes
   all_files                                   INCLUDE (<FILENAME *>)                 No                     No
JpgFilesRule                               INCLUDE (<FILENAME *.jpg>)                 No                     No
                 INCLUDE (<INACTIVE_OVER 30> AND <UNCHANGED_OVER 30>)

Lists all cloud external migration rules on this cluster.

migration-cloud-rules-list JpgFilesRule


                  Rule ID: c0e90c1a-32c5-11d1-9003-c1122ef0b8b6
               Rule label: JpgFilesRule
         Rule description: JPG files not accessed or modified in 30 days
                     Rule: INCLUDE (<FILENAME *.jpg>)
                           INCLUDE (<INACTIVE_OVER 30> AND <UNCHANGED_OVER 30>)
           Case Sensitive: No
       In use by policies: No

List the cloud external migration rule named 'JpgFilesRule'.

Applies To

Global security context
