virusscan-exclusion-list-enable

Enable the virus scanning exclusion list

Syntax

virusscan-exclusion-list-enable

Description

The virusscan-exclusion-list-enable command enables the use of the virus scan exclusion list in determining whether or not to scan a file.

In this state, if a file's extension appears in the exclusion list it is not scanned. This is true regardless of the "scan all" state or its appearance in the inclusion list.

Applies To

EVS
