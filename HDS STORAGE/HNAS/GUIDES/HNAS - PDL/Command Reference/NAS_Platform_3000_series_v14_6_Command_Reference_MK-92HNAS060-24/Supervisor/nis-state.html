nis-state

Configure the NIS client to be active or not

Syntax

nis-state [on | off]

Description

Displays and manipulates whether the NIS client is active. This affects both traditional and LDAP-based NIS client modes (see nis-ldap-mode).

Examples

To display the current setting of NIS client state:

nis-state

To enable the NIS client:

nis-state on

To disable the NIS client:

nis-state off

Applies To

EVS
