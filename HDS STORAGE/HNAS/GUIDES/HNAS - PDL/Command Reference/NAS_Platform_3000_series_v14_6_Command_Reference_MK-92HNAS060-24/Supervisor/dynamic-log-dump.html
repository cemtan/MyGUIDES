dynamic-log-dump

Dumps dynamic log

Syntax

dynamic-log-dump

Description

The dynamic-log-dump command will dump to the console the contents of the server's dynamic log. This file is created when checkfs, fixfs, and other disk management utilities have been run. It contains useful information about any problems with the filesystem the utility was run against.

Applies To

Cluster node

See Also
