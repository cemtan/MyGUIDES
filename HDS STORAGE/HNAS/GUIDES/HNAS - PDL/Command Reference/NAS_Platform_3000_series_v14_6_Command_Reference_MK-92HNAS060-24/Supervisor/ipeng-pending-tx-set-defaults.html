ipeng-pending-tx-set-defaults

Reset pending request tx buffer limits to default values

Syntax

ipeng-pending-tx-set-defaults

Description

The ipeng-pending-tx-* commands control a feature that limits the number of buffers used to store transmit data. This is intended to avoid transmission on TCP connections starving all other TCP connections of buffers. It is a global setting, acting on all ports.

This feature shouldn't usually come into play, so configuration is unlikely to be useful but it is turned on by default to catch the times when it will help. Only use these commands under instruction from your support provider.

Parameters are in thousands (strictly x1024) of buffers. Newer platforms have buffers of 8KB. Older platforms have 2KB buffers.

The commands available are as follows.

ipeng-pending-tx-conn-limit [<number>]

If a parameter is given, sets the threshold of tx buffers each TCP connection is allowed before new buffers should be discarded. The parameter is in binary thousands (1024) of buffers.

With no parameter, displays the current and default settings.

ipeng-pending-tx-total-limit [<number>]

If a parameter is given, sets the threshold of free tx buffers before new buffers should be discarded. The parameter is in binary thousands (1024) of buffers.

With no parameter, displays the current and default settings.

ipeng-pending-tx-limit-enable

Turns on the tx buffer limiting feature.

ipeng-pending-tx-limit-disable

Turns off the tx buffer limiting feature.

ipeng-pending-tx-set-defaults

Sets the pending tx buffer limiting feature back to default values. The default values can be found by typing the command "ipeng-pending-tx-conn-limit" and "ipeng-pending-tx-total-limit" with no parameters.

Examples

To turn on buffer limit enable, then set the free level to 500 Ki buffers:

    $ ipeng-pending-tx-limit-enable
    $ ipeng-pending-tx-total-limit 500

To set transmit buffering available for a single connection to 8 Ki buffers:

    $ ipeng-pending-tx-conn-limit 8

To disable buffer limit enable:

    $ ipeng-pending-tx-limit-disable

To set the feature back to factory settings:

    $ ipeng-pending-tx-set-defaults

To set *all* ipeng features back to factory default settings:

    $ ipeng -d

Applies To

Cluster wide
