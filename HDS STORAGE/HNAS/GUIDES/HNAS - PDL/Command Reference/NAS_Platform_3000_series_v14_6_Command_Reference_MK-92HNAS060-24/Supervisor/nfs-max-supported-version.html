nfs-max-supported-version

Set maximum supported version of NFS protocol

Syntax

nfs-max-supported-version [3 | 4]

Description

The nfs-max-supported-version command sets and displays nfs-max-supported-version setting. This setting controls the maximum version of nfs that the server will make available to clients that mount nfs exports. Note that the portmapper will no longer advertise NFS version 4 services to clients if its usage is disabled.

The following options are available:

nfs-max-supported-version

Display current setting.

nfs-max-supported-version [ 3 | 4 ]

Change setting to NFSv3 or NFSv4 respectively.

Examples

nfs-max-supported-version


nfs-max-supported-version 3

Applies To

Cluster wide
