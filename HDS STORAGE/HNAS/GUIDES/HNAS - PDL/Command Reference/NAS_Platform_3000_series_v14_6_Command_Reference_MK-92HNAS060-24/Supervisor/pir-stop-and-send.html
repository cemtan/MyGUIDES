pir-stop-and-send

Finish a performance info report (PIR) early

Syntax

pir-stop-and-send

Description

Aborts a currently running PIR and sends the results.

Note that this command will not work if the PIR running is a Continuous PIR (i.e. it was started with "pir --start"). The correct way to stop a Continuous PIR is with "pir --stop".

Applies To

Cluster node

See Also

performance-info-report pir-cancel pir-status

Privilege Level

Supervisor
