ndmp-session-stats

Display performance stats for the current transfer of the NDMP session

Syntax

ndmp-session-stats [-r|--reset] [-p|--perf] [-m|--memory] [<session_number>]

Description

The command controls various statistics for an NDMP session identified by the session number parameter or for all sessions on this node. The command usually shows numbers of files/directories processed. The -m parameter can be used to show some details of memory usage and the -p parameter can be used to show performance statistics. The transfer stats and performance stats for the session(s) may be reset using the -r parameter. Note that in some cases the statistics may not exist and will not be displayed.

Examples

Show the numeric statistics and memory use for all sessions:

ndmp-session-stats -m

Display the current performance statistics for NDMP session 22:

ndmp-session-stats -p 22

Reset the statistics for session 24. This might be used at the start of a time period which is to be scrutinized.

ndmp-session-stats -r 24
