cifs-log-access-administrators-only

Set cifs logs access to administrators only

Syntax

cifs-log-access-administrators-only

Description

The cifs-log-access-administrators-only command is used to restrict access to the cifs logs to administrators.

Examples

cifs-log-access-administrators-only

Applies To

EVS security context

See Also

cifs-log-access-mode cifs-log-access-unrestricted

Privilege Level

Supervisor
