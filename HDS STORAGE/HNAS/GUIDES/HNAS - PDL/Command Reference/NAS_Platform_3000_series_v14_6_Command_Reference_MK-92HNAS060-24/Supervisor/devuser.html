devuser

Limit access to the top level user

Syntax

devuser [on|off|serialonly]

Description

The devuser command controls whether the 'dev' user is enabled. This user is a special user used by your support provider to run very low level diagnostic and debugging tools. It is recommended that you leave access to this user on in case it is needed. Passwords for the dev user are only generated by a tool that is not publicly available, and are valid for 1 day only.

The options are

on

Allows dev user access from any console (e.g. ssc, serial port).

off

Completely disables the dev user.

serialonly

Allows dev user access only via the serial port.

Examples

To enable the dev user but only when accessing the server via the serial port:

devuser serialonly

Applies To

Cluster wide

Privilege Level

Supervisor
