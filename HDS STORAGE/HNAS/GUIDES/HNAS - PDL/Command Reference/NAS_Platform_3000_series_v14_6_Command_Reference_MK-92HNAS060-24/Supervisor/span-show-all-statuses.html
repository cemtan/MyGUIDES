span-show-all-statuses

Display the statuses of all spans

Syntax

span-show-all-statuses [--verbose] [--separate]

Description

'span-show-all-statuses' produces a list of all spans visible to the server, displaying the status of each span.

The --verbose or -v option decodes the 'span status' field, showing any problems with the span. Without --verbose, the status is shown only as a hexadecimal number.

The --separate or -s option displays separate results for each cluster node. Without --separate, each fault is displayed if it is present on any server. Most faults are either present on all nodes or absent from all nodes, but a few can be present on one node and not on another. For example, on a gateway server, if you disconnect all the Fibre Channel ports, all spans will fail on that server but not on its peers.

Here are the fields displayed by this command:

Span instance name

The 'storage-based-snapshot-label' man page explains the three kinds of span label.

Permanent ID

This long hexadecimal number uniquely identifies the span.

Span Status

This hexadecimal number is a bit-field showing all statuses of the span.

The --verbose option displays the status more readably.

Color

This is the 'traffic light' color representation of the status: green = OK, amber = Warning, red = Severe.

Pnode

This field, which is shown only when you specify --separate, shows the physical node -- the server -- for which information is displayed on this line.

Examples

server:$ span-show-all-statuses
Span instance name             Permanent ID       Span status   Color
------------------             ------------       -----------   -----
Foo                            7BFCCF4E5E832C89   0000002000    Amber
Run 'trouble storage' to start diagnosis
server:$ span-show-all-statuses -v
Span instance name             Permanent ID       Span status   Color
------------------             ------------       -----------   -----
Foo                            7BFCCF4E5E832C89   0000002000    Amber
                                                  Path validation is in progress on some SDs


Run 'trouble storage' to start diagnosis
server:$ span-show-all-statuses -s
Span instance name             Permanent ID       Pnode   Span status   Color
------------------             ------------       -----   -----------   -----
Foo                            7BFCCF4E5E832C89
                                                      1   0000000000    Green
                                                      2   0000000000    Green
                                                      3   0000002000    Amber
Run 'trouble storage' to start diagnosis
server:$ span-show-all-statuses -sv
Span instance name             Permanent ID       Pnode   Span status   Color
------------------             ------------       -----   -----------   -----
Foo                            7BFCCF4E5E832C89
                                                      1   0000000000    Green
                                                          Status OK


                                                      2   0000000000    Green
                                                          Status OK


                                                      3   0000002000    Amber
                                                          Path validation is in progress on some SDs


Run 'trouble storage' to start diagnosis
server:$
