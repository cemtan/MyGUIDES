new-battery-fitted

Set the relevant fields in the Serial EEPROM for a new battery

Syntax

new-battery-fitted <--field|--factory> --confirm

Description

Updates the battery fitted date in the MCP serial eeprom. Will also mark the battery for a conditioning cycle, which will be triggered next time the chassis monitor is started.

Parameters:

--field

Specify this option when fitting a new battery in the field. This will ensure the battery-fitted-date is set immediately and a conditioning cycle will be triggered next time the chassis monitor is started.

--factory

Specify this option when fitting a new battery in the factory. This will not set the battery-fitted-date immediately, it will be set next time the chassis monitor is started. A conditioning cycle will also be triggered next time the chassis monitor is started.

--confirm

Confirm command is to be run

Examples

new-battery-fitted --field --confirm

New battery fitted to the server in the field.

new-battery-fitted --factory --confirm

New battery fitted to the server in the factory.
