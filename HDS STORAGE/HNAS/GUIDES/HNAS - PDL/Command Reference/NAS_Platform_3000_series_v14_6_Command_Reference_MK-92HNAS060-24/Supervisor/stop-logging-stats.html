stop-logging-stats

Stops logging statistics whose names contain the optional string

Syntax

stop-logging-stats [<string>]

Description

Stops logging statistics whose names contain the given string. Logged statistics are gathered on a per board basis.

Examples

To stop logging all statistics with "monkey" in their name:

stop-logging-stats monkey

And on all boards

for-each-board stop-logging-stats monkey

Applies To

Cluster node

See Also

dump-logged-stats start-logging-stats stats-logger

Privilege Level

Supervisor
