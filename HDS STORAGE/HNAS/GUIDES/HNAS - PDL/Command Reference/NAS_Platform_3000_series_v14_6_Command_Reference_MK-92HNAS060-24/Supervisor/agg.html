agg

Configure/Display aggregation interfaces

Syntax

agg [show] | create agN | delete agN

Description

The agg command is used to manipulate an aggregated interface. An aggregation is a collection of one or more physical interfaces that are bundled together to behave as a single logical interface. Note that all physical interfaces participating in the same aggregation must be of the same speed.

Examples

To display existing aggregated interfaces:

    agg

To create a new aggregated interface:

    agg create ag1

To delete an existing aggregated interface:

   agg delete ag1

Note that an interface cannot be deleted if it has any assigned IP addresses, if packet capturing is in use, if is referenced by an aggregate Linux interface or if it is referenced by a VLAN interface.

Applies To

Cluster wide
