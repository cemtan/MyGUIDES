namespace-showlink

Displays the options for a link within a cluster namespace

Syntax

namespace-showlink <cluster-namespace> <path>
Alias:       cns-showlink

Description

Displays the options that will be applied to NFS and CIFS clients that traverse through the cluster namespace path specified.

Usage:

namespace-showlink <cluster-namespace> <path>

Applies To

Cluster wide

See Also

namespace-mklink namespace-modlink

Privilege Level

Supervisor
