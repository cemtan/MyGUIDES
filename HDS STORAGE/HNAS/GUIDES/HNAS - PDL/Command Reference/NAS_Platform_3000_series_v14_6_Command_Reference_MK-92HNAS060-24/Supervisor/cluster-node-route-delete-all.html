cluster-node-route-delete-all

Delete all cluster-node-route configuration entries

Syntax

cluster-node-route-delete-all [--confirm] -p pnode-id

Description

The cluster-node-route-delete-all command removes all the cluster-node-routes from a single pnode.

To delete all cluster-node-routes from a specified pnode:

cluster-node-route-delete-all [--confirm] -p pnode-id

-p <pnode>

The pnode from which the cluster-node-routes will be removed.

-c, --confirm

Confirms that all cluster-node-routes for the specified pnode should be removed.

Examples

To delete all the cluster-node-routes from pnode 5:

cluster-node-route-delete-all --confirm -p 5

Applies To

Cluster node
