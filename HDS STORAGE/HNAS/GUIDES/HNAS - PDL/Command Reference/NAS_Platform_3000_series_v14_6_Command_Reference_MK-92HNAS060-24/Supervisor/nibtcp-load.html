nibtcp-load

Load information for NIBTCP FPGA

Syntax

nibtcp-load [--block-list|--profile-list] | [-s|--summary] [--profile [<threshold%>]]

Description

The *-load commands are used to diagnose some performance problems by showing how busy parts of the hardware are.

Only use under instruction from your support provider.

--block-list

This option lists the names of all of the blocks. For those blocks which support vlsi profiling, the name of the profiler instance is also included in brackets.

--profile-list

This option lists the names of the profiler instances supported by this command (see --profile option).

-s, --summary

Display results without the bar graph.

--profile [<threshold%>]

This option will also execute the profiler for those blocks for which the load exceeds the supplied threshold (0 - 100). The default 'threshold' is 80%. The command execution time will depend upon the number blocks which are profiled.

Applies To

Cluster node
