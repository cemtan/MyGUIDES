ndmp-device-trace-remove

Stop per-device tracing for the specified NDMP subsystem

Syntax

ndmp-device-trace-remove <tape-req|scsi-req|tape-proc|tape-wr|tape-rd|stats|tape-error|state|position>

Description

Removes the specified NDMP device subsystem from the output of the NDMP device trace.

It is advisable to consult your Support Organisation before using this command to change the subsystems being traced.

Examples

Remove tracing for the NDMP device SCSI request subsystem:

ndmp-session-trace-remove scsi-req

Applies To

Cluster node

See Also

ndmp-device-trace ndmp-device-trace-add

Privilege Level

Supervisor
