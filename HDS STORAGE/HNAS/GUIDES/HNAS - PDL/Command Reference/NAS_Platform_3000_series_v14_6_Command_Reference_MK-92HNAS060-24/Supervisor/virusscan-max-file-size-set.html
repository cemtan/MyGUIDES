virusscan-max-file-size-set

Configure the virus scanning maximum file size

Syntax

virusscan-max-file-size-set <size>

Description

Set the maximum file size for virus scanning. Use "virusscan-max-file-size-show" to check the current setting.

Scanning large files for viruses can take a noticeable time and/or consume limited bandwidth. Use this command to restrict virus scanning only to files smaller than a specified maximum.

Sizes may be specified in bytes or in larger multiples by using an appropriate suffix. Note that sizes are scaled in binary units, rather than decimal. For example, 1 MB is 1048576 bytes, not 1000000. For more information on units, see "man units".

Examples

To set a maximum file size of 1234 bytes

virusscan-max-file-size-set 1234

To set a maximum file size of 3 Megabytes

virusscan-max-file-size-set 3MB

To set a maximum file size of 2.45 Gigabytes

virusscan-max-file-size-set 2.45GB

Applies To

EVS
