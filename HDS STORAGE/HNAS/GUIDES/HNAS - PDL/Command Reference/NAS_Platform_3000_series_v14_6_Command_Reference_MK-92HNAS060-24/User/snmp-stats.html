snmp-stats

Report or reset SNMP statistics

Syntax

snmp-stats [-r]
Alias:       snmpstats

Description

snmp-stats reports or resets the various statistics with respect to requests to the server's built-in SNMP agent.

-r, --reset

Reset all counters to zero.

Examples

Show current statistics:

snmp-stats

Reset current stats:

snmp-stats -r

Applies To

Cluster node
