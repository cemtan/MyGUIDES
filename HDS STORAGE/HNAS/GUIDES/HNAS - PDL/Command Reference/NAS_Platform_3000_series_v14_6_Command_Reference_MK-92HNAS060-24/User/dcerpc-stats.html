dcerpc-stats

Show DCE/RPC Statistics.

Syntax

dcerpc-stats [--verbose]

Description

The dcerpc-stats command reports the number and type of DCE/RPC packets that have been processed by the server. Use "dcerpc-stats-reset" to reset the statistics.

Usage:

dcerpc-stats [OPTION]

-v, --verbose

Shows additional statistics tables describing returned statuses and fault code counts.

Examples

dcerpc-stats

Displays statistics for PDU Types and total packet counts.

dcerpc-stats -v

Displays statistics for PDU Types and packet counts as well as returned status and fault code counts.

Applies To

Cluster node

See Also

dcerpc-stats-reset

Privilege Level

User
