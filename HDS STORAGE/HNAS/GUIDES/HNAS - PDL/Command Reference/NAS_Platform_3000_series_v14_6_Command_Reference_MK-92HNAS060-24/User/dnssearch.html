dnssearch

Define DNS search list

Syntax

dnssearch operation
where 'operation' is one of
	list
	add domain1 domain2 ...
	del domain

Description

The dnssearch command allows the search list to be manipulated. When a name is looked up, the DNS client appends the name to each of the domains in the search list in turn until a match is found. This functionality is similar to the search facility in resolv.conf in other OSs.

The search list is limited to six domains. There is also a combined limit of 255 characters for all domains, including an extra character (internal field-separator) per domain.

Examples

To list out the names in the search list:

dnssearch list

To add a name to the search list:

dnssearch add example.com

To delete a name from the search list:

dnssearch del example.com

Applies To

EVS security context
