snapshot-rule-list

Display details of the snapshot rules.

Syntax

snapshot-rule-list [<rule-name>]

Description

This command displays details about the specified snapshot rule. If no rule name is specified, the command displays details about all registered snapshot rules.

<rule-name>

Display rule by name.

Examples

server:$ snapshot-rule-list rule1
 Rule Name                        Queue Size   File System
 ------------------------------   ----------   -----------
 rule1                                     4   fs1

Displays details about rule1.

Applies To

EVS

See Also

crontab snapshot-rule-create snapshot-rule-delete snapshot-rule-modify
