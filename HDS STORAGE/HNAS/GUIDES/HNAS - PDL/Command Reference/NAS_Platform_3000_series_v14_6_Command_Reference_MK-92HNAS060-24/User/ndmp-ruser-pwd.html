ndmp-ruser-pwd

Change NDMP/adc restricted user password

Syntax

ndmp-ruser-pwd -o <oldpwd> -n <newpwd> <ruser_name>

Description

Options:

-o <oldpwd>

The old password associated with this restricted user

-n <newpwd>

The new password to be set

<ruser_name>

The restricted user to change

This command is a user level command that is intended to allow restricted users to change their own passwords. The old password needs to be provided to ensure that other ordinary users cannot change the password.

Examples

ndmp-ruser -o passwd -n XYZ987 johnx

johnx changes the administrator set password "passwd" to one of his own choosing "XYZ987".

Applies To

EVS
