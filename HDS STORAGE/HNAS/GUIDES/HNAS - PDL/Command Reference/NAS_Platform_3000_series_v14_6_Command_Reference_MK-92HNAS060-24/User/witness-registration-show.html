witness-registration-show

Show Witness Registration

Syntax

witness-registration-show <registration_key>

Description

Display a Witness Registration specified by registration key (UUID).

Examples

witness-registration-show 0292bf59-d6a9-d011-9094-01e6ec337fb6
Witness Registration:
  Key           : 0292bf59-d6a9-d011-9094-01e6ec337fb6
  Version       : v2
  Client Name   : hostname.example.com
  Net Name      : hostname
  IP Address    : 192.0.2.1
  Share Name    : -
  Notifications : NetName
  Keep Alive    : 120000 ms
  AsyncNotify   : registered, expires in 97194 ms

Applies To

EVS
