ndmp-devices-list

Display NDMP device information

Syntax

ndmp-devices-list [-t|--type all|tape|changer] [-p|--pnode <pnode_id>] [-v|--vnode <vnode_id>]

Description

The command displays media changer and tape drives available for use via NDMP. See the ndmp-devices man page for an overview of NDMP devices.

Before a backup device is listed by this command it must be made available for use and assigned to an EVS or all EVSs. The backup-devices-list command can be used to show all the tape drive and media changer devices that are visible to the machine. These devices can then be enabled using backup-device-allow-access and assigned to an EVS using backup-device-set-evs.

By default the command lists all such devices in the cluster. Optionally, parameters can be used to restrict the list of devices to those of particular types or those visible to a specific cluster node or EVS. After enabling use of devices the ndmp-devices-update should be used to rescan the devices.

Options:

-t, --type

Specifies the type(s) of devices to be listed.

-p, --pnode

List devices visible from a specific physical node.

-v, --vnode

List devices visible from a specific EVS.

Examples

List tape drive and media changer devices available to NDMP:

ndmp-devices-list

List only media changer devices:

ndmp-devices-list -t changer

Check which devices can be seen from physical node 2:

ndmp-devices-list --pnode 2

Applies To

Cluster wide

See Also

backup-device-allow-access backup-device-set-evs backup-devices-list ndmp-devices ndmp-devices-update

Privilege Level

User
