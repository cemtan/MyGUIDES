units

Description

This man page does not refer to a specific command. It explains how the system reports large quantities.

Storage sizes are scaled and printed in binary units (eg TiB), even where the output might suggest decimal units (eg TB). The CLI presentation can be altered to display eg TiB with:

server:$ set useStandardBinaryPrefixes true
server:$

Revert to the default with:

server:$ unset useStandardBinaryPrefixes
server:$
