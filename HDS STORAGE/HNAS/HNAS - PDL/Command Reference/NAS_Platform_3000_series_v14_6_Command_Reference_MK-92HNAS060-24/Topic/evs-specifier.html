evs-specifier

Description

Many commands require the user to uniquely specify an evs. This may be done using one the following methods.

evs id

The evs id is a numeric identifier. It is shown in the EVS ID column by evs list.

evs label

The evs label is a textual identifier. It is shown in the Label column by evs list.

Examples

evs list

Show evs information for a system:

server:$ evs list
