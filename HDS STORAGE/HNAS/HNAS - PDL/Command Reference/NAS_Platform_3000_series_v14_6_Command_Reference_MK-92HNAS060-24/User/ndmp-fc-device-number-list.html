ndmp-fc-device-number-list

Display NDMP numbers assigned to FC devices

Syntax

ndmp-fc-device-number-list

Description

List the mapping between Fibre Channel Devices that hold tape and autochanger devices and the Device Numbers used in the autogenerated NDMP device names. This mapping is usually automatically generated by NDMP but the ndmp-fc-device-number-set command can be used to request specific device numbering.

See the ndmp-devices man page for details of NDMP device naming.

Examples

ndmp-fc-device-number-list

Applies To

Cluster wide

See Also

ndmp-devices ndmp-devices-list ndmp-fc-device-number-delete ndmp-fc-device-number-set

User
