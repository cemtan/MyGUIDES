cifs-stats

Report or reset CIFS statistics

Syntax

cifs-stats [-r] [-b] [-v]

Description

The cifs-stats command reports the number of SMB requests that have been processed by the server, along with the current number of clients that are connected and the current number of shares that are mapped.

Usage:

cifs-stats [OPTION]

-b, --boot

Shows the statistics since boot. Resetting the statistics does not affect the values returned with the -b option.

-r, --reset

Resets the statistics. The request counts are returned to 0.

-v, --version <version>

Shows the statistics for the particular SMB version.

The supported version options are 1, 2, 2.1, 3 and the default value all.

Examples

cifs-stats

Displays statistics for all supported SMB versions.

cifs-stats -v 2

Displays statistics for SMB2 only.

Applies To

Cluster node
