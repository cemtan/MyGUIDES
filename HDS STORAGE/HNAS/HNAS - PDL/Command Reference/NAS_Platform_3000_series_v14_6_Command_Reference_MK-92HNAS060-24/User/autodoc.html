autodoc

Auto document commands

Syntax

autodoc [-a] [-u] [-s] [-f] [-c] [-w <word>]

Description

autodoc produces a formatted list of commands as specified by the options.

-a, --all

Documents commands for all user levels.

-u, --user

Documents commands for user level.

-s, --supervisor

Documents commands for supervisor level.

-f, --filter

Documents filters instead of commands.

-c, --csv

Documents commands in csv format.

-w, --search

Documents commands containing the specified word in the command, syntax or description.

Examples

autodoc -a -c
