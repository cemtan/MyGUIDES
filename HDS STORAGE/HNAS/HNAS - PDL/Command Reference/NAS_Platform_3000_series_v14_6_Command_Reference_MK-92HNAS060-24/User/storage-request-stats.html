storage-request-stats

Show storage request stats

Syntax

storage-request-stats [-r]

Description

This command is used to display or reset statistics on I/O requests processed by the HNAS storage interface layer fast path.

Options

-r, --reset

reset statistics

Description of output

Total Requests & Responses

These figures relate to requests issued to the storage interface layer from higher-level code - mainly the file system layer.

I/O Requests

This section gives the numbers of fast-path SCSI I/O commands sent from the server to the storage. A response will not be counted for a command that times out or whose reponse has to be handled on the slow path. This will therefore create a discrepancy between the request and response counts that persists even when there are no requests in flight.

Cache

Hit and miss statistics for the sector cache. These relate to 32K cache buffers, but a hit is only recorded if all the blocks needed for a request are present in the buffer.

I/O Status Counters

The number of requests to the storage (fast-path only) that have been failed, after exhausting all retries; and the number of retries. For more detailed information on the types of errors occurring, see scsi-io-errors.
