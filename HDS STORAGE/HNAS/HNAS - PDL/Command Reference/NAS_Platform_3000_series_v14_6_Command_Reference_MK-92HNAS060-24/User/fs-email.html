fs-email

File system contact list management (outside a virtual volume)

Syntax

fs-email <get|set|delete>
	fs-email get <file-system>
	fs-email set <file-system> <email>
	fs-email delete <file-system>

Description

File system contact list management (outside a virtual volume)

Usage:

fs-email get <file-system>

Get file system's email contact list

fs-email set <file-system> <email>

Set file system's email contact list

<email> is a list of contacts for the file system.

  • Addresses with a leading '*' can be specified to email users directly about events affecting them ('*' is replaced with the user's NFS/NT account name).
  • Separate multiple addresses with ';' and enclose entire list in quotes.
fs-email delete <file-system>

Remove file system's email contact list

Examples

Send emails for events on fs0 to admin@example.com and the affected user: e.g. if the user 'joe' exceeds his quota, an email will be sent to admin@example.com and joe@example.com

fs-email set fs0 "admin@example.com; *@example.com"

Applies To

EVS
