rest-server-status

Retrieves the status of the RestAPI server.

Syntax

rest-server-status [--json]

Description

Retrieves the current status of the RestAPI server.

For further information about the RestAPI server, use 'man rest-server'.

rest-server-status [OPTIONS]

OPTIONS

--json

    Print the status report in the JSON format. This format is for internal testing purposes.

Output details

The 'rest-server-status' command shows the following information:

Status

If the RestAPI server is running or not.

Supported API Versions

A list of the supported API versions that the server will respond to.

Mode

Which mode the RestAPI server is running in.

Examples

  server:$ rest-server-status


  RestAPI server is running
   supported API versions:
    07.2.1
    08.0.1


  RestAPI server current mode is native.

Applies To

Cluster wide
