cluster-node-route-status

Show the cluster-node-route configuration status

Syntax

cluster-node-route-status

Description

The cluster-node-route-status command is used to display the cluster-node-route operating status.

The cluster-node-route-enable and cluster-node-route-disable commands are used to enter or leave cluster-node-route mode. See the cluster-node-route manpage for more details.

Note: It is advised that RIP and IRDP are disabled when cluster-node-route functionality is in effect.

Examples

To display the cluster-node-route operating status:

$ cluster-node-route-status
Using cluster-node routing

Applies To

Cluster wide

See Also

cluster-node-route cluster-node-route-disable cluster-node-route-enable cluster-node-route-list

Privilege Level

User
