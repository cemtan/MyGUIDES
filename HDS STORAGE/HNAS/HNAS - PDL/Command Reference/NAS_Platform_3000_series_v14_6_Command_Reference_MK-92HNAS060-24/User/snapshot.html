snapshot

Snapshot management (deprecated, please see man page)

Syntax

snapshot <operation ls|mk|mv|rm|rmall|rule-ls|rule-mk|rule-mod|rule-rm|recover-fs>
	ls         [--all] [--freeable-space] (--file-system <file-system> [<snapshot-name>]) | (--rule <rule-name> [--file-system <file-system>])
	mk         (--file-system <file-system> <snapshot-name>) | (--rule <rule-name> [--file-system <file-system>])
	mv         --file-system <file-system> <current-snapshot-name> <new-snapshot-name>
	rm         (--file-system <file-system> <snapshot-name>) | (--rule <rule-name> [--file-system <file-system>]) | (--file-system <file-system> --checkpoint <checkpoint-number>)
	rmall      --file-system <file-system> [--confirm]
	rule-ls    [<rule-name>]
	rule-mk    --file-system <file-system> [--queue-size <size>] <rule-name>
	rule-mod   [--file-system <file-system>] [--queue-size <size>] [--new-rule-name <new-rule-name>] <rule-name>
	rule-rm    <--no-file-system | --file-system <file-system>> [--confirm] <rule-name>
	recover-fs [--undo] --file-system <file-system> [--confirm] [<snapshot-name>]

Description

This command is deprecated. See the man pages for the preferred commands listed below.

NOTE: Some sub-commands require supervisor privilege level.
