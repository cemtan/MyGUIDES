iscsi-target

Manages iSCSI targets

Syntax

iscsi-target operation
Where operation is:
list [<alias>]
add [-c <comment>] [-m <access-configuration>] <alias> <secret>
addlu <alias> <lu> <lun>
del [-f] [<alias> | -g <globally unique name>]
delall [-c]
delalllu [-c] <alias>
dellu [-f] <alias> <lun>
mod [-n <alias>] [-g] [-c <comment>] [-s <secret>] [-a enable|disable] [-m <access-configuration>] <alias>

Description

This command manages iSCSI targets. A target is the destination to which an iSCSI initiator connects.

Usage:

iscsi-target list [<alias>]

Lists one/all configured targets.

alias

The alias (name) of the target to list.

iscsi-target add [-c <comment>] [-m <access-configuration>] <alias> <secret>

Adds a new target.

--comment, -c

Set comment.

--access-configuration, -m

Set the client access restrictions for the target. The default is '*(rw)', meaning that all clients have read and write access to the target.

See mount-point-access-configuration for more details.

alias

The alias (name) of the target to create.

secret

The secret that an initiator must provide when logging in to this target.

iscsi-target del [-f] [<alias> | -g <globally unique name>]

Deletes a target.

--force, -f

Force deletion of target even if in use.

alias

The alias (name) of the target to delete.

globally unique name

The globally unique name of the target to delete.

iscsi-target delall [-c]

Deletes all targets.

--confirm, -c

Don't ask for confirmation.

iscsi-target mod [-n <alias>] [-g] [-c <comment>] [-s <secret>] [-a enable|disable] [-m <access-configuration>] <alias>

Modifies a target.

--new-alias, -n

Update the alias.

--regenerate-gun, -g

Regenerates the globally unique name.

--comment, -c

Update comment.

--secret, -s

Update shared secret.

--authentication, -a <enable/disable>

Enable/disable authentication.

--access-configuration, -m

Update the client access restrictions for the target. The default is '*(rw)', meaning that all clients have read and write access to the target.

See mount-point-access-configuration for more details.

alias

The alias (name) of the target to modify.

iscsi-target addlu <alias> <logical unit> <lun>

Adds a logical unit to the target, enabling initiators to access the logical unit via the logical unit number (LUN) specified.

alias

The alias (name) of the target.

logical unit

The alias (name) of the logical unit to add.

lun

The LUN to assign to the logical unit (0-255).

iscsi-target dellu [-f] <alias> <lun>

Removes a logical unit from a target, making the specified LUN unavailable to initiators.

--force, -f

Force deletion of logical unit even if in use.

alias

The alias (name) of the target.

lun

The LUN to remove.

iscsi-target delalllu [-c] <alias>

Removes all logical units from a target.

--confirm, -c

Don't ask for confirmation.

alias

The alias (name) of the target.

Examples

iscsi-target add -c "My target" t1 secretsecret1
iscsi-target del t1
iscsi-target delall -c
iscsi-target mod -s secretsecret2 t1
iscsi-target mod -a disable t1
iscsi-target addlu t1 lu1 0
iscsi-target dellu -f t1 0
iscsi-target delalllu -c t1

Applies To

EVS
