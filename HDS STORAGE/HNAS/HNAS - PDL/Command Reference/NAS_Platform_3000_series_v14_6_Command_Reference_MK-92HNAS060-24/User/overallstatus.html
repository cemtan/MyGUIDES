overallstatus

Report overall system status

Syntax

overallstatus

Description

Displays the status of all the cluster nodes, the quorum device (if this is a cluster) and any storage controllers known to the server. There are three levels of status: OK, Warning and Critical.

If you want more information on the status of an individual item in the overall status type 'status -a' for more information about the cluster nodes and 'cluster-show -q' for more information about the quorum device.

Examples

overallstatus

See Also

cluster-show status trouble
