tail

Show just the last part of output

Syntax

... | tail [NUM]

Description

Command output can be piped through tail to display just the last few lines of the output. 'tail' may not be used as a command.

It takes an optional unsigned numeric parameter that specifies the number of lines to output. The default is 10.

If a command has a built-in way to limit the amount of output it produces, using that facility is likely to be more efficient than piping the command's output through 'tail'.

Examples

To show the fifty lines most recently added to the dblog:

dblog | tail 50
