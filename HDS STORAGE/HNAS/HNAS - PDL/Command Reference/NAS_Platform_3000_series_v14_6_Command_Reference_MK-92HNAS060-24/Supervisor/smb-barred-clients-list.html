smb-barred-clients-list

display the SMB barred clients list

Syntax

smb-barred-clients-list

Description

Displays the list of client addresses that are barred from SMB access.

Examples

smb-barred-clients-list
192.168.0.1
fdc7:682a:c330::1

Applies To

EVS security context

See Also

Privilege Level

Supervisor
