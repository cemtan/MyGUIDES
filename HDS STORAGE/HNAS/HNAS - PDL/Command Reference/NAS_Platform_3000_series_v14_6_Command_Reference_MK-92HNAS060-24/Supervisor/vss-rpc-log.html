vss-rpc-log

Display or clear the log of VSS RPCs received by this pnode

Syntax

vss-rpc-log  ( list | clear )

Description

This command is used to support debug of VSS hardware provider operation. If you are not using VSS, ignore this command.

Each cluster node maintains a log of recent VSS management requests submitted by external VSS hardware provider DLLs. This command permits you to list the contents of the log, and to clear it.

Examples

List most recent 20 entries:

vss-rpc-log list 20

List all entries:

vss-rpc-log list

Clear the log of VSS RPC requests:

vss-rpc-log clear

Applies To

Cluster node
