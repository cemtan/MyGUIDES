ndp-dump

dumps ndp

Syntax

ndp-dump

Description

The ndp-dump command displays internal NDP tables maintained in accordance with RFC4861. There are three tables that we maintain:

Neighbors

The neighbor table provides information on what MAC address we'll use when sending to an on-link IPv6 address.

Routers

The router table provides the IPv6 address of routers that have been discovered.

Prefixes

The prefix table provides information about what prefixes are on link.

Per Link Data

The per link data table provides information about the value of the retransmission timer, valid lifetime, preferred lifetime and hop limit for each IPv6 capable interface.

Both the discovered routers and prefixes are propagated to the routing module.

Examples

To display the internal NDP tables:

server:$ ndp-dump
neighbors:

eth1: fe80::224:14ff:feb7:cc13 resolves to 00-24-14-b7-cc-13 reachability: Stale

routers:

eth1: address: fe80::224:14ff:feb7:cc13 mtu: 1500 lifetime: 1800s
ag1: address: fe80::224:14ff:feb7:cc13 mtu: 1500 lifetime: 1800s
prefixes:

eth1: prefix: fdca:f995:220a:480::/64 valid lifetime: 2592000s preferred lifetime: 604800s
ag1: prefix: fdca:f995:220a:480::/64 valid lifetime: 2592000s preferred lifetime: 604800s
