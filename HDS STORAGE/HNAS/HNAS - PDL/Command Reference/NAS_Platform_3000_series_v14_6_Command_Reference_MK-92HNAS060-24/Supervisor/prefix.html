prefix

Runs a command prefixing each line of output with the supplied string

Syntax

prefix <prefix> <command>

Description

Runs a command, prefixing a given string to the command's output. Errors are not prefixed.

ssrun -c can be useful when trying to run complicated commands.

Examples

To list the contents of a directory with each line of output containing a prefix:

server:$ prefix "[contents of home directory] " ls /home/fred
[contents of home directory] foo
[contents of home directory] bar
[contents of home directory] baz
[contents of home directory] qux

Applies To

Cluster node

See Also
