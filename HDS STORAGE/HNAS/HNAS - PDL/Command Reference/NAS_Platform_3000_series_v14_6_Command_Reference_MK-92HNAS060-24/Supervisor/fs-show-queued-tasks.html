fs-show-queued-tasks

display all queued tasks

Syntax

fs-show-queued-tasks

Description

The fs-show-queued-tasks command dumps the names of all the currently scheduled file system tasks for all file systems to the console. Tasks that are currently running are not shown.

Note: fs-show-queued-tasks may be run with the file system mounted or unmounted.

Applies To

EVS

See Also

checkfs fs-cancel-queued-task

Privilege Level

Supervisor
