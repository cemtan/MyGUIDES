show-stats

Shows statistics whose names contain the optional substring

Syntax

show-stats [<substring>]

Description

Dumps the current values of low-level system statistics. The statistics themselves are not documented, and vary between releases.

Examples

To see the current values of all statistics on the current board:

show-stats

To see the current values of statistics that case insensitively match the string "FS1"

  show-stats FS1

Privilege Level

Supervisor
