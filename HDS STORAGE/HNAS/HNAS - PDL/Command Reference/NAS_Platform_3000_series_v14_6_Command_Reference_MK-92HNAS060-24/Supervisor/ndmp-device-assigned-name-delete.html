ndmp-device-assigned-name-delete

Delete a name assigned to an NDMP device

Syntax

ndmp-device-assigned-name-delete <name>

Description

The command removes a device name that has previously been added using ndmp-device-assigned-name-set. The device name will revert to the name automatically assigned by the system.

See the ndmp-devices man page for a discussion of NDMP device names.

Examples

ndmp-device-assigned-name-delete tape1

Applies To

Cluster wide

See Also

ndmp-device-assigned-name-list ndmp-device-assigned-name-set ndmp-devices ndmp-devices-list

Privilege Level

Supervisor
