ndmp-device-trace-add

Start per-device tracing for the specified NDMP subsystem

Syntax

ndmp-device-trace-add <tape-req|scsi-req|tape-proc|tape-wr|tape-rd|stats|tape-error|state|position>

Description

Adds the specified NDMP device subsystem to the output of the NDMP device trace.

Not all NDMP device subsystems are traced by default because there is a performance penalty in doing so for certain subsystems.

For this reason, it is advisable to consult your Support Organisation before using this command to add extra tracing.

Examples

Add tracing for the NDMP device SCSI request subsystem:

ndmp-device-trace-add scsi-req

Applies To

Cluster node

See Also

ndmp-device-trace ndmp-device-trace-remove

Privilege Level

Supervisor
