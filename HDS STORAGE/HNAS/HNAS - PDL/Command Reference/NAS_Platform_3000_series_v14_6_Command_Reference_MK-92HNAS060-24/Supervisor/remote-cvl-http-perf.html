remote-cvl-http-perf

Display remote CVL HTTP performance statistics

Syntax

remote-cvl-http-perf [--reset] [--graph] [<server name>]

Description

The remote-cvl-http-perf command displays performance statistics about remote cross volume link HTTP activity. Performance statistics can be displayed as ASCII graphs, showing the round trip response times for each CVL HTTP operation.

Options:

-r, --reset

Reset performance statistics for all remote HTTP servers, or just for the specified one.

-g, --graph

Display statistics graphically.

Examples

Display the remote CVL HTTP performance statistics graphically for remote server weasel:

remote-cvl-http-perf --graph weasel

Reset the remote CVL HTTP performance statistics for remote server dobson:

remote-cvl-http-perf -r dobson

Applies To

Cluster node
