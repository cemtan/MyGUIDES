namespace-mv

Renames an object within a cluster or EVS namespace

Syntax

namespace-mv <cluster or EVS namespace> <from-path> <to-path>
Alias:       cns-mv

Description

Renames a link or directory within the specified cluster or EVS namespace. If the path specifies a link then only the link will be removed from the file system; the file system itself will not be modified.

This command cannot be used to move objects within a file system. It is purely for renaming objects created using the namespace-mklink and namespace-mkdir commands.

Usage:

namespace-mv <cluster or EVS namespace> <from-path> <to-path>

Applies To

Cluster wide

See Also

namespace-ls namespace-mkdir namespace-mklink namespace-rm

Privilege Level

Supervisor
