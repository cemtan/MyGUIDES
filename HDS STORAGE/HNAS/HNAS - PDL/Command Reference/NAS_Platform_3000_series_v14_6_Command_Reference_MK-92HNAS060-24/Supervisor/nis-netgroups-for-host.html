nis-netgroups-for-host

test nisclient::api::GetNetgroupsForHost

Syntax

nis-netgroups-for-host <host>
Alias:       GetNisNetgroupsForHost

Description

The nis-netgroups-for-host command is used to determine the set of netgroups to which a host belongs.

This command works only in classic NIS mode, as opposed to NIS/LDAP mode, and only when using the NIS netgroup.byhost map. It does not check local netgroups, only NIS netgroups, and makes use of the NIS cache.

Examples

To find the netgroups to which a host belongs:

    nis-netgroups-for-host 172.15.14.100


    nis-netgroups-for-host fdca:f995:220a:0:58c1:e76e:d2c4:9f1c
