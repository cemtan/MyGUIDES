fc-network-monitor-dump

dump the internal state of the network monitor

Syntax

fc-network-monitor-dump [hport id]

Description

The fc-network-monitor-dump command allows you to query the internal state of the fibre channel network monitor. It will report the number of errors that it has recorded for all visible connections at the last poll. Command options are:

[hport id]

If specified, restricts the output to the specified host port. By default all statistics are dumped for all (enabled) host ports.

Examples

Report the internal state of the fibre channel network monitor as seen from host port 1:

fc-network-monitor-dump 1

Report the internal state of the fibre channel network monitor as seen from all host ports:

fc-network-monitor-dump
