vlsi-profiler-threshold-set

Set the minimum busy percentage for a state machine in order for it to be profiled.

Syntax

vlsi-profiler-threshold-set <profile threshold %>

Description

This command is used to specify the minimum percentage of busyness a state machine must reach before the vlsi-profiler profiles it. It is not possible to specify the percentage for individual state machines, the value applies to all.

The default value is 75%.

Note that the percentage only applies to that node, and is not persistent between resets.

Examples

Set the profile threshold to 80%:

vlsi-profiler-threshold-set 80

Applies To

Cluster node
