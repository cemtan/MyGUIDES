replication-target-revert-from-read-write-fs

Reverts a replication target file system to only be modifiable by object replication

Syntax

replication-target-revert-from-read-write-fs <file-system>

Description

This command prevents the specified file system from being modified by anything other than the object replication engine. The specified file system must be unmounted.

It is valid to use this command on any file system. However it is unlikely that subsequent object replications to the file system will be successful unless it was originally a replication target, and the live version of the file system has been rolled back to a valid replica of the object replication source file system.

Examples

To enable a file system to be modifiable by the object replication engine only:

  replication-target-revert-from-read-write-fs fs1

Applies To

EVS
