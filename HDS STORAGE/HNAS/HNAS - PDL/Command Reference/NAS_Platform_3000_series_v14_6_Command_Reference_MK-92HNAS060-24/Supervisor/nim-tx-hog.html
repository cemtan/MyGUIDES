nim-tx-hog

Lists the number of NIM TX buffers consumed by each connection

Syntax

nim-tx-hog [<maximum line count>]

Description

This command is used to diagnose some performance problems. It shows how busy the connections to network clients are by listing how much data is waiting to be sent.

Using this command may momentarily reduce the performance of the server as it gathers the results.

Only use under instruction from your support provider.

The following options are available:

<maximum line count>

If given, limit the number of lines output.

Applies To

Cluster node

Privilege Level

Supervisor
