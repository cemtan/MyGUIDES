fs-stat

Report file system state/statistics

Syntax

fs-stat [--file-system <fs-label>] [-r|--reset]

Description

Displays information about the current/specified file system.

Usage:

fs-stat [--file-system <fs-label>] [--reset]

-f, --file-system

Selects a file system

-r, --reset

Resets statistics
