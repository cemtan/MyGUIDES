namespace-rm

Removes an object within a cluster or EVS namespace

Syntax

namespace-rm [-r] <cluster or EVS namespace> <path>
Alias:       cns-rm

Description

Removes a path from within the specified cluster or EVS namespace. If the path specifies a directory then it must be empty. If the path specifies a link then only the link will be removed from the file system, the file system itself will not be modified.

This command cannot be used to remove objects within a file system. It is purely for removing objects created using the namespace-mklink and namespace-mkdir commands.

Usage:

namespace-rm [--recursive] <cluster or EVS namespace> <path>

-r, --recursive

Remove the contents of directories recursively.

Applies To

Cluster wide

See Also

namespace-ls namespace-mkdir namespace-mklink

Privilege Level

Supervisor
