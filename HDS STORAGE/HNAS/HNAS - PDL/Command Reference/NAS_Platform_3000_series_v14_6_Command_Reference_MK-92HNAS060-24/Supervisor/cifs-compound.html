cifs-compound

Display SMB compound command combinations

Syntax

cifs-compound [--reset|-r] [--version|-v (all|2|2.1|3)]

Description

The cifs-compound command displays the SMB compound command combinations that have been received by the server. It provides information that may be useful when analyzing the workload of SMB2 and later.

Usage:

--reset, -r

Clears the log.

-v, --version <version>

Shows the statistics for the particular SMB version.

The supported version options are 2, 2.1, 3 and the default value all.

Applies To

Cluster node

See Also

cifs-errors cifs-stats

Privilege Level

Supervisor
