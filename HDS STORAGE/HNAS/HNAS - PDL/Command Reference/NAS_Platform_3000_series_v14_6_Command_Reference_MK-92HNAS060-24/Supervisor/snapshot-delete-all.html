snapshot-delete-all

Delete all snapshots on a file system.

Syntax

snapshot-delete-all --file-system <file-system> [--confirm]

Description

This command deletes all snapshots of a specified file system.

--file-system, -f <file-system>

Select file system by label.

--confirm, -C

Suppress interactive confirmation.

Note: Deleting a snapshot does not immediately free the disk space which it occupies - the space is returned over time by a background task. To recover space quickly, refer to fs-recover-space man page.

Note: Certain types of snapshot, if they are required by the HNAS for correct operation, may be protected from deletion. In this case, a message will be displayed indicating why a snapshot cannot be deleted.

Examples

server:$ snapshot-delete-all --file-system fs1 --confirm
Information: Snapshot marked for deletion ("2007-09-08_1700+0000.hourly" on fs1)
Information: Snapshot marked for deletion ("test" on fs1)
Information: Snapshot marked for deletion ("2007-09-08_1800+0000.hourly" on fs1)
