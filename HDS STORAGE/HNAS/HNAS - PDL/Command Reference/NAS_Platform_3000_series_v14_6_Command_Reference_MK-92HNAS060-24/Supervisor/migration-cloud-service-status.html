migration-cloud-service-status

Shows the current status of Data Migrator to Cloud processes

Syntax

migration-cloud-service-status [--verbose] [--memory] [--pnode <pnode>]

Description

Shows the status of Data Migrator to Cloud processes for all physical nodes.

--verbose, -v

List all the processes related to Data Migrator to Cloud that currently run.

--memory, -m

List the maximum seen memory consumption used by the cloud processes.

--pnode, -n <nodeid>

List all related to Data Migrator to Cloud processes that run on physical node 'nodeid'.

Examples

migration-cloud-service-status --memory --verbose

Shows the detailed status of Data Migrator to Cloud processes including maximum reached memory usage.

Applies To

Cluster wide

See Also
