fs-quarantine-release

Release one or more objects from file system quarantine

Syntax

fs-quarantine-release --all | -f <file-system> [-o object-number]

Description

The fs-quarantine-release command is used to remove one or more objects from quarantine. An individual object can be removed; all the objects for a given file system may be removed; or quarantine itself can be completely emptied. If the target file system is mounted, then the removal will take immediate effect, else the object(s) will not be quarantined when the file system is next mounted.

It is a syntax error to invoke this command without supplying either the '--all' option or a file system id.

Examples

To remove everything from quarantine on this node:

fs-quarantine-release --all

To remove all objects everything from quarantine for file system 'cake':

fs-quarantine-release -f cake

To remove the object with object-number 0x30000 on file system 'biscuit' from quarantine:

fs-quarantine-release -f biscuit -o 0x30000

Applies To

EVS
