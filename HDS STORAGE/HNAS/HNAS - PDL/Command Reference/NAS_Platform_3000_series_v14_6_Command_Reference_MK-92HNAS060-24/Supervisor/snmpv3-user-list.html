snmpv3-user-list

List all SNMPv3 users

Syntax

snmpv3-user-list

Description

The snmpv3-user-list command outputs the names of all known SNMPv3 users that have been added with the snmpv3-user-add command into a table. These are the users that can be used to authenticate SNMPv3 requests when the agent's protocol is set to SNMPv3 (man snmp-protocol).

This command cannot be used to retrieve passwords. If a user forgets their password they should delete (man snmpv3-user-delete) and then re-add (man snmpv3-user-add) the SNMPv3 user.

Examples

List all SNMPv3 users:

snmpv3-user-list


Users
-----
user1
user2

Applies To

Cluster wide
