xfps

Dump data from XFPs (deprecated)

Syntax

xfps [--diags | --data] [--port-family (net|cluster|fc)] [--port-number <port-number>] [--port-name <port-name>] [--type (xfp|sfp)] [--hex]

Description

Note - this command is deprecated in favour of pluggable-transceiver-dump.

Dumps information on all XFPs on the current board, from the devices' EEPROM, including diagnostics.

This command is only valid on boards that have XFPs.

Examples

xfps

See Also

pluggable-transceiver-dump

Privilege Level

Supervisor
