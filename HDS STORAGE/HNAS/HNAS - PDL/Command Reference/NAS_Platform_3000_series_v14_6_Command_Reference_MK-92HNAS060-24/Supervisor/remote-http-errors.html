remote-http-errors

Report remote HTTP protocol errors

Syntax

remote-http-errors [-r|--reset]

Description

The remote-http-errors command is used to display errors returned from remote HTTP servers.

Usage:

--reset, -r

Resets the error log.

Applies To

Cluster node
