wlog-stats-control

Control wlogm1 statistics

Syntax

wlog-stats-control [-r|--reset] [--enable [int32|int128|ext]|--disable [int32|int128|ext]]

Description

This command controls the VLSI statistics gathering.

When no options are supplied the command displays the current configuration settings.

Only use under instruction from your support provider.

-r, --reset

Reset the statistics data items.

Resetting the statistics while a performance information report (PIR) is being generated may result in misleading results.

--enable [int32|int128|ext]

Enable the statistics collection.

--disable [int32|int128|ext]

Disable the statistics collection.

Applies To

Cluster node
