ls

Lists directory entries

Syntax

ls [-a] [-d] [--full-time] [-h] [-l] [-r] [-i] [-s <sort option>] [--show-cookies] [--show-handles] [--show-name-flags] [--show-shortnames] [--utc] [-C <resume cookie>] [<path>]

Description

The ls command is used to list files/directories. By default it lists the current directory, as set with the cd command. However, you can also provide it with the path of a directory to list.

Usage:

-a, --all

Do not hide entries starting with the '.' character

-C, --resume

Resume on the entry with the specified cookie

-d, --directory

Show the directory itself, instead of its contents

--full-time

Show nanosecond resolution in file times

-h, --human-readable

Show human-readable file lengths

-i, --case-insensitive-sort

Sort names case insensitively

-l, --long

Use a long listing format. The second column in the listing represents file type. Supported file types are C=file-clone, X=CVL/XVL, I=ingested and -=regular/others.

-r, --reverse-sort

Reverse the sort direction

-s, --sort <sort option>

Sort the list. Options are none, name, time, atime and size. The default is to sort by name.

--show-cookies

Show cookies

--show-handles

Show file handles

--show-name-flags

Show name flags

--show-shortnames

Show DOS short file names

--utc

Show file times in UTC (Universal Time Coordinated)

-v, --verbose

Verbose listing format
