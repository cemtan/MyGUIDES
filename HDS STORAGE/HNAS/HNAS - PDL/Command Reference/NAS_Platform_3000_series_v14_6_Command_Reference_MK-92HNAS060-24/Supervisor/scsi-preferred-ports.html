scsi-preferred-ports

Display preferred pathing information.

Syntax

scsi-preferred-ports [-d device|-p port|-a all]

Description

Display preferred path information for SCSI ports connected to the SCSI devices that the server has discovered. The information displayed includes the following: a list of online/offline ports; which port is active, ie. the port that is currently open and used by the server to access the device; ports that are on the preferred processor (if applicable); ports that are on the preferred controller; ports that are not preferred; the granularity of the available best path and a column (labelled "best") which shows if the best possible preferred path is being used. The "Best" column will not be output if no SD supports preferred path. If none of the SCSI devices that have been discovered support preferred path, the command will show a message to indicate this and not display any further information. The -a flag can be used to override this.

Options:

-d, --device <id>

Display preferred path information for the given device ID.

-p, --port <id>

Display preferred path information for the given SCSI port ID. Displays the devices connected to it.

-a, --all

Display preferred path information for all devices including those that do not support preferred path.
