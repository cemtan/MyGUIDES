ldap-security

Display / update LDAP security settings

Syntax

ldap-security -u username | -w password | -t (on | off)

Description

The ldap-security command allows you to configure NIS-over-LDAP security settings.

With no arguments, the ldap-security command displays current NIS-over-LDAP security settings. Or you may specify any combination of the following options:

-u, --username username

Set DN (distinguished name) used to bind to LDAP server(s). For anonymous binds, leave this blank (or set to '').

-w, --password password

Set password used to authenticate as DN when binding (see -u).

-t, --tlsenable on | off

Enable or disable use of TLS security for LDAP.

In order to successfully connect to and query an LDAP server it is also necessary to configure search bases. This is done using the domainname and/or ldap-search-config commands.

Examples

Display the current NIS-over-LDAP security settings:

ldap-security

Add or change LDAP bind credentials:

ldap-security -u 'uid=testadmin,ou=Special Users' -w secret

Enable LDAP use of TLS:

ldap-security -t on

Applies To

EVS security context
