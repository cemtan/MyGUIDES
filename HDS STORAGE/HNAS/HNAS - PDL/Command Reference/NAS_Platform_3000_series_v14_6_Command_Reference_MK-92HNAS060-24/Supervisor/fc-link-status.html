fc-link-status

Show FC link status and config

Syntax

fc-link-status -i [1|...|N]
Alias:       fclinkstatus

Description

This command provides information about the server's Fibre Channel links. It displays each one's current speed, its port type, whether it is enabled, and the current status of the link.

The command takes a single, optional argument:

-i

Specify the FC interface to display. If this option is not provided the command displays information about all the available interfaces. Interfaces are numbered from 1.

Possible statuses are:

Up

Down

(including when because disabled)

DOWN

Down since boot, for more than 1 minute

Isolated

(Arbitrated Loop only) The link is up, but has recently been unstable. The server has isolated itself from the loop to avoid unnecessary disruption. It will automatically attempt to rejoin the loop once the link has been stably up for 30 s, or when it comes up after being down continuously for at least 5 minutes. To force reparticipation sooner, run "fc-link enable".

ISOLATED

(Ditto) The link has been persistently unstable over an extended period. It will remain isolated until cleared manually, as above.

Examples

server:$ fc-link-status
FC 1:  4 Gbps  N-port  Enabled   Up
FC 2:  4 Gbps  N-port  Enabled   Up
FC 3:  4 Gbps  N-port  Enabled   DOWN
FC 4:  4 Gbps  N-port  Enabled   Down


server:$ fc-link-status -i 1
FC 1:  4 Gbps  N-port  Enabled   Up

Applies To

Cluster node
