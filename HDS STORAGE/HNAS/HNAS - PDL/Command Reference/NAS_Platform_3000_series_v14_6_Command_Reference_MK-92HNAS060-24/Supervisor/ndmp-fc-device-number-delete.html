ndmp-fc-device-number-delete

Delete assignment of a number to a Fibre Channel Device used by NDMP

Syntax

ndmp-fc-device-number-delete <number>

Description

Delete the mapping between the node name of a Fibre Channel Device and its Device Number used in NDMP device names. This will only be possible if the relevant device has been removed from the Fibre Channel and there are no active devices referring to it.

We recommend that you do not use this command if the device will be re-attached later. If the mapping has not been deleted then the device will retain its NDMP device name when re-attached.

The ndmp-devices man page explains how the device names are created.

Examples

Delete the Device Number associated with old tape library 1. A newly attached tape library may now be assigned this number.

ndmp-fc-device-number-delete 1

Applies To

Cluster wide
