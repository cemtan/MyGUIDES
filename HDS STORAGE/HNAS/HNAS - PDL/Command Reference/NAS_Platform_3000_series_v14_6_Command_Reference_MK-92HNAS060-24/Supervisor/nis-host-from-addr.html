nis-host-from-addr

test nisclient::api::HostFromAddr

Syntax

nis-host-from-addr address
Alias:       NisHostFromAddr

Description

The nis-host-from-addr and nis-host-from-name commands are used to find a host record given its name or its address

The address can be an IPv4 address or an IPv6 address.

Examples

To find a hostname given an IP address:

    nis-host-from-addr 10.1.1.7


    nis-host-from-addr fdca:f995:220a:0:58c1:e76e:d2c4:9f1c

To find a host given its name:

    nis-host-from-name test

Applies To

EVS
