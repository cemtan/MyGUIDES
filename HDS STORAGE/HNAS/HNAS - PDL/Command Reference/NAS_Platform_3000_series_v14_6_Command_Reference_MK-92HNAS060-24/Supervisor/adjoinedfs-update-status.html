adjoinedfs-update-status

While adjoined filesystems are deprecated, the command updates the status of a given one.

Syntax

adjoinedfs-update-status -f <fs label>

Description

Adjoined filesystems are deprecated and administrators are encouraged to transition away from them.

Options:

-f, --fs <file system label>

Main file system through which the adjoined file system is accessed.

Update the status of the secondary file systems adjoined to the specified file system. The software will automatically update the status when files are placed on the adjoined file system. However, it will not notice automatically when the last link to the file systems is deleted. This command will mark the file system as not in use if no files are found there.

Examples

adjoinedfs-update-status --fs fs1

Update status of secondary files systems of the adjoined file system fs1.

Applies To

EVS
