pir-cancel

Cancel a performance info report (PIR)

Syntax

pir-cancel

Description

Cancels a currently running PIR. The server does not allow any attempts to start a new PIR while an existing one is running.

Note that this command will not work if the PIR running is a Continuous PIR (i.e. it was started with "pir --start"). The correct way to stop a Continuous PIR is with "pir --stop".

Applies To

Cluster node
