alert-summary

Send alert summary now

Syntax

alert-summary

Description

The alert-summary command sends an alert summary now, rather than waiting for the next scheduled summary time. It takes no arguments.

This command only sends emails to recipients in profiles which are configured to receive summary emails.

Examples

To send a summary email now

alert-summary

Applies To

Cluster node
