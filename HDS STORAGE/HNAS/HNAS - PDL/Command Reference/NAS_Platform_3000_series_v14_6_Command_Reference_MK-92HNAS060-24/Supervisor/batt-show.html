batt-show

Display battery information

Syntax

batt-show

Description

Displays the status of the chassis battery.

This includes fitted state, monitor state (eg. normal, charging, conditioning, testing), voltage, and current. A negative current indicates the battery is charging, a positive current indicates it is discharging.

Examples

batt-show

Displays a limited set of battery attributes, eg.

Battery is fitted, Normal, 7.97V, -0.03A

Privilege Level

Supervisor
