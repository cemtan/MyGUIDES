group-mappings-delete

Deletes an existing group mapping

Syntax

group-mappings-delete --unix-name <name> | --unix-id <id> | --nt-name <name> | --nt-id <id> |
 --nfsv4-name <name>

Description

Deletes an entry from the group mappings table.

Options:

--unix-name

Lookup the entry to delete by Unix name

--unix-id

Lookup the entry to delete by Unix group id

--nt-name

Lookup the entry to delete by NT name

--nt-id

Lookup the entry to delete by NT SID

--nfsv4-name

Lookup the entry to delete by NFSv4 name

Only specify one field to lookup the entry. If the entry exists, it will be deleted. An error will be printed if the entry cannot be found.

Examples

Delete Unix group badgers:

$: group-mappings-delete --unix-name badgers

Applies To

EVS security context

See Also

group-mappings-add group-mappings-list group-mappings-modify group-mappings-refresh
