virtualization-path-journal-show

Displays journal of the deleted files.

Syntax

virtualization-path-journal-show -t <fs label> -n <association name> [--start-date <YYYY-MM-DD>] [--end-date <YYYY-MM-DD>]

Description

Displays journal of files deleted from remote server for a virtualization path association.

Options:

-t, --target <fs label>

Target file system.

-n, --name <association name>

Logical name of the association.

-s, --start-date <YYYY-MM-DD>

First log date to display (optional). If not provided, defaults to first log file existing.

-f, --end-date <YYYY-MM-DD>

Last log date to display (optional). If not provided, defaults to current date.

Examples

virtualization-path-journal-show -t fs1 -n assoc1 --start-date 2012-09-01 --end-date 2012-09-30

Displays log of all deleted files in September for the virtualization path for file system fs1.
