chown

Change the owner associated with a file

Syntax

chown owner <file1> [<file2>...]

Description

The chown command is used to change ownership of a file.

Usage:

owner

Specify the unix id or unix name of the new owner of the file(s).

Examples

To change ownership of file 'bar' to user 'foo':

chown foo bar

EVS
