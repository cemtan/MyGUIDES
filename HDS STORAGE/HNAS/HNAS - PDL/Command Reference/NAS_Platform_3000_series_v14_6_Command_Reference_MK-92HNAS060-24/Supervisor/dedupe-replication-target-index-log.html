dedupe-replication-target-index-log

Show or reset the log of dedupe index access on replication target file systems.

Syntax

dedupe-replication-target-index-log [--reset]

Description

The dedupe-replication-target-index-log command shows or resets the log of dedupe index access on replication target file systems.

This is a diagnostic command, intended for use by your service organization.

Usage:

  dedupe-replication-target-index-log [-r]

Options:

-r, --reset

Reset the log.

Examples

  dedupe-replication-target-index-log

Shows the log.

  dedupe-replication-target-index-log -r

Resets the log.

Applies To

Cluster node.

Privilege Level

Supervisor
