lstat

Displays file information, doesn't follow symlinks

Syntax

lstat <file1> [<file2>...]

Description

The lstat command is used to display information about files, directories or symlinks. Although it is similar to the stat command, the lstat doesn't follow symlinks, so it can be used to display information about the links themselves.

Examples

To display information about the symlinks 'link1', 'link2' and 'link3':

lstat link1 link2 link3
