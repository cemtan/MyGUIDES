dblog

View console output

Syntax

dblog [last|current] [sincemarker] [all]

Description

The server maintains a set of logs containing extra information over and above the event log. These are accessible using this command.

Options

last

Shows the logs from the previous server boot.

current

Shows the logs from this server boot (default).

sincemarker

Shows the logs since the last time this command was run with this option. This only works on the current logs, and will not function correctly if multiple processes are running this.

Examples

To print the log from the last session:

dblog last

To print the current log since the last time this command was run:

dblog sincemarker

To set up a cron job to mail daily additions to this log:

crontab add -m user@example.com @daily "dblog sincemarker"

Applies To

Cluster node

Privilege Level

Supervisor
