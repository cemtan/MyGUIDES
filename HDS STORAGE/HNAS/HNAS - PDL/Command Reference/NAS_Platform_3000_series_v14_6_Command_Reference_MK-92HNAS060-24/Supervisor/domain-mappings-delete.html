domain-mappings-delete

Deletes an entry from the domain mapping table

Syntax

domain-mappings-delete --map-unix-names | --nt <nt domain> | --nfsv4 <NFSv4 domain> | --kerberos <kerberos realm>

Description

Deletes an entry from the domain mappings table.

Options:

--map-unix-names

Delete the entry that is set to map to/from Unix names

--nt

Delete the entry that is set to map to/from the NT domain

--nfsv4

Delete the entry that is set to map to/from the NFSv4 domain

--kerberos

Delete the entry that is set to map to/from the Kerberos realm

Examples

$: domain-mappings-delete --nt EXAMPLE

Applies To

EVS security context

See Also

domain-mappings-add domain-mappings-list domain-mappings-replace domain-mappings-use-default

Privilege Level

Supervisor
