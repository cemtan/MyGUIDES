replication-xvl-as-links

Checks the status of or sets a file system to allow object replication to replicate XVLs as links

Syntax

replication-xvl-as-links <file-system> [--set]

Description

This command checks or sets the specified file system so that XVLs are replicated by object replication as links instead of being rehydrated on the target file system.

--set

Set all subsequent object replication sessions from the specified file system to replicate XVLs as links. This flag, once set, cannot be unset without reformatting the file system.

Examples

To check whether the file system will replicate XVLs, with object replication, as links:

$ replication-xvl-as-links fs1
XVLs on this file system will not be replicated by object replication as links.

To set the file system to replicate XVLs as links in object replication:

$ replication-xvl-as-links fs1 --set
XVLs on this file system will be replicated by object replication as links.

Applies To

EVS
