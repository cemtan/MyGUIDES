for-each-cnode

Run a command in the context of each configured cnode

Syntax

for-each-cnode [--quiet] <command> [args]

Description

The for-each-cnode command runs the specified command once for each cnode, setting the console context appropriately first.

Options:

--quiet|-q

Don't display each cnode id as commands are run

ssrun -c can be useful when trying to run complicated commands.

Examples

To show the state of link aggregation ag1 on each cluster node:

for-each-cnode ifconfig ag1

Applies To

Cluster wide

See Also

console-context ssrun

Privilege Level

Supervisor
