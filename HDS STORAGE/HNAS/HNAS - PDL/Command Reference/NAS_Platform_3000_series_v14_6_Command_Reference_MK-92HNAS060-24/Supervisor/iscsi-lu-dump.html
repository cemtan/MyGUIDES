iscsi-lu-dump

Show iSCSI LU diagnostic information

Syntax

iscsi-lu-dump

Description

The iscsi-lu-dump command is a diagnostic command used to display the internal configuration and state of iSCSI logical units. This information can be useful when debugging iSCSI connectivity problems or initiator failures.

Examples

iscsi-lu-dump

Applies To

Cluster node

See Also

iscsi-lu iscsi-lu-pr-dump

Privilege Level

Supervisor
