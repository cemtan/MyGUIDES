batt-log-show

Display contents of battery log

Syntax

batt-log-show [test|condition]

Description

Displays the battery log file

There are three battery log files: the normal log, the test log and the conditioning log. The normal log is the battery log produced during normal running. The test log file is the output for the periodic battery test. The conditioning log is the output from a conditioning cycle.

Parameters:

[test|condition]

Display test or conditioning log

Examples

batt-log-show

Displays the battery log for normal running

batt-log-show test

Displays the battery log from the last test cycle

batt-log-show condition

Displays the battery log from the last conditioning cycle

Privilege Level

Supervisor
