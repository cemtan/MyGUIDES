ingested-file-target

Displays the target of an ingested file

Syntax

ingested-file-target <path>

Description

Usage:

<path>

Path name of the file to be checked.

The ingested-file-target command tests whether a file is an ingested file, and if so displays the status and target details of the ingested file.

Examples

To show the ingested file target details of a file, "file1.txt":

ingested-file-target file1.txt
