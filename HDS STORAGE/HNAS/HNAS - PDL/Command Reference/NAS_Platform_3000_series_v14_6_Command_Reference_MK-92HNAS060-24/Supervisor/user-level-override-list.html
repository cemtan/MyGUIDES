user-level-override-list

List commands configured with overrides for the user access level required to execute the command.

Syntax

user-level-override-list

Description

The 'user-level-override-list' command lists the commands for which the default user access level required to execute the command has been overridden to further restrict who can execute the command.

To configure the user access level for commands contact your support provider for further information.

Examples

To list the configured user access level overrides:

$ user-level-override-list
Command     Access Level
-------     ------------
 status       SUPERVISOR

Applies To

Cluster wide

See Also

Privilege Level
