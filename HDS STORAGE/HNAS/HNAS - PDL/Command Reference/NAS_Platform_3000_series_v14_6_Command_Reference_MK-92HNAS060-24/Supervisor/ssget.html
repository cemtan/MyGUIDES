ssget

Get file

Syntax

ssget <server file> <local file>

Description

The ssget command transfers data to a client running ssc from the server flash. This command is only available on ssc connections.

The command takes two parameters - the client filename and the server flash filename.

Examples

ssc 192.168.0.1 ssget sd0.cod backup/sd0.cod

Applies To

Cluster node

See Also

ssfs ssput

Privilege Level

Supervisor
