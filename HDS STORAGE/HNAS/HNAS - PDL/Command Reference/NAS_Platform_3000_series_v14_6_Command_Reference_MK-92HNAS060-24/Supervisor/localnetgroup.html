localnetgroup

Add/remove local netgroup membership

Syntax

localnetgroup [ -l | -R | [ -v | -e | -a | -s | -r netgroup [member ...] ] ]

Description

Manipulates a local set of netgroup maps that are maintained on the server independently of any NIS or LDAP server.

These maps are used for permission checking when performing remote mounts.

-a, --add <netgroup> <member>

Adds <member> to localnetgroup <netgroup>.

-s, --set <netgroup> <member(s)>

Creates/resets localnetgroup <netgroup> so it contains (only) <member(s)>.

-r, --remove <netgroup> <member>

Removes <member> from localnetgroup <netgroup>.

-R, --removeall

Removes all localnetgroups.

-v, --view <netgroup>

Displays the definition of <netgroup>.

-e, --expanded <netgroup>

Displays the expanded definition of <netgroup>. A <netgroup> consists of a set of <members>, any of which may have further definitions. An expanded definition is one in which all members have been replaced by their definitions. The --expanded option lists a localnetgroup's definition with all members replaced by their definitions (recursively if necessary).

-l, --list

Displays the definitions of all localnetgroups.

Examples

Add member1 and member2 to localnetgroup netgroup1:

localnetgroup -a netgroup1 member1 member2

Update (overwrite) localnetgroup netgroup1 to leave it with just the members member1 and member2

localnetgroup -s netgroup1 member1 member2

Remove member1 from localnetgroup netgroup1:

localnetgroup -r netgroup1 member1

Remove all localnetgroups completely:

localnetgroup -R

View a localnetgroup's definition:

localnetgroup -v netgroup1

View a localnetgroup's expanded definition:

localnetgroup -e netgroup1

View all localnetgroups:

localnetgroup -l

Applies To

Cluster wide
