sd-luids-list

Produce a table of SD device IDs and Luids

Syntax

sd-luids-list [--alphabetic]

Description

Prints a lookup table translating system drive device IDs to Luids. This table can help to relate server device IDs to system drive IDs in certain storage configurators.

By default, the list is sorted by device ID. If you supply the '--alphabetic' or '-a' switch, the list is sorted by Luid instead.

Examples

sd-luids-list


sd-luids-list -a

Applies To

Cluster wide

See Also
