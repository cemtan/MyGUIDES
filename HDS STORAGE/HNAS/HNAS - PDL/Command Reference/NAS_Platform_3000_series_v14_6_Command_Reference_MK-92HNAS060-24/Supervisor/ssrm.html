ssrm

Delete file(s) from flash

Syntax

ssrm <filespec> [<filespec> ...]

Description

Delete a file from the server flash.

The command takes a number of parameters for the names of the files to be deleted. The filenames may include the flash where the file resides. Wildcards may be used in the filenames.

Examples

ssrm test.txt

Deletes test.txt from local flash on current board.

ssrm //simm/test.txt

Deletes test.txt from Simm flash.

ssrm test*

Deletes all files beginning with "test" in the local flash

ssrm //fsm/test*

Deletes all files beginning with "test" in the FSM flash

ssrm test1* //fsm/test2*

Deletes all files beginning with "test1" in the local flash and all files beginning with "test2" in the FSM flash

The above wildcard examples will prompt you to delete the individual files.

Applies To

Cluster node
