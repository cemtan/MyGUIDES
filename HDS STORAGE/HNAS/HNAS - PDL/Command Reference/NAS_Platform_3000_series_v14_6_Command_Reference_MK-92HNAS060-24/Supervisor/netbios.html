netbios

Used to enable/disable NetBIOS

Syntax

netbios [--on|--off]

Description

The netbios command is used to enable or disable NetBIOS. NetBIOS is an API that provides three services - a name service, a session service and a datagram distribution service.

NetBIOS has traditionally been used by Microsoft to carry SMBs (Server Message Blocks) used by the CIFS protocol over TCP/IP. The protocol that transports NetBIOS over TCP/IP is called NBT. NetBIOS has also provided a name service for CIFS clients and servers and a datagram distribution service for such things as Domain Controller discovery.

However, nowadays CIFS can run directly over TCP/IP, without requiring the NBT transport, and DNS can be used both as a name service and for Domain Controller discovery. So, NetBIOS isn't required in some environments. This being the case, it may be desired for NetBIOS support to be disabled altogether.

NetBIOS is disabled when in multi-tenancy mode.

Apple Mac clients used to need NetBIOS to be enabled on the server in order to connect over CIFS. However, Apple Mac clients since at least OS X 10.6.8 (Snow Leopard) can connect to the server over CIFS with NetBIOS disabled.

Usage:

netbios

Display whether NetBIOS is currently enabled or disabled.

netbios --on

Enable NetBIOS.

netbios --off

Disable NetBIOS.

Applies To

Cluster wide
