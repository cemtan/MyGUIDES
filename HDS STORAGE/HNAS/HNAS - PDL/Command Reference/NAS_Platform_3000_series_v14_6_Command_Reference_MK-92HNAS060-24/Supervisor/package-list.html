package-list

List installed packages

Syntax

package-list

Description

The package-list command lists the current packages in flash.

One of two warnings may be displayed following the package name in the list.

"package installed, but link to flash file broken"

There is a fault in the installed package, the package cannot be selected as default or run.

"package installed, new flash will cause reboot"

The package can be selected as default, but when first run will cause a complete system reboot in order to re-program the flash.

Examples

package-list

Applies To

Cluster node
