tls-certificate-show

Show the installed SSL/TLS certificate, trust chain certificates and private key

Syntax

tls-certificate-show [--show-private-key] [--pem] [--all]

Description

The tls-certificate-show command allows the installed certificate, trust chain certificates and private key to be viewed.

Options:

--all

Shows all the certificates and trust chains that have been imported.

--pem

When specified certificates will be printed in base64 PEM format. When not specified the output will be human readable.

--show-private-key

Shows the private key of the certificate in base64 PEM format. This only works when specified with --pem.

Examples

tls-certificate-show

Shows only the certificate in human readable format, no private key will be shown.

tls-certificate-show --pem --show-private-key

Shows the certificate and private key in base64 PEM format.

tls-certificate-show --all --pem

Shows all the certificates in base64 PEM format. No private keys are shown.

tls-certificate-show --all

Shows all the certificates in a human readable format. No private keys are shown.

Applies To

Cluster wide
