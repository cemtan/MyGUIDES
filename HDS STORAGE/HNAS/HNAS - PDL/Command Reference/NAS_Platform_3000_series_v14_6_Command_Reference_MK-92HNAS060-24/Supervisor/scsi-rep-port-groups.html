scsi-rep-port-groups

Send a SCSI Report Target Port Groups command to the target device

Syntax

scsi-rep-port-groups <-p,  --port id>[-l, --lun][-d, --dump][-q, --qtag]

Description

Sends a SCSI Report Target Port Groups command to the target device on the given SCSI port. This command is used to request that a SCSI RAID controller return the target port groups.

Options:

-p, --port <port>

SCSI port id of target device.

-l, --lun <LUN>

LUN of the target device.

-d, --dump

Hex dumps the returned data.

-q, --qtag <tag>

Uses this queue tag value, which must be one of:

Simple = 0

Head of q = 1 (default)

Ordered q = 2

ACA q = 3

Untagged = 4

Examples

To send a SCSI report target port groups command to SCSI port 3 and hex dump the result:

scsi-rep-port-groups -p 3 -d

Applies To

Cluster node
