send-quota-report

Send a quota details report now

Syntax

send-quota-report <email address>

Description

send-quota-report requests an email with a .csv report of the configuration and usage figures for all quotas on all virtual volumes on all file systems known to the server.

For each quota, the report contains the following information:

File System

The file system hosting the virtual volume.

Virtual Volume

The virtual volume to which the quota applies.

Path

The path in the file system leading to the root of the virtual volume.

Email Contacts

The list of contacts to whom emails will be sent when quota limits are exceeded.

User/Group Account

The user or group to which the quota's values apply.

Quota Type

Whether the quota is for a user, group or the entire virtual volume.

Created By

Whether the quota was created manually, or automatically as a result of quota defaults having been configured for the virtual volume.

Usage

The current usage of the virtual volume for the owner of the quota.

Usage Limit

The quota's usage limit.

Usage Hard Limit

Whether the usage limit is rigidly enforced, or just triggers an alert.

Usage Reset (%)

Amount by which usage must fall below a threshold before re-logging its event (%).

Usage Warning (%)

Usage warning threshold (%).

Usage Critical (%)

Usage critical threshold (%).

File Count

The current number of files in the virtual volume owned by the owner of the quota.

File Count Limit

The quota's file count limit.

File Count Hard Limit

Whether the file count limit is rigidly enforced, or just triggers an alert.

File Count Reset (%)

Amount by which file count must fall below a threshold before re-logging its event (%)

File Count Warning (%)

File count warning threshold.

File Count Critical (%)

File count critical threshold.

The command requires a single argument, <email address>, which is the single email address to which the report will be sent.

Examples

send-quota-report dsmith@example.com

The received email will contain a .csv attachment giving information of the following form:
