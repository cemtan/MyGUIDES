fc-link-speed

Set or display FC link speed

Syntax

fc-link-speed [-i <1|...|N>] [-s <1|2|4>]
Alias:       fclinkspeed

Description

The fc-link-speed command allows you to query and set the link speed of the Fibre Channel interfaces. Possible settings are 1, 2, 4, 8 and 16 gigabits/second; the speeds actually available depend on the hardware. The settings apply to all nodes in a cluster. Command options are:

-i <interface_number>

Specify the FC interface to query or set. If this option isn't set then the command applies to all available interfaces. Interfaces are numbered from 1.

-s <speed>

If this option is passed then the specified interface (see -i option) or all interfaces will be set to the specified speed.

Examples

View the configured link speed on all interfaces:

fc-link-speed

Set the link speed on interface FC1 to 4G:

fc-link-speed -i 1 -s 4

Set the link speed on all interfaces to 8G:

fc-link-speed -s 8

Applies To

Cluster wide
