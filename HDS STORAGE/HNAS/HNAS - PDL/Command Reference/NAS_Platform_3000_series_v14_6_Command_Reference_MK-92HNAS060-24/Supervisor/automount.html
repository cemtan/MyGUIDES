automount

Queries/sets the current automount setting

Syntax

automount [--enable | --disable]

Description

Displays and changes the automatic mount behavior. In all normal circumstances the server should be allowed to automatically mount volumes, making them available to the configured file serving protocols. Note this setting is persistent across server reboots.

Examples

To display the current setting:

automount

To disable automount:

automount --disable

See Also

mount
