login

Login to console

Syntax

login [<username> [<password>]]

Description

The login command allows you to log into a console. Normally when a console is started you will automatically be prompted for a login username and password, but you can use this command. Options:

username

Optionally specify the username you want to login as

password

Optionally specify the password you want to login with

If username or password are not supplied, they are prompted for. It is recommended that password is not supplied, as if it is prompted for it is hidden with * characters.

Examples

login user

Applies To

Console

See Also

logout who whoami

Privilege Level

Any
