cluster-rename

Rename a Cluster

Syntax

cluster-rename [-r | --rename-cluster-only] <new name>

Description

The cluster-rename command will rename the cluster. It will also rename each cluster node, using the new cluster name with the cluster node ID appended, separated with a hyphen ('-'). (Note: renaming the cluster nodes can be prevented using the --rename-cluster-only option.)

Cluster name may be up to 13 alpha-numeric or hyphen ASCII characters long.

Usage:

--rename-cluster-only | -r

Only rename the cluster. Do not also rename the cluster nodes.

<new name>

The new cluster name

Note:

Changing the cluster name will trigger creation of a new self-signed SSL/TLS certificate for the SOAP HTTPS management server and (if present) for the embedded SMU. However, if you had installed a custom certificate it will not be affected (but you may want to consider updating it to match the new domain name).

Examples

server:$ cluster-rename NEWCLUSTER
Configuring QD with new cluster name: NEWCLUSTER
Cluster successfully renamed to: NEWCLUSTER

Applies To

Cluster wide
