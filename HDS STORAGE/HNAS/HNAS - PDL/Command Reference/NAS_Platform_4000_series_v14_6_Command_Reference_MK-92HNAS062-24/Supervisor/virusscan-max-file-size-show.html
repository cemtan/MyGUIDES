virusscan-max-file-size-show

Show the virus scanning maximum file size configuration

Syntax

virusscan-max-file-size-show

Description

Show the current configuration of the maximum file size for virus scanning.

Examples

To display the current status

  virusscan-max-file-size-show

Example output

  Maximum file size limit is disabled.
  If enabled, files larger than 3.226 GB (3464364634 B) would be excluded from scanning.

Applies To

EVS

See Also

virusscan-max-file-size-disable virusscan-max-file-size-enable virusscan-max-file-size-set

Privilege Level

Supervisor
