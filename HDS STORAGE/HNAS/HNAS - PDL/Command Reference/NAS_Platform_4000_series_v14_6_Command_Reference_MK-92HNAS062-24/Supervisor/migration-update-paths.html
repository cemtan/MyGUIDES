migration-update-paths

Checks if paths for this file system are active and updates status.

Syntax

migration-update-paths -s <fs label>

Description

Options:

-s, --source <fs label>

Source file system.

Update the status of Data Migration paths from the specified source file system. All paths for the file system itself and any Virtual Volumes will be updated. The software will automatically update the status when the path is used. However, it will not notice automatically when the last cross-file-system link on the path is deleted. This command will mark the path as not in use if no files are found at the target.

Examples

migration-update-paths --source Accounts

Update path status for all paths on file system Accounts. This includes paths associated with Virtual Volumes on that file system.

Applies To

EVS
