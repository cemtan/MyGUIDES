ca-certificate-delete

Remove a custom certificate from the trusted certificate store.

Syntax

ca-certificate-delete <certificate-name>

Description

The ca-certificate-delete command deletes a certificate from the system trusted certificate store.

Examples

To delete the certificate named 'hcp8-uk-01.example.com.crt':

$ ca-certificate-delete hcp8-uk-01.example.com.crt

Applies To

Cluster wide

See Also

ca-certificate-delete-all ca-certificate-import ca-certificate-show

Privilege Level

Supervisor
