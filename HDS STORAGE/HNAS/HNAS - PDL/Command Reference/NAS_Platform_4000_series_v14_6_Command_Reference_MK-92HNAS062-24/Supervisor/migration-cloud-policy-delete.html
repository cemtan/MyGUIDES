migration-cloud-policy-delete

Delete an external cloud migration policy

Syntax

migration-cloud-policy-delete <policy-name>|<policy-id> [--stop-active] [--delete-history|-h]

Description

Deletes a policy for cloud migration and all its schedules.

<policy-id>

ID of the policy.

<policy-name>

Name of the policy.

--stop-active

Stops all currently running migrations for this policy and then deletes a policy record. Note: migration details may be lost.

--delete-history, -h

Deletes the cloud migration history for this policy. By default the external migration history is preserved.

Examples

migration-cloud-policy-delete DefaultPolicy --delete-history

Deletes the cloud migration policy named 'DefaultPolicy'. The schedules and history entries associated with the policy are also deleted; a dump of the history entries can be found in the dblog.

Applies To

EVS security context
