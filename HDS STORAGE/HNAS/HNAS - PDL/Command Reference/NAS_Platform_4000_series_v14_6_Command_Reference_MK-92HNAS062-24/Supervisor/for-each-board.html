for-each-board

Run a command on each board

Syntax

for-each-board [--quiet] <command> [args]

Description

The for-each-board command runs the specified command once for each board.

Options:

--quiet|-q

Don't display each board name as commands are run

ssrun -c can be useful when trying to run complicated commands.

Examples

Look for the word "failed" in the debug log for each board

for-each-board dblog | grep -i failed

Applies To

Cluster node

See Also

console-context ssrun

Privilege Level
