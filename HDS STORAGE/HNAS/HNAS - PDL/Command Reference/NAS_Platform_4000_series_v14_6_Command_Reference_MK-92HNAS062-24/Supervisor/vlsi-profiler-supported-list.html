vlsi-profiler-supported-list

List the state machines supported by automated vlsi profiling.

Syntax

vlsi-profiler-supported-list

Description

This command displays the names of all of the state machines that can be made "eligible for profiling" by the automated vlsi-profiler.

Examples

Listing state machines:

vlsi-profiler-supported-list


Chip Name     State Machine Name
---------     ------------------
      fdp       nim_msg_out_pull
      fdp       nim_msg_out_push
      fdp                  nv_in
      fdp              sim_in_rx
      fsa                  cache
      fsa                control
      fsa                    dwb
      fsa                    map

Applies To

Cluster node
