network-protocol-performance-monitor-auto-log-disable

Set spare logging capacity to remain unused.

Syntax

network-protocol-performance-monitor-auto-log-disable

Description

Enable and disable the automatic distribution of spare logging capacity across file systems that would otherwise not be logged.

Default is enabled.

The network-protocol-performance-monitor-logsize command can be used to override the configuration for specific file systems.

Examples

Allow spare logging capacity to be used to monitor file systems that would otherwise not be logged:

    network-protocol-performance-monitor-auto-log-enable

Only log what is specifically asked:

    network-protocol-performance-monitor-auto-log-disable

Applies To

Cluster wide
