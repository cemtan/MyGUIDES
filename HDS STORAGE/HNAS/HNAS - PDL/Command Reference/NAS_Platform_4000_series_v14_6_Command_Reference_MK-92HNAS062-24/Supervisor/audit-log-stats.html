audit-log-stats

Display basic file system audit log statistics

Syntax

audit-log-stats [--reset] [<file-system>]

Description

The audit-log-stats command displays basic statistics about file system audit log activity.

Options:

-r, --reset

Reset statistics.

file-system

File system label.

Examples

Reset all file system audit statistics:

audit-log-stats -r

Display all file system audit log statistics:

audit-log-stats

EVS
