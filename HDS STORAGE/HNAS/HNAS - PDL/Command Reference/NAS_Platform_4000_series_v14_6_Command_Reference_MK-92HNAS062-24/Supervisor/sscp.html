sscp

Copy file

Syntax

sscp <filename> <filename>

Description

Copy a file in server flash.

The command takes two parameters - the source and destination filenames, both of which may include the flash name.

Examples

sscp test1 test2

Copies file test1 to test2 in local flash on current board.

sscp test1 //simm/test2

Copies file test1 in local flash on current board to test2 in Simm flash.

sscp //simm/test1 //fsm/test2

Copies file test1 in Simm flash to test2 on local flash on FSM.
