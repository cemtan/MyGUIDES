lacp-show

Show LACP configuration

Syntax

lacp-show [<interface>]

Description

The 'lacp-show' command displays the LACP configuration parameters for one or more aggregations. Without any options, the command will display the configuration parameters for all aggregations.

To enable LACP on an aggregation use the 'aggedit' command.

Examples

To display LACP configuration parameters for an aggregation:

    lacp-show ag1

Applies To

Cluster wide

See Also

aggedit lacp lacp-set-timeout-default lacp-set-timeout-long lacp-set-timeout-short

Privilege Level

Supervisor
