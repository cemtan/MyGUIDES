ndmp-device-assigned-name-list

List names assigned to NDMP devices

Syntax

ndmp-device-assigned-name-list

Description

The command lists the names that have been assigned to NDMP devices using the ndmp-device-assigned-name-set command.

See the ndmp-devices man page for a discussion of NDMP device naming.

Examples

Applies To

Cluster wide

See Also

ndmp-device-assigned-name-delete ndmp-device-assigned-name-set ndmp-devices ndmp-devices-list
