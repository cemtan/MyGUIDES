dsb-set-file-counts

Set file counts on the latest DSB

Syntax

dsb-set-file-counts <file-counts> <fs>

Description

Set the file count values of the specified file system's latest dynamic super block. This command only works on WFS-2 format file systems. The file system needs to be unmounted for this operation.

This command should be run if checkfs reports that the file counts stored in the DSB do not match the actual file counts.

<file-counts>

Space separated values of file counts to be set, in the following order: Non Regular Directory Block Character SymLink Socket Fifo HardLinked

Examples

To set file counts to (Non:1 Regular:2 Directory:3 Block:4 Character:5 SymLink:6 Socket:7 Fifo:8 HardLinked:9) on file system "fs1"

dsb-set-file-counts 1 2 3 4 5 6 7 8 9 fs1
