nfsv4-compound

Display NFSv4 COMPOUND sub-command combinations

Syntax

nfsv4-compound [--reset|-r]

Description

The nfsv4-compound command displays the NFSv4 COMPOUND command combinations that have been received by the server, as well as any errors that have been returned. It provides information that may be useful when debugging NFSv4 connectivity issues.

Usage:

--reset, -r

Clears the log.

Applies To

Cluster node

See Also

nfs-stats

Privilege Level

Supervisor
