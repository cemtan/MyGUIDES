console-logging

Set/show logging of console commands

Syntax

console-logging [on|off]

Description

The console-logging command controls whether CLI commands are logged to the server's management log. This log can be retrieved at a later date to provide a history of commands to enable trouble-shooting, so it is recommended that you leave it on.

Specify either "on" or "off" as the only parameter to switch logging on or off, or no argument to show the current state.

Examples

To switch logging off:

console-logging off

Applies To

Console

See Also
