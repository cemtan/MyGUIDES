nis-netgroups-lookup-depth

Show/set maximum recursive lookup depth for NIS netgroups

Syntax

nis-netgroups-lookup-depth [depth]

Description

Displays or sets the maximum netgroup lookup depth.

While in classic NIS mode, this setting applies only when the "nobyhost" functionality is enabled. The setting always applies in NIS/LDAP mode.

e.g. If we have the following netgroup map

group6 group5
group5 group4
group4 group3
group3 group2
group2 group1
group1 (host,,)

With the depth set to 5 host will appear in groups group1 through group5 but it will not appear to be in group6 (since to check that requires a 6th lookup in the netgroup map and we set the maximum to 5)

Examples

Display the current maximum depth:

        $ nis-netgroups-lookup-depth
        Maximum recursive lookup depth: 20
        $

Set a new maximum depth:

        $ nis-netgroups-lookup-depth 30
        Maximum recursive lookup depth: 30
        $

Applies To

EVS security context
