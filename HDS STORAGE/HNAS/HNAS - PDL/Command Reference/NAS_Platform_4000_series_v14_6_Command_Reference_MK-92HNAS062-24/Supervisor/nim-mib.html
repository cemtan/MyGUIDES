nim-mib

Display NIM MIB stats

Syntax

nim-mib [-n stat-name] [-p port-number] [-r] [-t]

Description

The nim-mib command displays MIB information gathered by the hardware.

-n, --name <stat-name>

Displays value of the MIB stat named <stat-name>.

-p, --port <number>

Displays MIB stats for the specified interface. Note that the interfaces are numbered internally from 0 onwards.

-r, --rx

Displays RX MIB stats.

-t, --tx

Displays TX MIB stats.

Examples

To display all the information for all interfaces:

nim-mib

To display all information for the second interface:

nim-mib -p 1

To display all information for RX stats:

nim-mib -r

To display all information for TX stats:

nim-mib -t

To display a specific statistic:

nim-mib -n RX_MIB_0_UDP_BYTES

Applies To

Cluster node

Privilege Level

Supervisor
