diagemail

Send a diagnostic email

Syntax

diagemail [--dump] [--caseno <number>] [recipient|all]

Description

The diagemail command packages up various logs, debugging, performance and configuration options and emails it to the specified recipient.

If the recipient "all" is specified, diagnostic emails will be sent to all recipients configured in the email profiles, otherwise a single email address must be specified.

Options:

-c, --caseno

specify the case number, if available

-d, --dump

do a diagnostic dump first. Only use under instruction from your support provider

Note that the command returns to the prompt immediately and generates the diagnostic in the background. On a busy cluster it may be a couple of minutes before the email is actually sent.

Examples

Send a diagnostic email to support@example.com noting that this relates to case 1234

diagemail --caseno 1234 support@example.com

Applies To

Cluster node
