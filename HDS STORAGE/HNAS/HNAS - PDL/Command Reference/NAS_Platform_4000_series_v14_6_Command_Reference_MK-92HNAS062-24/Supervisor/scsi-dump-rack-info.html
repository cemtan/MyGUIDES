scsi-dump-rack-info

dump engineering debug level scsi racks information

Syntax

scsi-dump-rack-info [-p, --scsi-port]

Description

Displays debug troubleshooting information about a storage rack or all storage racks attached to the server. The information output by this command is for use by engineering and services to troubleshooting storage issues and should not be required in normal operation.

Options:

-p, --port

Display troubleshooting information about the storage rack that hosts the given SCSI port number.

If no SCSI port number is entered troubleshooting information about all storage racks is displayed.

Examples

Show the debug information for the storage rack on SCSI port 1:

  scsi-dump-rack-info --port 1
