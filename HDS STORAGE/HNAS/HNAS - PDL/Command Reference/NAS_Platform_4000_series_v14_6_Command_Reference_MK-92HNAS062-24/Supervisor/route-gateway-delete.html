route-gateway-delete

Delete a gateway route

Syntax

route-gateway-delete <gateway>

Description

The route-gateway-delete command deletes a gateway from the routing table.

Options

The following options are recognised:

<gateway>

This is the gateway that identifies the route. It must be supplied.

Examples

$ route-gateway-delete 10.1.2.3
Route cache flushed.
$

Deletes a gateway of 10.1.2.3.

Applies To

Cluster wide
