cifs-server-comment-get

Syntax

cifs-server-comment-get

Description

Displays the CIFS comment of the server (also sometimes referred to as the description). The CIFS comment describes the server in a NetrServerGetInfo response and can be retrieved from an SMB server using the DCERPC call. Customers should assess whether their environment requires this value to be set. This command replaces the "comment" field of the deprecated "id" command. Its default value will be whatever was set in the comment field when the HNAS was running older firmware.

cifs-server-comment-get
    Display the CIFS comment.

Options

    None.

Examples

server:$ cifs-server-comment-get
Archive server

Applies To

Cluster wide.
