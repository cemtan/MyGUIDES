xfp-support

Report/manage the supported parts list

Syntax

xfp-support [list] | [add <vendor-name-str> <vendor-part-no-str>] | [reset]

Description

The xfp-support command reports/manages the supported XFP list.

Options:

list

Lists supported XFPs.

add <vendor-name-str> <vendor-part-no-str>

Adds an XFP to the supported list. This option is only available to the dev user.

reset

Resets the supported list to its default state. This option is only available to the dev user.

Applies To

Cluster wide

Privilege Level

Supervisor
