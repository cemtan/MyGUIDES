apikey-list

List all configured API Keys

Syntax

apikey-list

Description

apikey-list displays the configured API Key prefixes, descriptions, current state of the key, and the access level granted to the key.

A full API Key is formatted as follows, with the prefix being the characters preceeding the first . character:

SzKdKXQxur.GU0RAlCfxyQ8TJYc7twueT3.x.8VvvvlXBXBP6.bvH3

For security reasons, the full API Key is only displayed upon creation and can't be displayed again.

Examples

server:$ apikey-list


    Prefix     Description        State                 Access Level
----------     -----------     -------- ----------------------------
SzKdKXQxur             one      Enabled             native-rest-read
hsxq97IKyR             two     Disabled                  full-access
3H7xGA3Qsb           three      Enabled other-all | native-rest-read

Applies To

Cluster
