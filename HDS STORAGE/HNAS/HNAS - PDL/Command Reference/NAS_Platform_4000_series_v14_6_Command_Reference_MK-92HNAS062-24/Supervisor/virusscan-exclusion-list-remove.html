virusscan-exclusion-list-remove

Remove one or more file extensions from the exclusion list

Syntax

virusscan-exclusion-list-remove <file-types>

Description

The virusscan-exclusion-list-remove command removes one or more file extensions from the list of file extensions which are excluded from virus scanning.

Options:

<file types>

A comma-separated list of file type extensions.

The case of file extensions is ignored.

Removal of extensions is explicit - wildcards cannot be used to match multiple variations of a given extension.

There must be no whitespace between consecutive types.

Examples

Remove the extension "ABC" from the list:

  virusscan-exclusion-list-remove ABC

Remove "AB1" and "AB2" from the list:

  virusscan-exclusion-list-remove AB1,AB2

Note that wildcards cannot be used to remove multiple variations of an extension.

Remove "AB1" and "XY?" from the list:

  virusscan-exclusion-list-remove AB1,XY?

Applies To

EVS
