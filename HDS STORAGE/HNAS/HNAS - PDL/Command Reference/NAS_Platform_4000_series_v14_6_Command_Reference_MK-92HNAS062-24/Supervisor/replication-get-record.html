replication-get-record

Displays the replication record for the specified replication.

Syntax

replication-get-record -f <fs_label> -n <replic_name> [-c] [-d]

Description

Displays the record of a replication. We recommend that the SMU be used for all replication configuration. This command is only intended for use by SMU scripts and support.

Options:

-f, --file-system <fs_label>

Label of the file system addressed.

-n, --name <replic_name>

Name of the replication.

-c, --check_snapshot

Checks that the snapshot associated with the replication record exists.

-d, --destination

If set, gets the destination record.

Examples

replication-get-record --file-system Accounts --name Accounts2Standby

Displays details of the replication Accounts2Standby from its source file system Accounts.

Applies To
