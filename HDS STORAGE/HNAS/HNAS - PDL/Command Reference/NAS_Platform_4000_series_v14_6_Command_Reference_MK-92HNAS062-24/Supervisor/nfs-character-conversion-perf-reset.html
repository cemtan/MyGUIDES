nfs-character-conversion-perf-reset

Reset the character conversion performance statistics

Syntax

nfs-character-conversion-perf-reset

Description

The nfs-character-conversion-perf-reset command resets the performance statistics which are displayed by nfs-character-conversion-perf-show.

Applies To

Cluster node

See Also

nfs-character-conversion-perf-show

Privilege Level

Supervisor
