tcp-per-connection-statistics-start

Start per-connection TCP statistics gathering

Syntax

tcp-per-connection-statistics-start  [-r] <duration | -f>
Alias:       tpcs-start

Description

Starts collection of per-connection TCP statistics.

-f, --free-run

Per-connection TCP statistics will be gathered until explicitly stopped with tcp-per-connection-statistics-stop or tcp-per-connection-statistics-cancel.

duration

A time specifying the period over which data should be gathered. The duration is taken to be in seconds unless modified with ms, s, m, h, d suffixes. Either duration or --free-run must be specified.

-r, --restart

Use this option when statistics are already being gathered to force the system to discard the currently gathering statistics, and start again with a new time specified.

Examples

To start collecting statistics until asked to stop:
    tcp-per-connection-statistics-start -f


Cancel statistics gathering currently in progress, and start gathering again for 10 minutes:
    tcp-per-connection-statistics-start -r 10m

Applies To

Cluster Node
