migration-cloud-cleanup-cancel-fs

Cancel background cleanup process on Cloud Gateway for a file system

Syntax

migration-cloud-cleanup-cancel-fs <fs-label>

Description

Cancels background cleanup process on Cloud Gateway for a file system. Note: the job will not be resumed automatically.

Note: Cleanup cannot be performed on object replication target file systems.

fs-label

File system label, to cancel cloud cleanup process for.

Examples

migration-cloud-cleanup-cancel-fs myFs

Stops cleanup process on the cloud for file-system 'myFs'.

Applies To

EVS security context

See Also

migration-cloud migration-cloud-cleanup migration-cloud-cleanup-cancel-all

Privilege Level

Supervisor
