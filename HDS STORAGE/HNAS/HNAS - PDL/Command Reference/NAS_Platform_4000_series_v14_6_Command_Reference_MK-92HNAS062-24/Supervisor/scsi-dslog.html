scsi-dslog

dump/clear the SCSI device status logs

Syntax

scsi-dslog [-d, --dump][-c, --clear][-s, --size][-h, --help]
Alias:       dslog

Description

Dump or clear the device state change log for all devices or for a specified device.

By default, the last 20 entries are dumped if no range is specified.

Console output may be stopped or aborted with a key press or by CTRL-C.

Changes in status of each device known by the server are logged. The status changes that are logged include:

  • When the device was opened and the nexus used.
  • When a failover occurred that caused a new nexus to be selected for I/O.
  • When the device was closed.
  • Error conditions.
  • Media check status changes.
  • FC link up/down events.

Options:

-d, --dump

Display the contents of the device status log for all devices or for the given device -- the last 20 entries are output if no range is specified.

-D <device ID>

Display the log for this device.

device list

Display the logs for the specified devices. See the 'sd-spec' man page for an explanation of how to specify a list of devices using sd-spec syntax. By default, the device list must specify system drives. To list tape devices the -t switch is required.

-C, --Conf

Suppress the message asking for confirmation before dumping all device logs.

-t, --type <disks | tapes>

Display only logs for disk or tape devices

-i, --ignore

When dumping all logs, ignore those that are empty, else a "Device State log is empty" message is displayed.

-a, --all | -n <[-]no of entries to dump>

Output all entries, last n entries or first n entries if n is negative if n = 0, dump entire log; default, dump last 20 entries.

-c, --clear

Clear the specified log or all device state change log if no device is specified.

-C, --Conf

Clear all logs, without requesting confirmation.

device ID

Clear the specified device log.

-s, --size
device ID

Show the current and maximum size of the specified log or all logs if no device is specified.

-h, --help

Show help.

Examples

Dump the last 20 entries from all logs:

scsi-dslog -d

Dump the entire log for all devices:

scsi-dslog -d -a

Dump all of device ID 1 log:

scsi-dslog -d 1 -a

Dump last 10 entries from system drive ID 2 log:

scsi-dslog -d 2 -n 10

Dump all logs, ignore empty logs, do not request confirmation:

scsi-dslog -d -c -i

Dump entire logs of system drives 1 to 10 :

scsi-dslog -d 1-10 -a

Dump entire logs for all tape devices, do not request confirmation:

dslog -d -a -t tapes -C

Dump entire log of tape device ID 10 :

scsi-dslog -d -D 10 -a

Dump entire log of tape devices 15 and 16 :

dslog -d 15,16 -a -t tapes

Clear all logs with no confirmation:

scsi-dslog -c -C

Clear device ID 1 log only, no conformation:

scsi-dslog -c -C 1

Applies To

Cluster node
