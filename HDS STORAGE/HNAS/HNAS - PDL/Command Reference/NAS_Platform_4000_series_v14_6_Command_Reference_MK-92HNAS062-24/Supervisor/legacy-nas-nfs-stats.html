legacy-nas-nfs-stats

Display statistics for legacy NAS NFS

Syntax

legacy-nas-nfs-stats [--reset]

Description

The legacy-nas-nfs-stats command displays statistics about legacy NAS activity using the NFS protocol.

Options:

-r, --reset

Reset statistics.

Examples

Reset all legacy NAS over NFS statistics:

legacy-nas-nfs-stats -r

Display all legacy NAS over NFS statistics:

legacy-nas-nfs-stats

Applies To

Cluster node
