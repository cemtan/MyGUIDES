sib-show-lock-owners

Display users of Span Manager, ID Manager and SCSI locks

Syntax

sib-show-lock-owners

Description

This command prints everything that the 'span-show-lock-owner' and 'idman-show-lock-owner' commands would print. See the relevant man pages.

Examples

sib-show-lock-owners

Applies To

Cluster node

See Also

idman-show-lock-owner span-show-lock-owner thread-association

Privilege Level

Supervisor
