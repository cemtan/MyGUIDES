home-directory-stats

Display basic home directory statistics

Syntax

home-directory-stats [--reset] | [--verbose]

Description

The home-directory-stats command displays basic statistics about file system home directory activity.

Options:

-r, --reset

Reset statistics.

-v, --verbose

Display verbose output.

Examples

Reset home directory statistics:

home-directory-stats -r

Display home directory statistics in verbose form:

home-directory-stats -v

Applies To

Cluster node

See Also

home-directory-perf

Privilege Level

Supervisor
