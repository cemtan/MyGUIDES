cifs-log-access-unrestricted

Allow unrestricted cifs log access

Syntax

cifs-log-access-unrestricted

Description

The cifs-log-access-unrestricted command is used to allow unrestricted access to the cifs log. Administrator access is required to clear the logs.

Examples

cifs-log-access-unrestricted

Applies To

EVS security context

See Also

cifs-log-access-administrators-only cifs-log-access-mode

Privilege Level

Supervisor
