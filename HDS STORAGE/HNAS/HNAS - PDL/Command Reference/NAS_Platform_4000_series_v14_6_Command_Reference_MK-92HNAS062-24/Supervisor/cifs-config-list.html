cifs-config-list

List currently-configured CIFS serving names for the EVS in context.

Syntax

cifs-config-list [ --passwords ]

Description

This command permits you to list CIFS name and CIFS domain records for an EVS. Optionally, you can also display the machine trust account passwords associated with each CIFS name.

The output format is somewhat cluttered since this command is meant for diagnostic purposes and not for routine CIFS configuration maintenance. For routine CIFS name operations, use the cifs-name command instead.

Examples

To display the CIFS-name configuration for EVS 2 without trust account passwords:

vn 2 cifs-config-list

To display the CIFS name configuration for EVS 2 with trust account passwords:

vn 2 cifs-config-list --passwords

To display the CIFS-name configuration with trust account passwords for all EVSs in the cluster:

for-each-evs cifs-config-list --passwords

Applies To

EVS security context
