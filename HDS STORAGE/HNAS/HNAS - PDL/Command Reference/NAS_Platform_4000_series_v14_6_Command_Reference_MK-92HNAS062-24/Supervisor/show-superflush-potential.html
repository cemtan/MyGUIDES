show-superflush-potential

Estimate the number of superflushable and other writes on a filesystem

Syntax

show-superflush-potential <filesystem-label>

Description

Superflush is a technique that the server uses to optimize writing to Raid 5 system drives (SDs). By writing a whole stripe line at a time, the server allows the Raid controller to generate parity more efficiently.

'show-superflush-potential' estimates how many areas of free space on a file system cover a whole stripeline, and how many don't. This gives a rough idea of how effective the superflush optimization will be on that file system. The file system must be mounted; all the SDs in the host span must have identical superflush parameters (stripesize and width), which must have been correctly configured into the server.

Examples

show-superflush-potential Accounts

Applies To

EVS
