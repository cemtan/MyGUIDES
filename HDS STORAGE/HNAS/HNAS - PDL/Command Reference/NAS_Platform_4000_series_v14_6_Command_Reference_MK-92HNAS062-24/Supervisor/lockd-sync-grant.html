lockd-sync-grant

Enables/disables using synchronous lockd GRANT requests.

Syntax

lockd-sync-grant [on|off]

Description

The lockd-sync-grant command configures the behavior of the network lock manager (NLM) with respect to GRANT messages.

If lockd-sync-grant is enabled, the server will await the result of each GRANT message synchronously. If it's disabled, the server won't await the result of each GRANT, but will handle results asynchronously as and when they're received.

Usage:

lockd-sync-grant [on | off]

on

Enables synchronous GRANT requests. This is the default.

off

Disables synchronous GRANT requests (i.e. enables asynchronous GRANT requests).

Applies To

EVS security context

Privilege Level

Supervisor
