evs-auto-failover

EVS management

Syntax

evs-auto-failover [disable | enable]

Description

Enables or disables automatic failover of EVSs in a cluster.

Usage:

enable

In the event of a cluster node, file system or Ethernet Aggregation failure the EVS will automatically failover to one of the remaining online cluster nodes.

disable

In the event of a cluster node, file system or Ethernet Aggregation failure the EVS will go into the "Offline" state. If the original cluster node is rebooted then the EVS will come back online on that node. Alternatively the EVS may be manually moved to another cluster node using the "evs migrate" command.

Examples

server:$ evs-auto-failover
Automatic EVS failover is enabled


server:$ evs-auto-failover disable
Automatic EVS failover is disabled

Applies To

Cluster wide
