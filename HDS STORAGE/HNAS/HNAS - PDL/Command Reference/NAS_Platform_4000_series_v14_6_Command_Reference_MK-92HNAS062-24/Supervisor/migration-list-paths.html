migration-list-paths

Lists all data migration destinations for a file system.

Syntax

migration-list-paths {-s <fs label> | -a}

Description

Lists all Data Migration paths defined for the specified file system, or for all file systems if the -a or --all option is used.

Options:

-s, --source <fs label>

Source file system.

-a, --all

All file systems. Produces a short summary at the end reporting on the number of file systems that contain a migration path, the number that do not contain a migration path and the number that cannot be reported upon, either because the file system is not mounted or the file system type does not support migration paths or because of an error in fetching the migration path information from the file system.

Examples

migration-list-paths -s Accounts

Lists migration paths on file system Accounts.

migration-list-paths --all

Lists migration paths for all file systems.
