unset-for-evs

Removes a variable

Syntax

unset-for-evs <variable> (--current-board | --all-boards)

Description

The unset-for-evs command clears a previously set evs variable for the evs currently in context. It should only be used under guidance from your support provider.

<variable name>

The name of the variable to be cleared.

--current-board | -c

Delete the variable defined in the current board scope. Typically this will be the "mmb" board on single-board platforms.

--all-boards | -a

Delete the variable defined in the "allBoards" board scope. Note that "allBoards" is a specific board designation just like "mmb" and using this switch will delete the variable in that scope if it exists. This will not cause the command to delete variables defined in all board scopes.

Examples

Clear "myvar1" for "allBoards".

unset --all-boards myvar1

Clear "myvar1" for the current board.

unset --current-board myvar1

Applies To

Cluster wide
