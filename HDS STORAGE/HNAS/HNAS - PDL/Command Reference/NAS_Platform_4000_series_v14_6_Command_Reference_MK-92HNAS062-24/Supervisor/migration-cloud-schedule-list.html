migration-cloud-schedule-list

List the external cloud migration schedules

Syntax

migration-cloud-schedule-list [<policy name> [<schedule id>]] [ --verbose | -v ]

Description

Lists schedules for cloud migration.

<policy name>

If given, display only schedule entries which apply this policy.

<schedule id>

If given, display only the schedule entry with this ID.

Examples

migration-cloud-schedule-list

Lists all cloud migration schedules on this EVS.

migration-cloud-schedule-list DefaultPolicy

Lists all cloud migration schedules for the policy named 'DefaultPolicy'.

migration-cloud-schedule-list DefaultPolicy 3

Lists the cloud migration schedule with ID 3 for the policy named 'DefaultPolicy'.

Applies To

EVS security context
