cifs-dc-broadcast

Used to enable/disable domain controller discovery using NetBIOS broadcast

Syntax

cifs-dc-broadcast [--on|--off]

Description

The preferred method for discovering domain controllers is by using DNS. However, if this isn't possible or a DNS lookup fails, the server will fall back to using WINS. If WINS isn't configured or a WINS lookup fails, the server is able, as a last resort, to send NetBIOS broadcasts onto the network to allow domain controllers to identify themselves. This command governs whether domain controller discovery via NetBIOS broadcast is enabled.

Usage:

cifs-dc-broadcast [--on | --off]

The --on and --off switches enable and disable DC discovery using NetBIOS broadcast. If no arguments are given, the command displays the current setting.

Examples

cifs-dc-broadcast

Domain controller discovery using NetBIOS broadcast is enabled.
