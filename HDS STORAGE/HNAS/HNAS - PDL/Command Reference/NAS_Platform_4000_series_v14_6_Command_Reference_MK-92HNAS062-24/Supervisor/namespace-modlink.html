namespace-modlink

Modifies the options for a link within a cluster namespace

Syntax

namespace-modlink [--remote-read-caching (disable|enable-all|enable-cvl)] [--local-read-caching (disable|enable-all|enable-cvl)] <cluster-namespace> <path>
Alias:       cns-modlink

Description

Modifies the options associated with an existing link in the specified cluster namespace.

Usage:

namespace-modlink [--remote-read-caching (disable|enable-all|enable-cvl)] [--local-read-caching (disable|enable-all|enable-cvl)] <cluster-namespace> <path>

-r, --remote-read-caching

Applicable to cluster namespaces only. Controls which files below the link are candidates for read caching, when accessed remotely:

disable

Disables read caching.

enable-all

All files are candidates for read caching.

enable-cvl

Only cross volume links are candidates for read caching.

-l, --local-read-caching

Controls which files below the link are candidates for read caching, when accessed locally. Values are the same as for --remote-read-caching.

Applies To

Cluster wide

See Also

Privilege Level

Supervisor
