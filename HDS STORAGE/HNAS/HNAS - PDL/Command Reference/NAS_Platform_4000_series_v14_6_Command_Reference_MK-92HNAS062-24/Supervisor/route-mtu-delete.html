route-mtu-delete

Delete an mtu route

Syntax

route-mtu-delete <network>

Description

The route-mtu-delete command removes a host route from the routing table.

Examples

$ route-mtu-delete 10.1.0.0/16
Route cache flushed.
$

Deletes an MTU route that specified an MTU that traffic sent from 10.1.0.0/16 would have used.

Applies To

Cluster wide
