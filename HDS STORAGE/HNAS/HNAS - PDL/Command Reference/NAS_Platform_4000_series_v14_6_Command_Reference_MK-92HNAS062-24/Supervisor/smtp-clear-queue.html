smtp-clear-queue

Discard all messages in the SMTP queue

Syntax

smtp-clear-queue

Description

The smtp-clear-queue command clears the queued SMTP messages without trying to send them. This should only be used in case of severe communication difficulties with the SMTP server.

Examples

smtp-clear-queue

Applies To

Cluster node

See Also

smtp-retry-connection smtp-statistics

Privilege Level

Supervisor
