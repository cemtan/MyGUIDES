cifs-smb2-signing-perf

Display SMB2 signing performance statistics

Syntax

cifs-smb2-signing-perf [--reset|--signing-rate|--summary]

Description

The cifs-smb2-signing-perf command reports SMB2 signing performance statistics. This includes statistics for the HNAS acting as both server (normal file-serving) and client (to Domain Controllers).

--reset, -r

Resets the statistics.

--signing-rate, -i

Shows the average signing throughput since the statistics were last reset.

--summary, -s

Disables the drawing of the performance stats graphs.

These fields apply to responses sent to an SMB2 client and requests to a server:

Send Data Transfer
Send Calculate Signature

Time spent sending data from FPGA to host and calculating the signature.

These fields apply to requests from an SMB2 client and responses from a server:

Receive Data Transfer
Receive Verify Signature

Time spent sending data from FPGA to host and verifying the signature.

Applies To

Cluster node
