ssmv

Rename ssfs file

Syntax

ssmv <oldname> <newname>

Description

Move a file in server flash.

The command takes two parameters - the source and destination filenames, both of which may include the flash name. The source and destination files must be in the same flash, i.e. you cannot move files between flashes.

Examples

ssmv test1 test2

Moves file test1 to test2 in local flash on the current board.

ssmv //fsm/test1 //fsm/test2

Moves file test1 in flash on FSM board to test2 in flash on FSM board.

See Also
