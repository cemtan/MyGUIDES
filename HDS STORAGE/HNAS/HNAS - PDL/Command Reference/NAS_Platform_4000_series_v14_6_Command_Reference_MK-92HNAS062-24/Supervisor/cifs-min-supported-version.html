cifs-min-supported-version

Set or display the minimum supported version of SMB

Syntax

cifs-min-supported-version [ --default | 1 | 2 | 2.1 | 3 ]

Description

The cifs-min-supported-version command has been obsoleted by the smb-min-supported-version command.

The function of the two commands is identical so you may continue to use cifs-min-supported-version for now, but smb-min-supported-version should be favored where possible.
