cifs-perf

Display CIFS performance statistics

Syntax

cifs-perf [[--reset] | [--summary] [--verbose] [--access-type (all|local|localcvl|remotecvl|legacynas)] [--type (elapsed|cpu|nvramreservation|lock|block|wfstime|wfscalls|remotetime|remotecalls|fscontext|security|accessandregionlock|readoplock|writeoplock|handleoplock|gns|renamelock|inq|requestsignatureverification|throughputcontrol|quotacheck|quotarecord|legacynastime|legacynascalls|sharedvirtualdisk|decrypt|encrypt|all)] [--file-system <label> | --all-file-systems]] [--version <1:2:2.1:3>] [<request-pattern>]

Description

The cifs-perf command reports SMB performance statistics for requests that are handled by the software. Requests that are auto-inquired (i.e. handled by the hardware) will not be shown in the command's output.

Usage:

cifs-perf [OPTION...] [<request-pattern>]

--access-type, -T all | local | localcvl | remotecvl | legacynas

Only displays statistics for accesses to objects of the given type.

--all-file-systems, -a

Shows per file system stats for all file systems.

--file-system, -f <label>

Shows per file system stats for the specified file system.

--reset, -r

Resets the statistics.

--summary, -s

Only displays a summary.

--type, -t elapsed | cpu | nvramreservation | lock | block | wfstime | wfscalls | remotetime | remotecalls | fscontext | security | accessandregionlock | readoplock | writeoplock | handleoplock | gns | renamelock | inq | requestsignatureverification | throughputcontrol | quotacheck | quotarecord | legacynastime | legacynascalls | sharedvirtualdisk | decrypt| encrypt | all

Only displays statistics of the given type.

--verbose, -v

Displays verbose information.

--version, -n <version>

Display stats for the specified SMB version only.

The supported version options are 1, 2, 2.1 and 3.

<request-pattern>

Only shows statistics for requests with names that contain the given string.

Examples

cifs-perf

Displays all available SMB performance statistics.

cifs-perf --version 2 negotiate

Displays performance statistics for SMB2 requests with names containing "negotiate".

Applies To

Cluster node
