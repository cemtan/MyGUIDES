cifs-smb2-status

Display the status of the SMB2 service (enabled or disabled).

Syntax

cifs-smb2-status

Description

The cifs-smb2-status command is used to display the status of the SMB2 service for the current evs. It has been superseded by the smb-min-supported-version and smb-max-supported-version commands.

Examples

cifs-smb2-status

Applies To

EVS

See Also

cifs-smb2-disable cifs-smb2-enable evs-select smb-max-supported-version smb-min-supported-version

Privilege Level

Supervisor
