tree-delete-job-list

List the status of tree delete jobs

Syntax

tree-delete-job-list [--active] [--terse] [--newest-first]

Description

Shows the status report of all the directory tree deleting jobs on the cluster node. The list includes all jobs that are active (running or waiting in the queue) and non-active (completed or aborted). However, it is limited to the most recent 160 jobs and will be erased on system reboot.

Note the details of the status report are described in the tree-delete-job-status man page.

--newest-first

This option causes newer jobs to be listed first. By default, older jobs are listed first.

--terse

Shows a short summary consisting of the file system, source tree, and the job state. By default, shows everything about the job.

--active

Lists active jobs only.

Examples

To show a detailed report for all jobs:

   $ tree-delete-job-list

To show a summary for all jobs:

   $ tree-delete-job-list --terse

To show the detailed report for active jobs only:

   $ tree-delete-job-list --active

Applies To

Cluster node
