package-diagnose

Try and diagnose why a package install failed

Syntax

package-diagnose <package-file>

Description

The package-diagnose command displays the last section of the install log for a package. It is intended to help understand why a package installation has failed.

Examples

package-diagnose mercury-package_6.5.1816.00.tar.gz

Applies To

Cluster node

See Also

package-get-status package-install

Privilege Level

Supervisor
