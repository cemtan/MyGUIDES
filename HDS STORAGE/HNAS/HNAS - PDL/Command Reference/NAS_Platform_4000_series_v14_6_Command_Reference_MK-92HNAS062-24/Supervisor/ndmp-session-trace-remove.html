ndmp-session-trace-remove

Stop per-session tracing for the specified NDMP subsystem

Syntax

ndmp-session-trace-remove <requests|env|statistics|backupq|notify|hsb-ds|ds|slave>

Description

Removes the specified NDMP subsystem from the output of the NDMP session trace.

Examples

Remove tracing for the NDMP request subsystem:

ndmp-session-trace-remove requests

Applies To

Cluster node

See Also

ndmp-session-trace ndmp-session-trace-add ndmp-trace-mask

Privilege Level

Supervisor
