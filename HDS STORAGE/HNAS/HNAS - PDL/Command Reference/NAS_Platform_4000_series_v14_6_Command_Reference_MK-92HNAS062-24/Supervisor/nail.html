nail

Send mail with attachment

Syntax

nail [-n] [-a <filename>] [-s <subject>] [to]

Description

nail is used to send a simple mail with (or without) an attachment from the 'flash' file system to a specified email address. All arguments are optional. The subject and destination email address will be prompted for if not specified. The file 'tmp' can also be attached.

Text can be entered for the body of an email unless the -n option is used. A single . on a line terminates the text input.

Options:

-n

Don't prompt for a body

-a <filename>

Add the attachment from the flash file system

-s <subject>

Set the subject for the email. Use quotes for subjects with spaces in

<to>

Email address to send to

The -a option can be used multiple times to attach multiple files and multiple recipients may be specified.

Examples

To send a recent packet capture file without a body

nail -n -a tmp -s "Your capture" support@example.com

Applies To

Cluster node

Privilege Level

Supervisor
