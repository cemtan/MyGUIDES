replication-schedule-list

List the object replication schedules

Syntax

replication-schedule-list [-n <num-entries>] [<policy name> <source fs label> [<schedule id>]]

Description

Lists schedules for object replication.

<policy name>

Name of the policy.

<source fs label>

Source file system label.

<schedule id>

ID of the schedule.

-n, --max <num-entries>

Lists at most num-entries schedules.

Examples

replication-schedule-list

Lists all object replication schedules on this EVS.

replication-schedule-list DefaultPolicy fs0

Lists all object replication schedules for the policy named 'DefaultPolicy' on source file system fs0.

replication-schedule-list DefaultPolicy fs0 3

Lists object replication schedule with ID 3 for the policy named 'DefaultPolicy' on source file system fs0.

Applies To

EVS
