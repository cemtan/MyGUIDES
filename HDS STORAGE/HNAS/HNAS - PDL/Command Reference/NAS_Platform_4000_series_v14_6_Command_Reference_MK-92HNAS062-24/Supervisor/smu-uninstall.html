smu-uninstall

Uninstall the SMU

Syntax

smu-uninstall

Description

Uninstalls the embedded SMU.

When a server is configured as a cluster, or joins an existing cluster, the embedded SMU is automatically uninstalled. This command allows the embedded SMU to be removed manually.

When it is complete, the uninstall process will leave an SMU backup in /root of the server's file system.

Examples

smu-uninstall

Applies To

Cluster wide
