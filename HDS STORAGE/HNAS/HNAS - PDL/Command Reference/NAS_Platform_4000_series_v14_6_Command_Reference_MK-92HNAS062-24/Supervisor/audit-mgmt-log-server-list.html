audit-mgmt-log-server-list

List configured management audit log servers

Syntax

audit-mgmt-log-server-list (no options)

Description

This command displays the currently configured list of CEF-aware syslog servers to which management audit events are sent.

Management audit events are sent to each configured CEF-aware syslog server.

Examples

To show the current configuration:

audit-mgmt-log-server-list
Management audit events are being sent to the following CEF server:
  logserver:612

Applies To

Cluster wide
