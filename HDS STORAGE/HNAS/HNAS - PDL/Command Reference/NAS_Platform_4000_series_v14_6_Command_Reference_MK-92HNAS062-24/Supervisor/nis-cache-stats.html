nis-cache-stats

Show/reset the NIS cache statistics

Syntax

nis-cache-stats [-r|--reset]
Alias:       ldap-cache-stats

Description

Reports statistics for the NIS cache. For nested queries, each netgroup successfully queried is treated as a distinct cache hit or cache miss. Performance indicates the time taken by each sample to perform a lookup. Entry size indicates the size of the entries added to the cache. In classic NIS mode, the size of an entry counts only its key, so each entry is of size 1. In NIS/LDAP mode, the size of an entry includes its comprising records as well as its key, so each entry is of size 1 or greater.

-r, --reset

Reset the statistics.

Examples

Display the NIS cache statistics since last reset:

        $ nis-cache-stats
        NIS cache hits:     2
        NIS cache misses:   6
        NIS cache lookups:  8
        NIS cache hit rate: 25%

         Last reset 1.662 min ago

                   |   |   |   |   |   |   |   |
                  1us  16 256u 4m 65ms 1s 17s 268s


        Entry sizes:
         Samples=       2  Total=       145  Min=    70  Av=    72  Max=    75
         Last reset 1.662 min ago

                 1|      #

                   |   |   |   |   |   |   |   |
                  1    16 256  4K  64  1M  16 256

Reset the NIS cache statistics:

        $ nis-cache-stats -r
        $
