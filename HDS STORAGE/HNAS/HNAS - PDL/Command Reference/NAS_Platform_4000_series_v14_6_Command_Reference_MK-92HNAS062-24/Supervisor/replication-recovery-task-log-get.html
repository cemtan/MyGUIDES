replication-recovery-task-log-get

Display a recovery task's log

Syntax

replication-recovery-task-log-get (-t|--task-id <task-id>) [--do-not-truncate]

Description

Display the contents of a recovery task's log.

-t, --task-id <task-id>

The task whose log should be displayed.

--do-not-truncate

Display the entire log, do not truncate it.

Examples

To display the contents of the log of a task with id 5afdcd94-4628-11cb-a005-ac2a04ffb80f:

replication-recovery-task-log-get --task-id 5afdcd94-4628-11cb-a005-ac2a04ffb80f
