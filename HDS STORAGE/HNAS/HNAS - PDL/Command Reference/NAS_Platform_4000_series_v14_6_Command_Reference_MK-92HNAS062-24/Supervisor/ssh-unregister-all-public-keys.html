ssh-unregister-all-public-keys

Delete all stored SSH public keys

Syntax

ssh-unregister-all-public-keys [<user-name>]

Description

Deletes all SSH public keys from the server's authorized list.

Options:

user-name

Name of user to delete

Examples

ssh-unregister-all-public-keys


ssh-unregister-all-public-keys johnsmith

Deletes public keys for johnsmith only.

See Also

ssh-unregister-public-key

Privilege Level

Supervisor
