virusscan-mode-set

Set virus scan mode

Syntax

virusscan-mode-set [--confirm] <RPC|ICAP>

Description

Set the virus scanning mode.

Options:

--confirm

Confirm that the virus scanning mode will be set without a further prompt.

<mode>

Virus scanning mode to set. Options are RPC and ICAP. The default mode is RPC. Switching virus scanning mode will cancel all outstanding scans on the current EVS and files can be denied access after the switchover until at least one virus scanner has successfully registered. In particular, RPC virus scanners will not re-register immediately after switchover.

Examples

To set the EVS in context to RPC virus scanning mode without a further prompt:

server:$ virusscan-mode-set --confirm RPC

Applies To

EVS
