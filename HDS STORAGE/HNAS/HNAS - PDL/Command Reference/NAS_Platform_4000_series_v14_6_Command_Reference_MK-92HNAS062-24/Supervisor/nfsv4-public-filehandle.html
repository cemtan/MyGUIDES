nfsv4-public-filehandle

nfsv4 public filehandle

Syntax

nfsv4-public-filehandle 
	nfsv4-public-filehandle set <pseudo-filesystem path>
	nfsv4-public-filehandle unset
	nfsv4-public-filehandle show

Description

This command is used to manage the nfsv4 public file handle. By default the public file handle is equivalent to the pseudo file system root.

nfsv4-public-filehandle [show | unset | set <pseudo-fs path>]

Examples

nfsv4-public-filehandle set /export1


nfsv4-public-filehandle show


Current public file handle: /export1


nfsv4-public-filehandle unset


nfsv4-public-filehandle show


Current public file handle: /

Applies To

EVS

Privilege Level

Supervisor
