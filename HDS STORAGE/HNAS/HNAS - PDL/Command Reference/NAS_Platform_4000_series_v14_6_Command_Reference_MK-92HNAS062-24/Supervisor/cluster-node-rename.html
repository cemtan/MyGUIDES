cluster-node-rename

Rename a Cluster Node

Syntax

cluster-node-rename <new name>

Description

The cluster-node-rename command will rename the cluster node on which it is run. The setting is persistent and takes immediate effect on most components. Reconnect to the server to see the effect reflected in the prompt.

Usage:

<new name>

The new cluster node name

Note:

Changing the cluster node name will trigger creation of a new self-signed SSL/TLS certificate for the SOAP HTTPS management server and (if present) for the embedded SMU. However, if you had installed a custom certificate it will not be affected (but you may want to consider updating it to match the new domain name).

Examples

server:$ cluster-node-rename MYNODE2
Cluster Node successfully renamed to: MYNODE2
server:$

Applies To

Cluster node
