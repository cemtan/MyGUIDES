tcpipperfstats

Report TCP/IP connection information.

Syntax

tcpipperfstats [-r] [-s] [-f]

Description

tcpipperfstats reports sequence numbers for all existing TCP connections.

The tcpipperfstats command takes the following options:

-s, --store

Remember current values as a base for relative sequence numbers.

-f, --forget

Stop using any base values that were set with --store.

-r, --relative

Report sequence numbers relative to the initial sequence numbers used in the TCP/IP handshake.

Examples

To get an idea of how much data has been used by tcpip connections:
     tcpipperfstats -r


To get an idea of how much data is being used per second on tcpip connections:
     tcpipperfstats -s
     sleep 1
     tcpipperfstats

Applies To

Cluster node
