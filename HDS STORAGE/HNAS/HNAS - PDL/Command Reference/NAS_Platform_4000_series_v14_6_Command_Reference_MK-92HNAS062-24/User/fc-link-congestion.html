fc-link-congestion

Show FC link congestion stats

Syntax

fc-link-congestion [-a]

Description

This command is used to display Fibre Channel link congestion statistics for the server.

Link congestion is defined as time during which the FC hardware is blocked from transmitting a frame by lack of buffer credit. The congestion percentage is measured over 2 minutes, based on a 10 Âµs timer. The 24-hour peak and average percentages are updated once every 24 hours.

Options

-a, --all

display statistics for all interfaces, not just enabled ones

Examples

server:$ fc-link-congestion
Link congestion (%):
 FC 1        Last: 0 Peak24: 0 Avg24: 0
 FC 3        Last: 0 Peak24: 1 Avg24: 0

Applies To

Cluster node
