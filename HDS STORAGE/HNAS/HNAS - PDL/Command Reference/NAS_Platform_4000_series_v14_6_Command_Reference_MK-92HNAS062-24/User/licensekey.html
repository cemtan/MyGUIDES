licensekey

Manipulate software license keys

Syntax

licensekey operation
Where operation is:
list   [-v] [service]
add    <key>
delete <key>
delall [--confirm]

Description

View, add or delete software license keys

Usage:

list

Display all of the license keys on the server, indicating what service(s) each license relates to and when it will expire. Select -v if you want to view the services vertically (down the side as opposed to across the top). Specify a service if you only want to view what is licensed on all keys for one service.

Note: As of version 13.1 the storage capacity license has been deprecated. The TB values in the licenses are not enforced by the license management system and the system will honour the maximum value supported for the specific platform.

add <key>

Add the specified license key. Contact your support provider for more information on obtaining license keys.

delete <key>

Delete the specified license key.

delall [--confirm]

Delete all the license keys. -C or --confirm must be specified to delete all license keys.
