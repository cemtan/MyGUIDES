iscsi-domain

Set, display or unset the iSCSI domain

Syntax

iscsi-domain [-d] [<DNS domain>]

Description

This command is used to set the DNS domain used by iSCSI. iSCSI qualified names should be unique across the world, so they're built, in part, from the DNS domain of an organization and the date at which the DNS domain was owned by the organization. For example, if a company owned the domain example.com in March 2004, the iSCSI qualified name for a target with that domain would include "2004-03.com.example".

Using this command provides the iSCSI service with a DNS domain from which to create its unique names, and does not update the DNS configuration in any way.

iscsi-domain [OPTIONS] [<domain>]
    Sets, displays or unsets the domain.
    domain
        The new domain name.


    OPTIONS
    --delete, -d
        Unset the domain.

Examples

iscsi-domain
iscsi-domain example.com
iscsi-domain -d

Applies To

EVS
