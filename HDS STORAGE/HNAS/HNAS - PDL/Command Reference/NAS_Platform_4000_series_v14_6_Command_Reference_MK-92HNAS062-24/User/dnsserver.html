dnsserver

Define DNS servers

Syntax

dnsserver operation
where 'operation' is one of
	list
	add server1 server2 ...
	del server

Description

The dnsserver command allows the DNS server list to be manipulated. When a name is looked up, the DNS client queries the servers in the order listed.

The number of DNS servers that can be configured is 3: this is a compromise between providing resilience and performance of host name lookup as each failed lookup impacts DNS lookup speed.

Examples

To list out the DNS servers being used:

dnsserver list

To add a DNS server:

dnsserver add 10.1.1.14

To delete a DNS server:

dnsserver del 10.1.1.14
