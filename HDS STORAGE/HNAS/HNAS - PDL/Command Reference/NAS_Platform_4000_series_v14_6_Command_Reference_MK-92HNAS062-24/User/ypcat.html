ypcat

Prints values of all keys in an NIS database

Syntax

ypcat [-k] mapname

Description

Displays the values of all keys in the specified yp database.

Use the -k option to display the relevant keys as well as the values.

Examples

Display all the keys and their values from the passwd.byname map:

ypcat -k passwd.byname

Display all the values from the group.byname map:

ypcat group.byname

Applies To

EVS

See Also

ypmatch ypwhich

Privilege Level

User
