ssrun

Run script

Syntax

ssrun -c <command line> | [-x] <script name>

Description

Runs a script file from the flash.

Options:

-x

Echo commands to standard error before running them.

-c <command line>

Execute a single command line instead of a script file.

The -c form is useful with commands like time and repeat to allow them to execute more than one command at once.

Examples

Run the script file test.scr, echoing it to the invoking console:

ssrun -x test.scr

This performs much the same function as time bash -c "ls; ls" on a Unix box:

time ssrun -c "ls; ls"

One way of nesting arguments which require quotes is illustrated by:

server:$ ssrun -c "ssrm 'foo "'"bar"'" baz'"
Failed to delete foo "bar" baz, Invalid flash file name or path
server:$

Cluster node
