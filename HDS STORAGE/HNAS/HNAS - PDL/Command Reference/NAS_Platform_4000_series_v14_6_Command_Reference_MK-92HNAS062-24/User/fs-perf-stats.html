fs-perf-stats

Report average file system performance statistics

Syntax

fs-perf-stats [--by-fs [-p <pnode-id>] [-v]] [--by-pnode [-f <fs-label>] [-v]] [--disable] [--enable] [--reset]

Description

Displays average file system performance statistics.

Usage:

fs-perf-stats [OPTION]...

--by-fs [--pnode-filter <pnode-id>] [--verbose]

Display stats by file system.

--pnode-filter, -p

Only display stats for the specified pnode.

--verbose, -v

Display stats for all operations.

--by-pnode [--fs-filter <fs-label>] [--verbose]

Display stats by the pnode from which operations originated.

--fs-filter, -f

Only display stats for the specified file system.

--verbose, -v

Display stats for all operations.

--reset, -r

Reset statistics.

--enable
--disable

Enable or disable statistics gathering.
