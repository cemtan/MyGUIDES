exit

Exit the current shell

Syntax

exit

Description

The exit command disconnects from the current console. If this is a remote console (such as ssc), then the connection will be closed. The command history is discarded. Ctrl-D has much the same effect.

Examples

server:$ whoami
User supervisor currently logged in.
server:$ devuser on; logout; login dev ed6nO9rTw0CJOu5o
server:$ whoami
User dev currently logged in.
server:$ exit
Connection closed by foreign host.
$

Applies To

Console

See Also

Any
