filesystem-specifier

Description

Many commands require the user to uniquely specify a filesystem. This may be done using one the following methods.

filesystem name

The filesystem name is a textual identifier. It is shown in the Name column by filesystem-list and filesystem-object-list.

device id

The device id is a numeric identifier. It is shown in the Dev column by filesystem-list and filesystem-object-list.

permanent id

The permanent id is a alphanumeric identifier. It is shown in the Permanent ID column by filesystem-object-list.

Examples

filesystem-list

Show filesystem information for a system with filesystem-list:

titannet1-a(FSA):$ filesystem-list
 Instance name      Dev   On span            State  EVS  Cap(GB)  Confined Flag
 -----------------  ----  -----------------  -----  ---  -------  -------- ----
 Alpha              1062  Span0               UnMnt   1       12        -
 Beta               1060  Span1               UnMnt   1     1023        -
 Gamma              1061  Span2               UnMnt   1       19        -

filesystem-object-list

Show filesystem information for a system with filesystem-object-list:

titan1-a(FSA):$ filesystem-object-list

 ----  ---  ----------------  --------------------  --------------------
