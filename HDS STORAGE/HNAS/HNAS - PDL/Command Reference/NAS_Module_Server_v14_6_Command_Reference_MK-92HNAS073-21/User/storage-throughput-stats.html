storage-throughput-stats

Show storage throughput statistics

Syntax

storage-throughput-stats [-a] [-r]

Description

This command is used to display or reset statistics on the I/O throughput of the HNAS storage interface layer fast path.

Options

-a, --all

no effect on unified hardware platform

-r, --reset

reset (non-instantaneous) statistics

Description of output

The figures for instantaneous and peak throughput are based on a 1-second measurement interval. The second section shows total byte counts.

Examples

server:$ storage-throughput-stats
Throughput (B/s):         RX  (Peak)                  TX  (Peak)
 Backplane           4600832  ( 107742208)      13137408  ( 163515392)


Total:                                 RX                          TX
 Backplane                  1825436553216               2269002734080


Last reset: 2015-12-08 11:58:47+00:00

Applies To

Cluster node
