nis-server

Configure NIS servers

Syntax

nis-server [show] | add [-p N] server | remove server | remove-all

Description

Displays and manipulates the servers which the NIS client will attempt to contact.

The following command forms are permitted:

nis-server [show]

nis-server add [-p 1|2|3] server-ip

nis-server remove server-ip

nis-server remove-all

Priority 1 is the highest (i.e. the most likely to be chosen), priority 3 is the lowest. By default servers are added with priority 3.

For servers of equal priority, the server chosen will be the one with the fastest response time.

Manually configured servers will be chosen in preference to broadcast servers.

The server-ip can be an IPv4 address or an IPv6 address.

Examples

To display the current setting of NIS client servers:

nis-server
--[Configured Servers]---------------
-- 3 10.1.16.41
--[Responded to Broadcast]-----------
sb 4 172.18.0.1

s indicates the currently bound server. b indicates a server located via broadcast.

To add an NIS client server:

nis-server add -p 1 10.1.16.31


nis-server add -p 1 fdca:f995:220a:0:58c1:e76e:d2c4:9f1c

To remove an NIS client server:

nis-server remove 10.1.16.31

To remove all NIS client servers (for example, in order to use only future broadcast servers):

nis-server remove-all

Applies To

EVS
