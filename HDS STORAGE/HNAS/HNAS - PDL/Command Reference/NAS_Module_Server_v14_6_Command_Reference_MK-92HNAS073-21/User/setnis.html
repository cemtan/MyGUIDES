setnis

Configure NIS support.

Syntax

setnis [ -d [domainname]] [-b 0|1] [-e 0|1] [-t [<timeout>]] [-T [<ttl>]] [-w [<wait>]] [-a <addr> [-p 1|2|3]] [-r <addr>] [-R] [-s] [-m] [-l] [-L]

Description

setnis displays and modifies the NIS client configuration.

setnis is deprecated and has been replaced by a series of simpler commands:

setnis -d is replaced by domainname.

setnis -b is replaced by nis-broadcast-mode.

setnis -T is replaced by nis-cache-ttl.

setnis -s is replaced by nis-rebind.

setnis -a,-r,-R are replaced by nis-server.

setnis -l,-L are replaced by nis-show.

setnis -e is replaced by nis-state.

setnis -t,-w are replaced by nis-set.

Applies To

EVS security context
