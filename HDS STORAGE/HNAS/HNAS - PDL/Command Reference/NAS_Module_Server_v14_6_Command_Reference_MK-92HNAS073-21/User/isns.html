isns

Manages iSNS servers

Syntax

isns operation
where 'operation' is one of:
  add [--port <port number>] <ip address>
  del <ip address>
  list
  refresh [--all]

Description

This command manages iSNS (Internet Storage Name Service) servers. iSNS provides a central database of iSCSI initiator and target names.

isns list

Lists all configured iSNS servers.

isns add [--port <port number>] <ip address>

Adds a new iSNS server.

-p, --port <port number>

The port number. Default is 3205.

ip address

The IP address of the iSNS server to add.

isns del <ip address>

Removes an iSNS server.

ip address

The IP address of the iSNS server to remove.

isns refresh [--all]

Refreshes iSCSI target registration.

--all, -a

Refresh all EVSs.

Examples

  $ isns list
  192.168.49.23:3205
  [fdca:f995:220a:480:2::1]:3205


  $ isns add 192.168.49.23
  $ isns add fdca:f995:220a:480:2::1
  $ isns add --port 49152 192.168.49.23
  $ isns add --port 48653 fdca:f995:220a:480:2::1


  $ isns del 192.168.49.23
  $ isns del fdca:f995:220a:480:2::1

Applies To

EVS
