sleep

Pause for the specified time

Syntax

sleep number[suffix]
Dev-level options: --support-mse-abort|-s
Alias:       delay

Description

The sleep command sleeps for the specified amount of time (in seconds by default). Useful in scripts, or multi-line commands.

The parameter is a number followed by an optional suffix. The number can be integer or floating-point. The suffix can be any of: "ms" for milliseconds, "s" for seconds (the default), "m" for minutes, "h" for hours, or "d" for days.

Examples

# 10 seconds
 sleep 10


# 0.1 seconds
 sleep 0.1


# 2 days
 sleep 2d


ps -r;sleep 30;ps -s

Applies To

Cluster node

Privilege Level

User
