snmp-communities

Manipulate authentic SNMP communities

Syntax

snmp-communities operation
where 'operation' is one of:
list
add <CommunityName>
del <CommunityName>
delall [--confirm]
modcom <NewCommunityName> <CurrentCommunityName>
authtrap
noauthtrap
Alias:       snmpauth

Description

This command is not supported on this platform. Use the Maintenance Utility to view and modify SNMP configuration.

Examples

Applies To

Cluster wide
