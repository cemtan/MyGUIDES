password

Change password of a user

Syntax

password

Description

This command can only be used for users named "root", "supervisor" or "manager".

Changes the password for a management user. This command takes no arguments but will prompt for a user name and new password.

To change the password of a management user other than the currently logged in one requires supervisor level user access.

The current password does not need to be specified when changing a management user password except when changing the root password. In this case the command will prompt for the current password.

Note: an external SMU needs to know about any password changes for users (usually supervisor) that it manages HNAS with. For this reason, if an external SMU is used, it is recommended that changes to passwords for those HNAS users are made using the page provided by the SMU, not this command, so that the SMU continues to be able to manage HNAS.

Applies To

Cluster wide
