ver

Report server versions

Syntax

ver [-h|-s|-v|-t]

Description

The ver command displays software and hardware versions. Without any options it gives an overall summary. The following options can be used to obtain more detail:

-h, --hw

Display detailed hardware versions

-s, --sw

Display detailed software versions

-v, --vlsi

Display detailed VLSI versions

-t, --type

Display board type summary

Examples

ver

ver -h

Applies To

Cluster node
