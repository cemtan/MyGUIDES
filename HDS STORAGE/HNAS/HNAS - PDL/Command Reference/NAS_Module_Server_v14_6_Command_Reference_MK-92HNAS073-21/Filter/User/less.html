less

Pause output every page

Syntax

... | less

Description

This is a very simple implementation of the familiar unix "less" filter. Command output can be piped through less to pause at the end of each page (25 lines) of output. It may not be used as a command.

less takes no options. Keystrokes when it pauses are:

q

don't output any further output from this command

enter

print just the next line

space

print another page

If left waiting for input for half an hour, then less times-out as if "q" had been pressed.

Examples

Show the event log a page at a time:

event-log-show | less
