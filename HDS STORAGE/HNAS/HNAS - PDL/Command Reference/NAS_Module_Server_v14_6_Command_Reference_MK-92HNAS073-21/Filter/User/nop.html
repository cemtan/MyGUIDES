nop

Copy input unchanged

Syntax

... | nop

Description

grep's output may be piped through nop to prevent it from coloring matches. This may be useful in scripts, where the escape sequences used for coloring make parsing harder.

Coloring also occurs in the output of many hex dumping commands or filters. Their output may be piped through nop to remove ANSI coloring escape sequences.

Examples

1. To look for connections which have serviced hundreds of requests:

      connection all | grep "[0-9][0-9][0-9] requests" | nop

2. Disable hex dump coloring:

      disk-read 100BLK --nbytes 1024 -f fs | nop

See Also
