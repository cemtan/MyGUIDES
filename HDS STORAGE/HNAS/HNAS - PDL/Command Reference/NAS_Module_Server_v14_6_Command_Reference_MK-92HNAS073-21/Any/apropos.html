apropos

Search command names and descriptions and man pages

Syntax

apropos <regular-expression>

Description

This command searches through man pages and CLI help for the specified POSIX extended regular expression, which uses the same syntax as 'grep -E'.

The regular expression must not be more than 256 bytes long. If non-Ascii characters are used, this length will correspond to fewer than 256 characters. If the maximum length is exceeded, 'apropos' will refuse to run.

Examples

apropos statistics

Applies To

Cluster node
