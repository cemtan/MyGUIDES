remote-nfs-errors

Report remote NFS protocol errors

Syntax

remote-nfs-errors [-r|--reset]

Description

The remote-nfs-errors command is used to display errors returned from remote NFS servers.

Usage:

--reset, -r

Resets the error log.

Applies To

Cluster node

See Also

protocol-error-logging remote-nfs-exports remote-nfs-perf remote-nfs-stats

Privilege Level

Supervisor
