arp

ARP table manipulation

Syntax

arp [-a <addr>] [-d <addr>] [-f] [-i <interface>]

Description

The 'arp' command displays and manipulates the Address Resolution Protocol (ARP) table.

Without any options, the 'arp' command displays the contents of the ARP table.

-f, --flush

Empties the ARP table.

-d, --delete <addr>

Deletes the ARP table entry for address <addr>.

-a, --address <addr>

Displays the ARP table entry for address <addr>.

-i, --interface <name>

Limits an operation to ARP entries associated with a specific interface.

Examples

To clear the ARP table of all entries:

    arp --flush

To delete an ARP entry:

    arp --delete 192.168.1.2

To display an entry for a specific IP address and interface:

   arp --address 192.168.1.2 --interface ag1

Applies To

Cluster node

Privilege Level

Supervisor
