fs-no-dos-offline-for-xvl

set or clear the No-DOS-Offline-for-XVL flag for a file system

Syntax

fs-no-dos-offline-for-xvl --file-system <file-system> [--set|--clear]

Description

Specifies whether or not a file system prevents the setting of the DOS offline attribute for externally migrated files by setting or clearing the No-DOS-Offline-for-XVL flag for the file system.

Setting the DOS offline attribute for an externally migrated file prevents the file's metadata from being retrieved from the remote server under certain circumstances, for example when a file is clicked on using Windows Explorer in Windows 7. Configuring a file system not to have the DOS offline attribute set for externally migrated files will therefore impose a performance overhead during file navigation as this metadata will no longer be prevented from being fetched from the remote server.

Usage:

-f, --file-system <file-system>

The file system to which this command refers.

-s, --set

(optional) Sets the No-DOS-Offline-for-XVL flag for the file system, which prevents the setting of the DOS offline attribute for externally migrated files.

-c, --clear

(optional) Clears the No-DOS-Offline-for-XVL flag for the file system, which allows the setting of the DOS offline attribute for externally migrated files.

If neither the --set nor --clear option is specified, the command reports whether the No-DOS-Offline-for-XVL flag is set or clear.

Examples

fs-no-dos-offline-for-xvl -f RNA --set

Prevents the setting of the DOS offline attribute for externally migrated files for file system RNA.

fs-no-dos-offline-for-xvl -f RNA

Reports if the No-DOS-Offline-for-XVL flag is set or clear for the file system RNA.

Applies To

File system
