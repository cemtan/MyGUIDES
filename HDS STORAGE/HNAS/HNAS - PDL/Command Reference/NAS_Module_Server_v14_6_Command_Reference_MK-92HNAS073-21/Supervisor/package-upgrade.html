package-upgrade

Upgrade software running on server

Syntax

package-upgrade [-q] [-r] local-file

Description

**** WARNING ****

On VSP with NAS modules (F/G4x0, F/G6x0, F/G8x0) only use this command under instruction from your support provider. NAS upgrades and downgrades should only be carried out using the Maintenance Utility on the MPC or SVP.

**** WARNING ****

The package-upgrade command upgrades the server software package.

The full path to the package name must be specified. Other options:

-r

Reset the server after uploading the package

-q

Don't display progress

The upgrade command only runs on ssc connections

Examples

Upgrade the server with 4.0.824q and reboot it to run the new software:

ssc 192.168.0.1 package-upgrade -r packages/4.0.824q.pkg

Applies To

Cluster node

See Also

package-list package-remove package-set-default

Supervisor
