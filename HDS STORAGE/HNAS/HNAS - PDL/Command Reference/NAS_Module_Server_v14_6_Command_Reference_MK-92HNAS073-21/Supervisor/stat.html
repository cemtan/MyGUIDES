stat

Displays file information

Syntax

stat <file1> [<file2>...]

Description

The stat command is used to display information about files or directories.

Examples

To display information about the file 'file1' and the directories 'dir1' and 'dir2' in the current working directory:

stat file1 dir1 dir2

Applies To

EVS

See Also
