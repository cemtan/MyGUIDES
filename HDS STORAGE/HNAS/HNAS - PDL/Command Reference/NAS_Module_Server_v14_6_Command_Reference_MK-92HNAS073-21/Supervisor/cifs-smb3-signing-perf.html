cifs-smb3-signing-perf

Display SMB3 signing performance statistics

Syntax

cifs-smb3-signing-perf [--reset|--signing-rate|--summary]

Description

The cifs-smb3-signing-perf command reports SMB3 signing performance statistics. This includes statistics for the HNAS acting as both server (normal file-serving) and client (to Domain Controllers).

--reset, -r

Resets the statistics.

--signing-rate, -i

Shows the average signing throughput since the statistics were last reset.

--summary, -s

Disables the drawing of the performance stats graphs.

These fields apply to responses sent to an SMB3 client and requests to a server:

Send Data Transfer
Send Calculate Signature

Time spent sending data from FPGA to host and calculating the signature.

These fields apply to requests from an SMB3 client and responses from a server:

Receive Data Transfer
Receive Verify Signature

Time spent sending data from FPGA to host and verifying the signature.

On Unified models, SMB3 signing is performed in FPGA which removes the need for transferring data from FPGA to the host:

Send VLSI Calculate Signature

Time spent calculating the signature for a response to a client or a request to a server.

Receive VLSI Verify Signature

Time spent verifying the signature for a request from a client or a response to a server.

Applies To

Cluster node
