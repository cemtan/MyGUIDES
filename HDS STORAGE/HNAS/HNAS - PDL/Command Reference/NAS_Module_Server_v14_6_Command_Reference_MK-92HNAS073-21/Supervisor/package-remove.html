package-remove

Delete a package

Syntax

package-remove [-f] <package-name>

Description

**** WARNING ****

On VSP with NAS modules (F/G4x0, F/G6x0, F/G8x0) only use this command under instruction from your support provider. NAS upgrades and downgrades should only be carried out using the Maintenance Utility on the MPC or SVP.

**** WARNING ****

The package-remove command removes a single package from flash.

Options:

-f, --force

Don't check if it is safe to remove the package before removing it. This is useful if the package is in a bad state, as in that circumstance the checks themselves can cause errors.

Examples

package-remove mercury-6.5.1839.00.tar.gz

Applies To

Cluster node
