virusscan-perf

Display Virus scan performance statistics

Syntax

virusscan-perf [[--reset] | [--summary] [--verbose]] [<request-pattern>]

Description

The virusscan-perf command reports the virusscan system's performance statistics.

Usage:

virusscan-perf [OPTION...] [<request-pattern>]

--reset, -r

Reset the statistics.

--summary, -s

Only display a summary.

--verbose, -v

Display verbose information.

<request-pattern>

Only show statistics for requests with names that contain the given string.

Applies To

Cluster node

See Also

enhanced-perf-stats virusscan virusscan-failures

Privilege Level

Supervisor
