span-show-lock-owner

Display users of Span Manager locks

Syntax

span-show-lock-owner

Description

The server has software that builds system drives into spans, divides spans into filesystems, stores span configurations on disk and reloads them at boot time. This software is called 'Span Manager'. Span Manager is protected by a system of locks, so that (for example) one user can't mount a filesystem that another user is deleting.

This command prints the IDs and backtraces of any threads that are using these locks.

Examples

span-show-lock-owner

Applies To

Cluster node

See Also

Privilege Level

Supervisor
