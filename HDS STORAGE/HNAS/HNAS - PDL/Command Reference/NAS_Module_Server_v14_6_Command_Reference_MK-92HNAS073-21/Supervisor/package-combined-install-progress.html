package-combined-install-progress

Check the progress of a combined package install.

Syntax

package-combined-install-progress  <PACKAGE FILE>

Description

The package-combined-install-progress command is used to check on the progress of the install of a combined package. The full path to the package name must be specified. The package-combined-install command only runs on ssc connections.

Examples

Check install progress for a combined package:

package-combined-install-progress hnas-package.tar

Applies To

Cluster node

See Also

package-combined-check package-combined-install package-list package-remove package-set-default

Privilege Level

Supervisor
