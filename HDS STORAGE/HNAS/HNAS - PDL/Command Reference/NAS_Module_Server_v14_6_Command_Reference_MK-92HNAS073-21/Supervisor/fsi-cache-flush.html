fsi-cache-flush

Flush FSI cache of all clean data

Syntax

fsi-cache-flush 
Alias:       cache-flush

Description

The FSI cache mainly contains file attributes and a path-to-file index but it is also used for cached credentials, cached symbolic link targets, cached security descriptors and for free-space bitmap caching in the file system repair utilities. The FSI cache shares memory with a general purpose heap. It can sometimes be advantageous to empty the FSI cache before a particularly heap-intensive operation or to verify that cached information can be regenerated.

Examples

To flush the FSI cache on every node in the cluster:

for-each-cnode fsi-cache-flush

Applies To

Cluster node
