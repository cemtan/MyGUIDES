replication-policy-delete

Delete an object replication policy

Syntax

replication-policy-delete [--force] [--preserve-history] <policy id>|<policy name> <source fs label>

Description

Deletes a policy for object replication and all its schedules and history entries.

<policy id>

ID of the policy.

<policy name>

Name of the policy.

<source fs label>

Source file system label.

--force, -f

Forcibly delete an object replication policy that is currently running.

--preserve-history, -p

Preserve the replication history for this policy. By default the replication history is also deleted.

Examples

replication-policy-delete DefaultPolicy fs0

Deletes the object replication policy named 'DefaultPolicy' for source file system fs0. The schedules and history entries associated with the policy are also deleted; a dump of the history entries can be found in the dblog.
