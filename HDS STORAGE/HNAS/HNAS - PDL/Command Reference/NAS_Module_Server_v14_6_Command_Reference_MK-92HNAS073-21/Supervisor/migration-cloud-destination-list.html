migration-cloud-destination-list

List the external cloud migration destinations

Syntax

migration-cloud-destination-list [<destination name>]|[<destination id>]

Description

Lists cloud external migration destinations. If no name or ID is specified all destinations are listed.

<destination name>

Lists details about the specified destination name only.

<destination id>

Lists details about the specified destination ID only.

Examples

migration-cloud-destination-list CloudDest

List details about destination CloudDest

Applies To

Global security context
