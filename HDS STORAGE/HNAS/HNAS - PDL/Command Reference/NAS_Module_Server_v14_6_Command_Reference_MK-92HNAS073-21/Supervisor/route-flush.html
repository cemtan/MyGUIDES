route-flush

Flush the router cache

Syntax

route-flush [--confirm]

Description

The route-flush command removes all dynamic host routes from the routing table.

This operation only applies to the cluster node on which the command is executed.

To flush other dynamic routes from the routing table use the 'irdp flush', 'ndp-flush' and 'rip flush' commands.

The following options are supported:

-c,--confirm

The --confirm option is required to verify that the command should delete dynamic host routes from the routing table.

Examples

To flush dynamic host routes from the route cache:

$ route-flush --confirm
Route cache flushed
$

Applies To

Cluster node
