audit-mgmt-log-server-add

Add a new management audit log server

Syntax

audit-mgmt-log-server-add <server> [ --port <port> ]

Description

This command configures the server to send management audit events to an external CEF-aware syslog server. Management audit events are always sent in CEF form.

Each use of this command specifies a single CEF-aware syslog server to add. Up to two such servers may be configured. Management audit events are sent to each configured CEF-aware syslog server.

The CEF-aware syslog server can be specified as a DNS hostname, or an IP address (IPv4 or IPv6). At this time, only simple TCP (non-TLS) transfers are supported.

If the port number for the CEF-aware syslog server is not specified, the UDP default of 514 is assumed, even though we send events over unsecured TCP.

Examples

To send management audit events to a CEF-aware syslog server "logserver" on port number 612:

audit-mgmt-log-server-add logserver --port 612

Applies To

Cluster wide
