cluster-node-delete

Remove a node from cluster

Syntax

cluster-node-delete <node name> <node id>

Description

The cluster-node-delete command will remove a server from a cluster and delete its configuration. This command may be run on any server in the cluster.

The "cluster-show -y" command will show all the <node id>/<node name> pairs in the cluster.

Usage:

<node name>

The name of the node to be removed

<node id>

The cluster node id of the node to be removed

Examples

server:$ cluster-node-delete NODE-2 2
This command will remove the configuration of (and reboot) node NODE-2
Do you want to continue?(Y/N)[N]:
y

Applies To

Cluster node

See Also

cluster-join cluster-show

Privilege Level

Supervisor
