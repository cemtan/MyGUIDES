rest-server-mode

Set the RestAPI server mode

Syntax

rest-server-mode [--native | --legacy]

Description

Displays or sets the mode that the RestAPI server runs in. This command will be removed from a future release, as legacy mode has been removed. The mode will always be reported as native, and can no longer be changed.

Options

--legacy

Set the RestAPI server to run in legacy mode next time it is restarted. Note that this option will always fail now that legacy mode has been removed.

--native

Set the RestAPI server to run in native mode next time it is restarted.

For further information about each mode, use 'man rest-server'.

Examples

$ rest-server-mode
The RestAPI server mode is: native


$ rest-server-mode --legacy
Legacy mode has been removed from the RestAPI server, and can no longer be enabled.

Applies To

Cluster wide
