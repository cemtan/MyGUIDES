smtp-statistics

Displays SMTP client statistics

Syntax

smtp-statistics [-r]

Description

Display or reset various statistics maintained with respect to communications with an SMTP server.

-r

Reset all counters to zero

Examples

Show current stats:

smtp-statistics

Reset current stats:

smtp-statistics -r

Applies To

Cluster node

See Also

smtp-clear-queue smtp-retry-connection

Privilege Level

Supervisor
