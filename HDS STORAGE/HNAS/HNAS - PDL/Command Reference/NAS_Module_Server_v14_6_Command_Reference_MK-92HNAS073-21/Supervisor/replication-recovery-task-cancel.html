replication-recovery-task-cancel

Cancel replication recovery tasks

Syntax

replication-recovery-task-cancel [--confirm] {--all | (-t|--task-id <task-id>) | (-f|--file-system <file-system>)}

Description

Cancel replication recovery tasks.

--confirm

Don't ask for confirmation.

--all

Cancel all active (running and queued) recovery tasks.

-t, --task-id <task-id>

Cancel a specific recovery task.

-f, --file-system <file-system>

Cancel the recovery task of a file system.

Examples

To cancel a task with id 5afdcd94-4628-11cb-a005-ac2a04ffb80f:

replication-recovery-task-cancel --task-id 5afdcd94-4628-11cb-a005-ac2a04ffb80f

Applies To

EVS
