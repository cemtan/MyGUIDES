replication-recovery-logging-policy-set

Set the logging policy for replication recovery tasks

Syntax

replication-recovery-logging-policy-set 
[--reset]
[--max-logs-per-pnode <value>]
[--max-log-size-in-bytes <value>]
[--max-entry-size-in-bytes <value>]

Description

Set the logging policy for replication recovery tasks.

--reset

Reset all to their default value.

--max-logs-per-pnode

Controls the maximum number of logs that will be kept on each physical node.

--max-log-size-in-bytes

Controls the maximum size of a log.

--max-entry-size-in-bytes

Controls the maximum size of each entry in a log.

Examples

To keep a maximum of 2 replication recovery logs per physical node:

replication-recovery-logging-policy-set --max-logs-per-pnode 2

Applies To

Cluster wide
