migration-delete-path

Deletes the specified migration path if it is inactive.

Syntax

migration-delete-path -s <fs label> [-v <vivol name>]

Description

Options:

-s, --source <fs label>

Source file system.

-v, --vivol <vivol name>

Name of source Virtual Volume (optional).

Deletes a Data Migration path. This command may only be completed when there are no cross-file-system links from the source of the path to the target. It may be necessary to update the status of the path using migration-update-paths before this will work.

Examples

migration-delete-path -s fs1

Delete the migration path for file system fs1.

migration-delete-path -s Accounts -v HR

Delete the migration path for Virtual Volume HR on file system Accounts.

Applies To

EVS
