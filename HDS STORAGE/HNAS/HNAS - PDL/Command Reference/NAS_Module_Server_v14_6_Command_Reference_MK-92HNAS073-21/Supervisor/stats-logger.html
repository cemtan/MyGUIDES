stats-logger

Turns statistics logger on or off

Syntax

stats-logger [on|off]

Description

This command is used to turn the internal statistics logging on or off. The statistics logger is turned on and off on a per board basis.

Examples

Turn statistics logging on:

stats-logger on

Turn statistics logging off on all boards

for-each-board stats-logger off

Applies To

Cluster node

See Also

dump-logged-stats start-logging-stats stop-logging-stats

Privilege Level

Supervisor
