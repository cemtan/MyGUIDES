filesystem-list-expansions

Show recent fs-expansions stored by ID Manager

Syntax

filesystem-list-expansions [<filesystems-see-'man fs-spec'>]

Description

Whenever a filesystem expands -- either automatically because it fills up, or because you expand it manually -- the server makes a note in its registry of the chunks being added. It remembers the hundred chunks added most recently to each filesystem.

The 'filesystem-list-expansions' command produces a list of those chunk numbers for one or more filesystems, which you can specify using the flexible syntax described in the 'fs-spec' man page. If you do not specify any filesystems, 'filesystem-list-expansions' lists all filesystems' recent expansions.

You can specify one or more filesystems, using the flexible syntax described in the 'fs-spec' man page.

Examples

To list recent expansions for filesystem Accounts:

filesystem-list-expansions Accounts

To list recent expansions for all filesystems on span SomeSpan:

filesystem-list-expansions SomeSpan

Applies To

Cluster wide
