msstats

Management server statistics

Syntax

msstats [<server>] [reset]

Description

The msstats command display statistics for the various management server interfaces to the server.

Valid servers are

HTTP

Remote management (API access)

HTTPS

Remote management (secure API access)

LoopbackHTTP

Local management via the loopback interface (API access)

rest

Native RestAPI management

ss

Statistics Server

ssc

Command line (CLI)

vss

VSS hardware provider DLL connection

If reset is specified either the given server's statistics are reset, or all the servers' are.

To see more detailed information about current connections to the management servers, use the msconnections command.

Examples

Show SSC server statistics:

msstats ssc

Reset SSC server statistics:

msstats ssc reset

Applies To

Cluster node

See Also

mscfg msconnections

Privilege Level

Supervisor
