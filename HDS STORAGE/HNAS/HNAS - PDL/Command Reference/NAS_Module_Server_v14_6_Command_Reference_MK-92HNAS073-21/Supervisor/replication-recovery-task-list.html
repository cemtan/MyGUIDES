replication-recovery-task-list

Display a replication recovery task's summary

Syntax

replication-recovery-task-list (-t|--task-id <task-id>) | ({--active | --all | (-f|--file-system <file-system>)} [--max <number-of-tasks>])

Description

Display a recovery task's summary.

--max <number-of-tasks>

The maximum number of tasks to be displayed.

--active

Display active (running/queued) tasks only.

--all

Display all tasks, including those that were completed/failed/cancelled.

-t, --task-id <task-id>

Display the summary of a specific task.

-f, --file-system <file-system>

Display the tasks of a file system.

Examples

To list the summary of a task with id 5afdcd94-4628-11cb-a005-ac2a04ffb80f:

replication-recovery-task-list --task-id 5afdcd94-4628-11cb-a005-ac2a04ffb80f
