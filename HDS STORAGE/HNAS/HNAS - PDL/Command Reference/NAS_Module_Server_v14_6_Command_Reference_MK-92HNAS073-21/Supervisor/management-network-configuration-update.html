management-network-configuration-update

Update management network configuration

Syntax

management-network-configuration-update [--verbose]

Description

The management-network-configuration-update command retrieves the management network configuration from the hypervisor, and updates the corresponding internal network configuration in the NAS module.

This command runs automatically whenever necessary. It is not intended to be run manually, although it is harmless to do so. (Note: The management-network-configuration-dry-run command may be used to verify that the NAS module's internal network configuration is up to date with the hypervisor's configuration).

Options:

-v, --verbose

This option displays each network configuration item. The default is to update the configuration silently.

Applies To

Cluster wide.

See Also

management-network-configuration-dry-run

Privilege Level

Supervisor
