file-filtering-failures-reset

Clear the failure log of file filtering failures

Syntax

file-filtering-failures-reset

Description

The file-filtering-failures-reset command clears the log of files that have resulted in file filtering failures. Use "file-filtering-failures-show" to show the failure log.

Examples

To clear the failure log:

file-filtering-failures-reset

Applies To

EVS

See Also

file-filtering-failures-show file-filtering-off file-filtering-on

Privilege Level

Supervisor
