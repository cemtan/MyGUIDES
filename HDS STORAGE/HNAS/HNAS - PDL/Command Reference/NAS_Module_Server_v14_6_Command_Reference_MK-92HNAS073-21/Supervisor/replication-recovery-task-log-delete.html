replication-recovery-task-log-delete

Delete a recovery task's log and summary

Syntax

replication-recovery-task-log-delete [--confirm] {--all | (-t|--task-id <task-id>) | (-f|--file-system <file-system>)} [--delete-summary]

Description

Delete a recovery task's log and summary.

--confirm

Don't ask for confirmation.

--delete-summary

Delete a task's summary in addition to its log. The summary will be deleted even when the log has been deleted already.

--all

Apply to all historical recovery tasks.

-t, --task-id <task-id>

Apply to a specific recovery task.

-f, --file-system <file-system>

Apply to all historical recovery tasks of a file system.

Examples

To delete the log of a task with id 5afdcd94-4628-11cb-a005-ac2a04ffb80f:

replication-recovery-task-log-delete --task-id 5afdcd94-4628-11cb-a005-ac2a04ffb80f

Note that the command above will not delete the summary of this task. To delete the log and the summary of a task with id 5afdcd94-4628-11cb-a005-ac2a04ffb80f:

replication-recovery-task-log-delete --delete-summary --task-id 5afdcd94-4628-11cb-a005-ac2a04ffb80f
