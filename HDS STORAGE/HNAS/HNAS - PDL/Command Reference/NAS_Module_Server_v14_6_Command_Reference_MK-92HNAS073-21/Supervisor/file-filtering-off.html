file-filtering-off

Turn the file filtering agent interface off

Syntax

file-filtering-off

Description

Turn off the file filtering agent interface. Use "file-filtering-state" to check the current setting.

Examples

To turn off file filtering:

file-filtering-off

Applies To

EVS
