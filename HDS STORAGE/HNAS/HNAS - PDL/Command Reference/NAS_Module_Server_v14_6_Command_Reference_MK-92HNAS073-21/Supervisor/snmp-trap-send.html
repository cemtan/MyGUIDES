snmp-trap-send

Send an SNMP trap with the specified message

Syntax

snmp-trap-send <message>

Description

snmp-trap-send allows a generic trap, containing specified text, to be sent to configured trap recipients. The command takes a single argument, the text (which should be contained in quotes if it consists of more than one word) to be sent.

This command is useful for simulating a trap that might be triggered by an event in the system's event log.

The trap is only sent from the node on which the command is run.

Examples

To send a trap that simply contains the text "Test message" in its bindings:

snmp-trap-send "Test message"

Applies To

Node
