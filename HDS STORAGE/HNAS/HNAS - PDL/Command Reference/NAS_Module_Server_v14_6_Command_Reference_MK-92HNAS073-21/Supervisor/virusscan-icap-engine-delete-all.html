virusscan-icap-engine-delete-all

Delete all ICAP virus scanning engines

Syntax

virusscan-icap-engine-delete-all [--confirm]

Description

The virusscan-icap-engine-delete-all command is used to remove all ICAP virus scanning engines from the current ICAP engine list. If the current virus scanning mode is RPC then no action will be taken. Note that if virus scanning is on and all virus scan engines are deleted, CIFS access to files might be denied.

Options:

--confirm

If the --confirm option is not supplied, the user will be prompted to confirm the request.

Examples

To remove all the ICAP engines from the engine list:

virusscan-icap-engine-delete-all

To remove all the ICAP engines from the engine list with no prompt:

virusscan-icap-engine-delete-all --confirm

Applies To

EVS
