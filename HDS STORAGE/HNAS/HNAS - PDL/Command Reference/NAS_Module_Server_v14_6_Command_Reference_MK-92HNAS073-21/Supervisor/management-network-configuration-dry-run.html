management-network-configuration-dry-run

Check if management network needs reconfiguring

Syntax

management-network-configuration-dry-run

Description

The management-network-configuration-dry-run command retrieves and displays the internal management network configuration from the hypervisor without actually applying it. Only information that will change is displayed.

Examples

management-network-configuration-dry-run
