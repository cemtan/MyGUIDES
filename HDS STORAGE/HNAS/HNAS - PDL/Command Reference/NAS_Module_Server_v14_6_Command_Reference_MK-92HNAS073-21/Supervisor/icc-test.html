icc-test

Loopback test for ICC ports

Syntax

icc-test <interface number>

Description

This is a command to test basic functionality of the cluster links. In order for it to work, you must attach an external loopback connection (or activate one of the hardware's internal loopback modes), and there should be no NVRAM mirroring or CNS traffic. Interfaces are numbered from 0.

Examples

icc-test 0

Applies To

Cluster node

Privilege Level

Supervisor
