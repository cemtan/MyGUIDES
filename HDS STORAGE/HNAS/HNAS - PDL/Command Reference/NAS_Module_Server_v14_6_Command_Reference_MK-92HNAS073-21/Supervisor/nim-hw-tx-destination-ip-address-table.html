nim-hw-tx-destination-ip-address-table

display the TX DEST IP ADDRESS table

Syntax

nim-hw-tx-destination-ip-address-table [<number to dump>] [-c]

Description

The nim-hw-tx-destination-ip-address-table command displays the contents of the hardware TX DEST IP table.

number to dump

Limit the number of displayed entries to this many.

The columns record the following information:

Index

The entry number in the TX DEST IP table.

TCP MTU

The MTU to be used for TCP traffic.

Other

The MTU to be used for non-TCP traffic.

Drop

If this field is 1, then at least one packet has been dropped by this entry.

Bufs

The number of TX buffers in use by this entry.

Next hop MAC

The MAC address to which all traffic for this entry will be sent. If the remote machine is within the same subnet, this will be a MAC address of that machine; otherwise it will be a MAC address of a local gateway. If the address is not yet resolved it will be 00-00-00-00-00-00.

Local IP

The server IP address to which this entry relates.

Peer IP

The IP address of a remote machine to which this entry relates.

VLAN

The VLAN, if any, which is recorded for this entry.

Agg

The internal aggregation number as recorded by the hardware. This is one less than the aggregation interface number, so ag2 will be recorded as 1.

Examples

$ nim-hw-tx-destination-ip-address-table
