virtualization-path-list

Lists the virtualization paths and current status for the target file system.

Syntax

virtualization-path-list {-t <fs label> | -a}

Description

Lists all NAS virtualization path associations defined for the specified file system, or for all file systems if the -a or --all option is used.

Options:

-t, --target <fs label>

Target file system.

-a, --all

All file systems. Produces a short summary at the end reporting on the number of file systems that contain a virtualization path, the number that do not contain a virtualization path and the number that cannot be reported upon, either because the file system is not mounted or the file system type does not support virtualization paths or because of an error in fetching the virtualization path information from the file system.

INFORMATION PROVIDED

Source URI

Legacy NAS path URI

Association

Logical name of the association

Mode

Association mode (configure, virtualize only, virtualize and migrate, paused)

Options

Optional parameters to association, ie. legacy NAS character set

Status

Runtime status of association

Seen Dirs

Number of directories found by virtualization process

Processed Dirs

Number of directories fully virtualized

Pending Dirs

Number of directories pending virtualization (including failed)

Failed Dirs

Number of directories which failed virtualization

Seen Files

Number of files found by virtualization process

Processed Files

Number of files fully migrated

Pending Files

Number of files waiting to start or re-start actively migrating

Migrating Files

Number of files actively migrating

Failed Files

Number of files which failed migration

Examples

virtualization-path-list -t Accounts

Lists the virtualization path on file system Accounts.

virtualization-path-list --all

Lists the virtualization path for all file systems.
