migration-list-ctrl-dir-requests

lists all request files contained in the control directory of the specified migration path

Syntax

migration-list-ctrl-dir-requests {-s <fs label> [-v <vivol name>] | -a}

Description

Options:

-s, --source <fs label>

Source file system.

-v, --vivol <vivol name>

Name of source Virtual Volume (optional).

-a, --all

All file systems.

Return a list of the migration request files that are currently in the control directories for the specified migration path. This command is similar to

selectfs fsname
ls /path/to/control_dir/*.request

repeated for each of the control directories, but it does not require the caller to know either the location of the control directories or the extension used for request files.

Naming Migration Request Files

Migration request files must be named with a specific extension in order to be recognized by the system. The required file name extension varies according to the type of migration to be performed.

internal migration

<file name>.request

external migration

<file name>.request-ext

Examples

migration-list-ctrl-dir-requests -s fs0

Display the requests specified in the control directory for file system fs0's data migration path, output would be of the form..

/path/to/control_dir1
 MigrationRequest1.request
 MigrationRequest2.request
/path/to/control_dir2
 MigrationRequest3.request
 MigrationRequest4.request

Where MigrationRequest1.request and MigrationRequest2.request are the only two request files in the first control directory, and MigrationRequest3.request and MigrationRequest4.request are the only two request files in the second control directory. The paths given are relative to the root directory of the specified file system or vivol.

If one of the specified control directories does not exist then the following output will be shown.

/path/to/control_dir1
 Directory does not exist.

If a directory exists but is empty then the output will be.

/path/to/control_dir1
 No Request Files.

migration-list-ctrl-dir-requests -all

Lists migration control directory requests for all file systems.

Applies To

EVS
