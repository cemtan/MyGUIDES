obj-stat

Shows information about a given OBJ_STORE object

Syntax

obj-stat <object number> [<checkpoint number> [<file-system>]]

Description

The obj-stat command displays information about the supplied object. The object's handle, sector address, and length are displayed.

The obj-stat command can only be successfully used on a mounted file system.

The command may take up to three arguments. The first is taken to be the object number; the second (if supplied) is taken to be the checkpoint number; the third (if supplied) is taken to be the file system name or number. In the absence of the optional arguments the latest checkpoint number and currently selected file system are the defaults.

Examples

To view information about the object whose object number is 0x8840, at checkpoint 0x34f29, on file system 4:

obj-stat 0x8840 0x34f29 4
