span-rename

Rename a span

Syntax

span-rename <from-base-or-instance-name> <to-base-name>
Alias:       mvspan

Description

Changes the base name of a span. If the span is a storage-based snapshot, all related snapshots' base names are changed; snapshot names stay unchanged; and new instance names are calculated.

The filesystems resident on the span are not affected and need not be unmounted. Nevertheless, the span's new name must be different from those of all other spans and all filesystems. The uniqueness rules are set out in the 'storage-based-snapshot-label' page.

If the server encounters two spans with the same name, it will refuse to load the second one, and will log an event giving the span name that's causing the problem. Rename the span that's already loaded, and then run 'sd-rescan-cod' to load the second span.

Examples

span-rename Accounts Finance

Applies To

Cluster wide

See Also
