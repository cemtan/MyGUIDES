replication-history-clear

Clears the object replication history for a specified policy.

Syntax

replication-history-clear <policy name> <source fs label>

Description

Clears replication history for a specified object replication policy.

<policy name>

Name of the policy.

<source fs label>

Source file system label.

Examples

replication-history-clear DefaultPolicy fs0

Clears the history for an object replication policy named 'DefaultPolicy' for source file system fs0.

Applies To

EVS

See Also

evs replication-history-list replication-policy-list

Privilege Level
