ndmp-enhanced-stats

Switch on/off enhanced stats

Syntax

ndmp-enhanced-stats {--enable|--disable|--begin-pir|--end-pir}

Description

Control the level of detail NDMP records in its performance statistics.

Do not use this command unless asked to do so by your support organization.

A single option should be supplied:

--enable Enable subdivision of stats in reports.

--disable Stats reports will be on complete operations.

--begin-pir Used internally by the pir command.

--end-pir Used internally by the pir command.
