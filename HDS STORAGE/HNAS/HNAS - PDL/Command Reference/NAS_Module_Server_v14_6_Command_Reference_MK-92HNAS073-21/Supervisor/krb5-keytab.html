krb5-keytab

Manage Kerberos keytab

Syntax

krb5-keytab import <keytab-path> | export <keytab-path> | delete | list

Description

The krb5-keytab command manages the keytab used by the Kerberos V5 GSS-API security mechanism.

To import an MIT-format keytab from a flash file:

krb5-keytab import <ssfs-path>

To export the current keytab into an MIT-format flash file

krb5-keytab export <ssfs-path>

To delete an EVS's keytab:

krb5-keytab delete

To list the current keytab on the CLI

krb5-keytab list

Examples

krb5-keytab import server.keytab

Applies To

EVS
