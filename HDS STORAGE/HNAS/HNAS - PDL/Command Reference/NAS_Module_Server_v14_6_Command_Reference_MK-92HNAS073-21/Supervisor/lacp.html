lacp

Display LACP interface diagnostics

Syntax

lacp  diag [agN|port] | history [agN|port] | frames [agN|port] |down-history [agN|port] | down-frames [agN|port]

Description

The lacp command provides diagnostic output describing the state of LACP for one or more aggregations.

Examples

To display diagnostic information about an aggregation:

    lacp diag ag1

To display recent history about an aggregation:

    lacp history ag2

To display recent LACP packets sent and received on an aggregation:

    lacp frames ag1

To display history about an aggregation up to the most recent link-down event:

    lacp down-history ag2

To display LACP packets sent and received on an aggregation up to the most recent link-down event:

    lacp down-frames ag2

Applies To

Cluster node

Privilege Level

Supervisor
