dailystatusreport

Send a daily status report now

Syntax

dailystatusreport [<email address>]

Description

The dailystatusreport is used to generate the daily status email on demand. If an argument is supplied it must be a valid email address to send the report to. If no argument is supplied, a daily status report is sent to all recipients configured to receive it.

Examples

To send a daily status email to fred@example.com now:

dailystatusreport fred@example.com

Applies To

Cluster wide
