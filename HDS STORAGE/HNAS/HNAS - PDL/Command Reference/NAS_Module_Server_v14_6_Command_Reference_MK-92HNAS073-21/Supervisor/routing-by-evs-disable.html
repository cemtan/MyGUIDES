routing-by-evs-disable

Disable routing by EVS

Syntax

routing-by-evs-disable

Description

The routing-by-evs-disable command reverts the effect of the routing-by-evs-enable command. In other words, when a sub-system hasn't specified it wants to use routing-by-evs, then the routing engine is at liberty to choose any source IP address for communications with a given destination.

Examples

To disable routing by EVS:

server:$ routing-by-evs-disable

Applies To

Cluster wide
