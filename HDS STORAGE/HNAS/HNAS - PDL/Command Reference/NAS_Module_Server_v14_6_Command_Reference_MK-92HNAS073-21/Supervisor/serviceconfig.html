serviceconfig

Configure or show services allowed

Syntax

serviceconfig [<service> {disable | enable}]
service = cifs | nfs | ftp | iscsi | cns | readcache

Description

Display or alter whether services are enabled or disabled.

serviceconfig [<service> {disable | enable}]

Where <service> can be 'cifs', 'nfs', 'ftp' 'cns' or 'iscsi' or 'readcache'

Examples

server:$ serviceconfig ftp disable
Service FTP is disabled - Stopping
FTP service configuration is disabled
server:$ serviceconfig
Service     Enabled?
NFS         Y
FTP         N
CIFS        Y
ISCSI       Y
CNS         Y
server:$ serviceconfig nfs
NFS Service Configuration
service enabled
server:$ serviceconfig ftp enable
Starting service FTP
FTP service configuration is enabled
server:$
