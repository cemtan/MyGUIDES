mfb-asserts

Displays information about asserts from the Alteras

Syntax

mfb-asserts [-r|--reset]

Description

Lists the historical and current VLSI asserts associated with the MFB.

This command is intended to be used under direction from your support provider.

-r, --reset

Resets historical information

Examples

To list asserts associated with the MFB:

mfb-asserts

To reset historical asserts associated with the MFB:

mfb-asserts -r

Applies To

Cluster node

Privilege Level

Supervisor
