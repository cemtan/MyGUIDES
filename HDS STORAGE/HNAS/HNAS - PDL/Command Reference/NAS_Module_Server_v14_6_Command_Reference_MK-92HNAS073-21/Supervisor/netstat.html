netstat

Show network statistics

Syntax

netstat [-i|--icmp] [-l|--listeners] [-t|--tcp] [-u|--udp] [-a|--all]

Description

netstat prints the network connections currently in use.

Options are:

-i, --icmp

Shows all ICMP sockets

-l, --listeners

Shows all listening TCP sockets

-t, --tcp

Shows all TCP sockets

-u, --udp

Shows all UDP sockets

-a, --all

Shows all sockets

Applies To

Cluster node

Privilege Level

Supervisor
