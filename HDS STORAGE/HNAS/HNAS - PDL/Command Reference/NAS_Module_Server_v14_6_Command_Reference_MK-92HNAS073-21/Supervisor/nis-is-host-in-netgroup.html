nis-is-host-in-netgroup

test nisclient::api::IsHostInNetgroup

Syntax

nis-is-host-in-netgroup <host> <netgroup>
Alias:       IsHostInNisNetgroup

Description

The nis-is-host-in-netgroup command is used to check whether a host is a member of a specified netgroup.

This command does not check local netgroups, only NIS netgroups, and makes use of the NIS cache, in both classic NIS and NIS/LDAP modes.

Examples

To check whether a host belongs to a netgroup:

    nis-is-host-in-netgroup 172.15.14.100 win3


    nis-is-host-in-netgroup fdca:f995:220a:0:58c1:e76e:d2c4:9f1c win3
