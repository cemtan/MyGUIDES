fs-browse-snapshots

Get/Set snapshot directory browsing configuration

Syntax

fs-browse-snapshots [on|off]

Description

Displays or sets the browsability of snapshots cluster-wide. If set to on, the local settings for each mount point on each file system are honored; if set to off, snapshot browsing is globally disabled and the local settings are ignored.

Usage:

fs-browse-snapshots [on | off]

Examples

To display the current setting:

fs-browse-snapshots

Applies To

Cluster wide

See Also

cifs-share nfs-export

Privilege Level

Supervisor
