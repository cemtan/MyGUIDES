cluster-extend-unconfigure

Unconfigure the extended cluster

Syntax

cluster-extend-unconfigure [-c | --confirm]

Description

Reverts an extended unified NAS module cluster to an ordinary two-node unified cluster. Use this command to unconfigure the extended cluster only when a cluster pair that joined using 'cluster-extend-join' is removed using 'cluster-extend-delete'. All the cluster nodes must be in ONLINE state to unconfigure the extended cluster. This command retains the EVS configuration, but removes all the configuration used to extend the cluster.

Options:

-c, --confirm

Do not prompt for confirmation.

Applies To

Cluster wide.

See Also

cluster-extend-create cluster-extend-delete cluster-extend-status

Privilege Level

Supervisor
