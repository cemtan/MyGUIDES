dlm-stats

Display Distributed Lock Manager statistics

Syntax

dlm-stats [-b] [-r]

Description

The dlm-stats command displays statistics about the distributed lock manager.

Usage:

dlm-stats [OPTION]

-r, --reset

Resets the statistics.

-b, --since-boot

Display statistics recorded since boot, irrespective of whether they've since been reset.

Applies To

Cluster node

See Also

dlm-errors

Privilege Level

Supervisor
