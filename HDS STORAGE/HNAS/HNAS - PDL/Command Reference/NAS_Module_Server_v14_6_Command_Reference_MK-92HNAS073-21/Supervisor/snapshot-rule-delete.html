snapshot-rule-delete

Delete a snapshot rule and associated snapshots/schedules.

Syntax

snapshot-rule-delete <--no-file-system | --file-system <file-system>> [--confirm] <rule-name>

Description

This command deletes an existing snapshot rule and its associated snapshots and schedules.

--no-file-system

Delete a rule on a previously deleted file system. File system label should not be specified if this option is used.

--file-system <file-system>

Select the file system by label.

--confirm

Don't ask for confirmation.

<rule-name>

Select rule by name.

Examples

server:$ snapshot-rule-delete --file-system fs1 rule1
 Successfully deleted rule 'rule1' and associated snapshots/schedules

Deletes the snapshot rule rule1 on file system fs1.

Applies To

EVS
