nim

Run command on other board

Syntax

nim <command>
Alias:       tcp

Description

The nim "command" is actually a prefix for another command that forces the other command to run specifically on the nim. For most commands this is not relevant, but a few things are specific to a board, e.g. flash files.

ssrun -c can be useful when trying to run complicated commands.

Examples

nim ssls

Applies To

Cluster node

See Also

fsm sim ssrun

Privilege Level

Supervisor
