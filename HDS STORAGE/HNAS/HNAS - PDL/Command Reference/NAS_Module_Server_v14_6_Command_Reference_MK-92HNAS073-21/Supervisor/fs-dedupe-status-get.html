fs-dedupe-status-get

List a file system's dedupe information

Syntax

fs-dedupe-status-get <file-system>

Description

Lists a file system's deduplication status:

  • whether the file system format supports deduplication (dedupe capable or not).

    • Note that dedupe can be enabled for Object replication target file systems.

  • whether the file system will be scheduled for deduplication (dedupe enabled or not).
  • how much data has been reduced so far. Reduced percentage is the reduced data divided by original size of the data. This is the same calculation as reported under the 'Reduction' column in the output of the 'df' command.
<file-system>

File system label.

Examples

List the deduplication status of file system 'fs0'.

fs-dedupe-status-get fs0

Applies To

File system
