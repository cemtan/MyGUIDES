test

test condition

Syntax

test <value> <condition> <value>

Description

The test command compares values and exits with the appropriate return code. Useful in conjunction with the && and || operators.

All tests are of the form

    value condition value

where values can be strings or numbers and the conditions are listed below.

Available comparisons are:

string1 = string2

The strings are equal

string1 != string2

The strings are not equal

integer1 -eq integer2

integer1 is equal to integer2

integer1 -ge integer2

integer1 is greater than or equal to integer2

integer1 -gt integer2

integer1 is greater than integer2

integer1 -le integer2

integer1 is less than or equal to integer2

integer1 -lt integer2

integer1 is less than integer2

integer1 -ne integer2

integer1 is not equal to integer2

Examples

Run the wfs_rs command, but only if the FSB board is rev 2 or greater:

test "$fsb_rev" -ge 2 && wfs_rs

Privilege Level

Supervisor
