package-install

Install a package

Syntax

package-install <package-file>

Description

**** WARNING ****

On VSP with NAS modules (F/G4x0, F/G6x0, F/G8x0) only use this command under instruction from your support provider. NAS upgrades and downgrades should only be carried out using the Maintenance Utility on the MPC or SVP.

**** WARNING ****

The package-install command installs a software package onto the server.

The package file must be present on the server in the ssfs area.

This command is similar to package-upgrade, but it doesn't copy the package to ssfs or set the package as default.

Examples

Install 6.5.1846.14 package:

package-install mercury-6.5.1846.14.tar

Applies To

Cluster node
