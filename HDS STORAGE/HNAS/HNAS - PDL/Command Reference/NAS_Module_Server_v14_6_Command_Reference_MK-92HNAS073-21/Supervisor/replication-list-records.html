replication-list-records

Lists replication records where the file system is the source or destination.

Syntax

replication-list-records -f <fs_label> [-d]

Description

Lists records of replications for the specified file system. We recommend that the SMU be used for all replication configuration. This command is only intended for use by SMU scripts and support.

Options:

-f, --file-system <fs_label>

Label of the file system addressed.

-d, --destination

If set, list records where this file system is the destination.

Examples

replication-list-records -f fs1

Lists records of replications which have fs1 as the source file system.

Applies To

EVS

Privilege Level

Supervisor
