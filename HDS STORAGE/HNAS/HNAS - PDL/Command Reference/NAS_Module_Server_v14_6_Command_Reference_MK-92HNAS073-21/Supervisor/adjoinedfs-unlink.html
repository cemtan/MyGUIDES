adjoinedfs-unlink

Unlinks adjoined file system leaving it as an ordinary file system.

Syntax

adjoinedfs-unlink -f <fs label>

Description

Adjoined filesystems are deprecated and administrators are encouraged to transition away from them.

Options:

-f, --fs <file system label>

File system to be made into an ordinary non-adjoined file system.

Removes the links between the main and secondary file systems. This command may only be completed when there is no longer any data on any of the secondary file systems.

Examples

adjoinedfs-unlink -f fs1

Delete the links to adjoined file systems from file system fs1.

Applies To

EVS
