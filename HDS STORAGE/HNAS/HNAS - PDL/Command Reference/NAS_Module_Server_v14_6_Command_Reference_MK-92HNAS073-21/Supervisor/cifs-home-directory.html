cifs-home-directory

Configure options for CIFS home directories

Syntax

cifs-home-directory [--restrict-shares | --unrestrict-shares]

Description

Configures options for the CIFS home directory feature.

Usage:

--restrict-shares, -r

Restrict share configurations so that only a single share per file system can be enabled for home directory operations. This setting affects the whole cluster. This is the default behaviour.

--unrestrict-shares, -u

Do not restrict the number of shares that can be enabled for home directory operations. This setting affects the whole cluster.

The cifs-home-directory command allows the administrator to configure the options to be applied to the home directory feature.

Examples

To restrict share configuration so only one share per file system can be used for home directories:

$ cifs-home-directory --restrict-shares

To unrestrict the home directory share configuration:

$ cifs-home-directory --unrestrict-shares

To show the cluster wide configuration:

$ cifs-home-directory
Cluster wide CIFS home directory options:
    Only one share can be configured for home directories per file system.

Applies To

Cluster wide
