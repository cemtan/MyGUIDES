nis-rebind

Cause the NIS client to rebind to a server

Syntax

nis-rebind [-f]

Description

Causes the NIS client to rebind to the NIS server.

If -f is specified, the NIS client will actively seek out the highest priority available NIS server rather than preferentially rebinding to the current NIS server.

Examples

To cause a rebind:

nis-rebind

To rebind to the best available NIS server:

nis-rebind -f

Applies To

EVS
