cifs-dc-errors

Report CIFS DC protocol errors

Syntax

cifs-dc-errors [-r|--reset]

Description

The cifs-dc-errors command is used to display errors seen on the Domain Controller connection. This information can be useful when debugging Domain Controller connectivity problems.

Usage:

--reset, -r

Resets the error log.

Applies To

Cluster node

See Also

cifs-dc cifs-dc-stats protocol-error-logging

Privilege Level

Supervisor
