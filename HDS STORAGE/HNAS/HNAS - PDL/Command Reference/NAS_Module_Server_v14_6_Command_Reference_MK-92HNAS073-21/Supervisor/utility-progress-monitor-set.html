utility-progress-monitor-set

Set up email monitoring of file system utilities

Syntax

utility-progress-monitor-set --email <email address list> [--period <minutes>]

Description

Sets up email monitoring of file system utilities running on a cluster. The monitor wakes up after the specified period and checks to see if any utilities are running and emails the output of the utility-progress command. If any utilities have completed running since the last time, the dynamic log is attached to the email.

When no utilities are running, the monitor is stopped to ensure stale data about monitoring requirements isn't held between instances.

--email

The full email address of the user(s) to mail output to. Multiple users can be specified, in quotes, in the form "user1;user2".

--period

Amount of time in minutes between polling. The default is 30 minutes if this is not specified.

Examples

utility-progress-monitor-set --email "info@example.com;admin@example.com" --period 60

Applies To

Cluster wide
