fs-reset-fixfs-resumption-point

Resets the point where fixfs should start checking the file system back to the beginning.

Syntax

fs-reset-fixfs-resumption-point <filesystem>

Description

Resets the starting point of fixfs back to the beginning of the file system. It only has any effect when fixfs is invoked with the '--non-critical-objects-only' switch, i.e. it has no effect on the behavior of fixfs when run offline.

Examples

To reset the starting point of fixfs for file system 'fs1':

fs-reset-fixfs-resumption-point fs1

Applies To

    File system
