quorumd

Add, remove and show the quorum device for a cluster

Syntax

quorumd add <name | ipaddr> | remove | show

Description

The quorumd command adds, removes and shows the quorum device for a cluster.

Usage:

add <name | ipaddr>

Adds a quorum device to a cluster. The quorum device can be specified by the DNS name or IP address. Adding by name determines the quorum device IP using a subnet broadcast.

remove

Removes the quorum device from the cluster. Use this command to remove a quorum device that was added using the add command.

show

Shows the quorum device information for the cluster.

Examples

quorumd add 10.4.1.1
quorumd add quorumdev
quorumd remove
quorumd show
