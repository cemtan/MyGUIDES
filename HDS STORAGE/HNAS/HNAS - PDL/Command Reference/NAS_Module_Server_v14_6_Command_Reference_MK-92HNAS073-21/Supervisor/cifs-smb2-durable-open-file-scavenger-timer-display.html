cifs-smb2-durable-open-file-scavenger-timer-display

Display the SMB2 durable open file scavenger timer

Syntax

cifs-smb2-durable-open-file-scavenger-timer-display

Description

The cifs-smb2-durable-open-file-scavenger-timer-display command is used to display the time that the durable open file scavenger waits before tearing down unclaimed orphaned durable open files. The time is given in milliseconds.

Examples

cifs-smb2-durable-open-file-scavenger-timer-display

Applies To

Cluster node

See Also
