oplock-stats

Show oplock statistics

Syntax

oplock-stats [--reset]

Description

The oplock-stats command displays the number and type of oplocks and leases that have been granted by the server, along with the number that have been broken and the levels to which they were broken.

Options:

-r, --reset

Reset statistics.

Examples

Reset oplock statistics:

oplock-stats -r

Applies To

Cluster node

See Also

cifs-stats

Privilege Level
