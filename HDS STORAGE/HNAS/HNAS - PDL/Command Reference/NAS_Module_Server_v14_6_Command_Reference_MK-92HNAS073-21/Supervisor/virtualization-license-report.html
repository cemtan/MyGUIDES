virtualization-license-report

Lists the amount of data currently virtualized by all virtualization paths

Syntax

virtualization-license-report --json

Description

The virtualization-license-report command lists each virtualization path along with the number of bytes that path has virtualized. The paths are grouped by file system. Virtualization paths that are in migration mode will report a virtualization usage of 0.

A summary line is produced reporting on the value of the Universal NAS Virtual Capacity license and the total usage of the currently active virtualization paths.

Options:

--json|-j

Produce the output in JSON format.

Examples

Report on the current virtualization usage:

$ virtualization-license-report
File System Name
================
 Association Name                                   Usage
 ================                                   =========
fs0
 onedir                                             ~766.1 GB
 twodir                                             ~221.4 MB


fs1
 second                                             ~5.133 TB


Universal NAS Virtual Capacity: Unlimited           ~5.882 TB

The same report in JSON format (with line breaks and spaces added):

$ virtualization-license-report --json
{
  "version":0,"reclist":
  [
    {"bytes_used":5644322234009,"fsname":"fs0","fsid":2620513515697171474,"associationname":"onedir"},
    {"bytes_used":822622226652,"fsname":"fs0","fsid":2620513515697171474,"associationname":"twodir"},
    {"bytes_used":223319987633,"fsname":"fs1","fsid":2620513541086972626,"associationname":"second"}
  ],
  "license_capacity":36028797018963968
}
