pir-status

Query the status of a performance info report (PIR)

Syntax

pir-status

Description

This command displays the status of any PIR currently running.

Examples

For a non-continuous PIR:

.$ pir-status

A 10-minute performance info report (entitled "Performance Info Report") is in progress.
Monitoring will end in 7.97 min.

.$

For a continuous PIR:

.$ pir-status

A continuous performance info report is running every 10 minutes.
The last run was number 1.
Emails are being sent every 10 runs.
The next run commences in 9.037 min.

.$

Applies To

Cluster node

See Also

performance-info-report pir-cancel pir-stop-and-send

Supervisor
