replication-history-list

Lists object replication history for a specified policy.

Syntax

replication-history-list [-v] [--all]|[[[--recent] <policy name>] <source fs label>]

Description

Lists the replication history for a specified object replication policy, for all policies on a specified file system or for all policies on all file systems on the current EVS.

<policy name>

Name of the policy.

<source fs label>

Source file system label.

--all

When listing the history of all policies on all file systems, show all history entries instead of the only the most recent 2 per policy. Each policy stores up to 20 history entries.

--recent

When listing the history of a specified policy, show only the most recent history entry.

-v

Verbose mode, dumps the object replication log.

Examples

replication-history-list -v DefaultPolicy fs0

Lists the replication history, including logs, for an object replication policy named 'DefaultPolicy' for source file system fs0.

Applies To

EVS

See Also
