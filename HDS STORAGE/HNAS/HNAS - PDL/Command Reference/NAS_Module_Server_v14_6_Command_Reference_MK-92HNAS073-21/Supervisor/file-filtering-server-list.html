file-filtering-server-list

Display the registered file filtering agents

Syntax

file-filtering-server-list

Description

This command displays a list of currently registered file filtering servers. Use "file-filtering-deregister" to remove a file filtering server from this list.

Examples

To list the currently registered file filtering servers:

server:$ file-filtering-server-list
Registered file filtering agents:


IP Address      Type  Status                         In Use  Agent ID
--------------- ----- ------------------------------ ------- --------
10.10.10.10     NSS   OK                             Yes     3805

Applies To

EVS

See Also

file-filtering-deregister file-filtering-off file-filtering-on file-filtering-reset

Privilege Level

Supervisor
