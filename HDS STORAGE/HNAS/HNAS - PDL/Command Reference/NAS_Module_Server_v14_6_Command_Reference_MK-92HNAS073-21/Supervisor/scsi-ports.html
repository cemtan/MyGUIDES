scsi-ports

Print info about SCSI ports

Syntax

scsi-ports [-a, --all] [-c, --clear] [<port id>]
Alias:       sports

Description

Displays information about the SCSI ports the server has discovered and the LUNs/devices reported by each port. If a device has multiple LUNs, summary information will show the current connected LUN postfixed with '+'.

Options:

-a, --all

Show all LUNs/devices, including those that are disconnected/removed

-c, --clear

clear SCSI port error information before displaying

[SCSI port id]

Show information about the given SCSI port

Examples

Clear SCSI port error information:

scsi-ports -c

Show information about SCSI port 2:

scsi-ports 2

Applies To

Cluster node

See Also

fc-ports scsi-devices

Privilege Level

Supervisor
