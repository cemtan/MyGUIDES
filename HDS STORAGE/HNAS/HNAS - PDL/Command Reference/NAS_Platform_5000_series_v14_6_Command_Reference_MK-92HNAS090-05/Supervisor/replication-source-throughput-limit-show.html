replication-source-throughput-limit-show

Shows the current object replication throughput limit

Syntax

replication-source-throughput-limit-show

Description

The replication-source-throughput-limit-show command reports the current aggregate throughput limit that's being applied to object replication per physical node. See the replication-source-throughput-limit-set man page for more details.

Examples

Show the current replication throughput limit:

$ replication-source-throughput-limit-show
Object replication throughput limit: 50 MiB/s

Applies To

Cluster wide

See Also

replication-show-active replication-source-throughput-limit-set

Privilege Level

Supervisor
