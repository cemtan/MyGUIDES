selectfs

Select/display the file system for this console

Syntax

selectfs [<label>]
Alias:       cv

Description

This command is used to select a file system for use by console commands such as cd, ls, rm, etc. When the current file system is changed, the current directory is changed to the root of the selected file system. If no arguments are provided, the command displays the console's current file system.

Examples

To set the console's current file system to 'fs0':

selectfs fs0

Applies To

EVS

See Also

cd unselectfs

Privilege Level

Supervisor
