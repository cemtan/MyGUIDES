remote-nfs-exports

Get a list of exports from the named NFS server, like "showmount -e server"

Syntax

remote-nfs-exports <server>

Description

Displays the NFS export information for an NFS server.

The remote-nfs-exports command displays the list of NFS exports from the named server along with their access restrictions. The output is similar to the linux "showmount -e <server>" command.

Examples

To display the NFS exports from server "wren":

$ remote-nfs-exports wren
Export list for server wren:
/                    *
/u30                 *
/var                 *
/tmp                 *
/var/chroot/ia32     *
/var/chroot/ia32/var *

Applies To

EVS
