virtualization-path-stats

Report virtualization path statistics

Syntax

virtualization-path-stats {-t <fs label> [-n <association>] | -a | -w} { [-r | --reset] | [-g | --graph] }

Description

Displays statistics associated with NAS virtualization paths, detailing the amount and time taken by the migration since the statistics were last reset, or start-up, whichever is the most recent.

Options:

-t, --target <fs label>

Show migration statistics for the target file system.

-n, --name <association>

Further narrow statistics to the association.

-a, --all

Show migration statistics for all associations on the EVS.

-w, --work

Show work team statistics.

-r, --reset

Reset statistics.

-g, --graph

Show a graphical representation of the time taken by the migration

Applies To

EVS
