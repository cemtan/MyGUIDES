snmpv3-user-delete-all

Delete all SNMPv3 users

Syntax

snmpv3-user-delete-all [--confirm]

Description

The snmpv3-user-delete-all command can be used to remove all SNMPv3 users from the system. This means that no SNMPv3 requests will be able to be authenticated until new users are added (man snmpv3-user-add). Running this command will prompt for confirmation unless the "--confirm" flag is specified. Confirmation defaults to "No" so simply pressing enter will not delete any users. Instead, press the "Y" key and enter.

Examples

Delete all SNMPv3 users:

snmpv3-user-delete-all
WARNING:
This command will delete all SNMPv3 users.
Delete all SNMPv3 users?(Y/N)[N]:

Delete all SNMPv3 users without being prompted for confirmation:

snmpv3-user-delete-all --confirm

Applies To

Cluster wide
