migration-cloud-policy-list

List the external cloud migration policies

Syntax

migration-cloud-policy-list [--policy-name-or-id <policy-name>|<policy-id>] [--source-fs-label <source-fs-label>]

Description

Lists cloud migration policies.

--policy-name-or-id <policy-id> or <policy-name>

List details about the specified policy ID or policy name only.

--source-fs-label, -s <source-fs-label>

Name of filesystem associated with policy.

Examples

migration-cloud-policy-list --policy-name-or-id DefaultPolicy

List details for policy named 'DefaultPolicy'.

Applies To

EVS security context

See Also
