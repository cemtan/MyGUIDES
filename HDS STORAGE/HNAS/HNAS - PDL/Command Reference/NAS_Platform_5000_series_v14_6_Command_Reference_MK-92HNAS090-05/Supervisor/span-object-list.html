span-object-list

List all span and filesystem IDs and names

Syntax

span-object-list [--no-lock | -L]

Description

This is a diagnostic command, intended for use by your support provider. It lists brief details of the spans and filesystems loaded into memory. The nearest customer-facing equivalent, which should be used in preference, is 'span-list --filesystems'.

The '--no-lock' or '-L' switch bypasses important safety mechanisms inside the server. It is not available to unprivileged users. Use of this switch is not supported.

Examples

span-object-list -L

Lists all loaded spans and filesystems, bypassing the span-management lock.

Applies To

Cluster node
