directory-entry-crc

Calculates the CRC of a directory entry

Syntax

directory-entry-crc -e <entry-name>

Description

The directory-entry-crc command verifies cyclic redundancy check hash values for directory entry names. This hash value is used by the directory tree object for fast look up of a directory entry. directory-entry-crc will work whether the supplied entry name exists or not.

Command was formerly named 'crc'.

Options:

-e <entry-name>

The full name of the directory entry

Examples

To determine the correct hash value for directory entry MyFile:

directory-entry-crc -e MyFile

Applies To

EVS

Privilege Level

Supervisor
