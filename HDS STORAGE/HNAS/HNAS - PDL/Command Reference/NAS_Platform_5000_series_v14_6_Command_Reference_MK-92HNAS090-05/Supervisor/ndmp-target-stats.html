ndmp-target-stats

Show NDMP target stats for this node

Syntax

ndmp-target-stats [-r|--reset]

Description

This command displays the accumulated statistics of file system accesses made by NDMP operations at the destination end of transfers. This includes NDMP recoveries, adc copies, replications and data migrations. It shows only the data related to writing to file systems.

By monitoring the increases in these figures the load on the file system from NDMP processes can be assessed.

These statistics can be reset using the -r parameter.

The statistics for reading from file systems can be displayed using the ndmp-source-stats command. Session level stats can be displayed using ndmp-session-stats, but that will only display stats for sessions that are currently active.

Examples

Applies To

Cluster node
