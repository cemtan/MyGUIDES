nim-tcp-analysis

Configure TCPAN data gathering

Syntax

nim-tcp-analysis [-o] [-s <local-addr>] [-p <local-port>] [-d <remote-addr>] [-g <remote-port>] [enable|restart|disable|clear]

Description

The nim-tcp-analysis command is used to select TCP traffic data which will subsequently be displayed by the nim-tcp-tcpan command.

To start TCP analysis gathering:

nim-tcp-analysis enable


nim-tcp-analysis restart

'restart' is equivalent to a 'disable' followed by an 'enable'.

The following options are available:

-o

Requests one-shot mode, stops gathering data once the buffer is full

-s

Specifies the source (local) address to select

-d

Specifies the destination (remote) address to select

-p

Specifies the source (local) port to select

-g

Specifies the destination (remote) port to select

To stop gathering further data:

nim-tcp-analysis disable

To discard accumulated data:

nim-tcp-analysis clear

Applies To

Cluster node

See Also

nim-tcp-tcpan

Privilege Level

Supervisor
