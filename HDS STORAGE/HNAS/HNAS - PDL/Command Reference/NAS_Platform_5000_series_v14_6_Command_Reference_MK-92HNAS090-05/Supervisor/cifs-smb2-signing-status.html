cifs-smb2-signing-status

Displays SMB2 signing required status.

Syntax

cifs-smb2-signing-status

Description

The cifs-smb2-signing-status command is used to display the current setting of signing required on the SMB2 service for the current EVS. Signing required is either disabled or enabled in a SMB2 server. If signing required is enabled all SMB2 packets will be signed (after the initial negotiation which signals the requirement). This is displayed as "SMB2 signing is required". If signing required is disabled it can be requested by the client on a per-session or per-packet basis. This is displayed as "SMB2 signing is NOT required".

EXAMPLE

$ cifs-smb2-signing-status


SMB2 signing is NOT required

Applies To

EVS
