scsi-dump-rack-states

dump engineering debug rack state information

Syntax

scsi-dump-rack-states [-p, --scsi-port]

Description

Displays debug troubleshooting information about the initialization state of a storage rack or all storage racks attached to the server. The information output by this command is for use by Engineering and Services to troubleshooting storage issues and should not be required in normal operation.

Options:

[-p, --port]

Displays troubleshooting information about the storage rack that hosts the given SCSI port number.

If no SCSI port number is entered, troubleshooting information about all storage racks is displayed.

Examples

To show the state information for the storage rack on SCSI port 0:

  scsi-dump-rack-states
  rack no 0

  consistent_system_id : true
  valid_host_config_options : true
  vendor_system_id : true
  validate_config_port_reset : false
  validate_config_outstanding : false
