dump-freed-blocks-list

Dumps the contents of a given Freed Blocks List

Syntax

dump-freed-blocks-list  [-f <file system>] <object-number>

Description

The dump-freed-blocks-list command is used to decode and display the on-disk contents of a Freed Blocks List. A Freed Blocks List must be specified by its object-number. It may be used to decode Freed Blocks Lists on WFS-2 file systems. If no file system is specified then the command works on the currently selected file system as shown by 'selectfs'.

This command may be run on a mounted or unmounted file system.

NB: Freed Blocks Lists may be very long, and the output consequently copious.

Usage:

<object-number> The object-number of the Freed Blocks List

-f <file-system> The file system on which the Freed Blocks List resides

Examples

To display contents of the Freed Blocks List whose object-number is 1024 on file system testfs:

dump-freed-blocks-list -f testfs 1024

Applies To

EVS

Privilege Level

Supervisor
