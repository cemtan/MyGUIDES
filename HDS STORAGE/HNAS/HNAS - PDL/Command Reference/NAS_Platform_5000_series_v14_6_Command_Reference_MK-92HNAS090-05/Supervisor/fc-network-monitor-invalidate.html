fc-network-monitor-invalidate

invalidate the internal state of the network monitor

Syntax

fc-network-monitor-invalidate [hport id]

Description

The fc-network-monitor-invalidate command invalidates the internal state of the fibre channel network monitor. This should be done after planned (or unplanned assuming the problem has been resolved) Fibre Channel network trauma to avoid the generation of events etc indicating potential backend problems. Command options are:

[hport id]

If specified, restricts the action to the specified host port. By default all statistics on all (enabled) host ports are cleared.

Examples

Invalidate the state of the fibre channel network monitor relevant to host port 1:

fc-network-monitor-invalidate 1

Invalidate the state of the fibre channel network monitor relevant to all host ports:

fc-network-monitor-invalidate
