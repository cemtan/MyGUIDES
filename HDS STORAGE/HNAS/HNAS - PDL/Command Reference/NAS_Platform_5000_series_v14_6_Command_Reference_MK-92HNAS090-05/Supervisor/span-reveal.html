span-reveal

Reveal a span's filesystems -- undo 'span-hide'

Syntax

span-reveal <span-or-filesystem-instance-name>

Description

The 'span-hide' command prepares a span for maintenance by making all its filesystems unavailable for checking, fixing, mounting and formatting. 'span-reveal' reverses this action. Use it after storage- or span-level maintenance is complete.

'span-reveal' queues file systems for mounting (or auto-mounting), but does not wait for them to mount. A script that runs this command may need to follow it with 'span-wait-for-loading --mount' to ensure that all the file systems on the span are ready to mount (or have already auto-mounted).

It is also possible to hide an individual filesystem. If you hide both a filesystem and its host span, you'll need to reveal them both (in either order) before any Filesystem I/O is possible.

If a filesystem or its host span is hidden, 'filesystem-list -a' shows the filesystem as 'Hiddn', and 'trouble' prints a warning.

Examples

span-reveal Admin

Applies To

Cluster wide
