migration-cloud-rule-delete-all

Delete all unused external migration rules

Syntax

migration-cloud-rule-delete-all [--confirm]

Description

Deletes all unused cloud migration rules.

--confirm

If specified, unused rules will be deleted without an additional prompt for a confirmation.

Examples

migration-cloud-rule-delete-all --confirm

Deletes all unused cloud migration rules.

Applies To

Global security context
