replication-source-perf

Show object replication source performance measures

Syntax

replication-source-perf [--reset [<replication id>] | <replication id>]

Description

Show performance statistics for the currently extant object replication sources. Most source performance statistics are discarded as soon as a replication completes so this command will not report much if no replication is running. replication-start --output-stats' provides an easier way to dump the stats for a specific replication.

Note that performance monitoring needs to be enabled with 'enhanced-perf-stats --enable' before this command will produce useful output.

Usage:

-r, --reset

Reset statistics. If a <replication id> is specified the stats for that replication will be reset, otherwise only the global stats will be reset. The global stats are stats that are not specific to any individual replication, e.g. global queue depths.

<replication id>

Dump stats for currently extant replication identified by specified id.

Applies To

Cluster node
