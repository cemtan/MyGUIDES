domain-mappings-list

lists domain mapping entries

Syntax

domain-mappings-list [--default] [--verbose]

Description

List entries in the domain mapping table.

Options:

--default

List the default domain mapping.

--verbose

Verbose listing.

The results are output in a table. The Unix column shows whether the entry will map Unix names. An asterisk (*) indicates mappings will occur. The NT column shows the NT domain name entries will be mapped to. The NFSv4 column shows the NFSv4 domain entries will be mapped to. Finally, the Kerberos field shows the Kerberos realm entries will be mapped to.

The default domain mapping is used to map Unix ids to the default NT domain.

Examples

$: # List configured domain mappings
$: domain-mappings-list


Unix     NT domain     NFSv4 domain     Kerberos realm
----     ---------     ------------     --------------
   *     EXAMPLE       example.com      example

Applies To

EVS security context
