fs-cancel-queued-task

Cancel a scheduled task that has not yet started

Syntax

fs-cancel-queued-task -f <file-system>

Description

The fs-cancel-queued-task command cancels a scheduled file system task for the supplied file system.

Note: fs-cancel-queued-task may be run with the file system mounted or unmounted.

Options:

<file-system>

The name or number of the file-system

Examples

To cancel the pending checkfs run for fs1:

fs-cancel-queued-task -f fs1

Applies To

EVS

See Also

checkfs fs-show-queued-tasks

Privilege Level

Supervisor
