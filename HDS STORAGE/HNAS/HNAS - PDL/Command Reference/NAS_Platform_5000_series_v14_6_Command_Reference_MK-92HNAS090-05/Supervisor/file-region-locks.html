file-region-locks

List a file's byte range locks

Syntax

file-region-locks <path>

Description

The file-region-locks command is used to display information about a file's byte range locks.

Examples

To display information about the file 'file1':

  $ file-region-locks file1
  FileRegion       : 0x0 - 0x20 (32 B)
    LockType       : Write or Exclusive
    LockSemantics  : Discretionary
    OwnerHandle    : 0x29E47EC8
    OwnerProcess   : 0x3b6d
    OwnerCorrelator: 0x17F520C8
  FileRegion       : 0x40 - 0x60 (32 B)
    LockType       : Write or Exclusive
    LockSemantics  : Discretionary
    OwnerHandle    : 0x29E47EC8
    OwnerProcess   : 0x3b6d
    OwnerCorrelator: 0x17F520C8
