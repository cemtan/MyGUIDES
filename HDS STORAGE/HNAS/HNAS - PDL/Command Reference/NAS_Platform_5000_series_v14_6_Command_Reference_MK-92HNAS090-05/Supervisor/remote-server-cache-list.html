remote-server-cache-list

List the remote server cache

Syntax

remote-server-cache-list [<server-key>]

Description

Lists the details of the servers configured in the remote server cache.

Usage:

remote-server-cache-list [<server-key>]

<server-key>

The key of the remote server whose details should be listed. If this option is not provided, then the details of all the remote servers in the cache are displayed.

Applies To

Cluster wide

Privilege Level

Supervisor
