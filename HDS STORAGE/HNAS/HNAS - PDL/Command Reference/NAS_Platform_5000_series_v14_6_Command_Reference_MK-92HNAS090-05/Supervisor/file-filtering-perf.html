file-filtering-perf

Display File filtering performance statistics

Syntax

file-filtering-perf [[--reset] | [--summary] [--verbose]] [<request-pattern>]

Description

The file-filtering-perf command reports NTP/NSS Software file filtering performance statistics. Supported file filtering agents (NTP Software, NSS) are very similar in terms of their function and messages; since only one type of agent may be in use at any time, only one set of common statistics is recorded.

Usage:

file-filtering-perf [OPTION...] [<request-pattern>]

--reset, -r

Reset the statistics.

--summary, -s

Only display a summary.

--verbose, -v

Display verbose information.

<request-pattern>

Only show statistics for requests with names that contain the given string.

Applies To

Cluster node
