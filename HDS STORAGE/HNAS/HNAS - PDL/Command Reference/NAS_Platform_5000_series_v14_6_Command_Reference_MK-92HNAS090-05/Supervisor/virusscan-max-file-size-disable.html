virusscan-max-file-size-disable

Disable the virus scanning maximum file size feature

Syntax

virusscan-max-file-size-disable

Description

Turn off the maximum file size for virus scanning. Use "virusscan-max-file-size-show" to check the current setting.

Examples

To disable the maximum file size limit:

virusscan-max-file-size-disable

Applies To

EVS

See Also

virusscan-max-file-size-enable virusscan-max-file-size-set virusscan-max-file-size-show

Privilege Level

Supervisor
