start-logging-stats

Starts logging statistics whose names contain the optional string

Syntax

start-logging-stats [<string>]

Description

Starts logging statistics whose names contain the given string. Note that "stats-logger" must be enabled for this to have any effect. Logged statistics are gathered on a per board basis.

Examples

To start logging all statistics with "monkey" in their name:

start-logging-stats monkey

And for all boards

for-each-board start-logging-stats monkey

Applies To

Cluster node

See Also

dump-logged-stats stats-logger stop-logging-stats

Privilege Level

Supervisor
