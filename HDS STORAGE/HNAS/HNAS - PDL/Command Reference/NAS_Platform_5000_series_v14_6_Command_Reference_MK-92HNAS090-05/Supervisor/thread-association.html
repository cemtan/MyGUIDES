thread-association

List all thread associations, or dump a specified association

Syntax

thread-association [<association-name> <association-name>...]
Alias:       club

Description

A thread association is a set of threads (processes running on the server) that are providing using a particular service, such as the spanning system. You can use the 'thread-association' command to list those threads and obtain stack dumps of them. These stack dumps are occasionally requested by your service organization.

The command runs on only a single board of a single cluster node. To get a complete picture, you may need to run it on all boards of all nodes in your cluster.

Examples

thread-association

Produces a list of the associations available in this release of the operating system.

thread-association span id

Produces stack dumps of all threads that implement or are using Span Manager or ID Manager on the board where the command is run.

Applies To

Cluster node
