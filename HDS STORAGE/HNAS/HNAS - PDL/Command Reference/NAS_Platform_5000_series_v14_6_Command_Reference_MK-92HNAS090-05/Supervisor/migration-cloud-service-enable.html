migration-cloud-service-enable

Enable the Data Migrator to Cloud service

Syntax

migration-cloud-service-enable [--force]

Description

Enable the Data Migrator to Cloud service.

--force, -f

Not recommended. Enables Data Migrator to Cloud service even when there is no enough uncommitted memory.

Examples

server:$ migration-cloud-service-enable
Attempting to apply modifications...
Data Migrator to Cloud is already enabled
Successfully applied configuration modifications.

Applies To

Cluster wide

See Also

migration-cloud migration-cloud-service-disable migration-cloud-service-status resource-config-show resource-memory-uncommitted-show

Privilege Level

Supervisor
