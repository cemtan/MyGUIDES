cluster-unconfigure

Unconfigure a cluster

Syntax

cluster-unconfigure [-c | --confirm]

Description

The cluster-unconfigure command is used to return a 1-node cluster to non-clustered operation.

Options:

-c, --confirm

Confirms cluster de-configuration.

Examples

server:$ cluster-unconfigure
This command will unconfigure the cluster and reboot the system.
Do you want to proceed?(Y/N)[N]:
y

Applies To

Cluster node

See Also

cluster-create cluster-node-delete cluster-show

Privilege Level

Supervisor
