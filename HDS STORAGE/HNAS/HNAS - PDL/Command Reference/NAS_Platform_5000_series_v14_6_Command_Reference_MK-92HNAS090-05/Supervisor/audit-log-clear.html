audit-log-clear

Clear (empty) the current file system audit log for a file system

Syntax

audit-log-clear [--confirm] <file-system>

Description

Options:

-c, --confirm

Do not prompt for confirmation.

file-system

File system label.

The audit-log-clear command removes all file system audit log records from the current audit log for a file system.

The command will prompt for confirmation before clearing the audit log unless the --confirm option is specified.

Examples

To clear the audit log for file system fs0:

audit-log-clear fs0

EVS
