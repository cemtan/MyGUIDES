cifs-max-supported-version

Set or display the maximum supported version of SMB

Syntax

cifs-max-supported-version [ --default | 1 | 2 | 2.1 | 3 ]

Description

The cifs-max-supported-version command has been obsoleted by the smb-max-supported-version command.

The function of the two commands is identical so you may continue to use cifs-max-supported-version for now, but smb-max-supported-version should be favored where possible.
