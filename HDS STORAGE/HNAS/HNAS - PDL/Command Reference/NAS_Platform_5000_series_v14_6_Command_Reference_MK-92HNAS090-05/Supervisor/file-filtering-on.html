file-filtering-on

Turn the file filtering agent interface on

Syntax

file-filtering-on

Description

Turn on the file filtering agent interface. Use "file-filtering-state" to check the current setting.

Note: It is no longer necessary to use "fsm set allow-ntp-file-filtering true" to enable file-filtering on the HNAS as a whole.

Examples

To turn on file filtering:

file-filtering-on

Applies To

EVS
