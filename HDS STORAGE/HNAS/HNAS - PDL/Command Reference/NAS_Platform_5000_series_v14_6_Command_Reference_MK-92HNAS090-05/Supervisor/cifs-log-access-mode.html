cifs-log-access-mode

Displays the current cifs log access mode

Syntax

cifs-log-access-mode

Description

The cifs-log-access-mode command is used to display the current cifs log access mode. The log access mode can be administrators only or unrestricted. Administrator access is required to clear the logs.

Examples

cifs-log-access-mode

Applies To

EVS security context

See Also

cifs-log-access-administrators-only cifs-log-access-unrestricted

Privilege Level

Supervisor
