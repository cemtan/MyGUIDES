backup-device-forget

Remove one or more backup devices from the ID Manager database

Syntax

backup-device-forget <backup-devices-see-'man sd-spec'>

Description

This command may be used to remove detail of backup devices (tape drives and media changers) from the server's registry. The command can only be used when the devices have been physically disconnected from the fibre channel network.

If the devices are being removed temporarily, it is recommended that this command is not used. If the device details are left in the registry the devices will retain their NDMP names when they are reconnected.

The parameter may be a single device identifier or a comma separated list of devices and may include device ranges. (See examples.)

Examples

backup-device-forget 800

Removes device 800 from the registry.

backup-device-forget 2,5-7

Removes devices 2, 5, 6 and 7 from the registry.

Applies To

Cluster wide
