fs-quarantine-list

Display a list of all quarantined objects for each file system

Syntax

fs-quarantine-list

Description

This utility will list all the currently quarantined objects, along with the file system number to which the quarantine applies, on the physical node it is invoked on.

If nothing is quarantined, then a message explaining as much will be displayed.

Applies To

EVS

See Also

fs-quarantine-add fs-quarantine-release

Privilege Level

Supervisor
