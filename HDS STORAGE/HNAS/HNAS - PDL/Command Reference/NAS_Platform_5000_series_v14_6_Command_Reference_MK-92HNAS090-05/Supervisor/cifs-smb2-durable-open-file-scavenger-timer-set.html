cifs-smb2-durable-open-file-scavenger-timer-set

Set the SMB2 durable open file scavenger timer

Syntax

cifs-smb2-durable-open-file-scavenger-timer-set <time(ms)>

Description

The cifs-smb2-durable-open-file-scavenger-timer-set command is used to set the time that the durable open file scavenger waits before tearing down unclaimed orphaned durable open files. The time is specified in milliseconds.

Examples

cifs-smb2-durable-open-file-scavenger-timer-set 10000

Applies To

Cluster node
