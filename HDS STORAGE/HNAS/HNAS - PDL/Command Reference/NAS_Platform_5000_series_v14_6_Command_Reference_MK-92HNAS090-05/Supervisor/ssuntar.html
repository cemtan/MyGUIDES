ssuntar

Extract files from an archive

Syntax

ssuntar <tarfile> [<file>] ...

Description

Extracts files from an archive file.

The command takes one or more parameters - the name of the archive file followed by an optional list of files to be extracted.

Examples

ssuntar test.tar

Extracts all files from archive file, test.tar.

ssuntar test.tar test1 test2

Extracts only test1 and test2 from the archive file, test.tar.
