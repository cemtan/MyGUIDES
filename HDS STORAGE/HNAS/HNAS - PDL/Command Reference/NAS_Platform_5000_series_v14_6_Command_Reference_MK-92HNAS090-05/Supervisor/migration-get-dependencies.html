migration-get-dependencies

Gets cross file system link dependencies for either source or target file system

Syntax

migration-get-dependencies {-s <source> | -t <target>}

Description

Options:

-s, --source <source>

Source file system label. List file systems that are targets of data migration paths on this source file system. (There may be multiple targets if several Virtual Volumes have separate migration paths.)

-t, --target <target>

Target file system label. List source file systems with migration paths which have this file system as target.

Examples

migration-get-dependencies --source Accounts

List all file systems which are targets of migration paths from the Accounts file system or one of its Virtual Volumes.

migration-get-dependencies --target vol2

List all file systems which have migration paths with vol2 as the target.

Applies To

EVS
