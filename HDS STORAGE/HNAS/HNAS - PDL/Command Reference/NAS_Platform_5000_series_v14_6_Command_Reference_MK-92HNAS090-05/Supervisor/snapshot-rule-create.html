snapshot-rule-create

Create a snapshot rule.

Syntax

snapshot-rule-create --file-system <file-system> [--queue-size <size>] <rule-name>

Description

This command creates a snapshot rule for the specified file system.

--file-system <file-system>

Select the file system by label.

--queue-size <size>

Specify the snapshot queue size.

<rule-name>

Specify rule by name.

Examples

server:$ snapshot-rule-create --file-system fs1 --queue-size 4 rule1

Creates a snapshot rule named rule1 for file system fs1 with queue size 4.

Applies To

EVS
