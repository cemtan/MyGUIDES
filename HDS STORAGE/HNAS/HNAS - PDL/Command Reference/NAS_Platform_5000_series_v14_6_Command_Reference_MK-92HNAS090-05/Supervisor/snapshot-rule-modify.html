snapshot-rule-modify

Modify a snapshot rule.

Syntax

snapshot-rule-modify [--file-system <file-system>] [--queue-size <size>] [--new-rule-name <new-rule-name>] <rule-name>

Description

This command modifies an existing snapshot rule.

--file-system <file-system>

Select the file system by label.

--queue-size <size>

Specify the snapshot queue size.

--new-rule-name <new-rule-name>

Specify a new name for the selected rule.

<rule-name>

Select rule by name.

Examples

server:$ snapshot-rule-modify --file-system fs1 --queue-size 6 --new-rule-name rule2 rule1

Modifies the queue size of snapshot rule rule1 on file system fs1 to 6 and changes its name to rule2.

Applies To

EVS
