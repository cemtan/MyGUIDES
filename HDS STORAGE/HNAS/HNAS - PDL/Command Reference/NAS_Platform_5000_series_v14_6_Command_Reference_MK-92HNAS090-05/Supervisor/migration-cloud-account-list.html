migration-cloud-account-list

List the external cloud migration accounts

Syntax

migration-cloud-account-list [<account name>]|[<account id>]

Description

Lists the cloud migration accounts. If no account name or ID is specified this command lists all accounts.

<account name>

Lists details about the specified account only.

<account id>

Lists details about the specified account only.

Examples

migration-cloud-account-list HNAS-HCPtarget

Lists details about account HNAS-HCPtarget.

Applies To

Global security context
