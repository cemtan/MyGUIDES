migration-cloud-agg-config-stats

Display statistics about Cloud Gateway's agg config queries

Syntax

migration-cloud-agg-config-stats [ -r ]

Description

Display or reset statistics about Cloud Gateway's aggregate network interface configuration queries.

These queries, and thus the statistics displayed by this command, are only collected when Cloud Gateway is configured to use the HNAS aggregate network interface(s).

-r

Reset these statistics.

Examples

migration-cloud-agg-config-stats

Display statistics about Cloud Gateway's aggregate network interface configuration queries.

migration-cloud-agg-config-stats -r

Reset the statistics.

Applies To

Cluster node

See Also

migration-cloud-service-mode-set migration-cloud-service-mode-show

Supervisor
