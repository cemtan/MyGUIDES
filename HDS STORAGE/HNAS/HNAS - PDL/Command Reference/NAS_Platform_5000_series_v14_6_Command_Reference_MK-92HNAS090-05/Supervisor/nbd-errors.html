nbd-errors

Report nbd protocol errors

Syntax

nbd-errors [-r|--reset]

Description

The nbd-errors command is used to display errors returned by the NBD server. The NBD protocol is used to allow the indexer process access to the dedupe index on replication target file systems.

Usage:

--reset, -r

Resets the error log.

Applies To

Cluster node

See Also

dedupe-replication-target-index-log dedupe-replication-target-index-stats deduplication

Privilege Level

Supervisor
