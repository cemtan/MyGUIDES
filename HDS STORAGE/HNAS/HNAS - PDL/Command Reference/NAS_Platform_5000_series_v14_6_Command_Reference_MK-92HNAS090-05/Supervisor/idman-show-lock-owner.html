idman-show-lock-owner

See who's using ID Manager

Syntax

idman-show-lock-owner

Description

The server uses a small database to ensure that each system drive, tape drive and filesystem (SD) always has the same device ID, and to store persistent information about these devices. The database is managed by software called 'ID Manager'. ID Manager is protected by a system of locks, so that one process can't modify information that another is using.

This command prints the IDs and backtraces of the threads, if any, that are using these locks.

Examples

idman-show-lock-owner

Applies To

Cluster node

See Also
