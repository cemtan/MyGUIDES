nfs-export-dump

Show NFS exports and/or mounts

Syntax

nfs-export-dump [--dns] [ exports | mounts | verbose ]
Alias:       showmount

Description

Displays all NFS exports and/or mount records belonging to all EVSs running on the current server. The list includes hidden CNS link exports as well as standard CNS and FS exports.

Usage:

nfs-export-dump [OPTION] [exports | mounts | verbose]

--dns

Display clients' DNS names. Only valid when used in combination with the verbose option.

exports

List exports only.

mounts

List mount records only.

verbose

List exports, mount records and clients.

<default>

List exports and mount records.

Applies To

Cluster node

See Also

nfs-export

Privilege Level

Supervisor
