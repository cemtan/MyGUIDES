multi-tenancy-disable

disable multi-tenant environment

Syntax

multi-tenancy-disable

Description

Disables the multi-tenant environment.

It is likely that this command will only be run when performing a major reconfiguration or decommissioning of an existing system. See the multi-tenancy manual page for further details.

To disable multi-tenancy, all EVSs must be deleted prior to calling this command.

Examples

Disabling multi-tenancy when no EVSs are present

server:$ multi-tenancy-disable
Operation successful.
server:$

Applies To

Cluster wide
