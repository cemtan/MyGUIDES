audit-log-consolidated-cache-show

Display the file system audit log consolidated cache file

Syntax

audit-log-consolidated-cache-show [-o] [count]

Description

Options:

-o, --oldfirst

Display file system audit records oldest to newest (default: newest to oldest).

count

Maximum number of audit records to display (optional).

The audit-log-consolidated-cache-show command displays file system audit log records from the audit log consolidated cache file.

If count is not specified and there are many records available to be displayed, the command will seek confirmation before continuing.

If there is more than one file system being audited on the same EVS, events from different file systems will be intermingled. However, events from each particular file system will be in time order.

Examples

To show 5 records from the audit log consolidated cache file:

audit-log-consolidated-cache-show 5

Applies To

EVS
