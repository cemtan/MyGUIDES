global-symlink-delall

Remove all global symlink definitions

Syntax

global-symlink-delall [-a] [-c]

Description

See the global-symlink-add man page for a general description of global symlinks.

The global-symlink-delall command is used to remove all path translations from the global symlink translation table.

Usage:

global-symlink-delall [OPTION]...

-a, --all-evss

Remove all global symlink translations from all configured EVSs. If the -a option is omitted, the command will target the current EVS only. This option cannot be used if multi-tenancy is enabled.

-c, --confirm

Don't prompt for confirmation.

Examples

To remove all global symlink translations from all EVSs (with a prompt for confirmation):

global-symlink-delall -a

Applies To

EVS

See Also
