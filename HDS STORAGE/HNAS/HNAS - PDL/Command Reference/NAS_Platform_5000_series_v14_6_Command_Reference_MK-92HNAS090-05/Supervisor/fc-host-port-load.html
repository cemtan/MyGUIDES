fc-host-port-load

Display allocation of devices to FC host ports

Syntax

fc-host-port-load [-v] [port]

Description

fc-host-port-load shows how many open system drives are currently assigned to each FC host port and optionally lists them. This will include open secondary drives, as load balancer does not make any distinction between primary and secondary drives.

-v --verbose

List SDs instead of just saying how many.

port

Specify a single host port. By default information for all enabled host ports is displayed.

Applies To

Cluster node

See Also

fc-target-port-load sdpath

Privilege Level

Supervisor
