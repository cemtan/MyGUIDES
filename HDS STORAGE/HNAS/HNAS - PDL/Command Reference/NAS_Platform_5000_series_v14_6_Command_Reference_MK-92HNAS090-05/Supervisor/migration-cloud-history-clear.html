migration-cloud-history-clear

Clears the cloud migration history for a specified policy.

Syntax

migration-cloud-history-clear <policy-name-or-id> [--source-fs-label <source-fs-label>] [--history-record-id <history-record-id>]

Description

Removes job history for specific cloud migration policy.

<policy-name-or-id>

Name or id of the policy for which to remove history.

--source-fs-label, -s <source-fs-label>

Source File System label for which to remove history.

--history-record-id, -i <history-record-id>

History record id to remove.

Examples

migration-cloud-history-clear DefaultPolicy

Removes history for policy 'DefaultPolicy'.

Applies To

EVS security context
