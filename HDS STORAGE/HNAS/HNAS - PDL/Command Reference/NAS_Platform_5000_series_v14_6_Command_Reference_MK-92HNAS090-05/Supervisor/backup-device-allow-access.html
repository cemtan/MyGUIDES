backup-device-allow-access

Allow access to one or more tape or media changer devices

Syntax

backup-device-allow-access <backup-devices-see-'man sd-spec'>

Description

The server will not allow use of a tape or media changer unless you have allowed access to it. After allowing access, you will need to assign the device to an EVS before it can be used. The device will be available on any node in the cluster where its assigned EVS is running.

Allowing more than one independent server to use a backup device may cause problems. You should be careful to ensure that servers accessing such a device can sucessfully share the device. Consult your support organization if you are unsure about whether sharing is possible.

Examples

backup-device-allow-access 800

Applies To

Cluster wide
