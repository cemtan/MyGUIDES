nfs-client-tab

List the clients accessing the specified export

Syntax

nfs-client-tab [--cns] <export name> [<IP address> | <client name>]

Description

The nfs-client-tab command displays the clients currently accessing the specified export and the time of the last access. The export name is resolved using the EVS currently in context.

The --cns flag must be specified if the export name is for a cluster namespace related export. If a CNS export is specified, then all nodes within the cluster are queried for clients that may be accessing it.

The output of this command can be further filtered by specifying an IPv4 address in dot notation, an IPv6 address in colon-notation or a name. If a name is specified then this must be able to be resolved to an IP address using the configured name services.

Examples

Display the list of clients accessing the export "root":

nfs-client-tab root


Client: 10.1.100.40, last access: 45.2 min ago, s/w requests: 25


Client: fdca:f995:220a:a00:250:56ff:fe06:d3b6, last access: 49.8 min ago, s/w requests: 23

Applies To

EVS
