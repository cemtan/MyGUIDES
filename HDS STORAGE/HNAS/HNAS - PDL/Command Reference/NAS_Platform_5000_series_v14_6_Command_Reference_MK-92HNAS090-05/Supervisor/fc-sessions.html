fc-sessions

Print sessions info

Syntax

fc-sessions [-h | --hport <hport no>]
Alias:       sessions

Description

The fc-sessions command prints discovered remote ports.

Options:

-h <hport no>

Print remote ports on this host port only

Examples

To show remote ports discovered on host port 2:

fc-sessions -h 2

Applies To

Cluster node

See Also

fc-ports

Privilege Level

Supervisor
