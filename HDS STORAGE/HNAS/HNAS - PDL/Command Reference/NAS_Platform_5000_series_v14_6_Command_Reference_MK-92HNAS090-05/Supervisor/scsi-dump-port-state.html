scsi-dump-port-state

Dump state information about this SCSI port

Syntax

scsi-dump-port-state <SCSI port ID | -a --all>

Description

Displays debug troubleshooting information about the state of a SCSI Port on the server. The information output by this command is for use by engineering and services to troubleshooting storage issues and should not be required in normal operation.

Options:

<Port ID>

SCSI port number on the rack for which the state information is required.

-a, --all

Displays the information for all SCSI Ports on the server.

Examples

Show the state information for the storage rack on SCSI port 1:
