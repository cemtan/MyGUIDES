audit-mgmt-log-server-connections

Show management audit log server connection status

Syntax

audit-mgmt-log-server-connections (no options)

Description

This command displays the connection status, on the present cluster node, for currently configured CEF-aware syslog servers to which management audit events are sent.

Note that while there is a single cluster-wide configured list of these servers, each cluster node maintains its own network connection to each CEF-aware syslog server.

Examples

To show connection status on the current cluster node:

audit-mgmt-log-server-connections
Configuration: logserver:612
       Status: Success
Using address: 172.20.180.4

To show connection status on cluster node 2:

cn 2 audit-mgmt-log-server-connections
Configuration: logserver:612
       Status: Success
Using address: 172.20.180.4

Applies To

Cluster node
