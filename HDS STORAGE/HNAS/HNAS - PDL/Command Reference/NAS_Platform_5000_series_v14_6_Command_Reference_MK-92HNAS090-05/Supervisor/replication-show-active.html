replication-show-active

Show active object replication sessions

Syntax

replication-show-active [-v]

Description

Lists active object replication sessions and displays the aggregate throughput across all sessions originating from this physical node.

Usage:

-v, --verbose

Provide more detailed information about the active object replication sessions.

Note that the time remaining is an estimate based on the file system data remaining. The latter shows "(calculating x%...)" if the estimate is not yet complete. The number of objects remaining is calculated separately and does not necessarily affect the time remaining.
