ssls

List flash directory contents

Syntax

ssls <filespec>

Description

Displays a listing of files in the server flash.

The command takes one optional parameter - the filespec, which may include the flash name.

Examples

ssls

Displays listing of files from local flash on current board

ssls t*

Displays listing of files that begin with 't' from local flash on current board

ssls //simm

Displays a listing of files from the Simm flash

ssls //fsm/s*

Displays a listing of files that begin with 's' from local flash on the FSM board

See Also
