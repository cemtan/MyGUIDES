nfsv3-rsize-get

Display the maximum number of bytes per NFSv3 READ request the client can receive.

Syntax

nfsv3-rsize-get --tcp | --udp | --all

Description

Displays the maximum number of bytes per NFSv3 READ request that a client can receive. These values may be set using nfsv3-rsize-set.

Usage:

nfsv3-rsize-get

nfsv3-rsize-get --all

Displays all the currently configured values.

nfsv3-rsize-get --tcp

Display the currently configured value for TCP transports.

nfsv3-rsize-get --udp

Display the currently configured value for UDP transports.

Applies To

Cluster wide

See Also

nfsv3-rsize-set nfsv3-wsize-get nfsv3-wsize-set

Privilege Level

Supervisor
