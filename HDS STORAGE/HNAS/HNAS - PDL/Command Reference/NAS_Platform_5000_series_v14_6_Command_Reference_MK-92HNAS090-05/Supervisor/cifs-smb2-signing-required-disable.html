cifs-smb2-signing-required-disable

Disables SMB2 signing required

Syntax

cifs-smb2-signing-required-disable

Description

The cifs-smb2-signing-required-disable command is used to disable signing required on the SMB2 service for the current EVS. Signing required is either disabled or enabled in a SMB2 server. If signing required is enabled all SMB2 packets will be signed (after the initial negotiation which signals the requirement). This is displayed as "SMB2 signing is required". If signing required is disabled it can be requested by the client on a per-session or per-packet basis. This is displayed as "SMB2 signing is NOT required".

Examples

$ cifs-smb2-signing-required-disable


SMB2 signing is NOT required on EVS 1

Applies To

EVS
