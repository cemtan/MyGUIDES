ndmp-session-trace-add

Start per-session tracing for the specified NDMP subsystem

Syntax

ndmp-session-trace-add <requests|env|statistics|backupq|notify|hsb-ds|ds|slave>

Description

Adds the specified NDMP subsystem to the output of the NDMP session trace.

Not all NDMP subsystems are traced by default because there is a performance penalty in doing so for certain subsystems.

For this reason, it is advisable to consult your Support Organisation before using this command to add extra tracing.

Examples

Add tracing for the NDMP request subsystem:

ndmp-session-trace-add requests

Applies To

Cluster node

See Also

ndmp-session-trace ndmp-session-trace-remove ndmp-trace-mask

Privilege Level

Supervisor
