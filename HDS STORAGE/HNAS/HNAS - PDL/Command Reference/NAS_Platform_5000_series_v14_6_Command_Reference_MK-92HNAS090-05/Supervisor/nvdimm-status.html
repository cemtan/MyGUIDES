nvdimm-status

Show NVDIMM status

Syntax

nvdimm-status

Description

Displays the status of the NVDIMM module.

This includes controller status, status of last operation, module health and energy source lifetime.

Examples

server:$ nvdimm-status
Controller status: Ready
Last command status: None
Last save status: None
Last restore status: Success
Last arm status: Success, Armed for catastrophic save
Last erase status: Success
Module Health Status: Backup system recovered
Energy Source lifetime: 94%

Privilege Level

Supervisor
