head

Show just the first part of output

Syntax

... | head [NUM]

Description

Command output can be piped through head to display just the first few lines of the output. 'head' may not be used as a command.

It takes an optional unsigned numeric parameter that specifies the number of lines to output. The default is 10.

If a command has a built-in way to limit the amount of output it produces, using that facility is likely to be more efficient than piping the command's output through 'head'.

Examples

To show the start of the dblog for the current boot session:

dblog | head 50
