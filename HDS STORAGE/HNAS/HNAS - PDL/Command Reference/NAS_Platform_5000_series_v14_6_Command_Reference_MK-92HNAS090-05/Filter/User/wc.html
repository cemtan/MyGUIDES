wc

Print the number of newlines, words, and bytes

Syntax

... | wc [-l] [-c]

Description

Command output may be piped through wc to count lines and characters in that output. It may not be used as a command. There is no UTF-8 support, so "character" actually means "byte".

Options:

-l

Count just lines

-c

Count just characters

By default both the line and character count are output

Examples

See how many exports we currently have:

nfs-export list | grep "Export name" | wc -l

Privilege Level

User
