icc-stats

Show ICC packet driver statistics

Syntax

icc-stats [-r]

Description

Displays simple packet and bytes counts for the cluster links. Only packets processed by the CPU network stack are counted; fast-path traffic is excluded from these statistics.

The output is presented in a table form, with cluster link interface names given as the columns. Externally connected interfaces are labelled from c1 upwards, on integrated platforms there is also a "c0" interface which is internal to the chassis.

The following option is available:

-r

Reset statistics

Examples

To show current accumulated statistics:

server:$ icc-stats
Interface    c0         c1         c2         c3         c4
tx  packets  5493       0          0          0          0
    bytes    1980730    0          0          0          0
rx  packets  0          0          0          0          0
    bytes    0          0          0          0          0
server:$

To show 10 seconds worth of statistics by resetting stats then waiting before display:

server:$ icc-stats -r; sleep 10; icc-stats
[sleep took 10 s.]
Interface    c1         c2
tx  packets  51         10
    bytes    7838       660
rx  packets  51         10
    bytes    16726      9546
server:$

Applies To

Cluster node
