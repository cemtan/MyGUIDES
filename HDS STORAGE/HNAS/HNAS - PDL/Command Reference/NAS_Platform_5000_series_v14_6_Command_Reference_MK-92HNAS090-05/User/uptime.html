uptime

Displays how long the system has been running

Syntax

uptime [-v] [-a]

Description

Displays the amount of time the system has been up.

Options:

-v, --verbose

Print more time details

-a, --all

Print details for all Cluster Nodes

Examples

uptime

Displays the system up-time.

uptime -v

Displays the system up-time, boot-time, and current time.

uptime -a

Displays the system up-time on all cluster nodes.

Applies To

Cluster node

See Also

setdate settime

Privilege Level

User
