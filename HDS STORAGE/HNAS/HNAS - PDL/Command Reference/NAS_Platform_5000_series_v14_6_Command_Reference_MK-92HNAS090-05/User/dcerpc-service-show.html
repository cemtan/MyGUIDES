dcerpc-service-show

Show DCE/RPC Service(s).

Syntax

dcerpc-service-show (--all | --uuid <uuid> | --name <name>)

Description

Display the list of registered DCE/RPC Services.

Usage:

--all

Show all registered DCE/RPC Services.

--uuid <uuid>

Show details for a registered DCE/RPC Service by UUID.

--name <name>

Show details for a registered DCE/RPC Service by name.

Examples

dcerpc-service-show --all
  Service Name          UUID                                  Version     Default State   State
  --------------------  ------------------------------------  ----------  --------------  --------------
  Endpoint Mapper       e1af8308-5d1f-11c9-91a4-08002b14a0fa  3.0         Enabled         Running
  Management            afa8bd80-7d8a-11c9-bef4-08002b102989  1.0         Enabled         Running
  Fsrvp                 a8e0653c-2744-4389-a61d-7373df8b2292  1.0         Enabled         Running
  Witness               ccd8c074-d0e5-4a40-92b4-d074faa6ba28  1.1         Enabled         Running
  --------------------  ------------------------------------  ----------  --------------  --------------


dcerpc-service-show --uuid e1af8308-5d1f-11c9-91a4-08002b14a0fa
  Service Name  : Endpoint Mapper
  UUID          : e1af8308-5d1f-11c9-91a4-08002b14a0fa
  Version       : 3.0
  Default State : Enabled
  State         : Running

Applies To

Cluster node
