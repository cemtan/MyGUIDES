default-quota

Manages default user/group quotas

Syntax

default-quota <operation get|set|clear>
	get   <target user|group> <file-system> [<vivol-name>]
	set   [--usage-limit <n[K|M|G|T|P|E]>] [--file-count-limit <n>]
	      [--usage-hard-limit <yes|no>] [--file-count-hard-limit <yes|no>]
	      [--usage-warn <n>] [--usage-critical <n>] [--usage-reset <n>]
	      [--fcount-warn <n>] [--fcount-critical <n>] [--fcount-reset <n>]
	      [--generate-events <yes|no>] [--ignore-domain-users <on|off>]
	      <target user|group> <file-system> [<vivol-name>]
	clear <target user|group> <file-system> [<vivol-name>]

Description

Manages default file system quotas.

Usage:

default-quota get <target user|group> <file-system> [<vivol-name>]

Gets the default user/group quota template for the specified file system or virtual volume.

default-quota set [--usage-limit <n[K|M|G|T|P|E]>] [--file-count-limit <n>] [--usage-hard-limit <yes|no>] [--file-count-hard-limit <yes|no>] [--usage-warn <n>] [--usage-critical <n>] [--usage-reset <n>] [--fcount-warn <n>] [--fcount-critical <n>] [--fcount-reset <n>] [--generate-events <yes|no>] [--ignore-domain-users <on|off>] <target user|group> <file-system> [<vivol-name>]

Sets the default user/group quota template for the specified file system or virtual volume.

default-quota clear <target user|group> <file-system> [<vivol-name>]

Clears the default user/group quota template for the specified file system or virtual volume.

-1, --usage-limit

Quota's space consumption limit (in bytes).

-2, --file-count-limit

Quota's file count limit.

-s, --usage-hard-limit

Sets hard usage limit: yes - hard; no - soft (default: hard).

-f, --file-count-hard-limit

Sets hard file count limit: yes - hard; no - soft (default: hard).

-w, --usage-warn

Usage warning threshold (%).

-p, --usage-critical

Usage critical threshold (%).

-r, --usage-reset

Amount by which usage must fall below a threshold before re-logging its event (%).

-a, --fcount-warn

File count warning threshold (%).

-b, --fcount-critical

File count critical threshold (%).

-c, --fcount-reset

Amount by which file count must fall below a threshold before re-logging its event (%).

-e, --generate-events

Add quota's events to the event log.

-i, --ignore-domain-users

Whether to apply default group quota to Domain Users group (not valid if setting default user quota).

Examples

To limit all users to 1G on fs0:

default-quota set --usage-limit 1G --usage-warn 80 --usage-critical 95 user fs0

To limit all groups to 10G on the virtual volume 'vivol0':

default-quota set --usage-limit 10G --usage-warn 80 --usage-critical 95 group fs0 vivol0

To get the default user quota template on fs0:

default-quota get user fs0

To remove the default user quota template on fs0:

default-quota clear user fs0
