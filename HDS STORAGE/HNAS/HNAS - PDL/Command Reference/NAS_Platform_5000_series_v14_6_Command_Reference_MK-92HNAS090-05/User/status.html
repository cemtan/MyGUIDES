status

Report server status

Syntax

status [-h|[-s] [-a]]

Description

Displays the status of the server.

Options:

-h, --hw

Display detailed hardware status

-s, --summary

Display summary status

-a, --all

Display status for all cluster nodes

-b, --both

Deprecated synonym for --all

Examples

status

Display general status of the server

status -h

Display hardware status of the server

status -s

Display summary status of the server

status -s -a

Display summary status of all cluster nodes

Privilege Level

User
