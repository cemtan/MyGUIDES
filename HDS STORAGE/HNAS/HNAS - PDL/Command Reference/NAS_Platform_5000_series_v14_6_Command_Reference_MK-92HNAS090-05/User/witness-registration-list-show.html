witness-registration-list-show

Show Witness Registration List

Syntax

witness-registration-list-show

Description

Display a list of Witness Registrations made to a Witness EVS.

Examples

witness-registration-list-show
Registered Witness Clients:
  Key                                   Client Name
  ------------------------------------  ----------------------------------------
  0292bf59-d6a9-d011-9094-01e6ec337fb6  hostname.example.com
  ------------------------------------  ----------------------------------------

Applies To

EVS
