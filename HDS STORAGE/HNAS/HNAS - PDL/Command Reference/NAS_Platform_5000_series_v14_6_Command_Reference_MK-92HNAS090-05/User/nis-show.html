nis-show

Display all NIS client configuration information

Syntax

nis-show

Description

Displays all of the available NIS client information

Examples

To display the current NIS client settings:

nis-show
State:                             enabled
Broadcast for servers:             enabled
Domain:                            uk
Current server:                    172.18.0.1
Rebind period (mins):              15
Max bind response time (ms):       300
Character Set:                     Latin-1
Domain transport:                  NIS classic
--[Configured Servers]---------------
-- 3 10.1.16.41
--[Responded to Broadcast]-----------
sb 4 172.18.0.1

s indicates the currently bound server.

b indicates a server located via broadcast.

Applies To

EVS

See Also

domainname nis-broadcast-mode nis-rebind nis-server nis-set nis-state ypwhich

Privilege Level

User
