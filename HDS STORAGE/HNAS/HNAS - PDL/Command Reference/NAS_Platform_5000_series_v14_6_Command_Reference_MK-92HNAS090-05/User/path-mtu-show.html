path-mtu-show

Display Path MTU Table

Syntax

path-mtu-show [-i <interface>]

Description

The 'path-mtu-show' command displays the dynamically discovered Path MTU information. Path MTU Discovery only applies to IPv6 traffic and operates in accordance with RFC-1981.

Without any options, the 'path-mtu-show' command displays all of the contents of the Path MTU table.

Options:

-i, --interface <name>

Displays table entries associated with the named interface.

Examples

To display the Path MTU table entries for interface 'ag1':

    path-mtu-show -i ag1

Applies To

Cluster node
