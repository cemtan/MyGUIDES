witness-unused-registration-timeout-show

Show Witness Unused Registration Timeout

Syntax

witness-unused-registration-timeout-show

Description

Display the time for which a Witness Registration is allowed to remain without registering for asynchronous notification. The timeout applies to the Witness protocol version 2 only.

Examples

witness-unused-registration-timeout-show
Witness Unused Registration Timeout : 60000 ms.

Applies To

Cluster node
