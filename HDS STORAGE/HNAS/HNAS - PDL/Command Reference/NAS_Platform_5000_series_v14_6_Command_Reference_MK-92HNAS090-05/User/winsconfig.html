winsconfig

Query/set WINS IPv4 addresses

Syntax

winsconfig operation
where 'operation' is one of:
list
add -p ip_address [-s ip_address]
	-p, --primary    Set primary IPv4 address
	-s, --secondary  Set secondary IPv4 address
del [-ps]
	-p, --primary    Delete primary address
	-s, --secondary  Delete secondary address

Description

When using netbios name lookup, you can specify the IPv4 addresses of primary and secondary WINS servers. IPv6 addresses are not supported.

To display the currently configured WINS servers:

winsconfig list

To add WINS servers:

winsconfig add [(--primary|-p) <primary WINS server IP>] [(--secondary|-s) <secondary WINS server IP>]

where

--primary or -p

allows you to specify the primary WINS server's IP address, and

--secondary or -s

allows you to specify the secondary WINS server's IP address.

To delete WINS servers:

winsconfig del [(--primary|-p)] [(--secondary|-s)]

Examples

To specify a primary WINS server: winsconfig add -p 10.1.1.7

To list existing WINS servers: winsconfig list

Applies To

EVS security context
