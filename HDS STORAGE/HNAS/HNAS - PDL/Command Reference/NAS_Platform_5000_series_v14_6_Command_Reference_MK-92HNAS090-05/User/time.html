time

Run a command and summarize system resource usage.

Syntax

time <command> [<args...>]

Description

The time command tells you how long a command takes to run.

ssrun -c can be useful when trying to run complicated commands.

Examples

server:$ time sleep 1


real    1000ms
server:$

Applies To

Cluster node

See Also
